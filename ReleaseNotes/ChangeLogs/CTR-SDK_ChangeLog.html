Changes in CTR-SDK 11.4.0 (2015/09/07 - for System Updater 0.24.66)

Additions and Changes

Additions and Changes37-S01:(BOSS) Changed document related to BOSS [SDK]

Changed the following document related to BOSS.

  • Added in the BossLotcheckTool Clear BOSS DB in the system explanation that expanded save data must be deleted when debugging without any BOSS information registered.
  • Added to the reference that nn::boss::FgOnlyTask must be specified in the nn::boss::RegisterImmediateTask function.
  • Changed Application ID notation in the nn::boss::ResultStorageNotFound class description to a more appropriate expression.

Additions and Changes37-S03:(CEC) Added function that obtains whether StreetPass box can be created [SDK]

Added nn::cec::MessageBox::CanCreateMessageBox function that obtains whether StreetPass box can be created.

Additions and Changes37-S04:(CEC) Added information on icon data and description data formats [SDK]

Added information on icon data and description data formats to the API reference.

Additions and Changes37-S05:(DevEnv) Updated nn.h [SDK]

Updated nn.h to allow use by all CTR-SDK API functions by using an include statement.

Additions and Changes37-S06:(DevEnv) Added definition for alignment restriction [SDK|SampleDemos]

Added definitions for alignment and size restrictions for the following libraries.

  • aacdec
  • aacenc
  • act
  • camera
  • cec
  • dlp
  • fs
  • gx
  • hid
  • http
  • ir
  • os
  • rdt
  • ro
  • snd
  • socket
  • uds

Additions and Changes37-S07:(DevEnv) Added and fixed system mode explanation [SDK]

The following system mode additions and fixes were done.

  • Added alert when system mode is set in the various functions that call system applets.
  • Fixed system mode explanation portion in Config.html.

Additions and Changes37-S09:(DevEnv) Added library for standard linking [SDK]

Added the following to the standard linked libraries in the CTR-SDK build system.

  • aacdec
  • aacenc
  • extrapad
  • qtm
  • webbrs

Additions and Changes37-S10:(DLP) Added warning in patch manual for when titles using fake clients are updated [SDK]

Added warning in patch manual for when titles using fake clients are updated.

Additions and Changes37-S12:(EC) Added nn::ec::CTR::PurchaseConfirmationApplet::RequestShowDialog member variable reference [SDK]

Clarified that nn::ec::CTR::PurchaseConfirmationApplet::RequestShowDialog returns Result::IsSuccess when users select the purchase button in the EC applet.

Additions and Changes37-S13:(EC)★ Added eShop REST API feature that provides functions for listing and purchasing service items through the independent server [SDK|SystemUpdater|SampleDemos]

Added eShop REST API feature that provides functions for listing and purchasing service items through the independent server.
In addition, the following was also added.

  • Manual containing required information for developing the independent server using the eShop REST API ($CTR_SDK/documents/TechnicalNotes/EShopRestApiManual)
  • eShop REST API ec/ServiceItemRestApi Demo

Additions and Changes37-S15:(EC) Added that Error 005-5637 will occur if local ticket is not deleted when reverting downloadable content [SDK]

Clarified in EDCK: Developing Applications That Support Downloadable Content that Error 005-5637 occurs if local ticket is not deleted when reverting downloadable content.

Additions and Changes37-S16:(ERREULA) Added description of value that specifies the nn::erreula::CTR::Config structure’s useLanguage member [SDK]

Added description of value that specifies the nn::erreula::CTR::Config structure’s useLanguage member.

Additions and Changes37-S17:(FS) Added operation for searching multiple media when mounting save data [SDK]

Added point about searching multiple media when mounting other titles with the nn::fs::MountSaveData function, and point about skipping from search with unformatted media.

Additions and Changes37-S18:(FS) Updated document related to the rollback prevention feature [SDK]

Added link to rollback prevention feature in the SaveDataFiler description.

Additions and Changes37-S20:(HIO) Updated the HostIO library manufactured by IS [SDK]

IS' HostIO library for the debugger has been updated to the 20150707 version.

Additions and Changes37-S21:(HIO) Added explanation of the nn::hio::SerialChannel::Flush function behavior [SDK]

Added explanation of the nn::hio::SerialChannel::Flush function behavior on the host side.

Additions and Changes37-S22:(NFP) Added 0x to items that display hexadecimal tag information in NfpUtil [SDK]

Changed such that 0x is added to items that display hexadecimal tag information in NfpUtil.

Additions and Changes37-S23:(NFP) Deleted nn::nfp::ResultDataAccessFailed [SDK]

Deleted nn::nfp::ResultDataAccessFailed from $CTR_SDK/include/nn/nfp/nfp_Result.h.
The NFP library function does not return nn::nfp::ResultDataAccessFailed.

Additions and Changes37-S24:(NFP) Added feature to rewrite NFP tag ROM regions in NoftWriter [SDK]

Added feature to rewrite NFP tag ROM regions in NoftWriter.
See the CTR Tools NoftWriter explanation page ($CTR_SDK/documents/ctools/NoftWriter.html) for details.

Additions and Changes37-S25:(NFP) Added feature to display NFT file Character Name and Character ID in NoftWriter [SDK]

Added feature to display NFT file Character Name and Character ID in NoftWriter.
See the CTR Tools NoftWriter explanation page ($CTR_SDK/documents/ctools/NoftWriter.html) for details.

Additions and Changes37-S26:(OS) Added warning about excessive calling of the nn::os::Thread::Sleep function [SDK]

Added that performance is reduced when the nn::os::Thread::Sleep function is called excessively within a short time interval and places a heavy load on the whole system.

Bug Fixes

Bug Fixes37-S02:(CEC) Fixed bug where the icon and title were broken in the StreetPass box imported with the CecBoxTool [SDK]

Fixed bug where icon and title information cannot be restored correctly when importing the StreetPass box with the CecBoxTool.

Bug Fixes37-S08:(DevEnv) Fixed bug where unbootable CCI are created [SDK]

Fixed bug where correct CCI were not created when size exceeded 16MB after static memory compression in ctr_makerom.

Bug Fixes37-S11:(EC) Fixed bug where downloadable content is not deleted when EcDevTool shop account is deleted [SDK]

Fixed bug where downloadable content was not deleted when EcDevTool shop account was deleted.

Bug Fixes37-S14:(EC) Fixed upper character limit to Details Information and Legal/Disclaimer Information displayed when purchasing a service item [SDK|SystemUpdater|SampleDemos]

Fixed upper limit to character count of Details Information and Legal/Disclaimer Information displayed when purchasing a service item.
For Details Information, 64 characters was increased to 128 characters, and there is no longer a limit for line breaks.
For Legal/Disclaimer Information, 512 characters was increased to 1024 characters, and there is no longer a limit for line breaks.
The EC library and ServiceItem sample was fixed along with this.

Bug Fixes37-S19:(HID) Fixed bug where Circle Pad Pro operates unintentionally during search [SDK|SampleDemos]

There was a bug where the Circle Pad Pro operated unintentionally during search.
In order to fix this, the working memory size passed to the nn::hid::CTR::ExtraPad::Initialize function was increased from 4096 to 12288 bytes.
Also, along with the working memory size increase, it was verified that the nn::hid::CTR::ExtraPad::Initialize function can receive this size.

Bug Fixes37-S27:(SND) Fixed bug that caused a deadlock when the sound library is under high load [SDK]

Fixed bug that caused a deadlock when the sound library function is called frequently within a short period of time.
