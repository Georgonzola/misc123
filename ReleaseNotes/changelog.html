CTR-SDK 11.4.0 (2015/09/07 - for System Updater 0.24.66) Change Logs

2015-09-07

Note that the System Updater for SNAKE that supports CTR-SDK 9.12 and later versions can only be applied to DP2 development hardware to which System Updater 0.22.32 or later versions has been applied, or to beta or later versions of IS-RAY-DEBUGGER.

Changes with a red star (★) in the title are of particular importance, so be sure to read them.

Changes in CTR-SDK 11.4.0 (2015-09-07 - for System Updater 0.24.66)

CTR-SDK

Additions and Changes

  • 37-S01: (BOSS) Changed document related to BOSS [SDK]
  • 37-S03: (CEC) Added a function that gets whether StreetPass box can be created [SDK]
  • 37-S04: (CEC) Added information on icon data and description data formats [SDK]
  • 37-S05: (DevEnv) Updated nn.h [SDK]
  • 37-S06: (DevEnv) Added a definition for alignment restriction [SDK | Sample Demos]
  • 37-S07: (DevEnv) Added and fixed the system mode explanation [SDK]
  • 37-S09: (DevEnv) Added a library for standard linking [SDK]
  • 37-S10: (DLP) Added a warning in the patch manual for when titles using fake clients are updated [SDK]
  • 37-S12: (EC) Added nn::ec::CTR::PurchaseConfirmationApplet::RequestShowDialog member variable reference [SDK]
  • 37-S13: (EC)★ Added an eShop REST API feature that provides functions for listing and purchasing service items through the independent server [SDK | System Updater | Sample Demos]
  • 37-S15: (EC) Added that Error 005-5637 will occur if the local ticket is not deleted when reverting downloadable content [SDK]
  • 37-S16: (ERREULA) Added a description of the value that specifies the nn::erreula::CTR::Config structure’s useLanguage member [SDK]
  • 37-S17: (FS) Added an operation for searching multiple media when mounting save data [SDK]
  • 37-S18: (FS) Updated the document related to the rollback prevention feature [SDK]
  • 37-S20: (HIO) Updated the IS HostIO library [SDK]
  • 37-S21: (HIO) Added an explanation of the nn::hio::SerialChannel::Flush function behavior [SDK]
  • 37-S22: (NFP) Added 0x to items that display hexadecimal tag information in NfpUtil [SDK]
  • 37-S23: (NFP) Deleted nn::nfp::ResultDataAccessFailed [SDK]
  • 37-S24: (NFP) Added a feature to rewrite NFP tag ROM regions in NoftWriter [SDK]
  • 37-S25: (NFP) Added a feature to display the NFT file Character Name and Character ID in NoftWriter [SDK]
  • 37-S26: (OS) Added a warning about excessive calling of the nn::os::Thread::Sleep function [SDK]

Bug Fixes

  • 37-S02: (CEC) Fixed a bug where the icon and title were broken in the StreetPass box imported with the CecBoxTool [SDK]
  • 37-S08: (DevEnv) Fixed a bug where unbootable CCI files are created [SDK]
  • 37-S11: (EC) Fixed a bug where downloadable content is not deleted when an EcDevTool shop account is deleted [SDK]
  • 37-S14: (EC) Fixed the upper character limit to Details Information and Legal/Disclaimer Information displayed when purchasing a service item [SDK | System Updater | Sample Demos]
  • 37-S19: (HID) Fixed a bug where Circle Pad Pro operates unintentionally during search [SDK | Sample Demos]
  • 37-S27: (SND) Fixed a bug that caused a deadlock when the sound library is under a high load [SDK]

System Updater

Additions and Changes

  • 37-U05: (EC) Added to the reference that Error 009-1400 occurs if Information by Language is not set when using the ec::RequestRedeemUniqueContentSetAuto function [SDK]
  • 37-U08: (EC)★ Added an eShop REST API feature that provides functions for listing and purchasing service items through the independent server [SDK | System Updater | Sample Demos]
  • 37-U12: (NFP) High frequency calling restrictions on the nn::nfp::StartDetection function [System Updater]

Bug Fixes

  • 37-U01: (APPLET) Fixed a bug where the screen is disrupted when transitioning to the HOME Menu and when returning to the application [System Updater]
  • 37-U02: (BOSS) Fixed a bug where an error is not returned if the NsData size is an invalid negative value [System Updater]
  • 37-U03: (CAMERA) Fixed a bug where a FATAL error infrequently occurs if the POWER button is pressed while the camera is in use [System Updater]
  • 37-U04: (DLP) Fixed a bug where an error occurs if scanning is stopped with the fake client and it immediately tries to join a session [System Updater]
  • 37-U06: (EC) Fixed the value returned by the EcApplet::RequestInitializeSession function [System Updater]
  • 37-U07: (EC) Fixed a bug where an error was not returned if a wireless switch was set to ON/OFF at a specific time during RequestInitializeSession [System Updater]
  • 37-U09: (EC) Fixed the upper character limit to Details Information and Legal/Disclaimer Information displayed when purchasing a service item [SDK | System Updater | Sample Demos]
  • 37-U10: (IR) Fixed a bug where Circle Pad Pro was inoperable after the HOME Menu crashed twice [System Updater]
  • 37-U11: (NFP) Fixed a bug where the system stopped responding when aging RemoveTag in CTR [System Updater]
  • 37-U13: (NFP) Fixed a bug where GetConnectResult returned ResultConnectCanceled after disconnection due to ResultIrFunctionError [System Updater]
  • 37-U14: (NFP) Fixed a bug where the UID for the Type1 tag could not be correctly obtained with SNAKE [System Updater]

Sample Demos

Additions and Changes

  • 37-D01: (BOSS) Added a supplement stating that deleting tasks each time with the BOSS sample is not recommended [Sample Demos]
  • 37-D02: (DevEnv) Added a definition for alignment restriction [SDK | Sample Demos]
  • 37-D03: (EC)★ Added an eShop REST API feature that provides functions for listing and purchasing service items through the independent server [SDK | System Updater | Sample Demos]

Bug Fixes

  • 37-D04: (EC) Fixed the upper character limit to Details Information and Legal/Disclaimer Information displayed when purchasing a service item [SDK | System Updater | Sample Demos]
  • 37-D05: (HID) Fixed error handling in hid/ExtraPad/AppRecommendedFlow sample [Sample Demos]
  • 37-D06: (HID) Fixed a bug where Circle Pad Pro operates unintentionally during search [SDK | Sample Demos]
  • 37-D07: (NFP) Fixed processing when the NFP sample demo POWER button is pressed [Sample Demos]
