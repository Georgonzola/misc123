Symptoms Due to Design Limitations of Specifications

The following list is of the currently confirmed symptoms that arise due to the design limitations of the specifications.

  1. [APPLET] The system sometimes stops on a black screen if a FATAL error occurs at the point the system is closed.
  2. [FS] The system may freeze if the CTR card is repeatedly inserted and removed.
  3. [FS] Removing a card while it is being accessed may cause a fatal error or cause the system to stop operating.
  4. [FS] The system may freeze if the SD card is inserted and removed when a downloadable application is starting.
  5. [HID] On rare occasions, input stops working from buttons other than the HOME Menu button.

Symptom

The system sometimes stops on a black screen if a FATAL error (including card removal error) occurs at the point the system is closed.

Frequency of Occurrence

Occurs very frequently if the time is right.

Affects

All applications.


Symptom

The system may freeze if you repeatedly insert and remove a CTR card so quickly that there is no time or just enough time for the icon for the CTR card to appear on the HOME Menu.

Frequency of Occurrence

Occurs very frequently if the time is right.

Affects

All card-based applications.


Symptom

If the card to be accessed is removed while the CTR Game Card or SD card is being accessed, the following may occur.

  • It causes a fatal error.
  • Pressing the HOME Button on the card removal error screen does not transition to the HOME Menu.
  • If the SD card is inserted and removed, data on the card will be destroyed.

Frequency of Occurrence

Occurs very frequently if the time is right.

Affects

All applications.


Symptom

The system can freeze with a black screen if an SD card is inserted and removed while starting a CTR application that was downloaded to the system.

Frequency of Occurrence

Occurs infrequently.

Affects

All downloadable applications.


Symptom

On rare occasions, input stops working from buttons other than the HOME Menu button. (Input from the touch panel also becomes impossible.)
If this occurs, you can restore functionality using either of these procedures.

  • Transition to sleep and then wake back up.
  • Change the headphone connection state.
    • If headphones are attached, remove them.
    • If headphones are not attached, plug them in.

Frequency of Occurrence

Occurs on rare occasions with PARTNER-CTR Capture and PARTNER-CTR Capture Debugger.

Affects

All applications.
