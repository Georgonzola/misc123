1. Introduction

A Nintendo Network account is an account for the use of Nintendo network features across devices and platforms. Users can register a Nintendo Network ID on the CTR starting from system version 7.0.0, enabling the authentication of a Nintendo Network account and the use of registration information.

Accordingly, it is now possible to distinguish users on an account by account basis rather than on the device by device basis that had been used previously. The following information is stored for each account.

  • User profile
  • E-commerce information
  • Miiverse information

One account (a system account) is automatically created for each system. If that system account is deleted, the system creates another system account.

System accounts can be associated with Nintendo Network accounts, which are managed on the Nintendo Network account server. System accounts that are associated with Nintendo Network accounts are known as network accounts, and system accounts that are not are known as local accounts.

Users must enter information such as a user ID, password, email address, date of birth, address, and time zone to create a Nintendo Network account. Once a CTR system account is associated with this Nintendo Network account, this information can be used from the CTR itself. Also, it will be possible to associate Nintendo Network accounts with various devices in the future, and users will be able to share their CTR account information with other devices in addition to share information with their Wii U system account all via their Nintendo Network accounts.

Friend accounts, which had been used to manage friend, matchmaking, and ranking information, will also exist on the system starting from system version 7.0.0. Services provided prior to system version 7.0.0 use friend accounts for authentication and reference the information bound to the friend accounts, excluding the information for e-commerce. Services provided with system version 7.0.0 and later allow you to choose whether you use the services unique to the use of a network account or to the use of a friend account. Basically, we recommend the use of a network account only when it is necessary to share information with other devices using the Nintendo Network account. In all other cases, the use of friend accounts is recommended.

On the CTR, Miiverse features cannot be used when the system account is not associated with a Nintendo Network account. To take full advantage of all of the network features available, you must associate the system account with a Nintendo Network account.
