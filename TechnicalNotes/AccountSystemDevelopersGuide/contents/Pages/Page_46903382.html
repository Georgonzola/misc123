3. Managing System Accounts

3.1. Creating and Deleting System Accounts

A system account is automatically created by the system. The system account is deleted and a new system account is automatically created at the following times.

  • When deleting network accounts
  • When initializing the system

3.2. Associating a System Account With a Nintendo Network Account

Associating a CTR system account with a Nintendo Network account enables the use of features provided directly with Nintendo Network accounts and services provided by using Nintendo Network accounts on the CTR. Once a system account is upgraded by associating it with a Nintendo Network account, it is known as a network account. A system account that has not been associated with a Nintendo Network account is known as a local account.

Figure 3-1. Network Accounts and Local Accounts.

On the CTR, new Nintendo Network accounts can be created and existing Nintendo Network accounts can be associated with system accounts from the Nintendo Network ID Settings application in System Settings.

There is a one-to-one relationship for each platform between system accounts and Nintendo Network accounts. A single system account cannot be associated with multiple Nintendo Network accounts, and multiple CTR platform system accounts cannot be associated with a single Nintendo Network account. Conversely, the network account associated with a CTR system account can also be associated with a Wii U system account. If the system account that a Nintendo Network account was originally associated with is deleted (for example, due to a system initialization), that Nintendo Network account may be associated with another system account as long as it is on the same console.

When transferring software and data between CTRs, you can use the system account and the associated Nintendo Network account on the target CTR. In this instance, the system account is deleted from the original system, and reassociation is no longer possible.

3.2.1. Default Age-Specific Account Restrictions

When a system account is associated with a Nintendo Network account, some restrictions are applied to the account in some regions depending on the user's age.

  • Europe
    • Users under 13
      • All items in Parental Controls are restricted.
  • Australia/New Zealand
    • Users under 15
      • Launch of applications with ratings other than G, PG, or M is restricted in Parental Controls.

If a PIN code for use with Parental Controls has not been configured on the console, it must be configured during account association.

This restriction can be removed later by a parent or guardian even if the user is not of age.

3.2.2. Disassociating Accounts

In most cases, it is not possible to disassociate system accounts and Nintendo Network accounts once they are associated.

However, a method for doing so is provided through assistance from user support.

Once a network account has been demoted, through disassociation, to a local account, that local account works exactly the same way as it did prior to association.
