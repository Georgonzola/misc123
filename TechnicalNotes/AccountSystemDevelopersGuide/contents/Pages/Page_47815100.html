6. Debugging Features for Developers

A website (the Account Developer Portal) is provided for attempting to simulate the errors that can occur when using the account server, and for checking how many times functions that get service tokens have been called. It includes the following functionality.

  • A feature for editing account information
  • A feature for simulating the steps taken for accounts or devices that are no longer used
  • A feature for simulating system upgrades, maintenance, or service termination
  • A feature for checking a call log for functions that get service tokens

Note:

For more information about how to access and use the Account Developer Portal, see the CTR Network Error Simulation Manual, which will be updated in the future. Some features are still being prepared.
