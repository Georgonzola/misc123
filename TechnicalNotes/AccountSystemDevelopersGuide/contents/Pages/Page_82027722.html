8. Appendix: Service Token V2 Sample Decoding and Decryption Program

This sample program decodes and decrypts service tokens V2 issued by an independent server.

This sample can also set an expiration period for a service token using the timestamp information included with the token when it is issued. This sample program treats expiration periods of more than 24 hours as invalid. You may change this specification for each title as necessary.

8.1. Perl Sample (Service Token V2)

DecodeServiceTokenV2Sample.pl

This code sample uses Perl to decode a service token V2. This code has been confirmed to run in the following environment.

  • Perl 5.10.1
  • Perl-Crypt-CBC-2.33
  • Perl-Crypt-OpenSSL-AES-0.02-9
  • Perl-Digest-SHA1-2.13

Set the client secret (a hexidecimal string) in $CLIENT_SECRET and the public key for verifying the NNAS server signature in $SERVER_PUBKEY. Then, pass the service token V2, initial vector, and signature received in the NNAS API response as arguments.

Code 8-1. Perl Sample Code
0001  #!/usr/bin/perl
0002  
0003  use strict;
0004  use warnings;
0005  use MIME::Base64;
0006  use Crypt::CBC;
0007  use Digest::SHA1 qw(sha1_hex);
0008  use File::Slurp;
0009  use Crypt::OpenSSL::RSA;
0010  
0011  my $CLIENT_SECRET = read_file("../secret/client_secret");
0012  my $SERVER_PUBKEY = read_file("../secret/server.key");
0013  
0014  if ($#ARGV != 2 && $#ARGV != 3) {
0015      print "perl DecodeServiceTokenSample.pl <SERVICE_TOKEN> <INITIAL_VECTOR> [SIGNATURE]\n";
0016      exit
0017  }
0018  my $service_token_base64 = $ARGV[0];
0019  my $initial_vector_base64 = $ARGV[1];
0020  my $signature_base64 = $ARGV[2];
0021  
0022  my $cipher = Crypt::CBC->new(
0023      -key => pack("H32", $CLIENT_SECRET),
0024      -keysize => 16,
0025      -literal_key => 1,
0026      -cipher => "Crypt::OpenSSL::AES",
0027      -iv => decode_base64($initial_vector_base64),
0028      -header => 'none',
0029      );
0030  
0031  my $decrypted_service_token = $cipher->decrypt(decode_base64($service_token_base64));
0032  
0033  my ($validation_string, $hash_string) = ($decrypted_service_token =~ /^(.+)\\h\\([0-9a-z]{40})$/);
0034  my $calchash = sha1_hex($validation_string);
0035  if ($calchash ne $hash_string) {
0036      print "Invalid hash ${calchash} != ${hash_string}\n";
0037      exit(1);
0038  }
0039  
0040  my $platform_id  = ($decrypted_service_token =~ /\\z\\(\d+)/)[0];
0041  my $principal_id = ($decrypted_service_token =~ /\\u\\(\d+)/)[0];
0042  my $account_id   = ($decrypted_service_token =~ /\\a\\([0-9a-zA-Z\-_\.]{6,16})/)[0];
0043  my $server_env   = ($decrypted_service_token =~ /\\s\\([A-Z]\d)/)[0];
0044  my $game_version = ($decrypted_service_token =~ /\\v\\([0-9A-F]{4})/)[0];
0045  my $unique_id    = ($decrypted_service_token =~ /\\e\\([0-9A-F]{5})/)[0];
0046  my $timestamp    = ($decrypted_service_token =~ /\\t\\(\d+)/)[0];
0047  
0048  my $now = time;
0049  if ($now - $timestamp > 24 * 60 * 60) {
0050      print "Warning: Token expired now=${now} timestamp=${timestamp}\n";
0051  }
0052  
0053  print "Platform_ID: ${platform_id}\n";
0054  print "Principal_ID(PID): ${principal_id}\n";
0055  print "Account_ID: ${account_id}\n";
0056  print "Server_ENV: ${server_env}\n";
0057  print "Game_Version: ${game_version}\n";
0058  print "Unique_ID: ${unique_id}\n";
0059  
0060  if ($signature_base64) {
0061    my $key = Crypt::OpenSSL::RSA->new_public_key($SERVER_PUBKEY);
0062    $key->use_sha256_hash();
0063    if ($key->verify($decrypted_service_token, decode_base64($signature_base64))) {
0064      print "Signature: correct\n";
0065    } else {
0066      print "Signature: verify failed\n";
0067    }
0068  }
0069  
0070  exit(0);

8.2. Python Sample (Service Token V2)

DecodeServiceTokenV2Sample.py

This code sample uses Python to decode a service token V2. This code has been confirmed to run in the following environment.

  • Python-2.7.6
  • PyCrypto-2.6.1

Set the client secret (a hexidecimal string) in $CLIENT_SECRET and the public key for verifying the NNAS server signature in $SERVER_PUBKEY. Then, pass the service token V2, initial vector, and signature received in the NNAS API response as arguments.

0014  CLIENT_SECRET = open("../secret/client_secret").read().rstrip()
0015  serverPublicKey = open("../secret/server.key").read()
0016  BLOCK_SIZE = 16
0017  def pkcs7unpad(text, blocksize):
0018      if text:
0019          padlen = ord(text[-1])
0020          if padlen and padlen <= blocksize:
0021              text = text[:-padlen]
0022          else:
0023              raise Exception
0024      return text
0025  usage = """
0026      DecodeServiceTokenSample.py [OPTIONS] service_token initial_vector [signature]
0027  """
0028  p = optparse.OptionParser(usage=usage)
0029  (opt, args) = p.parse_args()
0030  if len(args) != 2 and len(args) != 3:
0031      p.print_help()
0032      sys.exit(1)
0033  service_token_base64 = args[0]
0034  initial_vector_base64 = args[1]
0035  if len(args)==3:
0036      signature_base64 = args[2]
0037  else:
0038      signature_base64 = None
0039  crypt = Crypto.Cipher.AES.new(
0040                  binascii.unhexlify(CLIENT_SECRET),
0041                  Crypto.Cipher.AES.MODE_CBC,
0042                  base64.b64decode(initial_vector_base64)
0043                  )
0044  decrypted_service_token = pkcs7unpad(
0045                  crypt.decrypt(base64.b64decode(service_token_base64)),
0046                  BLOCK_SIZE
0047                  )
0048  r = re.compile(r'^(.+)\\h\\([0-9a-f]{40})$')
0049  m = r.search(decrypted_service_token)
0050  validation_string = m.group(1)
0051  hash_string = m.group(2)
0052  calchash = hashlib.sha1(validation_string).hexdigest()
0053  if calchash != hash_string:
0054      print "Invalid hash %s != %s"%(calchash, hash_string)
0055      sys.exit(1)
0056  platform_id  = re.compile(r'\\z\\(\d+)').search(validation_string).group(1)
0057  principal_id = re.compile(r'\\u\\(\d+)').search(validation_string).group(1)
0058  account_id   = re.compile(r'\\a\\([0-9a-zA-Z\-_\.]{6,16})').search(validation_string).group(1)
0059  server_env   = re.compile(r'\\s\\([A-Z]\d)').search(validation_string).group(1)
0060  game_version = re.compile(r'\\v\\([0-9A-F]{4})').search(validation_string).group(1)
0061  unique_id    = re.compile(r'\\e\\([0-9A-F]{5})').search(validation_string).group(1)
0062  timestamp    = re.compile(r'\\t\\(\d+)').search(validation_string).group(1)
0063  
0064  now = int(time.time())
0065  if now - int(timestamp) > 24 * 60 * 60:
0066      print "Warning: Token expired now=%d timestamp=%s"%(now, timestamp)
0067  print "Platform_ID: %s"%(platform_id,)
0068  print "Principal_ID(PID): %s"%(principal_id,)
0069  print "Account_ID: %s"%(account_id,)
0070  print "Server_ENV: %s"%(server_env,)
0071  print "Game_Version: %s"%(game_version,)
0072  print "Unique_ID: %s"%(unique_id,)
0073  if signature_base64:
0074      key = RSA.importKey(serverPublicKey)
0075      verifier = PKCS1_v1_5.new(key)
0076      digest = SHA256.new(decrypted_service_token)
0077      if verifier.verify(digest, base64.b64decode(signature_base64)):
0078          print "Signature: correct"
0079      else:
0080          print "Signature: verify failed"
0081  sys.exit(0)

This code sample uses PHP to decode a service token V2. This code has been confirmed to run in the following environment.

  • php-5.6.6
  • php-mcrypt-5.6.6

Signature verification for service token V2 is not supported in PHP versions prior to 5.4.8 because SHA256 is not available for the signature verification algorithm. (Decoding only is possible.) Take note of this.

Set the following variables in the source file before running the decoding process.

  • $CLIENT_SECRET_HEX variable: The client secret as a hexadecimal string.
  • $INITIAL_VECTOR_BASE64 variable: The initial vector included in the API response (left as BASE64 encoded).
  • $SIGNATURE_BASE64 variable: The signature included in the API response (left as BASE64 encoded).
  • $SERVICE_TOKEN_BASE64 variable: The service token V2 included in the API response (left as BASE64 encoded).
  • server.key file: The NNAS public key.

0021  function DecryptServiceTokenV2($token_base64, $clientSecret, $iv_base64)
0022  {
0023      $cipher = mcrypt_module_open(MCRYPT_RIJNDAEL_128, '', MCRYPT_MODE_CBC, '');
0024      $iv = base64_decode($iv_base64);
0025      if(mcrypt_generic_init($cipher, pack('H32', $clientSecret), $iv) == 0 )
0026      {
0027          $token = base64_decode($token_base64);
0028          //var_dump($data);

0032          $decryptedToken = mdecrypt_generic($cipher, $data);
0033         
0034          mcrypt_generic_deinit($cipher);
0035          mcrypt_module_close($cipher);
0036         
0037          return unpadPKCS7($decryptedToken, 128/8);
0038      }
0039      return '';
0040  }
