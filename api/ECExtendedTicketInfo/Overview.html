ECExtendedTicketInfo Structure

Syntax

struct ECExtendedTicketInfo
{
   unsigned short version;
   unsigned short ticketVersion;
   unsigned int deviceId;
   unsigned long long titleId;
   unsigned long long ticketId;
   unsigned long long accountId;
   time_t expiration;
   int isAccountIdFromTicket;
   unsigned int ticketType;
   ECItemRights * itemRights;
   unsigned int itemRightsLen;
   void * extension;
};

Description

Structure that stores ticket information.

Member Variables

version unsigned short The version of the ECExtendedTicketInfo structure.
ticketVersion unsigned short The ticket version.
deviceId unsigned int The device ID. Do not use this value.
titleId unsigned long long The title ID.
ticketId unsigned long long The ticket ID. Do not use this value.
accountId unsigned long long The account ID.
expiration time_t The expiration date. Specified in seconds since 1970/1/1 00:00:00 UTC.
isAccountIdFromTicket int Takes a value other than 0 if there were no account IDs listed on the ticket.
ticketType unsigned int Ticket type. One of the following: EC_TICKET_PERMENANT or EC_TICKET_SUBSCRIPTION.
itemRights ECItemRights * Pointer to the ECItemRights structure array. This is a list of item licenses for each item.
itemRightsLen unsigned int The number of elements in the array pointed by itemRights.
extension void * Reserved.
