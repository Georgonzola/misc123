ECItemRights Structure

Syntax

struct ECItemRights
{
   unsigned short type;
   unsigned int attr;
   unsigned char refId[EC_REF_ID_SIZE];
   unsigned int refIdAttr;
};

Description

Structure that stores item license information.

Member Variables

type unsigned short Item type. One of the following: EC_ITEM_PERMENANT, EC_ITEM_SUBSCRIPTION, or EC_ITEM_QUANTITY.
attr unsigned int Subscription/quantity license information. For subscription passes, this attribute stores the amount of time until the pass expires (in seconds since 00:00:00 UTC January 1, 1970). For consumable tickets, this attribute stores the number of uses purchased. (10 if 10 uses were purchased. It does not matter how many uses have actually been used.)
refId unsigned char Reference ID.
refIdAttr unsigned int Reserved.
