EC_GenerateRIVToken Function

Syntax

int EC_GenerateRIVToken(
     const char * privKey,
     const char * itemCode,
     const unsigned char refId[EC_REF_ID_SIZE],
     time_t validUntil,
     char ** rivToken,
     void *(*)(size_t) allocator,
     void(*)(void *) deallocator
);

Parameters

Name Description
in privKey A private key string.
in itemCode The item code for the requested item.
in refId[EC_REF_ID_SIZE] The reference ID for the requested item.
in validUntil The amount of time until the RIV token expires in seconds since 00:00:00 UTC January 1, 1970.
out rivToken The buffer to write the RIV token to.
in allocator An allocator for allocating a buffer for storing the license information for each item. Specify NULL to make the function use the system's malloc function.
in deallocator A deallocator for deallocating the buffer allocated by allocator. Specify NULL to make the function use the system's free function.

Return Values

Returns the result of the function.

Value Description
EC_OK Success.
EC_OUT_OF_MEMORY Insufficient memory to complete the process.
EC_CRYPTO_ERROR Internal encryption error while signing RIV token.

Description

Generates an RIV token based on the specified private key, item code, reference ID, and expiration date. The generated RIV token is a variable length, NULL-terminated string.

RIV tokens are used to verify that the specified item code and reference ID have not been illegally modified.

The caller of this function must pass an allocator/deallocator to allocate a buffer for storing the RIV token. The caller of this function is responsible for deallocating the allocated buffer. Deallocate the buffer after sending the RIV token to the client.
