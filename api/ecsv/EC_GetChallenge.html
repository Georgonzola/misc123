EC_GetChallenge Function

Syntax

int EC_GetChallenge(
     int challengeId,
     ECChallengeBlob cBlob_out,
     time_t secsTillExpire
);

Parameters

Name Description
in challengeId A challenge ID to associate with the challenge blob.
out cBlob_out A buffer to write the challenge blob to.
in secsTillExpire The amount of time until the challenge blob expires.

Return Values

Returns the result of the function.

Value Description
EC_OK Success.
EC_OUT_OF_MEMORY Insufficient memory to complete the process.
EC_GEN_KEY_ERROR Internal encryption error.
EC_INVALID_CERT_FILE Settings error. Invalid certificate authority file.
EC_CRYPTO_ERROR Internal encryption error while verifying the ticket envelope. The ticket envelope may be invalid or corrupted.

Description

Creates a new challenge blob for the specified challenge ID and writes it to the output buffer.

For the challenge IDs, use values that can uniquely identify secure communication channels between clients and the game server. The created challenge blob is sent to the clients.
