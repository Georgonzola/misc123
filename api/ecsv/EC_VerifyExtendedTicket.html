EC_VerifyExtendedTicket Function

Syntax

int EC_VerifyExtendedTicket(
     int challengeId,
     const void * extendEnvBlob,
     int extendEnvBlobSize,
     ECExtendedTicketInfo * extendedTicketInfo_out,
     void *(*)(size_t) allocator,
     void(*)(void *) deallocator
);

Parameters

Name Description
in challengeId The challenge ID associated with the ticket envelope.
in extendEnvBlob The ticket envelope sent by the client.
in extendEnvBlobSize The size of the ticket envelope.
out extendedTicketInfo_out A pointer to the instance of the ECExtendedTicketInfo structure to write the ticket information to.
in allocator An allocator for allocating a buffer for storing the license information for each item. Specify NULL to make the function use the system's malloc function.
in deallocator A deallocator for deallocating the buffer allocated by allocator. Specify NULL to make the function use the system's free function.

Return Values

Returns the result of the function.

Value Description
EC_OK Success.
EC_OUT_OF_MEMORY Insufficient memory to complete the process.
EC_GEN_KEY_ERROR Internal encryption error.
EC_INVALID_CERT_FILE Settings error. Invalid certificate authority file.
EC_INVALID_EBLOB_FORMAT The ticket envelope is formatted incorrectly or corrupted. Ticket envelope failed to authenticate challenge ID.
EC_CRYPTO_ERROR Internal encryption error while verifying the ticket envelope. The ticket envelope may be invalid or corrupted.
EC_INVALID_CHALLENGE Ticket envelope contains an invalid challenge key. The challenge ID may be invalid, the challenge key may have expired, or the challenge may be corrupted.
EC_INVALID_CERT Ticket envelope does not contain a valid certificate chain for verifying its authenticity.
EC_INVALID_SIGNATURE Ticket envelope does not contain a valid signature.
EC_INVALID_TICKET Invalid ticket information in ticket envelope.

Description

Verifies the challenge ID against the ticket envelope sent from the client. If verification is successful, the function writes the ticket information to extendedTicketInfo_out.

Each ticket contains a list of license information for each item. The license information list only includes currently valid licenses owned by the user. The license information for each item contains the item type, the expiration date or the total number of uses purchased for the item, and a reference ID. For subscription items, the item expiration date can be retrieved. Compare the expiration date against the current server time to determine whether the service item can still be used. For consumable service item tickets, the total number of uses purchased for the item can be retrieved. Compare the number of times the user has used the item with the total number of uses purchased to determine whether the service item can still be used.

The caller of this function must pass an allocator and deallocator to allocate a buffer for storing the license information list. Deallocating the allocated buffer is the responsibility of the caller of this function.

If this function returns an authentication error (an error other than EC_OUT_OF_MEMORY or EC_GEN_KEY_ERROR), display error 009-8001 in the Error/EULA display applet.
