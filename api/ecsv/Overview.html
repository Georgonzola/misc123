ECSV

Description

The ECSV library provides features for selling and managing service items. You can use the ECSV library by installing it on your independent server.
The ECSV library API function group follows below.
All API functions are thread-safe.

Structures

ECItemRights Structure that stores item license information.
ECExtendedTicketInfo Structure that stores ticket information.

Macros

EC_ITEM_PERMENANT Permanent item pass.
EC_ITEM_SUBSCRIPTION Subscription item.
EC_ITEM_QUANTITY Consumable service item.
EC_TICKET_PERMENANT Permanent ticket.
EC_TICKET_SUBSCRIPTION Subscription ticket (has an expiration date).

Functions

EC_GetChallenge Creates a new challenge blob for the specified challenge ID and writes it to the output buffer.
EC_VerifyExtendedTicket Verifies the challenge ID against the ticket envelope sent from the client. If verification is successful, the function writes the ticket information to extendedTicketInfo_out.
EC_GenerateRIVToken Generates an RIV token based on the specified private key, item code, reference ID, and expiration date. The generated RIV token is a variable length, NULL-terminated string.
EC_SetKey Call the EC_SetKey function once on each server instance before calling any of the other ECSV API functions.
