glBindRenderbuffer Function

Syntax

GL_APICALL void GL_APIENTRY glBindRenderbuffer(
     GLenum target,
     GLuint renderbuffer
);

Parameters

Name Description
in target Bind target.
in renderbuffer Render buffer object.

Return Values

No values are returned.

Description

Binds a render buffer object to the current bind target.

target is specified as GL_RENDERBUFFER.

Set renderbuffer to a render buffer object generated by glGenRenderbuffers. If you specify a nonexistent render buffer object, a new render buffer object of that name is created simultaneously.

This function generates the following errors.

GL_INVALID_ENUMThe target argument was set to an invalid value.
GL_OUT_OF_MEMORYFailed to allocate the management region.
