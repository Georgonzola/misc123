glCreateProgram Function

Syntax

GL_APICALL GLuint GL_APIENTRY glCreateProgram(
     void
);

Parameters

None.

Return Values

The created program object.

Description

Creates a program object.

Returns the program object's name. The name is represented by a 13-bit integer. Shader objects use a separate namespace.

This function generates the following errors.

GL_INVALID_OPERATIONThe number of generated program objects exceeded what can be represented by a 13-bit integer.
GL_OUT_OF_MEMORYFailed to allocate the management region.
