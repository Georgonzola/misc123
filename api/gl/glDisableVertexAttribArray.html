glDisableVertexAttribArray Function

Syntax

GL_APICALL void GL_APIENTRY glDisableVertexAttribArray(
     GLuint index
);

Parameters

Name Description
in index Indices of the attributes to disable.

Return Values

No values are returned.

Description

Disables the vertex attribute array.

You can set index to a value that is greater than or equal to 0 and at least one less than the value that can be obtained by glGetIntegerv with GL_MAX_VERTEX_ATTRIBS specified.

This function generates the following errors.

GL_INVALID_VALUEThe index argument was set to an invalid value.
