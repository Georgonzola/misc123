glGenRenderbuffers Function

Syntax

GL_APICALL void GL_APIENTRY glGenRenderbuffers(
     GLsizei n,
     GLuint * renderbuffers
);

Parameters

Name Description
in n Number of render buffer objects to generate.
out renderbuffers Array that stores the generated render buffer objects.

Return Values

No values are returned.

Description

Generates render buffer objects.

Set n the number of objects to generate. Set renderbuffers to the array in which to store the generated objects.

This function generates the following errors.

GL_INVALID_VALUEThe n argument is negative.
GL_OUT_OF_MEMORYFailed to allocate the management region.
