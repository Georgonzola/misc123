glIsBuffer Function

Syntax

GL_APICALL GLboolean GL_APIENTRY glIsBuffer(
     GLuint buffer
);

Parameters

Name Description
in buffer Value to use to determine whether the object is a valid vertex buffer object.

Return Values

Returns GL_TRUE when the object is a valid vertex buffer object. Otherwise, returns GL_FALSE.

Description

Checks whether an object is a valid vertex buffer object.

Returns GL_TRUE when the name specified by buffer indicates a valid vertex buffer object. Otherwise, returns GL_FALSE. No errors occur in this function.
