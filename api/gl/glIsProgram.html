glIsProgram Function

Syntax

GL_APICALL GLboolean GL_APIENTRY glIsProgram(
     GLuint program
);

Parameters

Name Description
in program Program object.

Return Values

Returns GL_TRUE if valid. Otherwise, returns GL_FALSE.

Description

Checks whether a program object is valid.

Returns GL_TRUE when the name specified by program indicates a valid program object. Otherwise, returns GL_FALSE. No errors occur in this function.
