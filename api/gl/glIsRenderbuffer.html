glIsRenderbuffer Function

Syntax

GL_APICALL GLboolean GL_APIENTRY glIsRenderbuffer(
     GLuint renderbuffer
);

Parameters

Name Description
in renderbuffer Render buffer object.

Return Values

Returns GL_TRUE if valid. Otherwise, returns GL_FALSE.

Description

Checks whether an object is a valid render buffer object.

Returns GL_TRUE when the name specified by renderbuffer indicates a valid render buffer object. Otherwise, returns GL_FALSE. No errors occur in this function.
