glTexParameterfv Function

Syntax

GL_APICALL void GL_APIENTRY glTexParameterfv(
     GLenum target,
     GLenum pname,
     const GLfloat * params
);

Parameters

Name Description
in target Texture target.
in pname The name of the parameter to set.
in params Array that stores the values to set for the parameters.

Return Values

No values are returned.

Description

Sets an array of texture parameter values as GLfloat-type values.

You can set target to the following values.

GL_TEXTURE_2D2D texture.
GL_TEXTURE_CUBE_MAPCube-map texture.

Set pname to one of the following values.

GL_TEXTURE_BORDER_COLORSpecifies the texture border color using 4 elements.
GL_TEXTURE_LOD_BIASSpecifies the texture LOD bias. Set params[0] to a value in the range from -16 to 16.
You can also specify parameters that can be specified with glTexParameteriv.

This function generates the following errors.

GL_INVALID_ENUMThe target or pname argument was set to an invalid value.
GL_INVALID_VALUEThe param parameter was set to an invalid value.
