glUniform1f Function

Syntax

GL_APICALL void GL_APIENTRY glUniform1f(
     GLint location,
     GLfloat x
);

Parameters

Name Description
in location Uniform location.
in x The value to set.

Return Values

No values are returned.

Description

Sets a single GL_FLOAT-type uniform value or a single element of an array type of GL_FLOAT uniform values for a vertex shader, geometry shader, or reserved fragment shader.

Set location to the uniform variable's location. Use x to specify the value to set.

Calls to this function are ignored when location is -1.

This function generates the following errors.

GL_INVALID_OPERATIONThe location argument is not a location in the current program.
GL_INVALID_OPERATIONThere is no current program object.
GL_INVALID_OPERATIONThis function was called on a location with a different data type or component count.
GL_INVALID_OPERATIONThe number of uniform arrays is less than the offset that was added to the location value obtained from glGetUniformLocation.
