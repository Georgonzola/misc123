CTR-SDK API

Overview

This section describes general assumptions regarding the CTR-SDK API.

Use of Private APIs Is Prohibited

Do not use private APIs.
Private APIs may be added, deleted, or changed without any prior or later notice.

Header include

Unless otherwise specified, put include headers directly under nn directories (for example, nn/os.h). The structure of headers lower in the hierarchy may change.
