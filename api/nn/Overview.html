nn Namespace

Description

The uppermost namespaces of the namespaces defined for the CTR-SDK API.

Namespaces

nn::aacdec Namespace for the AAC decoder (AACDEC) library.
nn::aacenc Namespace for the AAC encoder (AACENC) library.
nn::ac The namespace of the library used to connect to an access point.
nn::act Namespace for the account library.
nn::applet The namespace of the applet library.
nn::boss The namespace of the SpotPass communication library.
nn::camera The namespace of the camera library.
nn::cec The namespace of the StreetPass library.
nn::cfg The namespace of the System Settings library.
nn::crypto The namespace of the encryption library.
nn::cx The namespace of the data compression and decompression library.
nn::dbg The namespace of the debug assist library.
nn::dlp The namespace of the Download Play library.
nn::dsp The namespace of the library for executing processes using DSP.
nn::ec The namespace of the E-Commerce API.
nn::enc The namespace of the character code encoding library.
nn::erreula The namespace of the error/EULA applet call-up library.
nn::extrapad The namespace of the Circle Pad Pro calibration applet call-up library.
nn::fnd The namespace of the foundation library.
nn::font Namespace for the font library.
nn::friends Namespace for the friend library.
nn::fs The namespace of the file system library.
nn::gd The namespace of the GD (graphics) API.
nn::gr The namespace of the GPU register access library.
nn::gx The namespace of the graphics library.
nn::hid The namespace of the input device library.
nn::hio The namespace of the library for communicating with a host PC.
nn::http The namespace of the HTTP communication library.
nn::init The namespace of the application start-up library.
nn::ir The namespace of the library used for infrared communication between systems.
nn::jpeg The namespace of the JPEG data compression and decompression library.
nn::math The namespace of the arithmetic library.
nn::mic The namespace of the microphone library.
nn::midi The namespace of the MIDI library.
nn::ndm The namespace of the network daemon control library.
nn::news The namespace of the message library.

nn::olv The namespace of the OLV library.
nn::os The namespace of the operating system library.

nn::ptm The namespace of the power supply and clock library.
nn::qtm Namespace for the face tracking library.
nn::rdt The namespace of the library for reliable local communication.
nn::ro The namespace of the library providing DLL functionality.
nn::snd The namespace of the sound library.
nn::socket The namespace of the socket library.
nn::swkbd The namespace of the applet call-up library for the software keyboard.
nn::tpl The namespace of the library for loading textures from files created by the TexturePackager tool.
nn::ubl The namespace of the blocked user lists (blacklists).

nn::ulcd The namespace of the stereoscopic view library.
nn::util The namespace of the utility library.
nn::voicesel The namespace of the voice selector applet call-up library.
nn::webbrs Namespace for the Internet Browser API.
nn::y2r The namespace of the library for YUV-to-RGB conversion.

Classes

nn::Result Represents the result of an operation.
