nn::Result::operator== Member Function

Syntax

#include <nn/Result.h>
bool operator==(
     const Result & rhs
) const;

Parameters

Name Description
in rhs Specifies the right-hand operand.

Return Values

Returns true if the operands are equal, and false otherwise.

Description

Checks for equality.

Use to handle some errors for libraries other than nn::fs. For more information about nn::fs library error handling, see the reference materials for the nn::fs library.
