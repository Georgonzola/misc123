nn::aacdec::CTR::GetErrorCode Function

Syntax

u32 GetErrorCode(
     void
);

Parameters

None.

Return Values

Returns the error code.

Description

Returns an error code.

Here are the details about the errors.

Error CodeDetails
0Decoding was successful.
1This error occurs if a syntax error was detected in the AAC data, or if the data is out of specification.
2This error occurs if the sampling frequency or channel configuration has changed from the previous frame. When this is the case, you can decode the data by calling the initialization function and then passing the same data again.
