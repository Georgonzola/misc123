nn::aacdec::CTR::Initialize Function

Syntax

bool Initialize(
     Form form,
     Fs fs = FS_UNKNOWN,
     CrcFlag crcFlag = CRC_FLAG_ON,
     ReversePlayFlag revPlayFlag = REVERSE_PLAY_FLAG_OFF
);

Parameters

Name Description
in form The bitstream format.
in fs The sampling frequency index.
in crcFlag The CRC flag.
in revPlayFlag The reverse playback flag.

Return Values

Returns true if initialization succeeded; otherwise, returns false.

Description

Initializes the decoder.

To decode RAW data, you need to specify a sampling frequency for fs (specifying some value other than FS_UNKNOWN).

To decode a new file, call the nn::aacdec::CTR::Finalize function and then use this function to initialize the library again. If you begin decoding a new AAC file without first finalizing and re-initializing the library, the decoding result of the previous file will affect the decoding of the current file. For more information, see the Implementation Notes section in nn::aacdec::CTR.
