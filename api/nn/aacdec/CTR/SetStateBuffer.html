nn::aacdec::CTR::SetStateBuffer Function

Syntax

void SetStateBuffer(
     void * pBuffer
);

Parameters

Name Description
in pBuffer The buffer address to use to save state.

Return Values

None.

Description

Configures a buffer for saving the internal state of the decoder when the system enters sleep mode or transitions to the HOME menu.

Call this function after calling the nn::aacdec::CTR::Initialize function. If you do not configure a buffer using this function, the decoder's internal state is reinitialized when the system goes to sleep.

The buffer must be a STATE_BUFFER_ALIGNMENT (= 32)-byte aligned region in device memory that is NN_AACDEC_STATE_BUFFER_SIZE (32,768) bytes in size.

Do not free this buffer until you explicitly set NULL using this function. Calling the nn::aacdec::CTR::Finalize function automatically sets the buffer to NULL.
