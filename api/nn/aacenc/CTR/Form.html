nn::aacenc::CTR::Form Enumerated Type

Syntax

enum Form;

Value

Value Description
FORM_RAW Indicates the RAW format.
FORM_CRC_OFF Indicates the non-CRC ADTS format.
FORM_CRC_ON Indicates the CRC ADTS format.

Description
