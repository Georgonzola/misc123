nn::aacenc::CTR::Initialize Function

Syntax

bool Initialize(
     s32 nChannels,
     Fs fs,
     Rate bitrate,
     Form form
);

Parameters

Name Description
in nChannels The number of channels.
in fs The sampling frequency index.
in bitrate The bitrate index.
in form The bitstream format.

Return Values

Returns true if initialization succeeded; otherwise, returns false.

Description

Initializes the encoder.
