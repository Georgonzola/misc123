nn::aacenc::CTR::Rate Enumerated Type

Syntax

enum Rate;

Value

Value Description
RATE_H Indicates the highest bitrate.
RATE_M_H Indicates the second highest bitrate.
RATE_M_L Indicates the second lowest bitrate.
RATE_L Indicates the lowest bitrate.

Description

Enumerated type for bitrates.
