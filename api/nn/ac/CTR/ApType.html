nn::ac::CTR::ApType Enumerated Type

Syntax

enum ApType;

Value

Value Description
AP_TYPE_NONE None.
AP_TYPE_USER_SETTING_1 Network configuration 1.
AP_TYPE_USER_SETTING_2 Network configuration 2.
AP_TYPE_USER_SETTING_3 Network configuration 3.
AP_TYPE_NINTENDO_WIFI_USB_CONNECTOR Nintendo Wi-Fi USB Connector
AP_TYPE_NINTENDO_ZONE Nintendo Zone
AP_TYPE_WIFI_STATION Wi-Fi Station
AP_TYPE_FREESPOT FreeSpot
AP_TYPE_HOTSPOT Hotspot
AP_TYPE_TEMPORARY Application-specified temporary settings.
AP_TYPE_ALL All.

Description

Enumerated type indicating the type of access point to connect to.

Revision History

2011/04/18
From SDK 2.x, the type AP_TYPE_FREESPOT has been eliminated and merged with AP_TYPE_HOTSPOT. As of SDK 2.3 the enumerated type remains but it can no longer be used as a value. In the future it will be removed from the enumerated types.

Initial version.
