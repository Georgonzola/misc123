nn::ac::CTR::CreateDefaultConfig Function

Syntax

Result CreateDefaultConfig(
     Config * config
);

Parameters

Name Description
out config Instance of connection conditions.

Return Values

Returns the result.

Value Description
Result::IsSuccess Indicates that the call completed successfully.
ResultInvalidData NULL was specified as an argument.
Other Indicates failure for other reasons.

Description

Creates general connection conditions.

Normally the application passes the allocated config to this function. Specify this value unchanged in the arguments to the Connect function.

The config format may change in a future system update.
Avoid any coding that would change its content.
