nn::ac::CTR::DebugSetApType Function

Syntax

Result DebugSetApType(
     Config * config,
     u32 aptype
);

Parameters

Name Description
in config Instance of connection conditions.
in aptype Specifies the AP type.

Return Values

Returns the result.

Value Description
Result::IsSuccess Indicates that the call completed successfully.
ResultInvalidData NULL was specified as an argument.
ResultDebugOnly Indicates that debug mode was disabled.
Other Indicates failure for other reasons.

Description

Specifies the types of access points for network connections.

Specify an ApType value for aptype.
By taking a bitwise OR of ApType values, you can specify more than one type of access point at the same time.

Note: This function is provided for debugging purposes. It only works in Debug mode.
