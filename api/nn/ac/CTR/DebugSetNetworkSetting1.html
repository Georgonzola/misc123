nn::ac::CTR::DebugSetNetworkSetting1 Function

Syntax

Result DebugSetNetworkSetting1(
     const u8 ssid[],
     u8 ssidLength,
     SecurityMode securityMode,
     const u8 key[],
     u8 keyLen
);

Parameters

Name Description
in ssid[] Access point SSID.
in ssidLength Length of the access point SSID.
in securityMode Encryption mode of the access point.
in key[] Encryption key of the access point.
in keyLen Length of the access point's encryption key.

Return Values

Returns the result.

Value Description
Result::IsSuccess Indicates that the call completed successfully.
ResultOutOfMemory Failed to allocate memory.
ResultDebugOnly Indicates that debug mode was disabled.
Other Indicates failure for other reasons.

Description

Configures network configuration 1.

The settings are immediately applied to NAND memory.

Note: This function is provided for debugging purposes. It only works in Debug mode.

If you want to connect to the access point by using a setting specified from the product application, contact Nintendo support at support.com.
