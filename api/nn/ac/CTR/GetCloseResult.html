nn::ac::CTR::GetCloseResult Function

Syntax

Result GetCloseResult();

Parameters

None.

Return Values

Returns the result.

Value Description
Result::IsSuccess Indicates that the call completed successfully.
ResultProcessing Disconnection has yet to complete.
ResultNotConnecting Not connected.
Other Indicates failure for other reasons.

Description

Gets the processing results after closing an asynchronously established automatic connection.

Use this function to confirm results when connecting with CloseAsync, which signals an nn::os::Event.
