nn::ac::CTR::GetConnectingHotspotSubset Function

Syntax

Result GetConnectingHotspotSubset(
     HotspotSubset * hotspot
);

Parameters

Name Description
out hotspot Pointer that the hotspot is output to.

Return Values

Returns the result.

Value Description
Result::IsSuccess Indicates that the call completed successfully.
ResultInvalidData NULL was specified as an argument.
ResultNotConnecting Not connected to a network.
ResultInvalidLocation Not connected to a hotspot.
Other Indicates failure for other reasons.

Description

Gets information about the currently connected hotspot.
