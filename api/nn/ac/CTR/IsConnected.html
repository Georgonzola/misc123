nn::ac::CTR::IsConnected Function

Syntax

bool IsConnected();

Parameters

None.

Return Values

A Boolean value indicating whether the system is connected to an access point.

Description

Gets a value indicating whether the system is connected to an access point.

This function entails a substantial processing load if it is called every game frame.

This API returns false during connection processing.
During the finalization process, depending on the result branching of this function, if the Close function has not been called, a connection that was in the process of connecting may successfully connect. Take note of this.

This function can be used even if you have not called Initialize, but it returns a value indicating failure.
