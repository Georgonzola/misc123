nn::ac::CTR::NetworkArea Enumerated Type

Syntax

enum NetworkArea;

Value

Value Description
NETWORK_AREA_WAN A WAN connection is required.
NETWORK_AREA_AUTO A WAN connection is recommended. A LAN connection is also possible.
NETWORK_AREA_LAN Only a LAN connection is possible.

Description

Enumerated type indicating the connection level to request of the access point to connect to.
