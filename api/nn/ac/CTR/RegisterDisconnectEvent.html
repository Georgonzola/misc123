nn::ac::CTR::RegisterDisconnectEvent Function

Syntax

Result RegisterDisconnectEvent(
     nn::os::Event * event
);

Parameters

Name Description
out event Pointer to the initialized nn::os::Event that is notified.

Return Values

Returns the result.

Value Description
Result::IsSuccess Indicates that the call completed successfully.
ResultInvalidData Either NULL was specified as an argument or an uninitialized event was specified.
Other Indicates failure for other reasons.

Description

Sets which nn::os::Event is notified upon disconnection from an access point.
