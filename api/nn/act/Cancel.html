nn::act::Cancel Function

Syntax

Result Cancel(
     void
);

Parameters

None.

Return Values

Returns ResultSuccess on success.
Returns ResultNotInitialized when the account library is not initialized.

Description

Cancels communication tasks with the account server that are waiting for completion. Cancels all communication tasks with account servers that are waiting for completion that were registered by a process called by this function. Use in a different thread than the one from which the API function that communicates with the server was called.

The API function that canceled the task returns a Result and error code according to the state at that point. Note that if the cancellation is not in time, success may be returned. The failure error code that results from the cancellation does not need to be displayed in the error viewer.
