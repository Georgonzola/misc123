nn::act::Finalize Function

Syntax

Result Finalize();

Parameters

None.

Return Values

Returns the result of the operation.


Value Description
ResultSuccess Succeeded.
Other Other results can be returned depending on the internal state of the daemon.

Description

Finalizes the account library. If the nn::act::Initialize has been called multiple times, this function must be called the same number of times to completely finalize the account library. If this function is called more times than the nn::act::Initialize function was called, ResultNotInitialized is returned.
This call succeeds even if the library is not initialized.
