nn::act::GetCountry Function

Syntax

Result GetCountry(
     char * pCountry
);

Parameters

Name Description
out pCountry Specifies an out pointer to the buffer that stores the country setting.

Return Values

Returns ResultSuccess on success.
Returns ResultNotInitialized when the account library is not initialized.
Returns ResultInvalidPointer when an invalid pointer (such as a NULL pointer) is passed as an argument.

Description

Gets the country setting of the system account in ISO 3166-1 alpha-2 format. Gets the country setting of the Nintendo Network account associated with the system account in ISO 3166-1 alpha-2 format. However, this function gets the value the system cached for the locale, so calling this function does not trigger communication.

Pass a pointer to a buffer at least as big as NN_ACT_COUNTRY_SIZE bytes to pCountry.

When it is not a network account, the function succeeds and a blank string is copied to pCountry.

If the function fails, the value set in the buffer is undefined.
