nn::act::GetPrincipalId Function

Syntax

u32 GetPrincipalId(
     void
);

Parameters

None.

Return Values

Returns the principal ID for the account. Returns NN_ACT_INVALID_PRINCIPAL_ID if this is not a network account.

Description

Gets the principal ID for the system account. Gets the principal ID for the Nintendo Network account associated with the system account. However, this function gets the value the system cached for the locale, so calling this function does not trigger communication. The principal ID returned by this function is different from the friend account principal ID returned by the nn::friends::CTR::GetMyPrincipalId function.
