nn::act::GetSimpleAddressId Function

Syntax

u32 GetSimpleAddressId(
     void
);

Parameters

None.

Return Values

Returns a 32-bit ID that indicates the residence information for the account. The value corresponds to the id member in the nn::cfg::CTR::SimpleAddressId structure. Returns NN_ACT_INVALID_SIMPLE_ADDRESS_ID if this is not a network account.

Description

Gets the residence information for the system account. Gets the residence information for the Nintendo Network account associated with the system account. However, this function gets the value the system cached for the locale, so calling this function does not trigger communication. The values that can be obtained correspond to the id member in the nn::cfg::CTR::SimpleAddressId structure.
