Overloaded Member Functions

GetSimpleAddressId ( void ) Gets the residence information for the system account. Gets the residence information for the Nintendo Network account associated with the system account. However, this function gets the value the system cached for the locale, so calling this function does not trigger communication. The values that can be obtained correspond to the id member in the nn::cfg::CTR::SimpleAddressId structure.
GetSimpleAddressId ( nn::cfg::SimpleAddressId * ) Gets the residence information for the system account. Gets the residence information for the Nintendo Network account associated with the system account. Note that this function does not trigger any communication because it gets the value the system cached for the locale.

If the function fails, the value set in the buffer is undefined.
