nn::act::GetTimeZoneId Function

Syntax

Result GetTimeZoneId(
     char * pTimeZoneId
);

Parameters

Name Description
out pTimeZoneId Specifies an out pointer to the buffer that stores the time zone name.

Return Values

Returns ResultSuccess on success.
Returns ResultNotInitialized when the account library is not initialized.
Returns ResultInvalidPointer when an invalid pointer (such as a NULL pointer) is passed as an argument.

Description

Gets the name of the time zone set for the system account.

Gets the name of the time zone set for the Nintendo Network account associated with the system account in tz database format. However, this function gets the value the system cached for the locale, so calling this function does not trigger communication.

Pass a pointer to a buffer with at least as many elements as ACT_TIME_ZONE_ID_SIZE for pTimeZoneId.

When it is not a network account, the function succeeds and a blank string is copied to pTimeZoneId.

If the function fails, the value set in the buffer is undefined.
