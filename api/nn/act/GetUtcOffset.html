nn::act::GetUtcOffset Function

Syntax

s64 GetUtcOffset(
     void
);

Parameters

None.

Return Values

Returns a value in microseconds for the difference from UTC for the time zone set in the system account. Returns 0 if the account is not a network account or if the account library has not been initialized.

Description

Gets the time difference of the time zone set for the system account.

Gets a value in microseconds for the difference from UTC for the time zone set for the Nintendo Network account associated with the system account. However, this function gets the value the system cached for the locale, so calling this function does not trigger communication.

The function does take daylight saving time into account, but note that changes due to the start or end of daylight saving time might not be immediately applied to the value cached for the locale. Also, note that applying this function to a date in the past or future is highly likely to result in a value that differs from the actual time on that day. We recommend using an independent server and resolving the time difference from the time zone name obtained with the nn::act::GetTimeZoneId function if you need to deal with precise time difference information.
