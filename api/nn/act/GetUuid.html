nn::act::GetUuid Function

Syntax

Result GetUuid(
     Uuid * pUuid
);

Parameters

Name Description
out pUuid Specifies an out pointer to the buffer that stores the UUID.

Return Values

Returns ResultSuccess on success.
Returns ResultNotInitialized when the account library is not initialized.
Returns ResultInvalidPointer when an invalid pointer (such as a NULL pointer) is passed as an argument.

Description

Gets the UUID of the system account. The UUID for the system account is assigned when the account is created, and does not change until the system account is deleted. The ID space is separate for each title, so different titles calling this function get back different UUIDs from the same system account. Consult Nintendo support if you want to be able to identify a system account by its UUID across multiple titles.

Currently, the UUID retrieved from this function complies with the Version 1 specifications defined in RFC 4122, but the ID format could change to the extent that it does not impair the uniqueness of the ID. Be sure to handle the ID in its entirety as a single value. Do not use any individual parts of the ID as a timestamp or device identifier.

If the function fails, the value set in the buffer is undefined.
