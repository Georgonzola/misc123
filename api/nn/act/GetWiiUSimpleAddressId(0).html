nn::act::GetWiiUSimpleAddressId Function

Syntax

u32 GetWiiUSimpleAddressId(
     void
);

Parameters

None.

Return Values

Returns a 32-bit ID that indicates the residence information for the account. Returns NN_ACT_INVALID_SIMPLE_ADDRESS_ID if this is not a network account.

Description

Gets the system account's residence information in the SimpleAddressId format used by the Wii U. Gets the residence information for the Nintendo Network account associated with the system account in the SimpleAddressId format used by the Wii U. However, this function gets the value the system cached for the locale, so calling this function does not trigger communication. SimpleAddressId is passed directly to the Wii U by the application for communication purposes; there is no need to use this function beyond when the Wii U needs to parse its meaning.
