nn::act::GetWiiUSimpleAddressId Function

Syntax

nn::Result GetWiiUSimpleAddressId(
     nn::cfg::SimpleAddressId * pSimpleAddressId
);

Parameters

Name Description
out pSimpleAddressId Gets a point to a buffer that returns an ID for the account's residence information. When the account is not a network account, NN_ACT_INVALID_SIMPLE_ADDRESS_ID is set to pSimpleAddressId->id.

Return Values

Returns ResultSuccess on success.
Returns ResultNotInitialized when the account library is not initialized.
Returns ResultInvalidPointer when an invalid pointer (such as a NULL pointer) is passed as an argument.

Description

Gets the system account's residence information in the SimpleAddressId format used by the Wii U. Gets the residence information for the Nintendo Network account associated with the system account as the SimpleAddressId used by the Wii U. However, this function gets the value the system cached for the locale, so calling this function does not trigger communication.

SimpleAddressId is passed directly to the Wii U by the application for communication purposes; there is no need to use this function beyond when the Wii U needs to parse its meaning.

If the function fails, the value set in the buffer is undefined.
