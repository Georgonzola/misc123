Overloaded Member Functions

GetWiiUSimpleAddressId ( void ) Gets the system account's residence information in the SimpleAddressId format used by the Wii U. Gets the residence information for the Nintendo Network account associated with the system account in the SimpleAddressId format used by the Wii U. However, this function gets the value the system cached for the locale, so calling this function does not trigger communication. SimpleAddressId is passed directly to the Wii U by the application for communication purposes; there is no need to use this function beyond when the Wii U needs to parse its meaning.
GetWiiUSimpleAddressId ( nn::cfg::SimpleAddressId * ) Gets the system account's residence information in the SimpleAddressId format used by the Wii U. Gets the residence information for the Nintendo Network account associated with the system account as the SimpleAddressId used by the Wii U. However, this function gets the value the system cached for the locale, so calling this function does not trigger communication.

SimpleAddressId is passed directly to the Wii U by the application for communication purposes; there is no need to use this function beyond when the Wii U needs to parse its meaning.

If the function fails, the value set in the buffer is undefined.
