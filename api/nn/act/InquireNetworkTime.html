nn::act::InquireNetworkTime Function

Syntax

Result InquireNetworkTime(
     void
);

Parameters

None.

Return Values

Returns ResultSuccess on success.
Returns ResultNotInitialized when the account library is not initialized.
Returns ResultInvalidPointer when an invalid pointer (such as a NULL pointer) is passed as an argument.
Returns ResultInvalidSize when the array passed as an argument is either too long or too short.
Returns ResultAuthenticationError when authentication fails.
Returns ResultNotConnected when not connected to an access point

Description

Gets the time from the server, and activates and calibrates the network clock. Features that use the network clock, such as the nn::act::IsOverAge function, are available by calling this function.

When APIs that communicate with the server, such as the nn::act::applet::AcquireIndependentServiceToken function, have succeeded at least once in advance, there is no need to additionally call this function.

You cannot directly get the time with this function.
This function causes communication with the server, during which time processing is blocked.

For information about using this function, see the section on how to make API functions that trigger communication in the nn::act namespace.
