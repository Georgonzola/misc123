nn::act::IsOverAge Function

Syntax

bool IsOverAge(
     u8 age
);

Parameters

Name Description
in age Specifies the age to compare.

Return Values

Returns true if the account is the specified age or older, and returns false if it is younger. It returns false if used on a local account. It returns false when the network clock is not activated.

Description

Determines whether the system account user is at least the specified age. This function itself does not trigger any communication and can make this determination without an Internet connection by estimating the current date and time. However, you need to activate the network clock by calling a function that communicates with the account server in advance. In addition, you must reactivate the network clock every time the application runs.

Nintendo systems calculate the age of the user based on UTC+12 time, regardless of their country or region of residence. For users in Japan, age is calculated based on 9 P.M. of the day before the date of birth.

If you want to get the age of a user directly, please contact Nintendo support. Avoid identifying the ages of users through repeated calls to this function.
