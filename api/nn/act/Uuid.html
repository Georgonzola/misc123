nn::act::Uuid Type

Syntax

typedef struct nn::act::Uuid nn::act::Uuid;

Description

The structure that stores the UUID.

The Universally Unique Identifier (UUID) is an identifier that can be expected to be globally unique.

The UUIDs retrieved from the account library currently conform to the version 1 specifications stipulated in RFC 4122, but the format of these IDs can be modified in ways that do not diminish their uniqueness. Each of these IDs is intended to serve as a single, complete ID. Please do not use parts of the IDs for purposes such as time or device identifiers.
