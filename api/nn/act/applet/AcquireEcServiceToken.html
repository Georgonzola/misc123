nn::act::applet::AcquireEcServiceToken Function

Syntax

Result AcquireEcServiceToken(
     char * pEcServiceToken
);

Parameters

Name Description
out pEcServiceToken An out pointer to the buffer that stores the service token.

Return Values

Returns ResultSuccess on success.
Returns ResultNotInitialized when the account library is not initialized.
Returns ResultNotNetworkAccount when the account is not a network account.
Returns ResultInvalidPointer when an invalid pointer (such as a NULL pointer) is passed as an argument.
Returns ResultNotConnected when not connected to an access point
Returns ResultAuthenticationError when authentication fails.
Returns ResultCanceled when the operation was canceled.

Description

Gets a service token for an e-commerce service. Currently, this function can be used only to implement features for purchasing service items through an independent server.
No parental control checks are performed by this function. Configure the application as needed to implement functionality corresponding to Parental Controls settings.

For pEcServiceToken, pass a pointer to a buffer at least as big as NN_ACT_INDEPENDENT_SERVICE_TOKEN_SIZE bytes.

This function starts the applet, and blocks for the duration.

If the function fails, the value set in the buffer is undefined.
