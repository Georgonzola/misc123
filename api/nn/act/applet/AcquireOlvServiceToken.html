nn::act::applet::AcquireOlvServiceToken Function

Syntax

Result AcquireOlvServiceToken(
     char * pOlvServiceToken
);

Parameters

Name Description
out pOlvServiceToken Specifies an out pointer to the buffer that stores the service token.

Return Values

Returns the result of the operation.

Value Description
ResultSuccess Succeeded.
ResultNotNetworkAccount Indicates that the account is not a network account.
ResultInvalidPointer Indicates that an invalid pointer (such as a NULL pointer) was passed as an argument.
ResultNotConnected Not connected to an access point.
ResultAuthenticationError Indicates that authentication failed.
ResultCanceled Canceled.

Description

Gets a service token for the Miiverse.

No parental control checks are performed by this function. Configure the application as needed to implement functionality corresponding to Parental Controls settings.

Pass a pointer to a buffer at least as big as NN_ACT_INDEPENDENT_SERVICE_TOKEN_SIZE bytes to pOlvServiceToken.

This function sometimes returns a service token from the system's internal cache.

This function starts the applet, and blocks for the duration.

If the function fails, the value set in the buffer is undefined.

For information about using this function, see the section in the nn::act namespace that describes how to make API functions that trigger communication.
