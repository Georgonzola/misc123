nn::act::applet::CancelPreload Function

Syntax

void CancelPreload();

Parameters

None.

Return Values

None.

Description

Cancels the preloading of the login applet.

When preloading has not completed, this function does nothing. Consequently, prior to running this function, you must run either the WaitForPreload or tIsRegistered function to check whether preloading is complete.

This function cancels preloading for all library applets. Note that the preloading of library applets other that the login applet is also canceled by this function.
