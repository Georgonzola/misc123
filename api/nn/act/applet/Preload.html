nn::act::applet::Preload Function

Syntax

void Preload();

Parameters

None.

Return Values

None.

Description

Preloads the login applet.

By preloading, the login applet can have a comparatively shorter launch time. However, the following cautions are needed.

Preloading will not be finished even when returning from this function. When using this function to preload, always run either the WaitForPreload or IsRegistered function to check whether preloading is complete. After you verify preloading, you can then run the various request functions.

When another library applet other than the login applet is already preloaded, you must cancel the preload before running this function.

You must not terminate the application while an applet is in a preloaded state. Prior to application termination, use the CancelPreload function to cancel the preload.

After calling the Preload function, you must not process any HOME Button presses until the IsRegistered function returns true. In practical terms, when applet::IsExpectedToProcessHomeButton returns true, call applet::ClearHomeButtonState without calling applet::ProcessHomeButton.
