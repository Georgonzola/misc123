nn::act::applet::WaitForPreload Function

Syntax

void WaitForPreload();

Parameters

None.

Return Values

None.

Description

Waits for preloading of the login applet to complete.

You will not return from this function if it is called when you are not running a preload.

This function can be called just once in the interval between the running of the Preload function and the various request functions or the CancelPreload function.
