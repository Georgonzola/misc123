nn::applet::CTR::AppletAwakeCallback Type

Syntax

typedef void(* nn::applet::CTR::AppletAwakeCallback)(uptr arg);

Description

The type of the callback function invoked upon awaking from sleep states.

Note: The thread that calls this callback has a stack size of roughly 4 KB. Take care to avoid overflows.

Use the nn::applet::CTR::SetAwakeCallback function to set the callback.

The value specified in arg is passed as is when setting the callback. Use as appropriate.
