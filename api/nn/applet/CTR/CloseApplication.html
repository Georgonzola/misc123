nn::applet::CTR::CloseApplication Function

Syntax

Result CloseApplication(
     const u8 * pParam = NULL,
     size_t paramSize = 0,
     Handle handle = NN_APPLET_HANDLE_NONE
);

Parameters

Name Description
in pParam Specifies the parameter buffer.
in paramSize Specifies the parameter buffer size.
in handle Specifies the handle.

Return Values

Returns the result of the function. Returns one of the following Result values.

Value Description
Result::IsSuccess Indicates that the operation succeeded.

Description

Exits the program.

This function does not return. The function has a return value, but the process exits, so there is no way to actually return a value.

You can pass data to the next program started by specifying it in the pParam, paramSize, and handle arguments, but since this cannot be used by normal applications, use the default values instead. We recommend calling without any arguments.
