nn::applet::CTR::CloseApplicationForTrial Function

Syntax

Result CloseApplicationForTrial(
     const u8 * pParam = NULL,
     size_t paramSize = 0,
     Handle handle = NN_APPLET_HANDLE_NONE
);

Parameters

Name Description
in pParam Specifies the parameter buffer.
in paramSize Specifies the parameter buffer size.
in handle Specifies the handle.

Return Values

Returns the result of the function. Returns one of the following Result values.

Value Description
Result::IsSuccess Indicates that the operation succeeded.

Description

Exits the program.

This function is a substitute for the CloseApplication function. Use it in demo applications (SA/NW applications) that run on kiosks in stores. In normal applications, you close the application after receiving a request to do so. Demo applications that run on kiosks, however, are designed to close automatically.

The normal CloseApplication function checks whether a request to close the application has been received (the IsExpectedToCloseApplication returns true). This function does not check for a close application request.

Otherwise, this function works exactly the same as the CloseApplication function.

This function cannot be used in downloadable demos and general applications.
