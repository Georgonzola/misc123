nn::applet::CTR::GetScreenCapturePostPermission Function

Syntax

Result GetScreenCapturePostPermission(
     ScreenCapturePostPermission * pPermission
);

Parameters

Name Description
out pPermission Specifies the location to use to save the screenshot post permission settings.

Return Values

Returns the result of the function. Returns one of the following Result values.

Value Description
Result::IsSuccess Indicates that the operation succeeded.

Description

Sets the post permission setting for the screenshots the system gets at HOME Menu transitions.

The default value set automatically when the application starts is nn::applet::SCREEN_CAPTURE_POST_NO_SETTING (not set).
