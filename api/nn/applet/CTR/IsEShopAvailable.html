nn::applet::CTR::IsEShopAvailable Function

Syntax

bool IsEShopAvailable();

Parameters

None.

Return Values

Returns true if Nintendo eShop is installed.

Description

Determines whether Nintendo eShop is installed. In some cases, Nintendo eShop may not be installed on the system. This is the case if the system was shipped shortly after release and has never had a system update. This function can be used to check whether it is installed. Nintendo eShop is not installed on the system in the development environment. You can get this function to return true by importing the dummy Nintendo eShop included with the SDK. The dummy Nintendo eShop can be removed from DevMenu after it is imported.
