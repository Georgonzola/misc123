nn::applet::CTR::JumpToEShopTitlePage Function

Syntax

Result JumpToEShopTitlePage(
     bit32 uniqueId
);

Parameters

Name Description
in uniqueId The unique ID (the ID unique to each title).

Return Values

Returns success.

Description

Jumps to the software information page in Nintendo eShop. The titles displayed in Nintendo eShop are specified by their unique IDs. This function can only jump to the pages of titles that are publicly searchable. Publicly searchable titles are those titles that can be searched for by name and accessed on Nintendo eShop. The application ends when this function is called. If there is no information about the title on Nintendo eShop or if the value is invalid, an error displays on Nintendo eShop. You must use the nn::applet::IsEShopAvailable function to determine in advance whether Nintendo eShop has been installed. Always call this API after confirming that Nintendo eShop is installed. When Nintendo eShop is not present on the system, we recommend that you let the user know that a system update via the Internet is required. For example, you could display a message that says something like: "Cannot use Nintendo eShop. An Internet update of your system is needed. Proceed?" In the development environment, you can test the jump to Nintendo eShop feature by importing DummyNintendoEShop. For more information, see the detailed description of the DummyNintendoEShop tool. When using this function to make jumps, make an OMAS application for the destination title. For information about how to apply, see 3DS Guidelines: E-Commerce.
