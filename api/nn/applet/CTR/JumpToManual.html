nn::applet::CTR::JumpToManual Function

Syntax

void JumpToManual();

Parameters

None.

Return Values

None.

Description

Jumps to the e-manual. Jumping to the e-manual is similar to the HOME Menu transition process. Consequently, the application must wait for calls to this function in the same way it waits for calls to nn::applet::CTR::ProcessHomeButton() when the HOME Button is pressed. See the function reference for nn::applet::CTR::ProcessHomeButton(). Depending on the system mode setting, this function might not work correctly. For more information, see the documentation under CTR Tools > Configuration.
