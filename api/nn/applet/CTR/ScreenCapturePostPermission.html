nn::applet::CTR::ScreenCapturePostPermission Enumerated Type

Syntax

enum ScreenCapturePostPermission;

Value

Value Description
SCREEN_CAPTURE_POST_NO_SETTING Unset: The value is set to the default value when the application starts. Allows posting of the screenshot.
SCREEN_CAPTURE_POST_NO_SETTING_OLD_SDK Unset: The default value for titles released with the SDK prior to the introduction of this value become this value. Do not specify this value.
SCREEN_CAPTURE_POST_ENABLE Allows posting of the screenshot on both the upper and lower screens.
SCREEN_CAPTURE_POST_DISABLE Does not allow posting of the screenshot.
SCREEN_CAPTURE_POST_ENABLE_UPPER_LCD Allows posting of the screenshot on only the upper screen.
SCREEN_CAPTURE_POST_ENABLE_LOWER_LCD Allows posting of the screenshot on only the lower screen.

Description

The post permission for the screenshots the system gets at HOME Menu transitions.
