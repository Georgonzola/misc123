nn::applet::CTR::SetScreenCapturePostPermission Function

Syntax

Result SetScreenCapturePostPermission(
     const ScreenCapturePostPermission permission
);

Parameters

Name Description
in permission The value for the permission to post the screenshot you want to set.

Return Values

Returns the result of the function. Returns one of the following Result values.

Value Description
Result::IsSuccess Indicates that the operation succeeded.

Description

Sets the post permission for the screenshots the system gets at HOME Menu transitions.

Of the members in nn::applet::CTR::ScreenCapturePostPermission, specify either SCREEN_CAPTURE_POST_ENABLE or SCREEN_CAPTURE_POST_DISABLE in the permission argument.
It is not necessary to specify SCREEN_CAPTURE_POST_NO_SETTING.

Starting the application initializes the setting. The default value is SCREEN_CAPTURE_POST_NO_SETTING (not set). Calling the nn::applet::CTR::RestartApplication function also initializes the setting.
