nn::boss::DataStoreDownloadAction::Initialize Member Function

Syntax

nn::Result Initialize(
     u32 gameServerId,
     const wchar_t * pAccessKey
);

Parameters

Name Description
in gameServerId Specifies the ID used to identify game servers. (For more information, see the documentation in the NEX library.)
in pAccessKey Specifies the key used to access the game server. (Specify no more than 8 characters. For more information, see the documentation in the NEX library.)

Return Values

Returns the execution result of the function. Returns one of the following Result values.

Value Description
Result::IsSuccess Indicates that initialization succeeded.
ResultInvalidPointer The pointer to the access key string is NULL.
ResultInvalidSize The access key string length is 0 or exceeds the maximum number of characters allowed.
Other than those noted above An unexpected error. (For information about handling this error, see the Programming Manual.)

Description

Configures DataStore download tasks. Even if the DataStoreDownloadAction::Initialize function has already been run on an object, this function can be run again to allow the object to be reused.
