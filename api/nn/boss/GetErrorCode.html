nn::boss::GetErrorCode Function

Syntax

nn::Result GetErrorCode(
     u32 * pOut,
     TaskResultCode targetResult
);

Parameters

Name Description
out pOut Buffer for storing error codes.
in targetResult Target result codes.

Return Values

Returns the execution result of the function. Returns one of the following Result values.

Value Description
Result::IsSuccess Indicates successful retrieval.
ResultUnexpect An unexpected result code (result code for which an error code is not allocated) is passed.
(This result is also returned when pOut parameter is NULL.))
ResultIpcNotSessionInitialized Indicates that the session is not initialized.
The function returns this Result if it is called before Initialize is called.
Always call the Initialize function before you use the BOSS library.
Other than those noted above An unexpected error. (For information about handling this error, see the Programming Manual.)

Description

The error code is acquired from the task result code (TaskResultCode). The task result code is acquired with nn::boss::Task::GetResult. When the task result is UNKNOWN_ERROR, a value not included in the TaskResultCode list might be returned as a task result code (when it is UNKNOWN_ERROR, the task result code becomes a sum of the UNKNOWN_ERROR enumerator value and the error originating module ID, nn::Result::Module), but such a value can also be converted into an error code by this function.
