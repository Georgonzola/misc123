Overloaded Member Functions

GetStorageInfo ( size_t * ) Gets the size of the registered BOSS storage region.
This function can also be used to check whether BOSS storage (the save region on the game device) was registered to the system using RegisterStorage.
This function cannot be used to check for BOSS storage in the extended save data region. BOSS storage is automatically generated by the system as part of the extended save data. (For information about checking for extended save data, see nn::fs::MountExtSaveData.)
GetStorageInfo ( size_t *, u16 * ) Gets the size of the registered BOSS storage region and the maximum file count.
This function can also be used to determine whether BOSS storage (the save region on the game device) was registered to the system using RegisterStorage.
This function cannot be used to check for BOSS storage in the extended save data region. BOSS storage is automatically generated by the system as part of the extended save data. (For information about checking for extended save data, see nn::fs::MountExtSaveData.)
