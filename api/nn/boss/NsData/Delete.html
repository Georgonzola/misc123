nn::boss::NsData::Delete Member Function

Syntax

nn::Result Delete(
     void
);

Parameters

None.

Return Values

Returns the execution result of the function. Returns one of the following Result values.

Value Description
Result::IsSuccess Indicates that deletion succeeded.
ResultNsDataNotFound The specified NS data is not found. It might have been automatically deleted.
ResultIpcNotSessionInitialized Indicates that the session is not initialized. If this function is called before nn::boss::Initialize is called, this Result is returned. Always call the nn::boss::Initialize function before you use the BOSS library.
Other than those noted above An unexpected error. (For information about handling this error, see the Programming Manual.)

Description

Deletes an NS data file.

Note: Sometimes the target NS data cannot be found because of one of the following reasons: another application has already deleted the NS data, there was not enough space available when downloading the NS archive, or the data has been automatically deleted.
