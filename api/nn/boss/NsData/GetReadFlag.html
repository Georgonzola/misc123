nn::boss::NsData::GetReadFlag Member Function

Syntax

nn::Result GetReadFlag(
     bool * pFlag
);

Parameters

Name Description
out pFlag Returns the obtained flag. Returns true if already read.

Return Values

Returns the execution result of the function. Returns one of the following Result values.

Value Description
Result::IsSuccess Indicates successful retrieval.
ResultInvalidNsDataReadFlag NS data read flag storage region pointer is NULL.
ResultNsDataNotFound The specified NS data is not found. It might have been automatically deleted.
ResultStorageAccessPermission You do not have storage access rights. Cannot access the expanded save data region. Reconfirm your access rights.
ResultIpcNotSessionInitialized Indicates that the session is not initialized. If this function is called before nn::boss::Initialize is called, this Result is returned. Always call the nn::boss::Initialize function before you use the BOSS library.
Other than those noted above An unexpected error (see the Programming Manual for how to handle this error)

Description

Gets the read flag of the NS data.
