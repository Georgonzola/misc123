nn::boss::NsData Class

Syntax

class NsData

Description

Class for representing NS data. Use for NS data operations (such as reading).

Member Constants

S NN_BOSS_NSDATA_READ_ERROR_READ_DATA const s32 NS data ReadNsData error definition. Failed to load NS data.
S NN_BOSS_NSDATA_READ_ERROR_IPC const s32 NS data ReadNsData error definition. An IPC error occurred.
S NN_BOSS_NSDATA_READ_ERROR_UPDATED const s32 NS data ReadNsData error definition. The target NSD version was updated since the last time data was obtained.
