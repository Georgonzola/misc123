nn::boss::NsaList::GetDigest Member Function

Syntax

nn::Result GetDigest(
     u8 * pDigestBuf,
     size_t digestBufSize
);

Parameters

Name Description
out pDigestBuf The buffer storing NSA digest information.
in digestBufSize pDigestBuf buffer size. (40 alphanumeric characters are returned for the digest value, so the buffer must be 41 bytes or greater.)

Return Values

Returns the execution result of the function. Returns one of the following Result values.

Value Description
Result::IsSuccess Indicates successful retrieval.
ResultNSAListInvalidFormat The file format is invalid.
Other than those noted above An unexpected error. (For information about handling this error, see the Programming Manual.)

Description

Gets the digest value included in an NSA list.

Gets the digest value included in the NSA list. (In the CheckValidity function, compares the result of this function with the calculated digest value.) One use of this function is to check whether the NSA list has been updated. When the NSA is not frequently updated on the server, to shorten the time needed to check whether the NSA list has been updated, you can also use this function to get and remember a digest value for the downloaded NSA list. You can then compare just the digest value the next time you download the NSA list.
