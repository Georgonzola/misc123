nn::boss::RegisterImmediateTask Function

Syntax

nn::Result RegisterImmediateTask(
     Task * pTask,
     TaskAction * pAction,
     const char * pFilePath,
     TaskPolicy * pPolicy = NULL,
     TaskOption * pOption = NULL,
     u8 taskStep = DEFAULT_STEP_ID
);

Parameters

Name Description
in pTask Specify FgOnlyTask.
If any value other than FgOnlyTask is specified, the function returns ResultInvalidTaskId.
in pAction Specifies a task action.
in pFilePath File path. (Specifies the path for the file to be uploaded for the DataStore upload task. You cannot specify a ROM archive file.)
in pPolicy Specifies a task policy. (Optional.)
in pOption Specifies task options. (Optional.)
Note: This parameter is not currently supported because it is used with extended features. Do not specify it.
in taskStep Specifies a task step ID. (This parameter may be omitted if there is only one task step to be executed.)
Note: This parameter is not currently supported because it is used with extended features. Do not specify it.

Return Values

Returns the execution result of the function. Returns one of the following Result values.

Value Description
Result::IsSuccess Indicates that registration succeeded.
ResultInvalidTaskId This Result object is returned if the pTask parameter is set to NULL or any value other than FgOnlyTask.
ResultInvalidAction The pAction parameter is NULL.
ResultActionCodeOutOfRange The task action code is invalid. Confirm that it is in the range of Action.
ResultInvalidUrl The URL string pointer is NULL or a zero-length string.
ResultApInfoTypeOutOfRange The AP information type is invalid. Confirm that it is in the range of ApInfoType.
ResultCfgInfoTypeOutOfRange The Cfg information type is out of range. Confirm that it is in the range of ApInfoType.
ResultInvalidFilePath The file path string pointer is NULL or a zero-length string.
ResultInvalidOption The pOption parameter is uninitialized or the option code is out of range. Confirm that it is in the range of ExecOption.
ResultStorageNotFound The BOSS storage corresponding to the application is not registered. Determine whether the BOSS storage is registered.
ResultDatabaseFull BOSS storage and tasks cannot be registered because the maximum number of registered application IDs and tasks has been reached.
ResultFileAccess Failed to access a file.
This Result is returned when the SD card is removed or data is corrupted.
ResultIpcNotSessionInitialized Indicates that the session is not initialized.
The function returns this Result if it is called before Initialize is called.
Always call the Initialize function before you use the BOSS library.
ResultInvalidFilePath The file path string pointer is NULL or a zero-length string. (Note: This is probably operationally impossible, but there is an escape if the specified file cannot be found.)
Other than those noted above An unexpected error. (For information about handling this error, see the Programming Manual.)

Description

Registers an immediate-execution task to BOSS (to the save region on the game device) in concert with the file specified as the target. At the present time, this function can be used only to register DataStore upload tasks. For more information, see nn::boss::RegisterImmediateTask, which has the same specification but without the argument to take the file path.

The file corresponding to the specified file path is copied to BOSS storage during the task registration process. Copying can take a long time (upwards of 10 seconds), depending on file size and SD card type. As a result, it can take a long time for this function to respond.
