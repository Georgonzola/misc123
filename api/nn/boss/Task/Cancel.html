nn::boss::Task::Cancel Member Function

Syntax

nn::Result Cancel(
     void
);

Parameters

None.

Return Values

Returns the execution result of the function. Returns one of the following Result values.

Value Description
Result::IsSuccess Successfully canceled the running task.
ResultInvalidTaskId Indicates that the task ID pointer is NULL or a zero-length string.
ResultTaskNotExist The specified task ID was not found. Confirm whether it is registered yet.
ResultIpcNotSessionInitialized Indicates that the session is not initialized. The function returns this Result if it is called before Initialize is called. Always call the Initialize function before you use the BOSS library.
Other than those noted above An unexpected error. (For information about handling this error, see the Programming Manual.)

Description

Cancels a running task.

You can use the Start function to restart a task that has been canceled. If a task has been canceled, calling the GetState function returns TASK_STOPPED as the task's state. If the task was running when it was canceled, task execution is canceled and TASK_CANCELED is returned as the task's execution result.
