nn::boss::Task::GetError Member Function

Syntax

nn::Result GetError(
     TaskError * pTaskError,
     u8 * pStepId = NULL,
     u8 taskStep = CURRENT_STEP_ID
);

Parameters

Name Description
out pTaskError Returns the task error state.
out pStepId Returns the task step ID.
in taskStep Specifies a task step ID. (Defaults to the current step if omitted.)

Return Values

Returns the execution result of the function. Returns one of the following Result values.

Value Description
Result::IsSuccess Indicates successful retrieval.
ResultInvalidTaskError The pointer to the task error information is NULL.
ResultInvalidTaskId Indicates that the task ID pointer is NULL or a zero-length string.
ResultTaskNotExist The specified task ID was not found. Confirm whether it is registered yet.
ResultIpcNotSessionInitialized Indicates that the session is not initialized. The function returns this Result if it is called before Initialize is called. Always call the Initialize function before you use the BOSS library.
Other than those noted above An unexpected error. (For information about handling this error, see the Programming Manual.)

Description

Gets the task error state.

You must prepare a TaskError instance beforehand. Note that this instance requires approximately 136 bytes of memory.
