nn::boss::Task::GetResult Member Function

Syntax

TaskResultCode GetResult(
     u32 * pCount = NULL,
     u8 * pStepId = NULL
);

Parameters

Name Description
out pCount Returns the execution count. (Optional.)
out pStepId Returns the task step ID. (Optional.)

Return Values

Returns the task result code.

Description

Gets the task result code.

For more information about the task result codes, see TaskResultCode.
Notes about frequent task execution errors:
(1) If the infrastructure communication is disconnected,
HTTP_ERROR_RECV
HTTP_ERROR_CONNECT
(2) If there is a failure to connect to the server (cannot find server indicated by the URL),
HTTP_ERROR_CONNECT
(3) If there is a failure to connect to the server (failed to negotiate the HTTPS connection),
SSL_ERROR_FAILED
SSL_ERROR_VERIFY_COMMON_NAME
SSL_ERROR_VERIFY_ROOT_CA
SSL_ERROR_VERIFY_CHAIN
SSL_ERROR_VERIFY_DATE
(4) If the NSA download failed (NSA storage area insufficient),
NSA_ERROR_STORAGE_INSUFFICIENCY
(5) If the NSA download fails (not the NSA format, or downloaded a file with corrupted content due to a communications error),
NSA_ERROR_INVALID_FORMAT
NSA_ERROR_VERIFY_HASH
NSA_ERROR_VERIFY_SIGNATURE
NSA_NSD_ERROR_VERIFY_HASH
NSA_NSD_ERROR_VERIFY_SIGNATURE
NSA_NSD_ERROR_INVALID_FORMAT
