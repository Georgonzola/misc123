nn::boss::TaskActionBase::SetRootCa Member Function

Syntax

nn::Result SetRootCa(
     u32 inCaCertId
);

Parameters

Name Description
in inCaCertId Built-in CA certificate ID.

Return Values

Returns the execution result of the function. Returns one of the following Result values.

Value Description
Result::IsSuccess Successfully set.
ResultInvalidMaxRootCa The number of root CAs exceeds the maximum that can be set.
Other than those noted above An unexpected error. (For information about handling this error, see the Programming Manual.)

Description

Sets a built-in root certificate for use.

It can be used alongside a private root certificate.
You do not normally need to use this function to set a certificate unless you are using a data upload task.
(The BOSS library automatically sets the required root certificate. )
Contact Nintendo if you are using a data upload task and you want to use a special built-in root certificate.
