nn::boss::TaskPolicy::SetPriority Member Function

Syntax

nn::Result SetPriority(
     TaskPriority priority
);

Parameters

Name Description
in priority Specifies a task's execution priority.

Return Values

Returns the execution result of the function. Returns one of the following Result values.

Value Description
Result::IsSuccess Successfully set.
Other than those noted above An unexpected error. (For information about handling this error, see the Programming Manual.)

Description

Sets a task's execution priority.

For more information about priority types, see TaskPriority. Applications use the default value (PRIORITY_MEDIUM), and there is generally no need to change this. You must change this value to PRIORITY_STOPPED for tasks you want to run on a specific AP. A task's execution priority is adjusted by the scheduler according to circumstances and settings sent from the server.
