nn::boss::TaskQuery::GetValue Member Function

Syntax

nn::Result GetValue(
     u8 index,
     void * pValue,
     size_t size
);

Parameters

Name Description
in index Specifies the position of the HTTP query.
out pValue Specifies the value.
in size Specifies the size of the value.

Return Values

Returns the execution result of the function. Returns one of the following Result values.

Value Description
Result::IsSuccess Indicates successful retrieval.
ResultInvalidPointer The pointer to the value is NULL.
ResultNotFound Could not find a valid HTTP query at the specified position.
Other than those noted above An unexpected error. (For information about handling this error, see the Programming Manual.)

Description

Gets the value of the HTTP query.
