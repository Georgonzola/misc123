nn::boss::TaskServiceStatus Enumerated Type

Syntax

enum TaskServiceStatus;

Value

Value Description
SERVICE_TERMINATED 0: Service has terminated.
SERVICE_AVAILABLE 1: Service is available.
SERVICE_UNKNOWN 2: Cannot yet confirm service status.
GET_SERVICE_STATUS_ERROR 3: An error was detected while the service's status was being obtained.
SERVICE_MAX_VALUE (The enumerator used during internal processing. This enumerator cannot be used by applications.)

Description

Constants specifying task service statuses. (The numeric value of each enumerator is shown at the start of its description.)
