nn::boss::TaskStateCode Enumerated Type

Syntax

enum TaskStateCode;

Value

Value Description
TASK_STOPPED 0: The task is stopped.
TASK_WAITING_TIMER 1: The task is waiting for the time to start.
TASK_WAITING 2: The task is waiting to execute.
TASK_RUNNING 3: The task is executing.
TASK_PAUSED 4: The task is currently paused.
TASK_REGISTERED 5: The task is registered.
TASK_DONE 6: The task is done executing.
TASK_ERROR 7: Task execution ended in an error.
TASK_RETRY 8: Retrying the task after a communication error occurred during task execution.
GET_TASK_STATE_ERROR 9: Error getting the task state.
STATUS_MAX_VALUE (The enumerator used during internal processing. This enumerator cannot be used by applications.)

Description

Constants specifying task state codes. (The numeric value of each enumerator is shown at the start of its description.)
