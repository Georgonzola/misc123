nn::camera::CTR::ClearBuffer Function

Syntax

void ClearBuffer(
     Port port
);

Parameters

Name Description
in port Specifies the port to clear.

Return Values

None.

Description

Clears the buffer and the error flags.

This function is ignored if it is called during image capture (while the nn::camera::CTR::IsBusy function returns true). Do not call this function until you have used the nn::camera::CTR::StopCapture function to stop capturing and then waited for the nn::camera::CTR::IsBusy function's return value to change to false.
