nn::camera::CTR::Finalize Function

Syntax

void Finalize();

Parameters

None.

Return Values

None.

Description

Finalizes the CAMERA library.

Note: When shutting down the application, if this function is not called after stopping capture with the nn::camera::CTR::StopCapture function and stopping the cameras with the nn::camera::CTR::Activate(SELECT_NONE) function, then there may be audible noise while on the HOME Menu.

On SNAKE, the Super-Stable 3D feature is disabled from when the nn::camera::CTR::Initialize function is called until the nn::camera::CTR::Finalize function is called. In addition, note that these functions take longer to execute on SNAKE than on CTR due to the disabling and re-enabling of the Super-Stable 3D feature.
