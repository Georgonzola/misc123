nn::camera::CTR::FlipImage Function

Syntax

nn::Result FlipImage(
     CameraSelect select,
     Flip flip,
     Context context = CONTEXT_BOTH
);

Parameters

Name Description
in select Specifies which camera to change settings for.
in flip Specifies flipping.
in context Specifies which context to change settings for. If omitted, the settings are applied both to context A and context B.

Return Values

Returns the execution result of the function. Returns one of the following Result values.

Value Description
Result::IsSuccess Indicates that the operation succeeded.
ResultIsSleeping This function is unusable because the system cover is closed.
ResultFatalError There may be something physically wrong with the camera.

Description

Configures flipping of images.

The default values for each camera are shown below.

Inner CameraRight Outer CameraLeft Outer Camera
Context AFLIP_HORIZONTALFLIP_NONEFLIP_NONE
Context BFLIP_HORIZONTALFLIP_NONEFLIP_NONE

Note: This function may block for an extended period if a camera is restarting.
