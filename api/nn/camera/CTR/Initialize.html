nn::camera::CTR::Initialize Function

Syntax

nn::Result Initialize();

Parameters

None.

Return Values

Returns the execution result of the function. Returns one of the following Result values.

Value Description
Result::IsSuccess Indicates that the operation succeeded.
ResultAlreadyInitialized Specifies that the library is already initialized.
ResultUsingOtherProcess Cannot be used because another process is currently being used.
ResultIsSleeping This function is unusable because the system cover is closed.
ResultFatalError There may be something physically wrong with the camera.

Description

Initializes the CAMERA library and makes the cameras available for use.

Note: This function may block for an extended period if a camera is restarting.

On SNAKE, the Super-Stable 3D feature is disabled from when the nn::camera::CTR::Initialize function is called until the nn::camera::CTR::Finalize function is called. In addition, note that these functions take longer to execute on SNAKE than on CTR due to the disabling and re-enabling of the Super-Stable 3D feature.
