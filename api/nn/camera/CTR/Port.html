nn::camera::CTR::Port Enumerated Type

Syntax

enum Port;

Value

Value Description
PORT_NONE No port specified.
PORT_CAM1 Specifies the port for the inner camera and the right outer camera.
PORT_CAM2 Specifies the port for the left outer camera.
PORT_BOTH Both ports specified.

Description

Enumerated type that indicates a camera's connection target port.
