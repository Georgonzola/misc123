nn::camera::CTR::SetAutoExposureWindow Function

Syntax

nn::Result SetAutoExposureWindow(
     CameraSelect select,
     s16 startX,
     s16 startY,
     s16 width,
     s16 height
);

Parameters

Name Description
in select Specifies which camera to change settings for.
in startX Specifies the x-coordinate of the region's upper-left corner. If the x-coordinate of the image's upper-left corner is defined as 0, this argument can be specified in 40-pixel increments in the range from 0 through 600.
in startY Specifies the y-coordinate of the region's upper-left corner. If the y-coordinate of the image's upper-left corner is defined as 0, this argument can be specified in 30-pixel increments in the range from 0 through 450.
in width Specifies the region's width. This argument can be specified in 40-pixel increments in the range from 40 through 640. The sum of this value and startX must be no larger than 640.
in height Specifies the height of the region. This argument can be specified in 30-pixel increments in the range from 30 through 480. The sum of this value and startY must be no larger than 480.

Return Values

Returns the result of the function. Returns one of the following Result values.

Value Description
Result::IsSuccess Indicates that the operation succeeded.
ResultIsSleeping This function is unusable because the system cover is closed.
ResultFatalError There may be something physically wrong with the camera.

Description

Sets the region within the view to use as the basis for calculating auto-exposure (the response frame).

The rectangular region specified by startX, startY, width, and height must be no larger than 640 x 480.

The default values for each camera are shown below.

Inner CameraRight Outer CameraLeft Outer Camera
startX8000
startY6000
width480640640
height360480480

Although the cameras use the response frame that can be set by this function only when auto-exposure is enabled, you can set the response frame with this function even when auto-exposure is disabled. You must specify appropriate values when you call this function even if auto-exposure is disabled.

Note: This function may block for an extended period if a camera is restarting.
