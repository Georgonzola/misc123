nn::camera::CTR::SetNoiseFilter Function

Syntax

nn::Result SetNoiseFilter(
     CameraSelect select,
     bool on
);

Parameters

Name Description
in select Specifies which camera to change settings for.
in on Specify true to enable the noise filter. Specify false to disable it. The noise filter is enabled by default.

Return Values

Returns the execution result of the function. Returns one of the following Result values.

Value Description
Result::IsSuccess Indicates that the operation succeeded.
ResultIsSleeping This function is unusable because the system cover is closed.
ResultFatalError There may be something physically wrong with the camera.

Description

Sets the noise filter.

If a camera's noise filter is enabled, and the screen changes from bright to dark or vice versa, the camera module automatically eliminates noise by smoothing and other techniques. If you are using stereo cameras, this processing may cause only one of the stereo images to appear blurry. To avoid such situations, use this function to disable the noise filter for both cameras.

The default value is true for all cameras.

Note: This function may block for an extended period if a camera is restarting.
