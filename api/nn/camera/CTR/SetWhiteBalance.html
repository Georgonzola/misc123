nn::camera::CTR::SetWhiteBalance Function

Syntax

nn::Result SetWhiteBalance(
     CameraSelect select,
     WhiteBalance wb
);

Parameters

Name Description
in select Specifies which camera to change settings for.
in wb Specifies the white balance.

Return Values

Returns the execution result of the function. Returns one of the following Result values.

Value Description
Result::IsSuccess Indicates that the operation succeeded.
ResultIsSleeping This function is unusable because the system cover is closed.
ResultFatalError There may be something physically wrong with the camera.

Description

Sets the white balance.

The default value is WHITE_BALANCE_NORMAL for all cameras.

If synchronization with the stereo camera's brightness was enabled via the nn::camera::CTR::SetBrightnessSynchronization function, it will not be possible to set WHITE_BALANCE_NORMAL to the left external camera (SELECT_OUT2). Function execution will fail.

Note: This function may block for an extended period if a camera is restarting.
