Syntax

enum Size;

Value

Value Description
SIZE_VGA 640x480
SIZE_QVGA 320×240
SIZE_QQVGA 160×120
SIZE_CIF 352×288
SIZE_QCIF 176×144
SIZE_DS_LCD 256×192
SIZE_DS_LCDx4 512×384
SIZE_CTR_TOP_LCD 400×240
CAMERA_SIZE_MAX
SIZE_CTR_BOTTOM_LCD 320×240

Description

Enumerated type that indicates resolutions.

The SIZE_CIF and SIZE_QCIF aspect ratios are 11:9. If compared to images with a 4:3 aspect ratio resolution, these formats have their left and right sides cropped.

Similarly, the SIZE_CTR_TOP_LCD aspect ratio is 5:3. If compared to images with a 4:3 aspect ratio resolution, it has its top and bottom sides cropped.
