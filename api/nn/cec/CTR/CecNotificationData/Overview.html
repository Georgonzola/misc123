nn::cec::CTR::CecNotificationData Structure

Syntax

struct CecNotificationData
{
   size_t num;
   s32 count;
   CecNotificationParam param[MESSAGE_BOX_NUM_MAX];
};

Description

Receive information.

This information is the information that can be retrieved from GetCecInfoBuffer.

Member Variables

num size_t Number of items.
count s32 Communication event count (since system startup). This value is incremented each time communication occurs.
param CecNotificationParam Information about each box.
