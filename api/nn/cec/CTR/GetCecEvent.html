nn::cec::CTR::GetCecEvent Function

Syntax

nn::Result GetCecEvent(
     nn::os::Event * event
);

Parameters

Name Description
out event Specifies an uninitialized instance of nn::os::Event.

Return Values

nn::Result

Value Description
Result::IsSuccess Indicates success.

Description

Specifies the Event received as notification when StreetPass has occurred.

Specify an uninitialized instance of nn::os::Event.
You can use this function when you want to know in real time if any background communication occurred.
This specified Event is signaled when StreetPass has occurred (when any of the game title boxes sends or receives).
You can determine whether any receive data is addressed to foreground applications by using functions like GetCecInfoBuffer.
To release this Event, execute Finalize on the Event.
