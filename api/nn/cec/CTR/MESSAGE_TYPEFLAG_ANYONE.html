nn::cec::CTR::MESSAGE_TYPEFLAG_ANYONE Constant

Syntax

static const MessageTypeFlag MESSAGE_TYPEFLAG_ANYONE =  MESSAGE_TYPEFLAG_NON_FRIEND|MESSAGE_TYPEFLAG_FRIEND;

Description

Receivable without checking whether the sender is a friend or not a friend.

Use this to specify a MessageTypeFlag when creating a message (for example, Message::NewMessage).
