nn::cec::CTR::Message::GetExHeader Member Function

Syntax

nn::Result GetExHeader(
     MessageExHeaderType exhType,
     size_t * exhLen,
     void ** exhBody
) const;

Parameters

Name Description
in exhType Specifies the data type using the MessageExHeaderType enumerated type.
out exhLen The data length.
out exhBody The icon data when MESSAGE_EXHEADER_TYPE_ICON is specified for exhType. Icons are textures 40x40 pixels in size, and the pixel format is the native RGB565 PICA format. Gets the description data when MESSAGE_EXHEADER_TYPE_INFO is specified for exhType. For the description, up to two lines of text, each as wide as 16 characters of the widest system font, are displayed ("%" for Japan, the Americas, and Europe. Hiragana, kanji, Hangul, and other fonts for other region). The description may be up to 128 characters in length including newline characters and a terminator. The character encoding is UTF16-LE.

Return Values

nn::Result

Value Description
Result::IsSuccess Indicates success.
ResultNoData Specifies that there is no data.
Other Indicates failure.

Description

Gets the data from a message's extended header.
