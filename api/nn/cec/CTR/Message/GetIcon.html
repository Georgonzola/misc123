nn::cec::CTR::Message::GetIcon Member Function

Syntax

nn::Result GetIcon(
     void ** iconData,
     size_t * iconSize
) const;

Parameters

Name Description
out iconData The icon data. Icons are textures 40x40 pixels in size, and the pixel format is the native RGB565 PICA format.
out iconSize The data size (in bytes).

Return Values

nn::Result

Value Description
Result::IsSuccess Indicates success.
ResultNoData Indicates that there is no data.
Other Indicates failure.

Description

Gets the Icon of the Message.

Same as GetExHeader (MESSAGE_EXHEADER_TYPE_ICON, iconSize, iconData).
