nn::cec::CTR::Message::GetInfoText Member Function

Syntax

nn::Result GetInfoText(
     const wchar_t ** infoTextData,
     size_t * infoTextSize
) const;

Parameters

Name Description
out infoTextData The description data. For the description, up to two lines of text, each as wide as 16 characters of the widest system font, are displayed ("%" for Japan, the Americas, and Europe. Hiragana, kanji, Hangul, and other fonts for other region). The description may be up to 128 characters in length including newline characters and a terminator. The character encoding is UTF16-LE.
out infoTextSize The data size (in bytes).

Return Values

nn::Result

Value Description
Result::IsSuccess Indicates success.
ResultNoData Indicates that there is no data.
Other Indicates failure.

Description

Gets the information text of a message.

The same as GetExHeader(MESSAGE_EXHEADER_TYPE_INFO, infoTextSize, infoTextData).
