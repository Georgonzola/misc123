nn::cec::CTR::Message::GetMessageSize Member Function

Syntax

u32 GetMessageSize() const;

Parameters

None.

Return Values

Size value (in bytes).

Description

Gets the size of the message.

The total value of the header, signature, icon, body, and any other data is used as the message size.
Because some of this data is automatically placed in the header by the system, the message size can be roughly approximated by adding about 8 KB to the body size.
