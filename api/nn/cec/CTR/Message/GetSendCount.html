nn::cec::CTR::Message::GetSendCount Member Function

Syntax

u8 GetSendCount() const;

Parameters

None.

Return Values

Returns the send count.

Description

Gets the send count. The send count is decremented by one with each send. Data is no longer sent after the count reaches zero.

The message remains in the outbox even after it has been sent and its send count is 0.
