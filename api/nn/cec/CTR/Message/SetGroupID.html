nn::cec::CTR::Message::SetGroupID Member Function

Syntax

nn::Result SetGroupID(
     u32 groupId
);

Parameters

Name Description
in groupId Group ID.

Return Values

nn::Result

Value Description
Result::IsSuccess Indicates success.
Other Indicates failure.

Description

Sets the group ID for a message. Messages in the outbox with the same GroupID are grouped and sent together.

Anything in excess of the maximum size for a single send is not grouped.
When 0 is specified, nothing is grouped, and items are sent individually.

You can combine this with a MessageTypeFlag to send different data depending on the communication peer.
