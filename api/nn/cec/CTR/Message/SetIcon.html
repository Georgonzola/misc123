nn::cec::CTR::Message::SetIcon Member Function

Syntax

nn::Result SetIcon(
     const void * iconData,
     size_t iconSize
);

Parameters

Name Description
in iconData The icon data. Icons are textures 40x40 pixels in size, and the pixel format is the native RGB565 PICA format.
in iconSize Specifies the data length. The maximum value is 8 KB.

Return Values

nn::Result

Value Description
Result::IsSuccess Indicates success.
ResultTooLarge The data size is too large.
ResultNotAuthorized Attempted to edit a message that had already been established and assigned a MessageId.
Other Indicates failure.

Description

Sets a message icon.

The same as SetExHeader (MESSAGE_EXHEADER_TYPE_ICON, iconSize, iconData).
