nn::cec::CTR::Message::SetPropagationCount Member Function

Syntax

nn::Result SetPropagationCount(
     u8 propagationCount
);

Parameters

Name Description
in propagationCount Specifies the propagation count.

Return Values

nn::Result

Value Description
Result::IsSuccess Indicates success.

Description

Sets the propagation count. The propagation count is decremented by one upon reception. Data is copied to the outbox if the count is still 1 or greater.

Note: When 2 is specified, a message is sent twice: once from the original sender, and then once from recipients of the original sending. Note that data sent by the local host might be returned to the local host.
If the same message is already in the inbox, or if there is no room in the outbox, received messages are not copied to the outbox. In such cases, the message is not propagated further.
