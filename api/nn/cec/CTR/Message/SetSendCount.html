nn::cec::CTR::Message::SetSendCount Member Function

Syntax

nn::Result SetSendCount(
     u8 sendCount
);

Parameters

Name Description
in sendCount Specifies the send count.

Return Values

nn::Result

Value Description
Result::IsSuccess Indicates success.

Description

Sets the send count. The send count is decremented by one with each send. Data is no longer sent after the count reaches zero.

The message remains in the outbox even after it has been sent and its send count is 0.
