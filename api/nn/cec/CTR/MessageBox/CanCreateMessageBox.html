nn::cec::CTR::MessageBox::CanCreateMessageBox Member Function

Syntax

nn::Result CanCreateMessageBox(
     TitleId cecTitleId
);

Parameters

Name Description
in cecTitleId Specifies the StreetPass ID to check for.

Return Values

Returns the result of the operation.

Value Description
ResultSuccess A box can be created.
ResultBoxAlreadyExists The box already exists.
ResultBoxNumFull The maximum number of boxes has been created.
ResultInvalidArgument Indicates that one or more arguments are invalid.

Description

Checks whether a MessageBox for the specified StreetPass ID can be created.
