nn::cec::CTR::MessageBox::GetBoxSessionNum Member Function

Syntax

u32 GetBoxSessionNum(
     CecBoxType boxType
) const;

Parameters

Name Description
in boxType Specified using the CecBoxType enumerated type. Specify CEC_BOXTYPE_INBOX.

Return Values

Returns the number of Session IDs.

Description

Gets the Session ID count for (the number of items received in) the inbox.

The Session ID is generated using random numbers before searching for a communication partner.
This ID is passed to the partner once one is discovered. The local host and partner then begin communication.
The data receiver writes its partner's Session ID to received messages.
The same Session ID is written to all data received during the same CEC session.
Different Session IDs indicate that data was received in different communication sessions, even if the data has the same GroupID. You can find out how many times data was received by counting the unique Session IDs attached to the inbox messages.
