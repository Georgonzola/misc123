nn::cec::CTR::MessageBox::GetMessSizeByMessId Member Function

Syntax

size_t GetMessSizeByMessId(
     CecBoxType boxType,
     const MessageId & messageId
) const;

Parameters

Name Description
in boxType Inbox and outbox specification.
in messageId Message ID.

Return Values

Returns a value indicating the size.

Description

Gets the size of the message with the specified message ID.

This is the same as the value obtained from the GetMessageSizd function.
