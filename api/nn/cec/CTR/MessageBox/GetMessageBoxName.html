nn::cec::CTR::MessageBox::GetMessageBoxName Member Function

Syntax

nn::Result GetMessageBoxName(
     wchar_t * dataBuf,
     size_t dataBufSize
) const;

Parameters

Name Description

Encoding: UTF16-LE

in dataBufSize Specifies the buffer size.

Return Values

Returns an nn::Result value.

Value Description
Result::IsSuccess Indicates success.
Other Indicates failure.

Description

Gets the name of a MessageBox.
