nn::cec::CTR::MessageBox::SetBoxGroupNumMax Member Function

Syntax

nn::Result SetBoxGroupNumMax(
     u32 num,
     CecBoxType boxType
);

Parameters

Name Description
in num Maximum number of groups.
in boxType Inbox and outbox specification.

Return Values

Returns an nn::Result value.

Value Description
Result::IsSuccess Indicates success.
ResultNotAuthorized The box has not been opened.
ResultInvalidArgument Invalid number of groups.
Other Indicates failure.

Description

Sets the maximum number of message groups in the inbox or outbox.


The number of groups is not actually restricted by this value.
Use this value as a guideline for how many groups can be saved.
Inbox size (GetBoxSizeMax(CEC_BOXTYPE_INBOX)) divided by the size of one group,
or the maximum number of inbox messages (GetBoxMessNumMax(CEC_BOXTYPE_INBOX)) divided by the value for the number of messages for one group.
