nn::cec::CTR::MessageTypeFlag Type

Syntax

typedef u8 nn::cec::CTR::MessageTypeFlag;

Description

The flag for switching messages depending on the recipient peer. Specified in a message. (Such as Message::NewMessage.)

MESSAGE_TYPEFLAG_ANYONE = (MESSAGE_TYPEFLAG_NON_FRIEND | MESSAGE_TYPEFLAG_FRIEND)
Set this flag when sending messages without distinguishing whether the recipient is a friend.
This setting is usually used.

MESSAGE_TYPEFLAG_NON_FRIEND
Sends messages with recipients limited to non-friends.

MESSAGE_TYPEFLAG_FRIEND
Sends messages if the recipient is a friend.

It can be combined with the GroupId to send different messages depending on whether the recipient is a friend.
Example)
Place two messages in the outbox:
(1) [Group:1/MESSAGE_TYPEFLAG_NON_FRIEND/Body:MESSAGE1], and
(2)[Group:1/MESSAGE_TYPEFLAG_FRIEND/Body:MESSAGE2]
The result is that (1) is sent when a non-friend is found, and (2) is sent when a friend is found.
