Syntax

enum CfgCoppacsRestriction;

Value

Value Description
CFG_COPPACS_RESTRICTION_NONE No restrictions.
CFG_COPPACS_RESTRICTION_NEED_PARENTAL_PIN_CODE Allows restrictions to be released by entering the Parental Controls PIN code.
CFG_COPPACS_RESTRICTION_NEED_PARENTAL_AUTHENTICATION Requires COPPACS authorization procedures under System Settings (allows restrictions to be released by the application).
CFG_COPPACS_RESTRICTION_NUM

Description

Enumerated type representing COPPACS restrictions.
