nn::cfg::CTR::ConvertCountryCodeToIso3166a2 Function

Syntax

#include <nn/cfg.h>
nn::Result ConvertCountryCodeToIso3166a2(
     char * iso3166a2,
     CfgCountryCode countryCode
);

Parameters

Name Description
out iso3166a2 Stores the ISO 3166-1 alpha-2 string. Specify a buffer of 3 bytes or more.
If no ISO 3166-1 alpha-2 string corresponding to the country code exists, an empty string is stored here.
in countryCode Specifies the country code.

Return Values

The following Result values are returned as the execution result.

Value Description
Result::IsSuccess Indicates that the conversion succeeded.
ResultNotFound Specifies there is no ISO 3166-1 alpha-2 string corresponding to the country code.
This return value must always be handled.

Description

Converts the country code to an ISO 3166-1 alpha-2 string.
