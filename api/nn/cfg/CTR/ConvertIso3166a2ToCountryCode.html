nn::cfg::CTR::ConvertIso3166a2ToCountryCode Function

Syntax

#include <nn/cfg.h>
nn::Result ConvertIso3166a2ToCountryCode(
     CfgCountryCode * pCountryCode,
     const char * iso3166a2
);

Parameters

Name Description
out pCountryCode Stores the country code.
If the corresponding country code does not exist, CFG_COUNTRY_UNKNOWN is stored here.
in iso3166a2 Specifies the ISO 3166-1 alpha-2 string.

Return Values

The following Result values are returned as the execution result.

Value Description
Result::IsSuccess Indicates that the conversion succeeded.
ResultNotFound Indicates that there is no corresponding country code.
This return value must always be handled.

Description

Converts the ISO 3166-1 alpha-2 string to a country code.
