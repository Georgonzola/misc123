nn::cfg::CTR::ConvertToCtrSimpleAddressId Function

Syntax

#include <nn/cfg.h>
SimpleAddressId ConvertToCtrSimpleAddressId(
     SimpleAddressId wiiUSimpleAddressId
);

Parameters

Name Description
in wiiUSimpleAddressId Specifies the ID of the Wii U simple address information to be obtained.

Return Values

Returns the ID for the corresponding CTR simple address information.

Description

Converts Wii U simple address information IDs into CTR simple address information IDs.
