nn::cfg::CTR::ConvertToWiiUSimpleAddressId Function

Syntax

#include <nn/cfg.h>
SimpleAddressId ConvertToWiiUSimpleAddressId(
     SimpleAddressId ctrSimpleAddressId
);

Parameters

Name Description
in ctrSimpleAddressId Specifies the ID of the CTR simple address information to be obtained.

Return Values

Returns the ID of the simple address information of the corresponding Wii U.

Description

Converts CTR simple address information IDs into Wii U simple address information IDs.
