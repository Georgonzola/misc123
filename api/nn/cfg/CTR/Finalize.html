nn::cfg::CTR::Finalize Function

Syntax

#include <nn/cfg.h>
void Finalize(
     void
);

Parameters

None.

Return Values

None.

Description

Finalizes the use of the System Settings library.

After calling this function, you can no longer use the functions in the nn::cfg::CTR namespace (excluding the nn::cfg::CTR::GetCountryCodeA2 and nn::cfg::CTR::GetLanguageCodeA2 functions).
The function does nothing if the cfg library has not been initialized.
When the nn::cfg::CTR::Initialize function has been called multiple times, the cfg library will not be considered as finalized until the nn::cfg::CTR::Finaliz function is called an equal number of times.
