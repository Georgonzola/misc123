nn::cfg::CTR::GetSimpleAddressId Function

Syntax

#include <nn/cfg.h>
void GetSimpleAddressId(
     SimpleAddressId * pSimpleAddressId
);

Parameters

Name Description
out pSimpleAddressId Stores the obtained simple address information ID.

Return Values

None.

Description

Gets only the ID for the simple address information set in System Settings. For more information about this ID, see the nn::cfg::CTR::SimpleAddressId structure.

The country code is the same as the country code values that you can get using the nn::cfg::CTR::GetCountry function.
If no region has been set in System Settings, the region code is 0. If a country with no region selections is set in System Settings, the region code is 1.
If a region has been set in System Settings, the region code is 2 or higher.
