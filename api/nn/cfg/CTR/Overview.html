nn::cfg::CTR Namespace

Description

This is the namespace of the CTR system settings library. (It can also be written without the CTR portion.)

You must call nn::cfg::CTR::Initialize before any other function in the nn::cfg namespace that gets System Settings.

Classes

Result
nn::cfg::CTR::ResultOutOfMemory Insufficient memory.
nn::cfg::CTR::ResultNotFound Indicates that the target was not found.
nn::cfg::CTR::ResultMountContentFailed Failed to mount content.

Structures

nn::cfg::CTR::Birthday Stores a birthday.
nn::cfg::CTR::SimpleAddress Stores the user's simple address information configured in the System Settings.

Functions

Initialize Initializes the System Settings library.
Finalize Finalizes the use of the System Settings library.
GetUserName Gets the user name.
GetBirthday Gets the birthday.
GetCountry Gets the country code.
GetCountryCodeA2 Converts the country code to ISO 3166-1 alpha-2 format. This API is fast because it returns the value in memory. This API is scheduled for removal. In its place, please use the ConvertCountryCodeToIso3166a2 function.
ConvertCountryCodeToIso3166a2 Converts the country code to an ISO 3166-1 alpha-2 string.
ConvertIso3166a2ToCountryCode Converts the ISO 3166-1 alpha-2 string to a country code.
GetLanguage Gets the language setting in System Settings.
GetLanguageCodeA2 Converts a language code to ISO 639-1 alpha-2 format. This API is fast because it returns the value in memory.
GetSimpleAddress Gets the simple address information (such as the country and region name) that has been set by System Settings.
GetSimpleAddressId Gets only the ID for the simple address information set in System Settings. For more information about this ID, see the nn::cfg::CTR::SimpleAddressId structure.
ConvertToWiiUSimpleAddressId Converts CTR simple address information IDs into Wii U simple address information IDs.
ConvertToCtrSimpleAddressId Converts Wii U simple address information IDs into CTR simple address information IDs.
GetRegion Gets the region code of the system's market.
GetRegionCodeA3 Converts the region code of the system's market to the corresponding three-character string. This API is fast because it returns the value in memory.
GetSoundOutputMode Gets the sound output mode.
GetUserTimeOffset Gets RTC modification offset information.
IsCoppacsSupported Gets whether the current System Settings support COPPACS.
GetCoppacsRestriction Gets COPPACS restrictions. If the return value is CFG_COPPACS_RESTRICTION_NONE, COPPACS restrictions are not applied.
If the return value is CFG_COPPACS_RESTRICTION_NEED_PARENTAL_PIN_CODE, canceling the Parental Controls PIN code is viewed as canceling COPPACS restrictions.
Verify the PIN code using CheckParentalControlPinCode.
If the return value is CFG_COPPACS_RESTRICTION_NEED_PARENTAL_AUTHENTICATION, a jump must be made to system settings to perform COPPACS authentication.
IsParentalControlEnabled Gets whether Parental Controls settings are enabled.
CheckParentalControlPinCode Verifies the Parental Controls PIN code.
IsRestrictPhotoExchange Gets whether Parental Controls restrict the sending and receiving of pictures and images, sounds, videos, and long text (rich UGC).
IsRestrictAddFriend Gets whether adding friends is restricted by Parental Controls.
IsRestrictP2pInternet Gets whether Internet communication with other users is restricted by Parental Controls.
IsRestrictP2pCec Gets whether StreetPass communication with other users is restricted by Parental Controls.
IsRestrictShopUse Gets whether Parental Controls restrict credit card use and purchases of products and services in Nintendo eShop and elsewhere.
IsRestrictMiiverseBrowse Gets whether browsing of Miiverse is restricted by Parental Controls.
IsRestrictMiiversePost Gets whether posting to Miiverse is restricted by Parental Controls.
IsRestrictWatchVideo Gets whether Parental Controls restrict viewing video content.
IsAgreedEula Gets whether the user has agreed to the EULA.
GetTransferableId Gets the system ID.
