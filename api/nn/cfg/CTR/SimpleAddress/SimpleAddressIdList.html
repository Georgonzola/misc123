SimpleAddress Structure id Correspondence Table

SimpleAddress Structure id Correspondence Table (Japanese Prefectures / 50 US States)

The id member of the SimpleAddress structure stores the country code in the upper 8 bits and the region code in the lower 8 bits.
Here, we introduce the most frequently used region codes: those for the prefectures of Japan (country code: 1) and the 50 states of the United States (country code: 49).

Region code 0 indicates that Do not set was selected for Region Settings in the System Settings. Region code 1 indicates that Country Settings has been set to a country, such as Anguilla, for which region settings are not available.

Precautions:
The contents of this list may be updated at a later date due to changes in region names. If the list has been updated, the changes will be reflected in the System Settings when the CTR system is updated.
Accordingly, if you have used the contents of the list for in-game text, then it is possible that the names shown in your game will differ from the names selectable in the System Settings if the list gets updated after you have submitted the master list.
Moreover, updating the list may result in an increase in the number of regions that can be selected in the System Settings.

If you plan to use these codes, be careful that such changes do not present a problem in your game.
