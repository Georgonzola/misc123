nn::cfg::CTR::SimpleAddressId Structure

Syntax

struct SimpleAddressId
{
   u32 id;
};

Description

Stores the ID in simple address information.

The country code is stored in the upper eight bits of the ID, and the region code is stored in the following eight bits. The lower 16 bits are reserved.

Region code 0 indicates that no region has been selected in System Settings, while region code 1 indicates that the country has no region selections.
Region codes of 2 and higher indicate the corresponding specific regions.
For examples of region codes and the corresponding regions, see here.

Member Variables

id u32 Simple address information ID.

Member Functions

GetCountryCode Gets only the country code part of the simple address information ID.
GetRegionCode Gets only the region code part of the simple address information ID.
