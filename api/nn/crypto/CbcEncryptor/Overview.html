nn::crypto::CbcEncryptor Class

Syntax

template <size_t BlockSize>class CbcEncryptor : public nn::crypto::Encryptor

Template Arguments

Name Description
BlockSize
Block size of the targeted block cipher.

Description

Class for encrypting in CBC mode.

The cipher mode for CBC is as stipulated in NIST SP 800-38A.

This class is a template class that has the block size as a template parameter. Use the CbcDecryptor128 type when actually using.
