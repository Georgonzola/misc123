nn::crypto::GENERATE_HEADER_SIZE Constant

Syntax

const size_t GENERATE_HEADER_SIZE =  32;

Description

Amount by which data size increases when encrypted using the EncryptAndGenerateAes128Ccm function. When encrypting using the EncryptAndGenerateAes128Ccm function, the encrypted data is this number of bytes bigger than the original data.

When decrypting using the DecryptAndVerifyAes128Ccm function, the data returns to its original size.
