nn::crypto::GenerateRandomBytes Function

Syntax

void GenerateRandomBytes(
     void * pDst,
     size_t size
);

Parameters

Name Description
out pDst Pointer to the buffer storing the generated random number.
in size Size of the generated random number sequence.

Return Values

None.

Description

Generates a random number for encryption purposes.

This function is not suited to normal random number generation for the following reasons. Only use this function to generate random numbers for encryption purposes.
・This function is slow. ・The random numbers generated by this function cannot be reproduced.

Use the random number generator in nn::math for normal random number generation purposes.
