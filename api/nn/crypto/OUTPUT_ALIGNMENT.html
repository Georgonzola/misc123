nn::crypto::OUTPUT_ALIGNMENT Constant

Syntax

static const s32 OUTPUT_ALIGNMENT =  4;

Description

Specifies the number of bytes that the address that stores the result of the Sha1Context::GetHash, Sha256Context::GetHash, CalculateHmacSha1, CalculateHmacSha256, CalculateSha1, and CalculateSha256 functions must be aligned to.

Defined as 4 bytes.
