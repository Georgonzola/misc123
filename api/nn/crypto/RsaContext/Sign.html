nn::crypto::RsaContext::Sign Member Function

Syntax

nn::Result Sign(
     size_t * pSignLength,
     void * pDest,
     size_t destSize,
     const void * pSrc,
     size_t srcLength = SIZE_STRING_LENGTH
);

Parameters

Name Description
out pSignLength Length of signature data created.
out pDest The address of the buffer that stores the signature data.
in destSize The size of the buffer that stores the signature data (in bytes).
in pSrc The address of the buffer storing data to use for the signature.
in srcLength The length of data to use as the signature. (This parameter may be omitted. If omitted, the length is measured inside the function.)

Return Values

Returns an nn::Result value.

Description

Creates an RSA-PKCS1-v1.5-SHA256 signature from a string.
