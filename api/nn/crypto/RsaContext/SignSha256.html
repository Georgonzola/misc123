nn::crypto::RsaContext::SignSha256 Member Function

Syntax

nn::Result SignSha256(
     size_t * pSignLength,
     void * pDest,
     size_t destSize,
     const void * pSrc
);

Parameters

Name Description
out pSignLength Specifies the length of the created signature data.
out pDest The address of the buffer that stores the signature data.
in destSize The size of the buffer that stores the signature data (in bytes).
in pSrc Pointer to the address storing the SHA256 hash to be included in the signature.

Return Values

Returns an nn::Result value.

Description

Creates an RSA-PKCS1-v1.5-SHA256 signature from a hash.
