nn::cx::CompressGzip Function

Syntax

#include <nn/cx.h>
s32 CompressGzip(
     void * pDst,
     size_t dstSize,
     const void * pSrc,
     size_t srcSize,
     void * pWork
);

Parameters

Name Description
out pDst Buffer where compressed data is written.
in dstSize Size of the buffer pointed to by pDst.
in pSrc Buffer storing the data to be compressed.
in srcSize Size of the data to be compressed.
in pWork Specifies the work buffer. It requires a region equal to GZIP_COMPRESS_WORK_SIZE bytes.

Return Values

When compression succeeds, returns the data size after compression. Returns an ErrorCode value if unsuccessful.

Description

Performs deflate compression in gzip format.

You can get the size before compression for data compressed with this function by using the GetGzipUncompressedSize function.

Data compressed with this function can be decompressed with the UncompressGzip function.
