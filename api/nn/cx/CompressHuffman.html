nn::cx::CompressHuffman Function

Syntax

#include <nn/cx.h>
u32 CompressHuffman(
     const u8 * srcp,
     u32 size,
     u8 * dstp,
     u8 huffBitSize,
     void * work
);

Parameters

Name Description
in srcp Pointer to the compression source data.
in size Compression source data size.
out dstp Pointer to the compression target data.
in huffBitSize The number of bits to encode.
in work Temporary buffer to use for compression (requires a region of nn::cx::HUFFMAN_COMPRESS_WORK_SIZE bytes).

Return Values

Returns the size of the compressed data. If compressed data is larger than original data, compression is terminated and 0 is returned.

Description

Carries out Huffman compression.

Argument values must adhere to the following restrictions.

  • huffBitSize must be 4 or 8.
  • The address in work is aligned to SRC_ALIGNMENT_COMPRESS_HUFFMAN (4 bytes).
  • The value of size must be larger than 4.
  • The length of the dstp buffer must be equal to or greater than size. (This function never writes data larger than size to dstp.)
