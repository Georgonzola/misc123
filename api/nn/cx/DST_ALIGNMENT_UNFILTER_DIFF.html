nn::cx::DST_ALIGNMENT_UNFILTER_DIFF Constant

Syntax

static const s32 DST_ALIGNMENT_UNFILTER_DIFF =  2;

Description

Specifies the number of bytes that the address of the decompressed data specified in the UnfilterDiff or SecureUnfilterDiff function must be aligned to. Defined as 2 bytes.
