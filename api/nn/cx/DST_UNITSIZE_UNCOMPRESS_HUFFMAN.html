nn::cx::DST_UNITSIZE_UNCOMPRESS_HUFFMAN Constant

Syntax

static const size_t DST_UNITSIZE_UNCOMPRESS_HUFFMAN =  4;

Description

Specifies the number of bytes that the size of the decompressed data specified in the UncompressHuffman or SecureUncompressHuffman function must be a multiple of. Defined as 4 bytes.
