nn::cx::InitUncompContextGzip Function

Syntax

#include <nn/cx.h>
void InitUncompContextGzip(
     UncompContextGzip * pContext,
     void * pDst,
     size_t dstSize
);

Parameters

Name Description
out pContext The decompression context to initialize.
out pDst Buffer where decompressed data is written.
in dstSize Size of the buffer pointed to by pDst.

Return Values

None.

Description

Initializes the streaming decompression context for deflate compressed data in the GZIP format.
