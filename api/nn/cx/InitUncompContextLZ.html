nn::cx::InitUncompContextLZ Function

Syntax

#include <nn/cx.h>
void InitUncompContextLZ(
     UncompContextLZ * context,
     void * dest
);

Parameters

Name Description
in context Pointer to the decompressed context
in dest Destination address for uncompressed data.

Return Values

None.

Description

Initializes the streaming decompression context for LZ77 compressed data.

This API supports both LZ compression (backward compatibility) and LZ extended compression.
