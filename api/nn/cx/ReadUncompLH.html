nn::cx::ReadUncompLH Function

Syntax

#include <nn/cx.h>
s32 ReadUncompLH(
     UncompContextLH * context,
     const void * data,
     u32 len
);

Parameters

Name Description
in context Pointer to the decompressed context
in data Pointer to compressed data.
in len The size of the data.

Return Values

Returns the size of the remaining data to be decompressed. Returns a negative value if an error has occurred.

Description

Streaming decompression of LZ-Huffman compressed data.

If an error occurs, the negative value defined by nn::cx::ErrorCode returns.
