nn::cx::SRC_ALIGNMENT_UNCOMPRESS_LH Constant

Syntax

static const s32 SRC_ALIGNMENT_UNCOMPRESS_LH =  4;

Description

Specifies the number of bytes that the address of the compressed data specified in the UncompressLH or SecureUncompressLH function must be aligned to. Defined as 4 bytes.
