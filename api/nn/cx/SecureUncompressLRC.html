nn::cx::SecureUncompressLRC Function

Syntax

#include <nn/cx.h>
s32 SecureUncompressLRC(
     const u8 * srcp,
     u32 srcSize,
     u8 * destp,
     void * work
);

Parameters

Name Description
in srcp Address of the compressed data.
in srcSize Size of the compressed data.
out destp Address used to store decompressed data.
in work The work area (nn::cx::UNCOMPRESS_LRC_WORK_SIZE (36864) bytes are needed).

Return Values

If the conversion succeeds, returns 0; otherwise, returns a negative error code.

Description

Decompresses LZ-Range Coder compressed data. The compressed data address must be SRC_ALIGNMENT_UNCOMPRESS_LRC-byte aligned (4-byte aligned). The decompressed data address must be DST_ALIGNMENT_UNCOMPRESS_LRC-byte aligned (4-byte aligned).

Argument values must adhere to the following restrictions.

  • The address for srcp must be aligned to 4 bytes.
  • The address for work must be aligned to 4 bytes.
