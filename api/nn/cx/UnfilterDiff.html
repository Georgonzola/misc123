nn::cx::UnfilterDiff Function

Syntax

#include <nn/cx.h>
void UnfilterDiff(
     const void * srcp,
     void * destp
);

Parameters

Name Description
in srcp Address of the compressed data.
out destp Address used to store decompressed data.

Return Values

None.

Description

Restores data converted using a difference filter. The compressed data address must be SRC_ALIGNMENT_UNFILTER_DIFF-byte aligned (4-byte aligned). The decompressed data address must be DST_ALIGNMENT_UNFILTER_DIFF-byte aligned (4-byte aligned).

Argument values must adhere to the following restrictions.

  • The address for srcp must be aligned to 2 bytes.
  • The address for destp must have 2-byte boundary alignment.
