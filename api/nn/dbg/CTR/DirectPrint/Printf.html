Description of Printf ( const nn::math::VEC2 &, bool, bool, const char *, ... )

For pos, specify the value in a coordinate system where (0, 0) is the upper-left point of the display region specified by ChangeDisplaybuffer.

When autoLineFeed is set to true, text wraps when it reaches the edge of the display region.

The post-formatted string must fit within 255 characters.

Description of Printf ( const nn::math::VEC2 &, const char *, ... )

The version without the autoLineFeed and fillBg parameters. It acts the same as when true is passed.
