nn::dbg::CTR::DirectPrint::VPrintf Member Function

Syntax

#include <nn/dbg.h>
void VPrintf(
     const nn::math::VEC2 & pos,
     bool autoLineFeed,
     bool fillBg,
     const char * format,
     va_list v
);

Parameters

Name Description
in pos Upper-left coordinate where you are writing to.
in autoLineFeed Specifies whether to perform automatic text wrapping.
in fillBg Specify true to fill the text background with the specified background color. Specify false to make it transparent.
in format Specifies the format string.
in v String parameter.

Return Values

None.

Description

Formats strings and writes them to the display buffer. (The variable length argument version.)

See Printf.
