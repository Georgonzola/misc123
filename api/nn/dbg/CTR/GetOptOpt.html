nn::dbg::CTR::GetOptOpt Function

Syntax

#include <nn/dbg.h>
int GetOptOpt(
     void
);

Parameters

None.

Return Values

Unrecognized character during option analysis.

Description

Stores any character that could not be recognized during option analysis.

See the example for nn::dbg::CTR::GetOpt.

Always returns -1 in release builds.
