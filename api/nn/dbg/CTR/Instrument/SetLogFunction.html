nn::dbg::CTR::Instrument::SetLogFunction Member Function

Syntax

#include <nn/dbg.h>
static void SetLogFunction(
     void(*)(const char *,...) logFunc
);

Parameters

Name Description
logFunc Function for outputting dumps.

Return Values

None.

Description

Specifies the output function used to display buffer contents.

The function specified by logFunc is used for output by nn::dbg::CTR::Instrument::Dump function.

Although the default specifies the function for output to the PARTNER CTR-Debugger or IS-CTR-DEBUGGER console, you can do things like output to the SD card or filter results by changing the function as necessary.
