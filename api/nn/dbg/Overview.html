nn::dbg Namespace

Description

The namespace of the debug assist library.

Namespaces

nn::dbg::CTR The namespace of the CTR debug assist library. (It can also be written without the CTR portion.)

Enumerated Types

BreakReason Indicates the cause of a break.

typedef Definitions

BreakHandler User handler type for when assert fails or panic occurs.

Functions

Break Stops the program.
Panic Stops the program.
SetBreakHandler Sets the user handler for when the assert fails or a panic occurs.
