nn::dbg::SetBreakHandler Function

Syntax

#include <nn/dbg.h>
void SetBreakHandler(
     BreakHandler handler
);

Parameters

Name Description
in handler The handler to set.

Return Values

None.

Description

Sets the user handler for when the assert fails or a panic occurs.

Sets the user handler to call before stopping the program when the assert fails or a panic occurs.

The settings are released before the handler is called. In other words, the handler is not called even if an assert fails or a panic occurs in the handler.

This function is not limited to the case when the handler is reset from within the handler. There is no guarantee that the handler is called from only a single thread if the panic occurs simultaneously in multiple threads.

As it does when no handler is set, the program stops when it returns from the handler.
