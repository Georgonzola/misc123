nn::dlp::CTR::FakeClient::GetBufferSize Member Function

Syntax

static size_t GetBufferSize(
     u8 scanNum
);

Parameters

Name Description
in scanNum Specifies the number of child programs that can be received by a single scan request. The maximum value is MAX_SCAN_NUM.

Return Values

Returns the buffer size.

Description

Gets the buffer size required for fake client initialization.
