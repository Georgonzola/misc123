nn::dlp::CTR::FakeClient Class

Syntax

class FakeClient

Description

Class for fake clients for Download Play.

Member Constants

S BUFFER_ALIGNMENT const s32 Specifies the number of bytes that the working buffer address specified in the Initialize function must be aligned to. Defined as 4096 bytes.
S BUFFER_UNITSIZE const size_t Specifies the number of bytes that the size of the working buffer address specified in the Initialize function must be a multiple of. Defined as 4096 bytes.

Member Functions

S GetBufferSize Gets the buffer size required for fake client initialization.
S Initialize Initializes a fake client.
S Finalize Finalizes a fake client.
S GetEventDesc Gets arriving events from the queue.
S StartScan Starts scanning for servers.
S StopScan Stops scanning for servers.
S GetServerInfo Gets the server information from scan results. The linkLevel, nodeNum, and nodeInfo information changes dynamically and is only updated while scanning (CLIENT_STATE_SCANNING).
Use the GetConnectingNodes(), GetNodeInfo(), and GetLinkLevel() functions to get this information while connecting to the server (transitioning from CLIENT_STATE_WAITING_INVITE to CLIENT_STATE_REBOOTING).
S GetTitleInfo Gets child program information from the results of a scan.
S DeleteScanInfo Deletes the specified server's scan information (server information and child program information) from the scan results.
S StartFakeSession Starts a download session with no download.
S GetPassphrase Gets the UDS passphrase to use when an opponent client reconnects to a host.
S StopFakeSession Stops a participating session.
S GetMyStatus Gets the state of the local host.
S GetConnectingNodes Gets the node IDs of the nodes currently connected to the network.
S GetNodeInfo Gets the information for the nodes connected to the network.
S GetLinkLevel Gets the link level (signal strength).
