nn::dlp::CTR::Server::GetClientState Member Function

Syntax

static nn::Result GetClientState(
     ClientState * pClientState,
     u16 nodeId
);

Parameters

Name Description
out pClientState Returns the client's state.
in nodeId Specifies the client's node ID.

Return Values

The following Result values are returned as the execution result.

Value Description
Result::IsSuccess The client status was successfully obtained.
ResultInvalidState The state is invalid. This function can be used after the OpenSessions() function has succeeded (SERVER_STATE_OPENED_SESSIONS) as long as the state is not SERVER_STATE_ERROR.
ResultNoData No client with the specified node ID is connected.

Description

Gets the client's state.
