nn::dlp::CTR::ServerInfo Structure

Syntax

struct ServerInfo
{
   u8 mac[6];
   u8 channel;
   nn::util::SizedEnum1< nn::uds::LinkLevel > linkLevel;
   u8 maxNodeNum;
   u8 nodeNum;
   bit16 dlpVersion;
   NodeInfo nodeInfo[MAX_NODE_NUM];
   nn::os::Tick lastUpdateTick;
};

Description

Server information.

Member Variables

mac u8 Server MAC address.
channel u8 Channels that the server is on.
linkLevel nn::util::SizedEnum1< nn::uds::LinkLevel > The link level. It is updated every time a server is discovered while scanning (the CLIENT_STATE_SCANNING state).
maxNodeNum u8 Maximum number of connectable nodes, including the server.
nodeNum u8 The number of currently connected nodes, including the server. It is updated every time a server is discovered while scanning (the CLIENT_STATE_SCANNING state).
dlpVersion bit16 Download Play version.
nodeInfo NodeInfo Information about the nodes currently connected to the server. It is only updated while scanning (the CLIENT_STATE_SCANNING state).
lastUpdateTick nn::os::Tick The tick value when information was updated. This value is updated every time a server is discovered while scanning (the CLIENT_STATE_SCANNING state).
