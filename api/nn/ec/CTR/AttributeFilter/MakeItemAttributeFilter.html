nn::ec::CTR::AttributeFilter::MakeItemAttributeFilter Member Function

Syntax

static void MakeItemAttributeFilter(
     AttributeFilter outFilters[],
     const ItemId items[],
     const size_t numItems
);

Parameters

Name Description
out outFilters[] Specifies where to save the created filters.
items[]
in numItems Specifies the number of items.
in itemIds Specifies the items you want to set to the filters.

Return Values

None.

Description

Creates filters for searching for specified items. Use this function if you need to re-obtain the newest information for specific items. Get ItemId using the ContentSet::GetItemId function and the ServiceItem::GetItemId function. If you are handling multiple data titles, filter the data titles using the ContentSetCatalogFilter::SetTitleFilters function and the ServiceItemCatalogFilter::SetTitleFilter function to uniquely specify items.
