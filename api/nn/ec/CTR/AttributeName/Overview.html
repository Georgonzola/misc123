nn::ec::CTR::AttributeName Structure

Syntax

struct AttributeName
{
   char name[ATTRIBUTE_NAME_LENGTH];
};

Description

The structure that stores the item's attribute name.

Member Variables

name char A NULL-terminated UTF-8 string that specifies the attribute name.

Member Functions

S MakeNecessary Creates the name of the required attribute. The required attribute name contains the region code and language code. You can use this function to create a required attribute name including the specified region code and language code. Use MetaDataReader::GetRegionCode and MetaDataReader::GetLanguageCode to get a region code and language code specified in this function.
