nn::ec::CTR::ContentInfo::GetNumOccupiedBlocks Member Function

Syntax

size_t GetNumOccupiedBlocks() const;

Parameters

None.

Return Values

Returns the number of blocks on the SD card occupied by the content.

Description

Returns the number of blocks on the SD card occupied by the content.

Deleting content frees up room on the SD card by the number of blocks returned by this function.

The function returns 0 if the content has not been downloaded.
