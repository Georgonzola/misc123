nn::ec::CTR::ContentInfo::GetOccupiedSize Member Function

Syntax

s64 GetOccupiedSize() const;

Parameters

None.

Return Values

Returns the amount, in bytes, of SD card memory occupied by the content.

Description

Returns the amount of SD card memory occupied by the content.

Deleting content frees up room on the SD card by the amount of memory returned by this function. Use the block count obtained from ContentInfo::GetNumOccupiedBlocks to show the user the amount of space occupied by the content. Avoid displaying the occupied space obtained from this function in the application.

The function returns 0 if the content has not been downloaded.
