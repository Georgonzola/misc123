nn::ec::CTR::ContentInfo Class

Syntax

class ContentInfo

Description

Class representing local content information.

Use the DataTitle::ListContentInfos or ContentSet::ListContentInfos functions to get this information.

Member Functions

GetIndex Returns the content index.
IsOwned Determines whether the content has already been purchased.
IsOnDevice Determines whether the content has already been downloaded.
GetOccupiedSize Returns the amount of SD card memory occupied by the content.
GetNumOccupiedBlocks Returns the number of blocks on the SD card occupied by the content.
