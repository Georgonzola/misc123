nn::ec::CTR::ContentMeta Structure

Syntax

struct ContentMeta
{
   char title[CONTENT_META_TITLE_LENGTH];
   char text[CONTENT_META_TEXT_LENGTH];
   u32 iconIndex;
};

Description

The structure that stores metadata content information.

Member Variables

title char Stores the content name using a NULL-terminated UTF-8-encoded string.
text char Stores the content description using a NULL-terminated UTF-8-encoded string.
iconIndex u32 Index to the icon bound to the content.
