nn::ec::CTR::ContentSet::GetAttributeByName Member Function

Syntax

const char * GetAttributeByName(
     const char * name
) const;

Parameters

Name Description
in name NULL-terminated UTF-8 string that specifies the attribute name.

Return Values

Returns a pointer to a NULL-terminated UTF-8 string that indicates the attribute value.

Description

Gets a content set attribute value. Gets a content set attribute value specified by a NULL-terminated UTF-8 string attribute name. Returns NULL if the specified attribute name is not found. To include an attribute value in ContentSet, you must set an attribute name filter with ContentSetCatalogFilter::SetReturnAttributes and then call ContentSetCatalog::Retrieve. Specifying the set AttributeName::name in the name parameter of this function enables you to get the attribute value found in search. Note that the pointer to the attribute value is invalidated when the ContentSetCatalog obtained from the ContentSet is destroyed, or when the ContentSetCatalog::Retrieve function is called.
