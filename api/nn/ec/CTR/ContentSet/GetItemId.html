nn::ec::CTR::ContentSet::GetItemId Member Function

Syntax

ItemId GetItemId() const;

Parameters

None.

Return Values

Returns the item ID.

Description

Gets the item ID of a content set. Pass the ItemId obtained from this function as an argument to the EcApplet::RequesetRedeemContentSet function to have the EC applet purchase and download the content.

This function does not return a correct value when ContentSet::IsPurchasable returns false.
