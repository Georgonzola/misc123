nn::ec::CTR::ContentSet::GetReleaseDate Member Function

Syntax

fnd::DateTime GetReleaseDate() const;

Parameters

None.

Return Values

Returns date information.

Description

Gets the date a content set was released. The DateTime returned from this function is the new arrival reference date that is set as an item attribute and only the date information is valid. Zeros are set in all of the hour:minute:second fields. Note that this is different from the sales start date and time set in the item pricing. Note that although you can compare to the DateTime acquired with the Server::GetDateTime function and display it as a new arrival, the DateTime acquired with the Server::GetDateTime function is in Coordinated Universal Time (UTC).
