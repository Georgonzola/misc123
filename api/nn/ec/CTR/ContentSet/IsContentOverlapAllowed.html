nn::ec::CTR::ContentSet::IsContentOverlapAllowed Member Function

Syntax

bool IsContentOverlapAllowed() const;

Parameters

None.

Return Values

Returns true if duplicate purchases are allowed, and false otherwise.

Description

Determines whether purchase of duplicate content sets is allowed.

If purchase of duplicate content is allowed, the user can only purchase downloadable content if he or she does not already have licenses for all of that DLC as a whole set. If purchase of duplicate content is not allowed, the user can only purchase the downloadable content when he or she does not already have licenses for each individual piece of DLC.

For more information, see Sale of Sets in the 3DS Guidelines.
