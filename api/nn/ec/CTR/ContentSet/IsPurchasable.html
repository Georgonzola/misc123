nn::ec::CTR::ContentSet::IsPurchasable Member Function

Syntax

bool IsPurchasable() const;

Parameters

None.

Return Values

Returns true if the content is available for purchase, or false otherwise.

Description

Determines whether the content set is available for purchase. As one determination of whether purchase is possible, checks whether the sales start date and time has been reached. To determine whether it has been reached, the standard time for each country is used. Always returns true if ContentSet objects were searched with the SelectionFilter set to SELECTION_FILTER_PURCHASABLE.
