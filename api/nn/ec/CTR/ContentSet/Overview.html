nn::ec::CTR::ContentSet Class

Syntax

class ContentSet

Description

Class for getting content set information. Use the ContentSetCatalog::GetContentSet function to get an instance of the ContentSet class. Note that the ContentSet instance is invalidated when the ContentSetCatalog instance from which it was obtained is destroyed, or when the ContentSetCatalog::Retrieve function is called again.

Member Functions

GetTitleInfo Gets information about the data titles included in the content set.
IsPurchasable Determines whether the content set is available for purchase. As one determination of whether purchase is possible, checks whether the sales start date and time has been reached. To determine whether it has been reached, the standard time for each country is used. Always returns true if ContentSet objects were searched with the SelectionFilter set to SELECTION_FILTER_PURCHASABLE.
GetPrice Gets price information for a content set.
GetItemId Gets the item ID of a content set. Pass the ItemId obtained from this function as an argument to the EcApplet::RequesetRedeemContentSet function to have the EC applet purchase and download the content.
GetRating Gets rating information for a content set. There is generally no need to display the rating for a content set, and no need to use this function. Note that the pointer to the Rating object is invalidated when the ContentSetCatalog obtained from the ContentSet is destroyed.
GetReleaseDate Gets the date a content set was released. The DateTime returned from this function is the new arrival reference date that is set as an item attribute and only the date information is valid. Zeros are set in all of the hour:minute:second fields. Note that this is different from the sales start date and time set in the item pricing. Note that although you can compare to the DateTime acquired with the Server::GetDateTime function and display it as a new arrival, the DateTime acquired with the Server::GetDateTime function is in Coordinated Universal Time (UTC).
GetNumContents Gets the number of content items included in a content set.
ListIndexes Lists the indices of the content items included in a content set.
ListContentInfos Lists content information for a content set. If the SD card is inserted and removed while this function call is processing, or between calling the ContentSetCatalog::Retrieve function and calling this function, the function might return an unexpected Result.
CalculateRequiredSize Calculates the size required to download the content set. If the SD card is inserted and removed while this function call is processing, or between calling the ContentSetCatalog::Retrieve function and calling this function, the function might return an unexpected Result.
CalculateNumRequiredBlocks Calculates the number of blocks required to download the content set. If the SD card is inserted and removed while this function call is processing, or between calling the ContentSetCatalog::Retrieve function and calling this function, the function might return an unexpected Result.
GetAttributeByName Gets a content set attribute value. Gets a content set attribute value specified by a NULL-terminated UTF-8 string attribute name. Returns NULL if the specified attribute name is not found. To include an attribute value in ContentSet, you must set an attribute name filter with ContentSetCatalogFilter::SetReturnAttributes and then call ContentSetCatalog::Retrieve. Specifying the set AttributeName::name in the name parameter of this function enables you to get the attribute value found in search. Note that the pointer to the attribute value is invalidated when the ContentSetCatalog obtained from the ContentSet is destroyed, or when the ContentSetCatalog::Retrieve function is called.
IsContentOverlapAllowed Determines whether purchase of duplicate content sets is allowed.
