nn::ec::CTR::ContentSetCatalog::ContentSetCatalog Constructor

Syntax

ContentSetCatalog(
     Session session,
     void * pBuffer,
     size_t bufferSize
);

Parameters

Name Description
in session Specifies the Session object to use for initialization.
in pBuffer Pointer to the buffer to configure.
in bufferSize Size of the buffer to configure.

Description

Constructor for configuring the Session object and buffer. The Session object must have been initialized by the EcApplet::RequestInitializeSession function. The initialized data title maintained by the Session object is the one that is searched. The configured buffer is used when calling the ContentSetCatalog::Retrieve function.
