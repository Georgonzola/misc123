nn::ec::CTR::ContentSetCatalog Class

Syntax

class ContentSetCatalog : private nn::ec::CTR::detail::SessionHolder

Description

Class for searching for content sets.

Member Functions

ContentSetCatalog Constructor for configuring the Session object and buffer. The Session object must have been initialized by the EcApplet::RequestInitializeSession function. The initialized data title maintained by the Session object is the one that is searched. The configured buffer is used when calling the ContentSetCatalog::Retrieve function.
~ContentSetCatalog Destructor. After the instance is destroyed, the ContentSet obtained by ContentSetCatalog::GetContentSet is invalidated.
Retrieve Searches for content sets.
GetTotalNumMatched Returns the total number of content sets matching a search.
GetNumRetrieved Returns the number of content sets that were obtained in a single search query.
GetContentSet Gets content set information. Gets the ContentSet from the content set information searched by the ContentSetCatalog::Retrieve function. Specify a value for offset to select a specific ContentSet from the multiple content sets obtained from the ContentSetCatalog::Retrieve function. The offset value must be smaller than the number of content sets available from the ContentSetCatalog::GetNumRetrieved function. Operation is undefined when specifying a larger value for offset. Note that the ContentSet object is invalidated when the ContentSetCatalog instance from which it was obtained is destroyed.
