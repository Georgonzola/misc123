nn::ec::CTR::ContentSetCatalogFilter Class

Syntax

class ContentSetCatalogFilter : public nn::ec::CTR::detail::CatalogFilterImpl<ContentSetCatalogFilter>

Description

Filter for using ContentSetCatalog to search for ContentSet objects. Search by passing this class to the ContentSetCatalog::Retrieve function to filter the ContentSet search results.

Member Functions

ContentSetCatalogFilter Constructor for configuring a buffer for the filter. The configured buffer is used when calling the ContentSetCatalog::Retrieve function. You need the same number of buffers as you have configured filters. ContentSetCatalog::Retrieve returns ResultOutOfFilterMemory if the buffer is too small for the filters that were set. Fix the problem by such means as increasing the size of the buffer or setting fewer filters. A buffer of a specific size is required, even if nothing is set for the filter.
SetReturnAttributes Configures the attribute name for the attribute value to get. Call the ContentSetCatalog::Retrieve function using the filter set for the attribute name to get the attribute values corresponding to the relevant attribute names for each ContentSet object. Use the ContentSet::GetAttributeByName function to reference the attribute value to get. You can specify multiple attribute names in an array. ContentSetCatalogFilter only maintains references to the configured attribute name array and does not copy it, so be sure not to destroy the attribute name array until you are done using the configured filters.
SetAttributeFilters Configures the attribute filter. Call the ContentSetCatalog::Retrieve function using the filter set for the attribute filter to get just the ContentSet that matches the attribute filter. For more information about the attribute filter format, see AttributeFilter. You can specify multiple attribute filters in an array. ContentSetCatalogFilter only maintains a reference to the configured attribute filter array and does not copy it, so be sure not to destroy the attribute filter array until you are done using the configured filters.
SetTitleFilters Configures the title filter. Call the ContentSetCatalog::Retrieve function using the filter set for the title filter to get just the ContentSet included in the configured data title. If the title filter is not configured, all of the data titles initialized by the Session object configured by ContentSetCatalog are used as title filters. There is usually no need to configure a title filter. Specifying a data title not initialized in the Session object as a title filter causes the ContentSetCatalog::Retrieve function to return ResultSessionPreparedTitleNotFound and fail. You can specify multiple title filters in an array. ContentSetCatalogFilter only reads the configured title filter array and does not copy it. Be sure not to destroy the title filter array until you finish using the configured filters.
SetOrderByAttribute Configures the sort order for search results.
SetSelectionFilter Sets the search range.
