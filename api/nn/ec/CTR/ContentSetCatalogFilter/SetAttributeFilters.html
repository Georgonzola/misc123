nn::ec::CTR::ContentSetCatalogFilter::SetAttributeFilters Member Function

Syntax

ContentSetCatalogFilter & SetAttributeFilters(
     const AttributeFilter filters[],
     size_t numFilters
);

Parameters

Name Description
in filters[] Array of attribute filters to set.
in numFilters Number of attribute filters to set.

Return Values

Returns a reference to the filter set for the attribute filter.

Description

Configures the attribute filter. Call the ContentSetCatalog::Retrieve function using the filter set for the attribute filter to get just the ContentSet that matches the attribute filter. For more information about the attribute filter format, see AttributeFilter. You can specify multiple attribute filters in an array. ContentSetCatalogFilter only maintains a reference to the configured attribute filter array and does not copy it, so be sure not to destroy the attribute filter array until you are done using the configured filters.
