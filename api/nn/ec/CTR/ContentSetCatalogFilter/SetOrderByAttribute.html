nn::ec::CTR::ContentSetCatalogFilter::SetOrderByAttribute Member Function

Syntax

ContentSetCatalogFilter & SetOrderByAttribute(
     const OrderByAttribute * pOrder
);

Parameters

Name Description
in pOrder Pointer to the sort order setting.

Return Values

Returns a reference to the filter set for the sort order.

Description

Configures the sort order for search results.

By running ContentSetCatalog::Retrieve using the filter configured for the sort order for search results, you can sort the search results in order using a specific attribute as the key.

For more information about the sort order setting format, see OrderByAttribute.

Because you only maintain a reference to OrderByAttribute, configured by ContentSetCatalogFilter, without needing to copy it, there is no way to destroy the OrderByAttribute instance until the filter is used.
