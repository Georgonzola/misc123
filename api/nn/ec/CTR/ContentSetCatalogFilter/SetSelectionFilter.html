nn::ec::CTR::ContentSetCatalogFilter::SetSelectionFilter Member Function

Syntax

ContentSetCatalogFilter & SetSelectionFilter(
     SelectionFilter selection
);

Parameters

Name Description
in selection Search range to set.

Return Values

Returns a reference to the filter set for the search range.

Description

Sets the search range.

When SELECTION_FILTER_PURCHASABLE is specified, only items that can currently be purchased are searched. By default, this filter is applied even when this function is not run.

This filter targets both purchasable and downloadable items. There is no way to target just downloadable items.

When SELECTION_FILTER_ALL is specified, items that have not yet gone up for sales and items that are no longer for sale are added to the list of currently purchasable items. Specify this filter when you want to show the user items that will be on sale, or when you want to get information on items that could be redownloaded after sales have ended.

When searching for content using a sequence to redownload add-on content, use the SELECTION_FILTER_ALL filter to search all items that have been released until now, and not just what can be purchased currently.
