nn::ec::CTR::ContentSetCatalogFilter::SetTitleFilters Member Function

Syntax

ContentSetCatalogFilter & SetTitleFilters(
     const TitleInfo infos[],
     size_t numInfos
);

Parameters

Name Description
in infos[] Array of title filters to configure.
in numInfos Number of title filters to configure.

Return Values

Returns a reference to the filter set for the title filter.

Description

Configures the title filter. Call the ContentSetCatalog::Retrieve function using the filter set for the title filter to get just the ContentSet included in the configured data title. If the title filter is not configured, all of the data titles initialized by the Session object configured by ContentSetCatalog are used as title filters. There is usually no need to configure a title filter. Specifying a data title not initialized in the Session object as a title filter causes the ContentSetCatalog::Retrieve function to return ResultSessionPreparedTitleNotFound and fail. You can specify multiple title filters in an array. ContentSetCatalogFilter only reads the configured title filter array and does not copy it. Be sure not to destroy the title filter array until you finish using the configured filters.
