nn::ec::CTR::ContentSetGroup Class

Syntax

class ContentSetGroup

Description

Class for getting content set group information. Use the ContentSetGroupCatalog::GetContentSetGroup function to get an instance of the ContentSetGroup class. Note that the ContentSetGroup instance is invalidated when the ContentSetGroupCatalog instance from which it was obtained is destroyed, or when the ContentSetGroupCatalog::Retrieve function is called again.

Member Functions

GetName Gets the group name. Returns the group name as a NULL-terminated UTF-8 string. Note that the obtained pointer will become invalid when the ContentSetGroupCatalog instance that the ContentSetGroup was obtained from is destroyed.
GetSize Gets the number of content sets included in the group.
