nn::ec::CTR::ContentSetGroupCatalog::GetContentSetGroup Member Function

Syntax

ContentSetGroup GetContentSetGroup(
     u32 offset
) const;

Parameters

Name Description
in offset Offset of the content set group to get.

Return Values

Returns content set group information.

Description

Gets content set group information. Gets the ContentSetGroup from the content set group information retrieved by the ContentSetGroupCatalog::Retrieve function. Specify a value for offset to select a specific ContentSetGroup from the multiple content sets obtained from the ContentSetGroupCatalog::Retrieve function. The offset value must be smaller than the number of content sets returned by the ContentSetGroupCatalog::GetNumRetrieved function. Operation is undefined when specifying a larger value for offset. The ContentSetGroup object is invalidated when the ContentSetGroupCatalog instance from which it was obtained is destroyed.
