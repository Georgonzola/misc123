nn::ec::CTR::ContentSetGroupCatalog Class

Syntax

class ContentSetGroupCatalog : private nn::ec::CTR::detail::SessionHolder

Description

Class for searching for content set groups. Content set groups provide information about content sets grouped by user-defined keyword attribute values. Set a filter to group content sets by using ContentSetGroupCatalogFilter, and then call ContentSetGroupCatalog::Retrieve to get a ContentSetGroup as information grouped by ContentSetGroupCatalog::GetContentSetGroup. Content sets are grouped by attribute values of the specified length. For example, assume that the attribute name Color has four content sets, having the values BLACK, BLUE, GREEN, and BLACK. Grouping these content sets in ContentSetGroupCatalogFilter::SetAttributeGroupFilter with a filter specifying the attribute name "Color" and length "2" yields the following two groups. A group named BL containing 3 content sets. A group named GR containing 1 content set. If you specify a length of 0 in ContentSetGroupCatalogFilter::SetAttributeGroupFilter, content sets are grouped by the attribute value. In this example, the following three groups are returned. A group named BLACK containing 2 content sets. A group named GREEN containing 1 content set. A group named BLUE containing 1 content set.

Member Functions

ContentSetGroupCatalog Constructor for configuring the Session object and buffer. The Session object must have been initialized by the EcApplet::RequestInitializeSession function. The initialized data title maintained by the Session object is the one that is searched. The configured buffer is used when the ContentSetCatalogGroup::Retrieve function is run.
~ContentSetGroupCatalog Destructor. After the instance is destroyed, the ContentSetGroup obtained by ContentSetCatalogGroup::GetContentSetGroup is invalidated.
Retrieve Searches for content set groups.
GetTotalNumMatched Returns the total number of content set groups matching a search.
GetNumRetrieved Returns the number of content set groups that were obtained in a single search query.
GetContentSetGroup Gets content set group information. Gets the ContentSetGroup from the content set group information retrieved by the ContentSetGroupCatalog::Retrieve function. Specify a value for offset to select a specific ContentSetGroup from the multiple content sets obtained from the ContentSetGroupCatalog::Retrieve function. The offset value must be smaller than the number of content sets returned by the ContentSetGroupCatalog::GetNumRetrieved function. Operation is undefined when specifying a larger value for offset. The ContentSetGroup object is invalidated when the ContentSetGroupCatalog instance from which it was obtained is destroyed.
