nn::ec::CTR::ContentSetGroupCatalog::Retrieve Member Function

Syntax

ResultError Retrieve(
     u32 startOffset,
     size_t numItems,
     ContentSetGroupCatalogFilter & filter
);

Parameters

Name Description
in startOffset Offset to get the content set group from the search results matching the filter.
in numItems Number of content sets to get.
in filter Filter to use for the search.

Return Values

Returns the execution result of the function. For more information about handling ResultError objects, see the ResultError reference. The results returned by the ResultError::GetResult function are shown below.

Description

Searches for content set groups.

This function requires an Internet connection because it communicates with the Nintendo eShop server.

When this function is called, it first uses the buffer set by ContentSetGroupCatalogFilter to create a search query for the Nintendo eShop server. The function returns ResultOutOfFilterMemory if the buffer for creating the search query is too small.

Make sure to set the buffer size large enough when calling ContentSetCatalogFilter.

The search results returned by the Nintendo eShop server are extracted temporarily in system memory. The function returns ResultOutOfSystemMemoryError if there is not enough system memory for this operation. You can avoid running out of memory by adjusting the number of content set groups obtained each time, and calling this function multiple times with an offset specified to search through the content set groups a little at a time.

Use the ContentSetGroupCatalog::GetTotalNumMatched function to get the total number of content set groups that matched the filter.

The function then copies the content set group information from the search results extracted in system memory to the buffer configured by the constructor. Only part of the content set group information is copied if this buffer is too small to hold all of it.

Use the ContentSetGroupCatalog::GetNumRetrieved function to get the number of content set groups stored in the buffer.

The function returns ResultOutOfCatalogMemory or ResultOutOfCatalogMemoryError if the buffer is too small to hold even one item.

Note that adding and updating the items registered to the Nintendo eShop server can change the content set groups that match the filter and the content set groups that can be obtained with one search query.

If you do not plan for additions and updates, adding and updating items may not be possible in your application after it is released. To avoid this problem, adequately verify that you can add and update items under the expected conditions of actual operations.

The other member functions of this class must not be called if this function does not return successfully.

If this function is called multiple times, the ContentSetGroup objects obtained from previous calls are invalidated.
