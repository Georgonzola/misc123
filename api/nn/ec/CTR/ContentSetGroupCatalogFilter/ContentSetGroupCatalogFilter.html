nn::ec::CTR::ContentSetGroupCatalogFilter::ContentSetGroupCatalogFilter Constructor

Syntax

ContentSetGroupCatalogFilter(
     void * pBuffer,
     size_t bufferSize
);

Parameters

Name Description
in pBuffer Pointer to the buffer to configure.
in bufferSize Size of the buffer to configure.

Description

Constructor for configuring a buffer for the filter. The configured buffer is used when calling the ContentSetGroupCatalog::Retrieve function. You need the same number of buffers as you have configured filters. ContentSetGroupCatalog::Retrieve returns ResultOutOfFilterMemory if the buffer is too small for the filters that were set. Fix the problem by such means as increasing the size of the buffer or setting fewer filters. A buffer of a specific size is required, even if nothing is set for the filter.
