nn::ec::CTR::ContentSetGroupCatalogFilter Class

Syntax

class ContentSetGroupCatalogFilter : public nn::ec::CTR::detail::CatalogFilterImpl<ContentSetGroupCatalogFilter>

Description

Filter for using ContentSetGroupCatalog to search for ContentSetGroup objects. Search by passing this class to the ContentSetGroupCatalog::Retrieve function to filter the ContentSetGroup search results.

Member Functions

ContentSetGroupCatalogFilter Constructor for configuring a buffer for the filter. The configured buffer is used when calling the ContentSetGroupCatalog::Retrieve function. You need the same number of buffers as you have configured filters. ContentSetGroupCatalog::Retrieve returns ResultOutOfFilterMemory if the buffer is too small for the filters that were set. Fix the problem by such means as increasing the size of the buffer or setting fewer filters. A buffer of a specific size is required, even if nothing is set for the filter.
SetTitleFilters Configures the title filter. Call the ContentSetGroupCatalog::Retrieve function using the filter configured with the title filters to get just the ContentSetGroup objects included in the configured data titles. There is usually no need to configure title filters, because if none are configured, all of the data titles initialized in the Session object set in the ContentSetGroupCatalog object are used as title filters. Specifying a data title not initialized in the Session object as a title filter causes the ContentSetrGroupCatalog::Retrieve function to return ResultSessionPreparedTitleNotFound and fail. You can specify multiple title filters in an array. Do not destroy the title filter array until you are done using the configured filter, because ContentSetGroupCatalogFilter only maintains a reference to the specified title filter array and does not copy it.
SetAttributeGroupFilter Sets the attribute to group by. The attribute used for grouping must be a keyword attribute. For more information about grouping, see ContentSetGroupCatalog. Do not destroy the AttributeName object configured as the attribute name until you have used the configured filter, because the AttributeName object is not copied.
