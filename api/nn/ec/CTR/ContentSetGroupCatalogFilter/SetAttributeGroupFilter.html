nn::ec::CTR::ContentSetGroupCatalogFilter::SetAttributeGroupFilter Member Function

Syntax

ContentSetGroupCatalogFilter & SetAttributeGroupFilter(
     const AttributeName * name,
     size_t length
);

Parameters

Name Description
in name Specifies a pointer to an attribute name used for grouping.
in length Specifies a pointer to an attribute value length used for grouping.

Return Values

Returns a reference to the filter configured with the attribute to group by.

Description

Sets the attribute to group by. The attribute used for grouping must be a keyword attribute. For more information about grouping, see ContentSetGroupCatalog. Do not destroy the AttributeName object configured as the attribute name until you have used the configured filter, because the AttributeName object is not copied.
