nn::ec::CTR::DataTitle::DeleteContents Member Function

Syntax

ResultError DeleteContents(
     const u16 indexList[],
     size_t numIndexes
) const;

Parameters

Name Description
in indexList[] Index list of content to delete.
in numIndexes Number of indices.

Return Values

Returns the execution result of the function. The results returned by the ResultError::GetResult function are shown below.

Value Description
Result::IsSuccess Indicates that the operation succeeded.
ResultDataTitleContentNotFound Indicates that the specified content was not found.
ResultFsMediaWriteProtectedError The SD card is write-protected.
ResultFsMediaAccessFailedError An error occurred accessing the SD card.
ResultNoEnoughSpaceError There is not enough temporary space for deletion.
ResultDataTitleInUse One of the pieces of content of the specified data title is being mounted.
ResultDataTitleContentNotDeletable Non-deletable content was specified.

Description

Deletes content.

Required content with a content index of 0 cannot be deleted.
