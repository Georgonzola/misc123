nn::ec::CTR::DataTitle::GetNumContentInfos Member Function

Syntax

ResultError GetNumContentInfos(
     s32 * pOut
) const;

Parameters

Name Description
out pOut Buffer for getting the number of content information items.

Return Values

Returns the execution result of the function. For more information about how to handle ResultError objects, see the ResultError reference. The results returned by the ResultError::GetResult function are shown below.

Value Description
Result::IsSuccess Indicates that the operation succeeded.
ResultFsMediaAccessFailedError An error occurred accessing the SD card.

Description

Returns the number of content information items that can be obtained from the data title.

The number of content information items returned by this function is the number of content items defined by the data title. Note that this is not the number of downloaded content items.
