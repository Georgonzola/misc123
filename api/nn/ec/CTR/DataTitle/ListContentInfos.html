nn::ec::CTR::DataTitle::ListContentInfos Member Function

Syntax

ResultError ListContentInfos(
     s32 * pOut,
     ContentInfo * outInfoList,
     size_t numInfos,
     u32 offset
) const;

Parameters

Name Description
out pOut Buffer for getting the number of listed content information items.
out outInfoList Buffer for getting content information.
in numInfos Number of content information items to get.
in offset Offset of the content information to get.

Return Values

Returns the execution result of the function. For more information about how to handle ResultError objects, see the ResultError reference. The results returned by the ResultError::GetResult function are shown below.

Value Description
Result::IsSuccess Indicates that the operation succeeded.
ResultFsMediaAccessFailedError An error occurred accessing the SD card.

Description

Lists content information.

The content information listed by this function is the content information defined by the data title. Note that this includes more than just content information that has been downloaded.
