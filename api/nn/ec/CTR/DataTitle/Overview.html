nn::ec::CTR::DataTitle Class

Syntax

class DataTitle

Description

Class for getting content information from data titles.

You do not need a Session object to initialize DataTitle. You can use DataTitle without calling EcApplet::RequestInitializeSession so long as the data title is available locally. Invoking the EcApplet::RequestInitializeSession function may delete local data titles, because it synchronizes local licenses with the eShop server. Create a new DataTitle instance after invoking EcApplet::RequestInitializeSession.

Member Functions

Initialize Initializes an instance of the DataTitle class.
CalculateRequiredSizeOf Gets the size required to download the content.
CalculateNumRequiredBlocksOf Gets the number of blocks required to download the content.
GetNumContentInfos Returns the number of content information items that can be obtained from the data title.
ListContentInfos Lists content information.
FindContentInfos Searches content information.
DeleteContents Deletes content.
