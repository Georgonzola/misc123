nn::ec::CTR::ECardCatalog Class

Syntax

class ECardCatalog : private nn::ec::CTR::detail::SessionHolder

Description

Class for finding items that can be redeemed using a download voucher.

Member Functions

ECardCatalog Constructor for configuring the Session object and buffer. The Session object must have been initialized by the EcApplet::RequestInitializeSession function. The initialized data title maintained by the Session object is the one that is searched. The specified buffer is used when calling the ECardCatalog::Retrieve function.
~ECardCatalog The destructor. After the instance is destroyed, the ECardItem obtained by ECardCatalog::GetECardItem is invalidated.
Retrieve Searches for items that can be redeemed for a download voucher.
GetTotalNumMatched Returns the total number of items matching a search.
GetNumRetrieved Returns the number of items that were obtained in a single search query.
GetECardItem Gets information about items that can be redeemed using the download voucher. Gets an ECardItem object from the content set information retrieved by the ECardCatalog::Retrieve function. Specify a value for offset to select a specific ECardItem object obtained from the multiple content sets obtained from the ECardCatalog::Retrieve function. The offset value must be smaller than the number of content sets returned by the ECardCatalog::GetNumRetrieved function. Operation is undefined when specifying a larger value for offset. The ECardItem object is invalidated when the ECardCatalog instance from which it was obtained is destroyed.
