nn::ec::CTR::ECardCatalog::Retrieve Member Function

Syntax

ResultError Retrieve(
     u32 startOffset,
     size_t numItems,
     ECardCatalogFilter & filter
);

Parameters

Name Description
in startOffset Offset to get the item from the search results matching the filter.
in numItems The number of items to get.
in filter Filter to use for the search.

Return Values

Returns the result of the function. For more information about handling ResultError objects, see the ResultError reference. The results returned by the ResultError::GetResult function are shown below.

Value Description
Result::IsSuccess Indicates that the operation succeeded.
ResultSessionInvalid Invalid session.
ResultNewSessionRequired A new session is required.
ResultSessionPreparedTitleNotFound The title set in filter has not been initialized in this session, or no data titles have been initialized.
ResultECardNoRedeemableItemsError The item to be redeemed was not found.
ResultCatalogInvalidArgument An invalid filter was set for filter.
ResultOutOfFilterMemory The buffer set in the ECardCatalogFilter object is too small.
ResultOutOfCatalogMemory The buffer configured for the catalog is too small.
ResultOutOfCatalogMemoryError The buffer set in the catalog is too small to hold the search results.
ResultOutOfSystemMemoryError Insufficient system memory.
ResultAcNotConnectedError Either not connected to the Internet, or the connection was lost.
ResultInfraNeedsReconnect Infrastructure reconnection was requested.
ResultInfraReportError Infrastructure reported an error.
ResultInfraReceiveError Failed to receive from the infrastructure connection.
ResultInfraInvalidResponseError Invalid response from the infrastructure connection.
ResultInfraNeedsUpdateError Infrastructure was updated, so the system must be updated.
ResultInfraTimeoutError Infrastructure communication timeout.
ResultInfraBusyError The infrastructure connection is busy.

Description

Searches for items that can be redeemed for a download voucher.

This function requires an Internet connection because it communicates with the Nintendo eShop server.

When this function is called, it first uses the buffer set by ECardCatalogFilter to create a search query for the Nintendo eShop server. The function returns ResultOutOfFilterMemory if the buffer for creating the search query is too small.

Set a larger buffer in the ECardCatalogFilter object.

The search results returned by the Nintendo eShop server are extracted temporarily in system memory. The function returns ResultOutOfSystemMemoryError if there is not enough system memory for this operation. You can avoid running out of memory by adjusting the number of content sets obtained each time, and calling this function multiple times with an offset specified to search through the content sets a little at a time.

Use the ECardCatalog::GetTotalNumMatched function to get the total number of content sets that matched the filter.

The function then copies the content set information from the search results extracted in system memory to the buffer configured by the constructor.

Only part of the content set information is copied if this buffer is too small to hold all of it. Use the ECardCatalog::GetNumRetrieved function to get the number of content sets stored in the buffer.

The function returns ResultOutOfCatalogMemory or ResultOutOfCatalogMemoryError if the buffer is too small to hold even one item.

Note that adding and updating the items registered to the Nintendo eShop server can change the content sets that match the filter and the content sets that can be obtained with one search query.

If you do not plan for additions and updates, adding and updating items may not be possible in your application after it is released. To avoid this problem, adequately verify that you can add and update items under the expected conditions of actual operations.

The other member functions of this class must not be called if this function does not return successfully.

If this function is called multiple times, the ECardItem objects obtained from previous calls are invalidated.
