nn::ec::CTR::ECardCatalogFilter::SetAttributeFilters Member Function

Syntax

ECardCatalogFilter & SetAttributeFilters(
     const AttributeFilter filters[],
     size_t numFilters
);

Parameters

Name Description
in filters[] Array of attribute filters to set.
in numFilters Number of attribute filters to set.

Return Values

Returns a reference to the filter set for the attribute filter.

Description

Configures the attribute filter. Call the ECardCatalog::Retrieve function using the filter configured with the attribute filters to get just the ECardItem objects that match the attribute filters. For more information about how to code attribute filters, see AttributeFilter. Currently, the only AttributeFilter::filterType that can be set in a ECardCatalogFilter is "==". The ECardCatalog::Retrieve object will return an error if a different filter type is set. You can specify multiple attribute filters in an array. Do not destroy the attribute filter array until you are done using the configured filter, because ECardCatalogFilter only maintains a reference to the specified attribute filter array and does not copy it.
