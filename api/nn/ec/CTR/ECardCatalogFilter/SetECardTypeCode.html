nn::ec::CTR::ECardCatalogFilter::SetECardTypeCode Member Function

Syntax

ECardCatalogFilter & SetECardTypeCode(
     const ECardTypeCode * pCode
);

Parameters

Name Description
in pCode Pointer to download code.

Return Values

Returns a reference to the filter set for the sort order.

Description

Configures the download code. Specify the download code retrieved from the CardInfo::GetTypeCode function. The item to be redeemed for the set download code can be searched when ECardCatalog::Retrieve is run using the selected filter. If no download code is set in the filter, it is treated as an invalid filter.
