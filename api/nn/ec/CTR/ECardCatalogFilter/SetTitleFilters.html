nn::ec::CTR::ECardCatalogFilter::SetTitleFilters Member Function

Syntax

ECardCatalogFilter & SetTitleFilters(
     const TitleInfo infos[],
     size_t numInfos
);

Parameters

Name Description
in infos[] Array of title filters to configure.
in numInfos Number of title filters to configure.

Return Values

Returns a reference to the filter set for the title filter.

Description

Configures the title filter. Call the ECardCatalog::Retrieve function using a filter configured with the title filters to get just the ECardItem objects included in the configured data titles. There is usually no need to set title filters, because if none are configured, all of the data titles initialized in the Session object set in the ECardCatalog object will be used as title filters. Specifying a data title not initialized in the Session object as a title filter will cause the ECardCatalog::Retrieve function to return ResultSessionPreparedTitleNotFound and fail. You can specify multiple title filters in an array. ECardCatalogFilter only maintains a reference to the specified title filter array and does not copy it, so be sure not to destroy the title filter array until you are done using the configured filter.
