nn::ec::CTR::ECardInfo Class

Syntax

class ECardInfo : private nn::ec::CTR::detail::SessionHolder

Description

Class for getting download-voucher information.

Member Functions

ECardInfo Constructor for configuring the Session object and buffer. The Session object must have been initialized by the EcApplet::RequestInitializeSession function. The initialized data title maintained by the Session object is the one that is searched. The specified buffer is used when calling the ECardInfo::Retrieve function.
~ECardInfo Destructor.
Retrieve Gets download voucher information. This function requires an Internet connection because it communicates with the Nintendo eShop server. Copies the download voucher information to the buffer specified in the constructor. Returns ResultOutOfCatalogMemory or ResultOutOfCatalogMemoryError if the buffer is too small to hold all the download voucher information. Any lowercase letters included in the download voucher ID input to this function are converted to uppercase before being sent to the Nintendo eShop server. The other member functions of this class must not be called if this function does not return successfully.
GetTypeCode Gets the download voucher code. You can configure a search filter for the items associated with the download voucher by passing the download voucher code obtained here to ECardCatalogFilter::SetECardTypeCode.
