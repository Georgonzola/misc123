nn::ec::CTR::ECardInfo::Retrieve Member Function

Syntax

ResultError Retrieve(
     const char * eCardId
);

Parameters

Name Description
in eCardId NULL-terminated string holding the ID of the download voucher to retrieve.

Return Values

Returns the execution result of the function. For more information about how to handle ResultError objects, see the ResultError reference. The results returned by the ResultError::GetResult function are shown below.

Description

Gets download voucher information. This function requires an Internet connection because it communicates with the Nintendo eShop server. Copies the download voucher information to the buffer specified in the constructor. Returns ResultOutOfCatalogMemory or ResultOutOfCatalogMemoryError if the buffer is too small to hold all the download voucher information. Any lowercase letters included in the download voucher ID input to this function are converted to uppercase before being sent to the Nintendo eShop server. The other member functions of this class must not be called if this function does not return successfully.
