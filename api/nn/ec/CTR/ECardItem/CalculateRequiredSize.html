nn::ec::CTR::ECardItem::CalculateRequiredSize Member Function

Syntax

ResultError CalculateRequiredSize(
     s64 * pOut
) const;

Parameters

Name Description
out pOut Buffer for getting the required size.

Return Values

Returns the result of the function. For more information about handling ResultError objects, see the ResultError reference. The ResultError::GetResult function returns the following results.

Value Description
Result::IsSuccess Indicates that the operation succeeded.

Description

Calculates the size required to download the item (content set). If the SD card is inserted and removed while this function call is processing, or between calling the ECardCatalog::Retrieve function and calling this function, the function might return an unexpected Result.
