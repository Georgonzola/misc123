nn::ec::CTR::ECardItem::GetAttributeByName Member Function

Syntax

const char * GetAttributeByName(
     const char * name
) const;

Parameters

Name Description
in name NULL-terminated UTF-8 string that specifies the attribute name.

Return Values

Returns a pointer to a NULL-terminated UTF-8 string that indicates the attribute value.

Description

Gets a content set attribute value. Gets a content set attribute value specified by a NULL-terminated UTF-8 string attribute name. Returns NULL if the specified attribute is not found. Note that the pointer to the attribute value is invalidated when the ECardCatalog from which the ECardItem was obtained is destroyed, or when the ECardCatalog::Retrieve function is called.
