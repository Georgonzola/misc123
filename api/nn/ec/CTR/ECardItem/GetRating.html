nn::ec::CTR::ECardItem::GetRating Member Function

Syntax

Rating GetRating() const;

Parameters

None.

Return Values

Returns rating information.

Description

Gets rating information for an item. There is generally no need to display the rating for an item, and no need to use this function. Note that the data indicated by the pointer of the Rating object is invalidated when the ECardCatalog that the ECardItem was obtained from is destroyed.
