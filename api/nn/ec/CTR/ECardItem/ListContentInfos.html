nn::ec::CTR::ECardItem::ListContentInfos Member Function

Syntax

ResultError ListContentInfos(
     s32 * pOutNum,
     ContentInfo * outList,
     size_t numInfos,
     u32 offset
) const;

Parameters

Name Description
out pOutNum Buffer for getting the number of obtained content information items.
out outList Buffer for getting content information.
in numInfos Number of content information items to get.
in offset Offset of the content information to get.

Return Values

Returns the result of the function. For more information about how to handle ResultError objects, see the ResultError reference. The results returned by the ResultError::GetResult function are shown below.

Value Description
Result::IsSuccess Indicates that the operation succeeded.

Description

Lists content information for an item. If the SD card is inserted and removed while this function call is processing, or between calling the ECardCatalog::Retrieve function and calling this function, the function might return an unexpected Result.
