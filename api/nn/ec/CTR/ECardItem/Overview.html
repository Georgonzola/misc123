nn::ec::CTR::ECardItem Class

Syntax

class ECardItem

Description

Class for getting information about items that can be redeemed using a download voucher. Use the ECardCatalog::GetECardItem function to get an instance of the ECardItem class. Note that the ECardItem instance is invalidated when the ECardCatalog instance from which it was obtained is destroyed, or when the ECardCatalog::Retrieve function is called again.

Member Functions

GetTitleInfo Gets information about the data titles included in the item.
GetItemId Gets the item ID. Pass the ItemId obtained from this function as an argument to the EcApplet::RequesetRedeemContentSet function to request that the EC applet redeem a download voucher for content.
GetRating Gets rating information for an item. There is generally no need to display the rating for an item, and no need to use this function. Note that the data indicated by the pointer of the Rating object is invalidated when the ECardCatalog that the ECardItem was obtained from is destroyed.
GetReleaseDate Gets the date an item was released. The DateTime returned from this function is the new arrival reference date that is set as an item attribute and only the date information is valid. Zeros are set in all of the hour:minute:second fields. Note that this date and time differs from the sales start date and time set in the item pricing. Note that although you can compare to the DateTime acquired with the Server::GetDateTime function and display it as a new arrival, the DateTime acquired with the Server::GetDateTime function is in Coordinated Universal Time (UTC).
GetNumContents Gets the number of content items included in the item.
ListIndexes Lists the indices of the content included in an item.
ListContentInfos Lists content information for an item. If the SD card is inserted and removed while this function call is processing, or between calling the ECardCatalog::Retrieve function and calling this function, the function might return an unexpected Result.
CalculateRequiredSize Calculates the size required to download the item (content set). If the SD card is inserted and removed while this function call is processing, or between calling the ECardCatalog::Retrieve function and calling this function, the function might return an unexpected Result.
CalculateNumRequiredBlocks Calculates the number of blocks required to download the item. If the SD card is inserted and removed while this function call is processing, or between calling the ECardCatalog::Retrieve function and calling this function, the function might return an unexpected Result.
GetAttributeByName Gets a content set attribute value. Gets a content set attribute value specified by a NULL-terminated UTF-8 string attribute name. Returns NULL if the specified attribute is not found. Note that the pointer to the attribute value is invalidated when the ECardCatalog from which the ECardItem was obtained is destroyed, or when the ECardCatalog::Retrieve function is called.
GetType Gets the item type.
GetServiceItemType Gets the license type for the service item.
GetServiceItemLimit Returns the expiration or number of uses for the service item.
