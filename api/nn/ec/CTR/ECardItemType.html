nn::ec::CTR::ECardItemType Enumerated Type

Syntax

enum ECardItemType;

Value

Value Description
ECARD_ITEM_TYPE_CONTENT_SET A content set that can be exchanged for a download voucher.
ECARD_ITEM_TYPE_SERVICE_ITEM A service item that can be exchanged for a download voucher.
ECARD_ITEM_TYPE_UNKNOWN Type used by the system. Do not use.

Description

An enumerated type representing the types of items that can be exchanged using download vouchers.
