nn::ec::CTR::EcApplet::CancelPreload Member Function

Syntax

static void CancelPreload();

Parameters

None.

Return Values

None.

Description

Cancels the preloading of the EC applet. When preloading has not completed, this function does nothing. Consequently, you must either call the EcApplet::WaitForPreload function before calling this function, or use the EcApplet::IsRegistered function to confirm that all preloading is complete. This function cancels preloading for all library applets. Note that calling this function cancels preloading for any other library applets in addition to EC applets.
