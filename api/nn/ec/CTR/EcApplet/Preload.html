nn::ec::CTR::EcApplet::Preload Member Function

Syntax

static void Preload();

Parameters

None.

Return Values

None.

Description

Preloads the EC applet. Preloads the EC applet to make it start faster. However, the following cautions are needed. Preloading will not be finished even when returning from this function. Consequently, you must either call the EcApplet::WaitForPreload function when calling this function to preload, or use the EcApplet::IsRegistered function to confirm that preloading is complete before calling any request function. When a library applet other than the EC applet is preloading, you must cancel that preload before calling this function. You must not terminate the application while an applet is in a preloaded state. Call EcApplet::CancelPreload to cancel the preload before closing the application. After calling the Preload function, the application must not process the HOME Button until the IsRegistered function returns true. Specifically, if the IsExpectedToProcessHomeButton() function returns true, you must call the ClearHomeButtonState() function instead of the ProcessHomeButton() function.
