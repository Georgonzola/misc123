nn::ec::CTR::ItemId Structure

Syntax

struct ItemId
{
   bit32 id;
   char code[ITEM_CODE_LENGTH];
};

Description

The structure that indicates item IDs.

There is no need to reference normal ItemId members.

Member Variables

id bit32 Item ID.
code char NULL-terminated UTF-8 string that identifies the item. In actual practice, it works out to be 16 characters plus the NULL terminator.
