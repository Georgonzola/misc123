nn::ec::CTR::MetaDataReader::Finalize Member Function

Syntax

void Finalize();

Parameters

None.

Return Values

None.

Description

Destroys an instance of the MetaDataReader class.

The MetaDataReader::Initialize function mounts metadata content. After the MetaDataReader::Initialize function has been called, call this function to unmount the metadata content before making a call to EcApplet::RequestInitializeSession or DataTitle::DeleteContents.
If you do not call this function, ResultDataTitleInUse is generated and the process fails.
