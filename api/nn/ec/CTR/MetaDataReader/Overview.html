nn::ec::CTR::MetaDataReader Class

Syntax

class MetaDataReader : private nn::util::NonCopyable< MetaDataReader >

Description

Class for reading information from metadata content.

Each data title has its own metadata content, and this contains information such as add-on content name, description, and icon.

Call the EcApplet::RequestInitializeSession function to complete the initialization of a data title, and then automatically get the metadata content for that title on the SD card.

You do not need a Session object to initialize DataTitle. You can use DataTitle without calling EcApplet::RequestInitializeSession so long as the data title is available locally. Calling the MetaDataReader::Initialize function to initialize an instance of MetaDataReader consumes a resource that is shared by the system as a whole. You must not have more than one valid instance of MetaDataReader at any one time.

Member Functions
