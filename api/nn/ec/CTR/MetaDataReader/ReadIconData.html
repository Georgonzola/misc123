nn::ec::CTR::MetaDataReader::ReadIconData Member Function

Syntax

ResultError ReadIconData(
     IconData * pOut,
     u32 iconIndex
);

Parameters

Name Description
out pOut Buffer for getting icon data.
in iconIndex Icon data index.

Return Values

Returns the execution result of the function. For more information about how to handle ResultError objects, see the ResultError reference. The results returned by the ResultError::GetResult function are shown below.

Value Description
Result::IsSuccess Indicates that the operation succeeded.
ResultMetaDataIconNotFound Icon data not found.
ResultMetaDataInvalidFormat Invalid format.
ResultFsMediaAccessFailedError An error occurred while accessing the SD card.

Description

Gets icon data.

Each icon data item is assigned a separate index number different from the content index. Call this function specifying this index to get icon data.

The icon data index is included in the content metadata. Call the ContentMeta::iconIndex function to get this.
