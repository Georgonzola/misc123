nn::ec::CTR::Money Structure

Syntax

struct Money
{
   char amount[MONEY_AMOUNT_LENGTH];
};

Description

The structure that stores balance and price information. To display balance and price information on the screen, it is not enough to just display the string stored in the Money structure. You must also display any currency prefix and suffix. Use the Server::ConvertPrice function to convert the information in the Money structure into a PriceIndication structure. Make sure to display the converted PriceIndication on the screen.

Member Variables

amount char A NULL-terminated UTF-8 string that indicates the balance and price.
