nn::ec::CTR::NecessaryAttribute Enumerated Type

Syntax

enum NecessaryAttribute;

Value

Value Description
NECESSARY_ATTRIBUTE_NAME Represents a required attribute with a NA_NAME_ prefix.
NECESSARY_ATTRIBUTE_SORT_NAME Represents a required attribute with a NA_SORT_NAME_ prefix.
NECESSARY_ATTRIBUTE_DESCRIPTION_SHORT Represents a required attribute with a NA_DESCRIPTION_SHORT_ prefix.
NECESSARY_ATTRIBUTE_DESCRIPTION_LONG Represents a required attribute with a NA_DESCRIPTION_LONG_ prefix.
NECESSARY_ATTRIBUTE_DISCLAIMER Represents a required attribute with a NA_DISCLAIMER_ prefix.
NECESSARY_ATTRIBUTE_ICON_INDEX Represents a required attribute with a NA_ICON_INDEX_ prefix.

Description

An enumerated type indicating the required-attribute type.

You can create a required attribute name by passing this to the AttributeName::MakeNecessary function.
