nn::ec::CTR::OrderByAttribute Structure

Syntax

struct OrderByAttribute
{
   char orderByType[ATTRIBUTE_ORDER_BY_TYPE_LENGTH];
   char name[ATTRIBUTE_NAME_LENGTH];
   char dataType[ATTRIBUTE_DATA_TYPE_LENGTH];
};

Description

The structure for specifying the sort order of item search results.

It can be used to change the sort order based on the specified attribute as a key. You can specify a user-defined attribute name and a required attribute name acquired with MakeNecessary as a key. Attributes must by set on the server as a KEYWORD-type. Specify ascending or descending as the attribute type for sorting.

Member Variables

orderByType char Specifies the sort order using a NULL-terminated string. Specify ascending or descending.
name char Specifies the attribute name to use as the sort order key using a NULL-terminated string.
dataType char Specifies the data type using a NULL-terminated string. Specify string or integer.
