nn::ec::CTR::QuantityItemRight Class

Syntax

class QuantityItemRight

Description

Class representing a usage-based license of a service item.

Get this using ServiceTitle::ListLimitedRsourceItemRights.

This corresponds to an ECItemRights object of type EC_ITEM_QUANTITY in the ECSV library.

Member Functions

GetReferenceId Gets the reference ID.
GetLimit Gets the number of uses.
