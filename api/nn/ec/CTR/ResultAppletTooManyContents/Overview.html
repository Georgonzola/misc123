nn::ec::CTR::ResultAppletTooManyContents Class

Syntax

class ResultAppletTooManyContents

Description

The maximum number of content instances that can be saved in a data title has been exceeded. Content cannot be purchased or downloaded until some existing content is deleted.

A maximum of 2,000 pieces of content can be imported per data title. This result is returned when trying to import more than 2,000 pieces of content.
