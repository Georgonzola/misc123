nn::ec::CTR::ResultError::NeedsShowErrorCode Member Function

Syntax

bool NeedsShowErrorCode() const;

Parameters

None.

Return Values

Returns true if the error code must be displayed, and returns false otherwise.

Description

Determines whether the error necessitates displaying the error code.

When this function returns true, you must call ResultError::GetErrorCode to get the error code, and then display it using the Error/EULA applet.

For more information about how to use the Error/EULA applet, see the erreula library reference materials.
