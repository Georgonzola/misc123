nn::ec::CTR::ResultFsMediaAccessFailedError Class

Syntax

class ResultFsMediaAccessFailedError

Description

Media file system access failed.

This error is normally corrected during development. It may occur in cases such as when the SD card is inserted and then removed, or if it is damaged.

You must display the error code using the Error/EULA applet. The ResultError::NeedsShowErrorCode function returns true for ResultError instances wrapping this Result.
