nn::ec::CTR::Server::GetDateTime Member Function

Syntax

ResultError GetDateTime(
     fnd::DateTime * pOut
);

Parameters

Name Description
out pOut Buffer for getting the time.

Return Values

Returns the execution result of the function. The results returned by the ResultError::GetResult function are shown below.

Value Description
Result::IsSuccess Indicates that the operation succeeded.
ResultSessionInvalid Invalid session.
ResultNewSessionRequired A new session is required.
ResultOutOfSystemMemoryError Insufficient system memory.
ResultAcNotConnectedError Either not connected to the Internet, or the connection was lost.
ResultInfraNeedsReconnect Infrastructure reconnection requested.
ResultInfraReportError Infrastructure reported an error.
ResultInfraReceiveError Failed to receive from the infrastructure connection.
ResultInfraInvalidResponseError Invalid response from the infrastructure connection.
ResultInfraNeedsUpdateError Infrastructure updated, so the system must be updated.
ResultInfraTimeoutError Infrastructure communication timeout.
ResultInfraBusyError The infrastructure connection is busy.

Description

Gets the server time.

The DateTime acquired with this function is in Coordinated Universal Time (UTC).

This function requires an Internet connection because it communicates with the Nintendo eShop server.

The value obtained by this function is not used to show the time to the user, but rather for comparison purposes with the new-arrival reference date of the item. Frequently calling the function puts a heavy load on the Nintendo eShop server. Only call it when necessary, such as when updating item lists.
