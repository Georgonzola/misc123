nn::ec::CTR::Server::Initialize Member Function

Syntax

ResultError Initialize(
     Session session
);

Parameters

Name Description
in session Specifies the Session object to use for initialization.

Return Values

Returns the execution result of the function. The results returned by the ResultError::GetResult function are shown below.

Value Description
Result::IsSuccess Indicates that the operation succeeded.
ResultSessionInvalid Invalid session.

Description

Initializes a Server class object. Initializes a Server class object. You must have an instance of the Session class that has been initialized using the EcApplet::RequestInitializeSession function. The other member functions of this class must not be called if this function does not return successfully.
