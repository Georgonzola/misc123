nn::ec::CTR::Server Class

Syntax

class Server : private nn::ec::CTR::detail::SessionHolder

Description

Class for getting information from the Nintendo eShop server. Using the Server class requires a Session object that has been initialized using the EcApplet::RequestInitializeSession function. After initializing the Server object, calling the EcApplet::RequestInitializeSession function again, such as to reinitialize, causes the old session to be destroyed, with any further calls to any object functions returning a ResultError that includes ResultSessionInvalid. In such cases, use the newly initialized Session to initialize a new instance of the Server class, and then use that object.
