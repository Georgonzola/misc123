nn::ec::CTR::ServiceItem::GetAttributeByName Member Function

Syntax

const char * GetAttributeByName(
     const char * name
) const;

Parameters

Name Description
in name NULL-terminated UTF-8 string that specifies the attribute name.

Return Values

Returns a pointer to a NULL-terminated UTF-8 string that indicates the attribute value.

Description

Gets a service item attribute value.

Gets a service item attribute value specified as a NULL-terminated UTF-8 string attribute name. Returns NULL if the specified attribute is not found.

The pointer to the attribute value is invalidated when the ServiceItemCatalog from which the ServiceItem was obtained is destroyed, or when the ServiceItemCatalog::Retrieve function is called.
