nn::ec::CTR::ServiceItem::GetLimit Member Function

Syntax

u32 GetLimit() const;

Parameters

None.

Return Values

Returns the value corresponding to the license type.

Description

Returns the expiration or number of uses for the service item.

The value returned by this interface is valid only if the license type of the service item is time-based or usage-based.

For usage-based licenses, the return value represents the number of usages.

For time-based licenses, the return value is the time passed since January 1, 1970 00:00:00 UTC indicated in seconds.

The value for a service item representing a permanent license is invalid and cannot be used.
