nn::ec::CTR::ServiceItem::IsPurchasable Member Function

Syntax

bool IsPurchasable() const;

Parameters

None.

Return Values

Returns true if the content is available for purchase, or false otherwise.

Description

Determines whether the service item is available for purchase.

As one determination of whether purchase is possible, checks whether the sales start date and time has been reached. To determine whether it has been reached, the standard time for each country is used.

Always returns true if ServiceItem objects were searched with SelectionFilter set to SELECTION_FILTER_PURCHASABLE.

This function also returns search hits for items set to "initial purchase only" that have already been purchased. Handle items set to initial purchase only on IMAS either on your independent server or in your application so that users do not mistakenly purchase them again (so that no purchase requests are sent to the EC applet).

You can check whether an item is set to initial purchase only using the IsInitialPurchaseOnly function.

For consumable tickets or time-based (subscription) pass service items that are set to initial purchase only and have already been purchased, the EcApplet::RequestPurchaseServiceItem function returns ResultInfraReportError if you attempt to purchase them again. (See the Note below.)

Note: This error is different from the ResultAppletAlreadyPurchased error (attempted to purchase an item that has already been purchased). The ResultAppletAlreadyPurchased error only occurs when attempting to purchase items such as DLC and permanent pass service items that would result in the user having overlapping items.
