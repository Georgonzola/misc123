nn::ec::CTR::ServiceItem Class

Syntax

class ServiceItem

Description

Class for getting service item information.

Use the ServiceItemCatalog::GetServiceItem function to get an instance of the ServiceItem class. The ServiceItem instance is invalidated when the ServiceItemCatalog instance from which it was obtained is destroyed, or when the ServiceItemCatalog::Retrieve function is called again.

Member Functions

GetTitleInfo Gets information about the data titles that include the service item.
IsPurchasable Determines whether the service item is available for purchase.
GetPrice Gets the price information of a service item.
GetItemId Gets the item ID of a service item.
GetRating Gets rating information for a service item.
GetReleaseDate Gets the date a service item was released.
GetAttributeByName Gets a service item attribute value.
IsInitialPurchaseOnly Determines whether a service item can only be purchased one time.
GetMaxServiceDays Gets the maximum period until expiration that can be set for the time-based pass (subscription).
GetType Gets the license type for the service item.
GetLimit Returns the expiration or number of uses for the service item.
