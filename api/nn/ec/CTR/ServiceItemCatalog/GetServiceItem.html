nn::ec::CTR::ServiceItemCatalog::GetServiceItem Member Function

Syntax

ServiceItem GetServiceItem(
     u32 offset
) const;

Parameters

Name Description
in offset Offset of service item to get.

Return Values

Returns service item information.

Description

Gets service-item information.

Gets a ServiceItem from the service-item information retrieved by the ServiceItemCatalog::Retrieve function. Specify a value for offset to select a specific ServiceItem from the multiple service items obtained from the ServiceItemCatalog::Retrieve function. The offset value must be smaller than the number of service items obtained by the ServiceItemCatalog::GetNumRetrieved function. Operation is undefined when specifying a larger value for offset.

Note that the ServiceItem object is invalidated when the ServiceItemCatalog instance from which it was obtained is destroyed.
