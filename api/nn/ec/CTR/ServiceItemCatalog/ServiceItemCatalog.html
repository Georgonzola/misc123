nn::ec::CTR::ServiceItemCatalog::ServiceItemCatalog Constructor

Syntax

ServiceItemCatalog(
     Session session,
     void * pBuffer,
     size_t bufferSize
);

Parameters

Name Description
in session Specifies the Session object to use for initialization.
in pBuffer Pointer to the buffer to configure.
in bufferSize Size of the buffer to configure.

Description

Constructor for configuring the Session object and buffer.

The Session object must have been initialized by the EcApplet::RequestInitializeSession function. ServiceItemCatalog::Retrieve returns ResultOutOfFilterMemory if the buffer is too small for the filters that were set. Fix the problem by such means as increasing the size of the buffer or setting fewer filters.

The configured buffer is used when calling the ServiceItemCatalog::Retrieve function.
