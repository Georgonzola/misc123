nn::ec::CTR::ServiceItemCatalogFilter Class

Syntax

class ServiceItemCatalogFilter : public nn::ec::CTR::detail::CatalogFilterImpl<ServiceItemCatalogFilter>

Description

Filter for using ServiceItemCatalog to search for ServiceItem objects.

Search by passing this class to the ServiceItemCatalog::Retrieve function to filter the ServiceItem search results.

Member Functions

ServiceItemCatalogFilter Constructor for configuring a buffer for the filter.
SetReturnAttributes Configures the attribute name for the attribute value to get. Call the ServiceItemCatalog::Retrieve function using the filter configured with the attribute names to get the attribute values corresponding to the attribute names of matching ServiceItem objects. Use the ServiceItem::GetAttributeByName function to access the retrieved attribute value. You can specify multiple attribute names in an array. ServiceItemCatalogFilter only maintains references to the configured attribute name array and does not copy it, so be sure not to destroy this array until you are done using the configured filter.
SetAttributeFilters Configures the attribute filter.
SetTitleFilters Configures the title filter.
SetSelectionFilter Sets the search range. Specify SELECTION_FILTER_PURCHASABLE to only search content sets that are currently available for purchase. SELECTION_FILTER_PURCHASABLE is used as the default even if this function has not been called. Specify SELECTION_FILTER_ALL to search all items, including those that are not currently available for purchase such as those that have not yet come on sale, and those that are no longer being sold. Specify this value when you want to show a user items that will be for sale in future, or when you want to get information for items no longer for sale but that the user might want to download again.
