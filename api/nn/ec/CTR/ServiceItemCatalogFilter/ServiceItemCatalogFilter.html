nn::ec::CTR::ServiceItemCatalogFilter::ServiceItemCatalogFilter Constructor

Syntax

ServiceItemCatalogFilter(
     void * pBuffer,
     size_t bufferSize
);

Parameters

Name Description
in pBuffer Pointer to the buffer to configure.
in bufferSize Size of the buffer to configure.

Description

Constructor for configuring a buffer for the filter.

The configured buffer is used when calling the ServiceItemCatalog::Retrieve function. You need the same number of buffers as you have configured filters.

A buffer of a specific size is required, even if nothing is set for the filter.
