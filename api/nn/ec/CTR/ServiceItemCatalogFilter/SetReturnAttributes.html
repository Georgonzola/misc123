nn::ec::CTR::ServiceItemCatalogFilter::SetReturnAttributes Member Function

Syntax

ServiceItemCatalogFilter & SetReturnAttributes(
     const AttributeName names[],
     size_t numNames
);

Parameters

Name Description
in names[] Array of attribute names to configure.
in numNames Number of attribute names to configure.

Return Values

Returns a reference to the filter set for the attribute name.

Description

Configures the attribute name for the attribute value to get. Call the ServiceItemCatalog::Retrieve function using the filter configured with the attribute names to get the attribute values corresponding to the attribute names of matching ServiceItem objects. Use the ServiceItem::GetAttributeByName function to access the retrieved attribute value. You can specify multiple attribute names in an array. ServiceItemCatalogFilter only maintains references to the configured attribute name array and does not copy it, so be sure not to destroy this array until you are done using the configured filter.
