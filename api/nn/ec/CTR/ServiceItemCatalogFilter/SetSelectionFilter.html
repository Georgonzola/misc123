nn::ec::CTR::ServiceItemCatalogFilter::SetSelectionFilter Member Function

Syntax

ServiceItemCatalogFilter & SetSelectionFilter(
     SelectionFilter selection
);

Parameters

Name Description
in selection Search range to set.

Return Values

Returns a reference to the filter set for the search range.

Description

Sets the search range. Specify SELECTION_FILTER_PURCHASABLE to only search content sets that are currently available for purchase. SELECTION_FILTER_PURCHASABLE is used as the default even if this function has not been called. Specify SELECTION_FILTER_ALL to search all items, including those that are not currently available for purchase such as those that have not yet come on sale, and those that are no longer being sold. Specify this value when you want to show a user items that will be for sale in future, or when you want to get information for items no longer for sale but that the user might want to download again.
