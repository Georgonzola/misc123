nn::ec::CTR::ServiceItemCatalogFilter::SetTitleFilters Member Function

Syntax

ServiceItemCatalogFilter & SetTitleFilters(
     const TitleInfo infos[],
     size_t numInfos
);

Parameters

Name Description
in infos[] Array of title filters to configure.
in numInfos Number of title filters to configure.

Return Values

Returns a reference to the filter set for the title filter.

Description

Configures the title filter.

Call the ServiceItemCatalog::Retrieve function using a filter configured with the title filters to get just the ServiceItem object included in the configured data title.

If no title filter is set, it retrieves all service titles that the application can purchase. You can specify multiple title filters in an array. Be sure not to destroy the title filter array until you finish using the configured filters, because ServiceItemCatalogFilter only maintains a reference to the configured title filter array and does not copy it.
