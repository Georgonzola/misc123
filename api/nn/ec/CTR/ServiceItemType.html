nn::ec::CTR::ServiceItemType Enumerated Type

Syntax

enum ServiceItemType;

Value

Value Description
SERVICE_ITEM_TYPE_PERMANENT A permanent license for a service item.
SERVICE_ITEM_TYPE_SUBSCRIPTION A time-based license (subscription) for a service item.
SERVICE_ITEM_TYPE_QUANTITY A usage-based license for a service item.
SERVICE_ITEM_TYPE_UNKNOWN Type used by the system. Do not use.

Description

An enumerated type representing service-item license types.
