nn::ec::CTR::ServiceTitle::GetTicketEnvelopeSize Member Function

Syntax

ResultError GetTicketEnvelopeSize(
     size_t * pOut,
     size_t challengeSize
) const;

Parameters

Name Description
out pOut Buffer for getting the size of the ticket envelope.
in challengeSize Size of the challenge.

Return Values

Returns the execution result of the function. The results returned by the ResultError::GetResult function are shown below.

Value Description
Result::IsSuccess Indicates that the operation succeeded.
ResultNotSupported The specified challenge size is not supported.

Description

Gets the size of the ticket envelope. The size of the ticket envelope scales with the number of items purchased by the user.
