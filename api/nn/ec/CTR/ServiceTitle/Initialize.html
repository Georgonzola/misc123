nn::ec::CTR::ServiceTitle::Initialize Member Function

Syntax

ResultError Initialize(
     const TitleInfo & title
);

Parameters

Name Description
in title Title information for which to get licenses.

Return Values

Returns the execution result of the function. For more information about how to handle ResultError objects, see the ResultError reference. The results returned by the ResultError::GetResult function are shown below.

Value Description
Result::IsSuccess Indicates that the operation succeeded.
ResultServiceTitleNotOwned The specified data title has never been initialized.

Description

Initializes an instance of the ServiceTitle class.

Specifies a service title to get the license information, and initializes it. This function must be called before you use other member functions of the ServiceTitle class.

This function returns ResultServiceTitleNotOwned if the service title item has never been purchased.
