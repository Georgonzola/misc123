nn::ec::CTR::ServiceTitle Class

Syntax

class ServiceTitle

Description

Class for getting license information for a service title.

You do not need a Session object to initialize a ServiceTitle. You can use it without calling the EcApplet::RequestInitializeSession function so long as the license of the data title is available locally. Invoking EcApplet::RequestInitializeSession may delete local licenses, because it synchronizes local licenses with the eShop server. Create a new ServiceTitle instance after invoking EcApplet::RequestInitializeSession.

Member Functions

Initialize Initializes an instance of the ServiceTitle class.
GetNumPermanentItemRights Returns the number of permanent licenses that can be obtained from the service title.
GetNumSubscriptionItemRights Returns the number of time-limited licenses (subscriptions) that can be obtained from the service title.
GetNumQuantityItemRights Returns the number of usage-limited licenses that can be obtained from the service title.
ListPermanentItemRights Provides a list of permanent licenses from the service title.
ListSubscriptionItemRights Provides a list of time-based licenses (subscriptions) from the service title.
ListQuantityItemRights Provides a list of usage-based licenses from the service title.
GetTicketEnvelopeSize Gets the size of the ticket envelope. The size of the ticket envelope scales with the number of items purchased by the user.
GetTicketEnvelope Returns a ticket envelope. Use ServiceTitle::GetTicketEnvelopeSize to get the size of the ticket envelope.
