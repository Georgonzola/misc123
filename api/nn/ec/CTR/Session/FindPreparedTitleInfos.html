nn::ec::CTR::Session::FindPreparedTitleInfos Member Function

Syntax

Result FindPreparedTitleInfos(
     PreparedTitleInfo * outInfos,
     const TitleInfo infoList[],
     size_t numInfos
) const;

Parameters

Name Description
out outInfos Buffer for getting data title information.
in infoList[] List of title information for data titles to search.
in numInfos Number of data title information items to get.

Return Values

Returns the execution result of the function.

Value Description
Result::IsSuccess Indicates that the operation succeeded.
ResultSessionInvalid Invalid session. Use the EcApplet::RequestInitializeSession function to initialize a new session.
ResultSessionPreparedTitleNotFound No data title information was found for the specified title.

Description

Searches information for data titles that have been initialized this session. If service item information is passed to the EcApplet::RequestInitializeSession function, that information is included.
