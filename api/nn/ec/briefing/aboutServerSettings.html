ECDK: Configuring Access to the Development Server

For development using ECDK you need to have access to the development server.

DNS Configuration

To connect to the development server, you must configure special DNS settings for the development hardware.

To configure the DNS settings, use the Internet Settings in the System Settings of the development hardware.

DNS Address

Specify 205.166.76.187.

The DNS address may change in the future. If it is changed, refer to Nintendo's support page and update to the latest DNS address.

If You Cannot Connect

Contact Nintendo if the connection fails even when your network configuration is correct.

Using BOSS and ECDK at the Same Time

This address can also be used when developing applications using the BOSS library.

Note About Changing the DNS

If you are using the ECDK with a configured DNS address for development, an error occurs when you initialize the system and delete the Nintendo Network ID if you have not set the DNS to this specific address.

If error code 005-4701 occurs, set the DNS to the this address and then initialize the system and delete the Nintendo Network ID.
