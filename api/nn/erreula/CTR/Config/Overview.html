nn::erreula::CTR::Config Structure

Syntax

struct Config
{
   ErrorType errorType;
   s32 errorCode;
   s16 upperScreenFlag;
   u16 useLanguage;
   wchar_t errorText[ERROR_TEXT_LENGTH_MAX];
   bool homeButton;
   bool softwareReset;
   bool appJump;
   ReturnCode returnCode;
   u16 eulaVersion;
};

Description

The structure that is exchanged between the application and the error EULA applet that stores data.

You must always call the InitializeConfig function to initialize this structure before setting its member variables.

Member Variables

: Enter
errorType ErrorType The type of applet to call.
errorCode s32 An error code.
upperScreenFlag s16 Flag indicating the display method for the upper screen. Invalid setting. The stereoscopic view of the upper screen is inherited from application status, so this does nothing even if specified.
useLanguage u16 The language to use (from the UseLanguage enumerated type). If a language that cannot be specified in System Settings is specified here, Japanese is assumed for systems with a system market of Japan, and English is assumed for systems with a system market is the Americas or Europe.
errorText wchar_t Freely defined text string for displaying an error. Up to 1900 characters including the NULL terminator.
homeButton bool Specifies whether the error EULA applet exits when the HOME Button is pressed. The error EULA applet exits if true is specified and the HOME Button is pressed.
softwareReset bool Specifies whether to use software reset. The error EULA applet exits if true is specified and the button combination for software reset is pressed.
appJump bool After the error is displayed, if true is set for whether to display a dialog box and prompt an application jump to System Settings, a dialog box for guiding the user to System Settings is displayed. This just displays the message. The application must perform the application jump. This function only works when the ErrorType is ERROR_TYPE_ERROR_CODE or ERROR_TYPE_ERROR_CODE_LANGUAGE.
Output
returnCode ReturnCode Return code. Stores the return code that indicates the reason why the Error/EULA applet terminated.
eulaVersion u16 The EULA version that the end user has agreed to.
