nn::erreula::CTR::ErrorType Enumerated Type

Syntax

enum ErrorType;

Value

Value Description
ERROR_TYPE_ERROR_CODE Network error. Displays the infrastructure communications-related error message corresponding to the error code.
ERROR_TYPE_ERROR_TEXT System error. Displays free text made using the UI of this applet.
ERROR_TYPE_EULA Displays the EULA.
ERROR_TYPE_EULA_FIRST_BOOT Used for system settings. (Use is prohibited.)
ERROR_TYPE_EULA_DRAW_ONLY Used for system settings. (Use is prohibited.)
ERROR_TYPE_AGREE Used for system settings. (Use is prohibited.)
ERROR_TYPE_ERROR_CODE_LANGUAGE Displays a network error message in the specified display language.
ERROR_TYPE_ERROR_TEXT_LANGUAGE Displays a custom error message in the specified display language.
ERROR_TYPE_EULA_LANGUAGE Displays the EULA in the specified display language.
ERROR_TYPE_ERROR_TEXT_WORD_WRAP Displays a custom error message with automatic line wrapping. For more information, see the description below of the automatic line-wrapping feature.
ERROR_TYPE_ERROR_TEXT_LANGUAGE_WORD_WRAP Displays a custom error message in the specified display language with automatic line wrapping. For more information, see the description below of the automatic line-wrapping feature.
ERROR_TYPE_MAX_BIT Enumerator used during internal processing. This enumerator cannot be used by applications.)

Description

Enumerates error codes.

An automatic line-wrapping feature is provided for automatically handling newlines when the width of the message being displayed extends beyond the length of a single line. Wrapping is performed inside the message at a half-width space (U+0020) or full-width space (U+3000) position. Note, however, that it is necessary to insert returns ahead of time for long URLs and other text whose display width extends beyond the length of a line without including any returns or spaces, because the automatic wrap feature does not work with such messages. Use a no-break space (U+00A0) for spaces where you do not want a carriage return inserted.
