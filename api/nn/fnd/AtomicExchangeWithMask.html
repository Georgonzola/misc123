nn::fnd::AtomicExchangeWithMask Function

Syntax

#include <nn/fnd.h>
template <typename Int>Int AtomicExchangeWithMask(
     volatile Int * p,
     Int value,
     Int mask
);

Template Arguments

Name Description
T The target type. T must be an integer (8-bit, 16-bit, 32-bit, or 64-bit).

Parameters

Name Description
in p Pointer to variables in memory that are subject to atomic operations.
in value Value to assign.
in mask Specifies the mask value.

Return Values

Returns the value of *p immediately prior to assignment masked with the value in mask.

Description

Replaces values atomically with the masks specified.

Atomically, for only the bits set in the mask, assigns the value of value to *p and returns the value of *p just prior to the assignment masked with the value of mask. No changes occur in the bits that are not set in the mask.
