nn::fnd::AtomicFetchXor Function

Syntax

#include <nn/fnd.h>
template <typename Int>Int AtomicFetchXor(
     volatile Int * p,
     Int operand
);

Template Arguments

Name Description
T The target type. T must be an integer (8-bit, 16-bit, 32-bit, or 64-bit).

Parameters

Name Description
in p Pointer to variables in memory that are subject to atomic operations.
in operand Specifies the operand.

Return Values

Returns the value of *p prior to operations.

Description

Atomically takes the bitwise XOR of the specified values.

Atomically calculates *p ^= operand and returns the value of *p prior to the calculation.

Do not step through this function with a debugger. Doing so could result in an infinite loop within the function. If you inadvertently step into the function, get out of function by stepping back out.
