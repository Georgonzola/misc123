nn::fnd::AtomicStoreWithMask Function

Syntax

#include <nn/fnd.h>
template <typename Int>void AtomicStoreWithMask(
     volatile Int * p,
     Int value,
     Int mask
);

Template Arguments

Name Description
T The target type. T must be an integer (8-bit, 16-bit, 32-bit, or 64-bit).

Parameters

Name Description
in p Pointer to variables in memory that are subject to atomic operations.
in value Value to assign.
in mask Specifies the mask value.

Return Values

None.

Description

Atomically assigns values with a mask specified.

Atomically assigns the value of value to *p for only the bits that are set in mask. No changes occur in the bits that are not set in the mask. It works the same way as when you ignore the return value of AtomicExchangeWithMask(p, value, mask).

Do not step through this function with a debugger. Doing so could result in an infinite loop within the function. If you inadvertently step into the function, get out of function by stepping back out.
