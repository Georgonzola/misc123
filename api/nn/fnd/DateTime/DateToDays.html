nn::fnd::DateTime::DateToDays Member Function

Syntax

#include <nn/fnd.h>
static s32 DateToDays(
     s32 year,
     s32 month,
     s32 day
);

Parameters

Name Description
in year Specifies the year.
in month Specifies the month.
in day Specifies the day.

Return Values

Returns the number of elapsed days. Be sure to specify a valid date. The results are indeterminate if an invalid date is specified. You can use IsValidDate to check whether the specified date is valid.

Description

Calculates the number of days elapsed from 2000/01/01 until the specified date.
