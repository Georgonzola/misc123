nn::fnd::DateTime Class

Header File: nn/fnd.h

Syntax

class DateTime

Description

Indicates a date and time. Can handle dates and times from 1900/01/01 00:00:00.000 to 2189/12/31 23:59:59.999. The calendar is Gregorian, calculated from 2000/01/01 (Saturday). Each day is treated as having exactly 86,400 seconds. The DateTime class provides addition and subtraction with the TimeSpan class. You can get a DateTime object as the sum or difference of a DateTime and TimeSpan object. You can get a TimeSpan object as the difference between two DateTime objects. The upper and lower limits for DateTime are configured to prevent overflows in the TimeSpan object obtained as the difference between two DateTime objects.
