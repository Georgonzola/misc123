nn::fnd::ExpHeapBase::AllocationMode Enumerated Type

Syntax

enum AllocationMode;

Value

Value Description
ALLOCATION_MODE_FIRST_FIT Allocates a memory block from the first free region that is at least the size of the memory block to allocate.
ALLOCATION_MODE_BEST_FIT Searches for a free region nearest in size to the memory block to allocate, and then allocates the memory block from this free region.

Description

Mode in which memory blocks are allocated.
