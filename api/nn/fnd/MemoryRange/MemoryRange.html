nn::fnd::MemoryRange::MemoryRange Constructor

Syntax

MemoryRange(
     uptr begin,
     uptr end
);

Parameters

Name Description
in begin Specifies the starting address.
in end Specifies the ending address. Represents the final address within the region, plus four bytes.

Description

Constructs an object that represents the range of a memory region.
