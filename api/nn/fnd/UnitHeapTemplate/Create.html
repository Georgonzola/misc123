nn::fnd::UnitHeapTemplate::Create Member Function

Syntax

static UnitHeapTemplate * Create(
     HeapBase * parent,
     size_t unit,
     void * addr,
     size_t size,
     s32 alignment = DEFAULT_ALIGNMENT,
     bit32 option = 0,
     bit32 placement = HEAP_INFOPLACEMENT_HEAD
);

Parameters

Name Description
in parent Specifies the parent heap whose region contains addr.
in unit The size of the unit.
in addr The starting address of the memory used by the unit heap.
in size The size of the memory used by the unit heap.
in alignment The alignment of the unit.
in option Specifies the options Not yet implemented.
in placement Specifies the location where the management region (UnitHeap object) is located.

Return Values

Description

Creates a unit heap within a heap.

Heaps that are created with this function must be destroyed using HeapBase::Destroy. The actual UnitHeap object is created within the region defined by the values specified for the addr and size parameters. The actual size allocated to the heap is the size of the specified region minus sizeof(UnitHeap).

Use the placement parameter to specify where the UnitHeap object is placed. If you specify HEAP_INFOPLACEMENT_HEAD for placement, the actual UnitHeap object is placed at the start of the region (the region that starts at addr and ends at addr+sizeof(UnitHeap). If you specify HEAP_INFOPLACEMENT_TAIL for placement, the actual UnitHeap object is placed at the end of the region (the region that starts at addr+size-sizeof(UnitHeap) and ends at addr+size.
