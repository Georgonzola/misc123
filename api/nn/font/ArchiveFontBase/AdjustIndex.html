nn::font::ArchiveFontBase::AdjustIndex Member Function

Syntax

protected:
u16 AdjustIndex(
     u16 index
) const;

Parameters

Name Description
in index The glyph index to adjust.

Return Values

Returns the index after it has been corrected. The adjusted glyph index may be GLYPH_INDEX_NOT_FOUND even if the original glyph index was not.

Description

Adjusts a glyph index obtained from the CMAP block by the number of sheets that have not been loaded.

Use the unadjusted glyph index to get character width data from a CWDH block and the adjusted glyph index to extract a glyph image from a sheet.
