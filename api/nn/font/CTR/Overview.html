nn::font::CTR Namespace

Description

Namespace of the CTRfont library. (It can also be written without the CTR portion.)

Classes

nn::font::CTR::CharWriter A class that draws characters using the Font class.
nn::font::CTR::RectDrawer Class for drawing characters.
nn::font::CTR::TagProcessorBase A class that handles font resources that define groups of glyphs.
nn::font::CTR::TextWriterBase A class that draws strings using the nn::font::Font class.

Structures

nn::font::CTR::DispStringBuffer A class that manages a buffer for displaying strings.
nn::font::CTR::ColorBufferInfo Structure that holds color buffer information.
