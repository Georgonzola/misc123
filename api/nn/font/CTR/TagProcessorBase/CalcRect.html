nn::font::CTR::TagProcessorBase::CalcRect Member Function

Syntax

virtual Operation CalcRect(
     util::Rect * pRect,
     u16 code,
     PrintContext< CharType > * pContext
);

Parameters

Name Description
out pRect Pointer to the rectangle structure that stores the range of influence of tags.
in code Code of the control character that was taken as the starting point of tag processing. (0x0000 - 0x001F).
inout pContext Pointer to the drawing status information for a string.

Return Values

Returns the processes for the caller (TextWriter) to carry out. For a particular tag, the value returned here is normally the same as the value returned by the Process function.

Description

Calculates the range of influence of tags.
