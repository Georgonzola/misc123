nn::font::CTR::TagProcessorBase::Operation Enumerated Type

Syntax

enum Operation;

Value

Value Description
OPERATION_DEFAULT Do not leave space before the next character at the start of a line. Leave space before the next character anywhere except at the start of a line.
OPERATION_NO_CHAR_SPACE Do not leave space before the next character when a tag is processed somewhere other than at the start of a line. (The default functionality is to leave space.)
OPERATION_CHAR_SPACE Leave space before the next character when a tag is processed at the start of a line. (The default functionality is to not leave space.)
OPERATION_NEXT_LINE Adjust x-coordinates during a line break, but skip all other required processing. Do not adjust y-coordinates. Also, do not leave space before the next character.
OPERATION_END_DRAW Use this to stop the drawing of a string midway.

Description

An enumerated type for processing that must be carried out by the caller (TextWriter).
