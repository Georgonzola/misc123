nn::font::PackedFont::LockSheet Member Function

Syntax

bool LockSheet(
     CharCode c
);

Parameters

Name Description
in c Character code indicating the glyph included on the sheet to be locked.

Return Values

Returns true if it was locked or false if it could not be locked.
Locking cannot be performed once the maximum number of lockable sheets has been exceeded.

Description

Prevents sheets in the cache from being discarded.
