nn::font::ResFont Class

Syntax

class ResFont : public nn::font::ResFontBase

Description

A class that implements operations on font resources.

Member Constants

S BUFFER_ALIGNMENT const s32 Specifies the number of bytes that the buffer address specified in the SetDrawBuffer function must be aligned to. Defined as 4 bytes.
