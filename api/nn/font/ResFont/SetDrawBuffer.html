nn::font::ResFont::SetDrawBuffer Member Function

Syntax

void * SetDrawBuffer(
     void * buffer
);

Parameters

Name Description
in buffer Pointer to the buffer. Must be BUFFER_ALIGNMENT-byte aligned (4-byte aligned).

Return Values

Returns the previously configured contents. (The initial value is NULL).

Description

Sets the buffer required for drawing. The size of the buffer being set is obtained with the GetDrawBufferSize() function.

If you specify NULL, the drawing buffer returns to the unused state.
