Description of GetFriendMii ( nn::mii::StoreData *, const FriendKey *, size_t )

If there is no friend corresponding to the key passed in the arguments, the face library determines the Mii to be invalid. The function still returns success in this case.

To use the obtained Mii data, the separate Mii Face Library is required.
