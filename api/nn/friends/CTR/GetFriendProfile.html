Description of GetFriendProfile ( Profile *, const FriendKey *, size_t )

Returns empty profile information in this buffer if there is no friend corresponding to the key passed in the arguments. The function also returns empty profile information if the peer has never published its online status. The function itself still returns success in this case.

The platform code of empty profile information is always 0. If you cannot determine whether a value by itself is valid (for example, a region code), always use the platform code to determine validity.
