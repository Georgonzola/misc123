nn::friends::CTR::GetMyMii Function

Syntax

Result GetMyMii(
     nn::mii::StoreData * pMiiData
);

Parameters

Name Description
out pMiiData Specifies a pointer to a structure storing the local host's Mii data.

Return Values

Returns the result of the operation.

Value Description
Result::IsSuccess Indicates success.
ResultInvalidPointer An invalid pointer was passed as an argument.
ResultNotInitialized The friend presence library is not initialized.
Other Other results can be returned depending on the internal state of the daemon.

Description

Gets local Mii data.

If a Personal Mii is not configured in Mii Maker, the face library determines the Mii returned by this function to be invalid. The function still returns success in this case.

To use the obtained Mii data, the separate Mii Face Library is required.
