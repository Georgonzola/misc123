nn::friends::CTR::IsInitialized Function

Syntax

bool IsInitialized();

Parameters

None.

Return Values

Returns true if the friend presence library is initialized; otherwise returns false.

Value Description
true The friend presence library is initialized.
false The friend presence library is not initialized.

Description

Checks whether the friend presence library is initialized.
