nn::friends::CTR::NotificationType Enumerated Type

Syntax

enum NotificationType;

Value

Value Description
NOTIFICATION_NONE Invalid notification.
NOTIFICATION_ONLINE Indicates that the local host is currently online.
NOTIFICATION_OFFLINE Indicates that the local host is currently offline.
NOTIFICATION_FRIEND_ONLINE Indicates that a friend is currently online.
NOTIFICATION_FRIEND_PRESENCE Indicates that a friend's presence status has changed.
NOTIFICATION_FRIEND_MII Indicates that a friend's Mii has changed.
NOTIFICATION_FRIEND_PROFILE Indicates that a friend's profile has changed.
NOTIFICATION_FRIEND_OFFLINE Indicates that a friend is currently offline.
NOTIFICATION_BECOME_FRIEND Indicates that the local host has established a friend relationship with a friend.
NOTIFICATION_INVITATION Indicates that an invitation from a friend has arrived.

Description

Enumerated type representing notification types.
