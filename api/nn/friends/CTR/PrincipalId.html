nn::friends::CTR::PrincipalId Type

Syntax

typedef ::nnfriendsPrincipalId nn::friends::CTR::PrincipalId;

Description

Represents a principal ID.

Do not use the principal ID as an ID to show to the user. Use nn::cfg::CTR::GetTransferableId to get the Transferable ID to use as an ID to show to the user.

Revision History

2013/07/05
Indicated that this must not be used as an ID to show to the user.
