nn::friends::CTR::UpdateGameMode Function

Syntax

Result UpdateGameMode(
     const GameMode & gameMode,
     const char16 description[MODE_DESCRIPTION_SIZE]
);

Parameters

Name Description
in gameMode Specifies a game mode structure.
in description[MODE_DESCRIPTION_SIZE] Specifies a game mode description string.

Return Values

Returns the result of the operation.

Description

Updates the game mode.

Game mode information configured in this function is sent to friends when you are online, and is reflected in their friend lists.

Specify your own local system's online group participation status in the game mode. Parameters set here are shared among applications for which the same join-in game ID is specified. The friend list determines which games can be joined according to fixed logic based on the game mode parameter set here. Note that a friend list may erroneously display that join-in is possible if an inappropriate parameter is specified. For more information about how to use this feature, see the 3DS Programming Manual: Wireless and NEX Programming Manual: Server Services.

The value specified in icon data is used for the value of the join-in game ID and the value specified here is ignored. For information about specifying a join-in game ID in icon data, see the ctr_makebanner manual.

For the string configured in this function to be applied to the friend list and for the join-in game ID specified in the icon data to take effect, the application must be started from the HOME Menu or the development menu. Note that the string is not configured correctly if the application is started from the debugger software using PARTNER-CTR Debugger or IS-SNAKE-BOX. When an SD card is inserted in the system, the HOME Menu caches icon data on the SD card and therefore updates to that data sometimes do not take effect. In this case, delete the cached icon data as described in the HOME Menu Manual.

The size of the game mode description string that is passed as an argument can be as large as nn::friends::CTR::MODE_DESCRIPTION_SIZE characters, including newline and terminating characters, but the friend list is limited to the display of two lines of up to 16 double-byte characters. Anything exceeding that limit is not shown.

The strings that are displayed on the friends lists of other players have the following relationship with the system region.
Japan, the Americas, and Europe: The characters included in the Japanese, US, and Europe character sets for built-in fonts. China: Simplified Chinese characters included in the system fonts. Korea: Hangul characters included in the system fonts. Taiwan: Traditional Chinese characters included in the system fonts. Characters that are not included here are replaced with the Nintendo extended character "?" (L'\xE011') for display. For information about the character sets included in each font, see the 3DS Overview.

For newlines, use L'
'.

Do not call this function more frequently than the guidelines established in 3DS Guidelines, in the Information Update Frequency section.
