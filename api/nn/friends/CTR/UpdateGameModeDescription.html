nn::friends::CTR::UpdateGameModeDescription Function

Syntax

Result UpdateGameModeDescription(
     const char16 description[MODE_DESCRIPTION_SIZE]
);

Parameters

Name Description
in description[MODE_DESCRIPTION_SIZE] Specifies a game mode description string.

Return Values

Returns the result of the operation.

Value Description
Result::IsSuccess The process succeeded.
ResultRmcNotCalled The process succeeded. Due to connection status and limitations of frequency of sending data, however, notification to the server and friends is delayed.
ResultInvalidPointer Invalid pointer passed as an argument.
ResultNotInitialized The friend presence library is not initialized.
Other Another result may be returned, depending on the internal state of the daemon and other factors.

Description

Updates the game mode description string.

This string is sent to friends when you are online and is displayed in their friend lists.

To simply display the string in friend lists, you do not need to call the nn::friends::CTR::Login function in advance. If the string is updated while the system is offline, it is sent automatically by the presence feature in a timely way as soon as the system can go online, provided that background operations have not been suspended by the nn::ndm::Suspend, nn::ndm::SuspendDaemons, or nn::ndm::SuspendScheduler function.

To apply the string set with this function in friend lists, you must start the application from either the HOME Menu or the development menu. Note that it will not be set correctly if the application is started from the debugger software using the PARTNER-CTR Debugger or IS-SNAKE-BOX. The size of the string that is passed to the argument can be as large as nn::friends::CTR::MODE_DESCRIPTION_SIZE characters, including newline and terminating characters, but the friend list is limited to the display of 16 double-byte characters x 2 lines, and anything exceeding that is cut off and not displayed.

The strings that are displayed on the friend lists of other players have the following relationship with the system region.

Japan, the Americas, and Europe: The characters for Japan, the Americas, and Europe are included in the system fonts.

China: Simplified Chinese characters included in the system fonts.

Korea:Hangul characters included in the system fonts.

Taiwan: Traditional Chinese characters included in the system fonts.

Characters that are not included here are replaced with the Nintendo extended character "?" (L'\xE011') for display For details about the character sets included in each font, see 3DS Overview.

For newlines, use L'\n'.

This function does not change the game mode specified to the nn::friends::CTR::UpdateGameMode function.

Follow the rules set in Information Update Frequency of the 3DS Guidelines when determining how often to call this function.
