nn::fs::CommitSaveDataAndSetSecureValue Function

Syntax

Result CommitSaveDataAndSetSecureValue(
     const char * archiveName,
     bit64 value
);

Parameters

Name Description
in archiveName The name of the save data archive.
in value The secure value to write to the secure region.

Return Values

Returns the execution result.

Value Description
Result::IsSuccess Indicates that the operation succeeded.
Other Failed to access the media with the target save data. Unmount the archive and end the series of processes.

Description

Commits the save data and writes the values to a secure region used with the save data rollback prevention feature. Commits the save data archive specified by archiveName and writes the values specified with value to the save data rollback prevention feature secure region that corresponds to the save data archive specified by archiveName. If the target save data is not a duplicate, it is not committed.

For more information, see Overview of the Save Data Rollback Prevention Feature.
