Overloaded Member Functions

CommitSaveDataAndSetSecureValue ( const char *, bit64 ) Commits the save data and writes the values to a secure region used with the save data rollback prevention feature. Commits the save data archive specified by archiveName and writes the values specified with value to the save data rollback prevention feature secure region that corresponds to the save data archive specified by archiveName. If the target save data is not a duplicate, it is not committed.
CommitSaveDataAndSetSecureValue ( const char *, bit64, bit64 ) Commits the save data and writes the values with a mask specified to a secure region used with the save data rollback prevention feature. Commits the save data archive specified by archiveName, masks the values specified with mask, and writes the values specified with value to the save data rollback prevention feature secure region that corresponds to the save data archive specified by archiveName. If the target save data is not a duplicate, it is not committed.
