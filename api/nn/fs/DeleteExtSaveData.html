nn::fs::DeleteExtSaveData Function

Syntax

Result DeleteExtSaveData(
     ExtSaveDataId id
);

Parameters

Name Description
in id Specifies the ID of the expanded save data to delete.

Return Values

Returns the execution result.

Value Description
Result::IsSuccess Indicates that the operation succeeded.
ResultNotFound Indicates that expanded save data was not found. Create it by calling the CreateExtSaveData function.
Confirm that ExtSaveData is correct during development.
ResultMediaNotFound An SD card could not be found or recognized.
ResultOperationDenied The operation failed before it could finish because the specified expanded save data is currently being mounted.
ResultWriteProtected The SD card is write-protected.
ResultMediaAccessError An error occurred while accessing the SD card because of some cause such as a loose connection.
ResultBadFormat The SD card has an invalid format. The SD card must be formatted.
Other An application-side bug, or an unexpected error.

Description

Deletes the expanded save data.

Include the expanded save data ID (specified in the RSF file) as an argument.
