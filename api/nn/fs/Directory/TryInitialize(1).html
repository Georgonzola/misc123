nn::fs::Directory::TryInitialize Member Function

Syntax

Result TryInitialize(
     const char * pathName
);

Parameters

Name Description
in pathName Path of the directory to open

Return Values

Returns the execution result.

Description

Tries to open the specified directory.

Opens the directory in the specified path. This function returns an error when this object already has opened a directory.

Note: This function uses about 1 KB of buffer space on the stack because it converts the path name from a multibyte string into a wide string. Pay careful attention to the stack size.
