nn::fs::FileInputStream::FileInputStream Constructor

Syntax

explicit FileInputStream(
     const char * pathName
);

Parameters

Name Description
in pathName Path of the file to open. (The path can include ASCII characters only.)

Description

A constructor that constructs an object and opens the specified file.

It opens the file at the specified path with read-only access and enables reading.

Note: This function uses about 1 KB of buffer space on the stack because it converts the path name from a multibyte string into a wide string. Pay careful attention to the stack size.
