nn::fs::FileInputStream::GetSize Member Function

Syntax

virtual s64 GetSize() const;

Parameters

None.

Return Values

Returns the file size.

Description

Gets the file size.

This function is implemented to show an error screen when errors occur internally, regardless of whether such errors are fatal. Control does not return from the function in this case. Use nn::fs::FileInputStream::TryGetSize on everything other than ROM archives.

For more information about error handling, see Handling Errors During File and Directory Operations.
