nn::fs::FileInputStream::Read Member Function

Syntax

virtual s32 Read(
     void * buffer,
     size_t size
);

Parameters

Name Description
in buffer Specifies a pointer to the buffer being copied to.
in size Specifies the maximum number of bytes to copy.

Return Values

Returns the actual number of bytes that were copied. Returns 0 if it reached the end of the file.

Description

Loads the specified amount of data from a file into a buffer.

Up to size bytes of data are read from the current file position and copied to the region at the address specified by buffer.

Do not access the memory of the specified buffer region (neither reading nor writing) during this function call (for example, from another thread). File read operations may be incorrect and the program could stop working in a future version of the SDK.

This function is implemented to show an error screen when errors occur internally, regardless of whether such errors are fatal. Control does not return from the function in this case. Use nn::fs::FileInputStream::TryRead on everything other than ROM archives.

For more information about error handling, see Handling Errors During File and Directory Operations.
