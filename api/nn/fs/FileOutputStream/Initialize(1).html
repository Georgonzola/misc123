nn::fs::FileOutputStream::Initialize Member Function

Syntax

void Initialize(
     const char * pathName,
     bool createIfNotExist
);

Parameters

Name Description
in pathName Path of the file to open.
in createIfNotExist Specifies whether to create a file if it does not exist at the specified path.

Return Values

None.

Description

Opens the specified file or files.

Opens the file in the specified path with write-only access and enables writing.
If the createIfNotExist parameter has been set to true and no file exists in the specified path, this function creates the file and opens it with write-only access.
You cannot call this function if this object already has a file open.

Note: This function uses about 1 KB of buffer space on the stack because it converts the path name from a multibyte string into a wide string. Pay careful attention to the stack size.

Important

For expanded save data, you must explicitly create a file using the nn::fs::TryCreateFile function before calling this function. Note: Specifying createIfNotExist=true for expanded save data causes an error.
For more information about expanded save data, see Archives.
