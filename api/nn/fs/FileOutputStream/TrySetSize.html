nn::fs::FileOutputStream::TrySetSize Member Function

Syntax

virtual Result TrySetSize(
     s64 size
);

Parameters

Name Description
in size Specifies the file size.

Return Values

Returns the execution result.

Description

Sets the file size.

Error handling differs, depending on the archive. For more information, see Handling Errors During File and Directory Operations.

Note 1

After the size of the file has been extended, if nothing has been written to that region, in principle the process of reading that region returns nn::fs::ResultVerificationFailed.

Note 2

This function cannot be called for expanded save data. The function returns an error if called in this way.
For more information about expanded save data, see Archives.
