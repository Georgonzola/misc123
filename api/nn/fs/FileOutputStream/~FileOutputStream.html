nn::fs::FileOutputStream::~FileOutputStream Destructor

Syntax

virtual ~FileOutputStream();

Parameters

None.

Description

Destructor.

If the file is open, this function closes it and destroys the object.

If this function is called while written data has not been flushed, a panic results except with Release builds. Be sure to correctly flush written data by calling the TryFlush function before calling this function.
