nn::fs::FileStream::Seek Member Function

Syntax

virtual void Seek(
     s64 position,
     PositionBase base
);

Parameters

Name Description
in position Specifies the position to change the access position to.
in base Specifies the base position used to change the access position.

Return Values

None.

Description

Changes the access position in the file.

Sets the access position to position, relative to base.

This function is implemented to show an error screen when errors occur internally, regardless of whether such errors are fatal. Control does not return from the function in this case. Use nn::fs::FileStream::TrySeek on everything other than ROM archives.

For more information about error handling, see Handling Errors During File and Directory Operations.
