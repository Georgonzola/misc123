Description of TryInitialize ( const wchar_t *, bit32 )

Opens the file at the specified path in the specified openMode.
If no file exists at the specified path, this function creates a file and opens it in write-only mode. You cannot call this function if this object already has a file open.
For more information about the openMode parameter, see OpenMode.

This function returns any error that occurs while it is opening the file.

Important

For expanded save data, you must explicitly create a file using the nn::fs::TryCreateFile function before calling this function. Note: Specifying OPEN_MODE_CREATE for openMode when using expanded save data causes an error.
For more information about expanded save data, see Archives.
