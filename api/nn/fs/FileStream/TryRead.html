nn::fs::FileStream::TryRead Member Function

Syntax

virtual Result TryRead(
     s32 * pOut,
     void * buffer,
     size_t size
);

Parameters

Name Description
out pOut Stores the number of bytes that were actually copied. The value 0 is stored if the function reaches the end of the file.
in buffer Specifies a pointer to the buffer being copied to.
in size Specifies the maximum number of bytes to copy.

Return Values

Returns the execution result.

Description

Loads the specified amount of data from a file into a buffer.

Up to size bytes of data are read from the current file position and copied to the region at the address specified by buffer.

Do not access the memory of the specified buffer region (neither reading nor writing) during this function call (for example, from another thread). File read operations may be incorrect and the program could stop working in a future version of the SDK.

Error handling differs, depending on the archive. For more information, see Handling Errors During File and Directory Operations.
