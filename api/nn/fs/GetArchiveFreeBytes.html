nn::fs::GetArchiveFreeBytes Function

Syntax

Result GetArchiveFreeBytes(
     s64 * pOut,
     const char * archiveName
);

Parameters

Name Description
out pOut Returns the amount of free archive space.
in archiveName Specifies the archive name.

Return Values

Returns the execution result.

Value Description
Result::IsSuccess Indicates that the operation succeeded.
ResultNotFound The specified archive has not been mounted.
You must not allow this error to occur in retail products.
Other An application-side bug, or an unexpected error.

Description

Gets the amount of free space in the archive. This function can only be used on save data archives. The save data region manages data with a fixed size of 512 bytes. As a result, this function's return value is always a multiple of 512 bytes.
When this function returns 1024, you can create two files of 512 bytes. If you create one file of 513 bytes, the remaining available space goes to 0.
