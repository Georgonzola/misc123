nn::fs::GetSdmcSize Function

Syntax

Result GetSdmcSize(
     s64 * pTotal,
     s64 * pFree
);

Parameters

Name Description
out pTotal Returns the total SD card capacity.
out pFree Returns the amount of free space on the SD card.

Return Values

Returns the execution result.

Value Description
Result::IsSuccess Indicates that the operation succeeded.
ResultNotFound An SD card could not be found or recognized. This error is also returned if an SD card is inserted but the file system on the card cannot be accessed.
Other An application-side bug, or an unexpected error.

Description

Gets the total capacity and amount of free space on an SD card.

Important

The API Reference for CTR-SDK 0.14.16 and prior versions stated that this function returns the errors ResultMediaNotFound, ResultMediaAccessError, and ResultBadFormat. The actual error returned by the function is only ResultNotFound.

This function returns ResultNotFound if it is called when there is no SD card, and when there is no archive mounted on the SD card. This is a limitation of this function. To avoid this limitation, call this function after first calling either the nn::fs::MountSdmc, nn::fs::MountSdmcWriteOnly or nn::fs::MountExtSaveData function.
