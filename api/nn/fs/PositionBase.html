nn::fs::PositionBase Enumerated Type

Syntax

enum PositionBase;

Value

Value Description
POSITION_BASE_BEGIN Specifies the starting position relative to the beginning of the file.
POSITION_BASE_CURRENT Specifies the starting position relative to the current position in the file.
POSITION_BASE_END Specifies the starting position relative to the end of the file.

Description

Specifies constants that represent the positions of file read/write operations.
