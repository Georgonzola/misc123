nn::fs::RenameDirectory Function

Syntax

void RenameDirectory(
     const char * currentPath,
     const char * newPath
);

Parameters

Name Description
in currentPath The path to rename.
in newPath The renamed path.

Return Values

None.

Description

Renames a directory.

Note: This function uses about 1 KB of buffer space on the stack because it converts the path name from a multibyte string into a wide string. Pay careful attention to the stack size.

This function is implemented to show an error screen when errors occur internally, regardless of whether such errors are fatal. Control does not return from the function in this case. Use nn::fs::TryRenameDirectory on everything other than ROM archives.

For more information about error handling, see Handling Errors During File and Directory Operations.
