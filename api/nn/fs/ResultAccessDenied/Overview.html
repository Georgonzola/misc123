nn::fs::ResultAccessDenied Class

Syntax

class ResultAccessDenied

Description

No access rights. You must not allow this error to occur in retail products.

It indicates that you do not have access privileges to the target archive or for the attempted operation.

This may occur if you have not entered the required privileges under AccessControlInfo in the RSF file. To determine whether you have entered the correct information, see ctr_makerom in the API Reference Manual.
