nn::fs::ResultInvalidArgument Class

Syntax

class ResultInvalidArgument

Description

Invalid parameter. You must not allow this error to occur in retail products.

This indicates that an invalid value was passed as an argument.
This also occurs when the string passed as a path name uses an invalid format or is too long.

For information about the restriction on character counts and other restrictions, see Archives: Character Restrictions.
