nn::fs::ResultOutOfResource Class

Syntax

class ResultOutOfResource

Description

Insufficient resources. You must not allow this error to occur in retail products.

This indicates that there were not enough filesystem resources to complete the attempted operation.
This occurs when too many archives are mounted at the same time, when too many files and directories are open at the same time, and in other such situations.
