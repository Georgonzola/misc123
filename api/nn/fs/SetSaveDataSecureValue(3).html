nn::fs::SetSaveDataSecureValue Function

Syntax

Result SetSaveDataSecureValue(
     const char * archiveName,
     bit64 value,
     bit64 mask
);

Parameters

Name Description
in archiveName The name of the save data archive.
in value The secure value to write to the secure region.
in mask The mask for the secure value.

Return Values

Returns the execution result.

Value Description
Result::IsSuccess Indicates that the operation succeeded.
Other Failed to access the media with the target save data. Unmount the archive and end the series of processes.

Description

Writes the specified value masked with the specified mask to a secure region for use with the save data rollback prevention feature. Masks the value specified by mask and writes the value specified with value to the save data rollback prevention feature secure region that corresponds to the save data archive specified by archiveName.

For more information, see Overview of the Save Data Rollback Prevention Feature.
