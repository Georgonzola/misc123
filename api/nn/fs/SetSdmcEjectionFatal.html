nn::fs::SetSdmcEjectionFatal Function

Syntax

void SetSdmcEjectionFatal();

Parameters

None.

Return Values

None.

Description

Configures the system to produce the fatal error screen when an SD card is not inserted.

Configures the system to produce the fatal error screen when an SD card is not inserted. Once you have called this function, you cannot clear this setting in the application.

If an SD card is not inserted when this function is called, the fatal error screen is displayed immediately. Calling this function at a point where you expect an SD card to be inserted can prevent any associated processes from being executed if an SD card is not actually inserted at that time.

Conversely, do not call this function when using processes that assume that no SD card is inserted when they are executed.

To make a successful call to this function, you must not have any SD card ejection handlers registered. In other words, you must either not call the RegisterSdmcEjectedEvent function at all, or make sure to call the UnregisterSdmcEjectedEvent function after calling the RegisterSdmcEjectedEvent function.

Finally, once you call this function, you can no longer call the RegisterSdmcEjectedEvent or UnregisterSdmcEjectedEvent functions.
