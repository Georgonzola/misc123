Description of TryCreateDirectory ( const wchar_t * )

This function returns any error that occurs while it is creating the directory.

Error handling will differ, depending on the archive. For more information, see Handling Errors During File and Directory Operations.

Description of TryCreateDirectory ( const char * )

This function returns any error that occurs while it is creating the directory.

Note: This function uses about 1 KB of buffer space on the stack because it converts the path name from a multibyte string into a wide string. Pay careful attention to the stack size.

Error handling will differ, depending on the archive. For more information, see Handling Errors During File and Directory Operations.
