nn::fs::TryRenameFile Function

Syntax

Result TryRenameFile(
     const wchar_t * currentPath,
     const wchar_t * newPath
);

Parameters

Name Description
in currentPath The path to rename.
in newPath The renamed path.

Return Values

Returns the result of the function.

Description

Renames a file.

This function returns any error that occurs while it is renaming the file.

Which Result values need handling differ depending on the archive. For more information, see Handling Errors During File and Directory Operations.
