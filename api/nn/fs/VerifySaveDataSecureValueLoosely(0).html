nn::fs::VerifySaveDataSecureValueLoosely Function

Syntax

bool VerifySaveDataSecureValueLoosely(
     bit64 value
);

Parameters

Name Description
in value The secure value to compare.

Return Values

Returns true if the result of verification is correct; otherwise returns false.

Description

This function verifies a secure value stored in the secure region for use with the loose save data rollback prevention feature.

This function verifies that the specified value matches the value stored in the secure region for use with the save data rollback prevention feature. This function compares the values using the rules for loose save data rollback prevention and returns true for systems that have just been repaired.

You cannot use this function in combination with an overload that takes the mask parameter of the SetSaveDataSecureValue function. Use this function with overloads that take only the value parameter.

For more information, see Overview of the Save Data Rollback Prevention Feature.
