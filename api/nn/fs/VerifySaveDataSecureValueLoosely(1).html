nn::fs::VerifySaveDataSecureValueLoosely Function

Syntax

Result VerifySaveDataSecureValueLoosely(
     bool * pOut,
     const char * archiveName,
     bit64 value
);

Parameters

Name Description
out pOut Stores true in *pOut if the result of verification is correct; otherwise stores false.
in archiveName The name of the save data archive.
in value The secure value to compare.

Return Values

Description

Verifies a masked value stored in a secure region for use with the save data rollback prevention feature.

This function verifies that the value specified with value matches the value in the secure region for use with the save data rollback prevention feature that corresponds with the save data archive specified with archiveName. This function compares the values using the rules for loose save data rollback prevention and returns true for systems that have just been repaired.

You cannot use this function in combination with an overload that takes the mask parameter of the SetSaveDataSecureValue function. Use this function with overloads that take only the value parameter.

For more information, see Overview of the Save Data Rollback Prevention Feature.
