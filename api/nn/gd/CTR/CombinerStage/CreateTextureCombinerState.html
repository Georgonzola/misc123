nn::gd::CTR::CombinerStage::CreateTextureCombinerState Member Function

Syntax

static nnResult CreateTextureCombinerState(
     const CombinerDescription * description,
     CombinerState ** textureCombinerState
);

Parameters

Name Description
in description Describes how to construct the texture combiner state.
out textureCombinerState Specifies the address storing the pointer to the generated texture combiner state object.

Return Values

Returns the execution result.

Value Description
Result::IsSuccess Indicates that the operation succeeded.
ResultNullParameter NULL was specified in an argument.
ResultOutOfMemory Cannot obtain memory for the state object.
ResultInvalidParameter SOURCE_PREVIOUS has been specified for both m_SrcRGB[0] and m_SrcAlpha[0] of CombinerDescription. This result is also returned if neither SOURCE_CONSTANT, SOURCE_PREVIOUS, nor SOURCE_PREVIOUS_BUFFER has been specified for m_SrcRGB[1-5] and m_SrcAlpha[1-5].

Description

Creates a texture combiner state object.
