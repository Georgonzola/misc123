nn::gd::CTR::CombinerStage Class

Syntax

class CombinerStage

Description

The texture combiner pipeline stage.

The texture combiner calculates the ultimate fragment color based on a variety of inputs.

Enumerated Types

UnitId The texture combiner unit numbers.
Source The enum values for SrcRgb and SrcAlpha. The default value for the texture combiner unit 0 is SOURCE_CONSTANT. The default value for the other units is SOURCE_PREVIOUS.
OperandRgb The enum values for OperandRgb. The default value is OPERAND_RGB_SRC_COLOR.
OperandAlpha The enum values for OperandAlpha. The default value is OPERAND_ALPHA_SRC_ALPHA.
CombineRgb The enum values for CombineRgb. The default value is COMBINE_RGB_REPLACE.
CombineAlpha The enum values for CombineAlpha. The default value is COMBINE_ALPHA_REPLACE.
Scale The enum values for Scale. The default value is SCALE_1.
BufferInput The enum values for BufferInput. The default value is INPUT_PREVIOUS_BUFFER.

Member Constants

S COUNT_TEXTURE_COMBINER_UNITS const u32 The number of texture combiner units.

Member Functions

S CreateTextureCombinerState Creates a texture combiner state object.
S ReleaseTextureCombinerState Releases a texture combiner state object.
S SetTextureCombinerState Sets a texture combiner state object to OutputStage.
S SetTextureCombinerUnitConstantColor Sets a constant color to the specified texture combiner unit. To use this constant color, you must define the combiner state source RGB and source alpha values in CombinerStage::SOURCE_CONSTANT. This function is an immediate function. (An immediate function directly generates commands and inserts them into the command buffer.)
