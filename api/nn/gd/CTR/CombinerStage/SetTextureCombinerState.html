nn::gd::CTR::CombinerStage::SetTextureCombinerState Member Function

Syntax

static nnResult SetTextureCombinerState(
     CombinerState * textureCombinerState
);

Parameters

Name Description
in textureCombinerState Specifies the pointer to the state object created by CreateTextureCombinerState.

Return Values

Returns the execution result.

Value Description
Result::IsSuccess Indicates that the operation succeeded.
ResultAlreadyReleased The specified object has already been released. Create an object using CreateTextureCombinerState.

Description

Sets a texture combiner state object to OutputStage.

If NULL is specified for textureCombinerState, the default combiner state object is set.
Fragment light is enabled or disabled for registers 0x1c6 and 0x8f depending on the fragment primary color and secondary color settings configured for m_SrcRGB and m_SrcAlpha.
Texture is enabled or disabled for register 0x80 depending on the texture color settings configured for m_SrcRGB and m_SrcAlpha.
