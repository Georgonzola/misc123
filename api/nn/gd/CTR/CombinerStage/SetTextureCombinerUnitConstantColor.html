nn::gd::CTR::CombinerStage::SetTextureCombinerUnitConstantColor Member Function

Syntax

static nnResult SetTextureCombinerUnitConstantColor(
     UnitId unit,
     u8 colorR,
     u8 colorG,
     u8 colorB,
     u8 colorA
);

Parameters

Name Description
in unit The unit number of the texture combiner for which constant color is being set.
in colorR The red color value.
in colorG The green color value.
in colorB The blue color value.
in colorA The alpha value.

Return Values

Returns the execution result.

Value Description
Result::IsSuccess Indicates that the operation succeeded.
ResultInvalidTextureUnitId The texture unit ID is invalid.

Description

Sets a constant color to the specified texture combiner unit. To use this constant color, you must define the combiner state source RGB and source alpha values in CombinerStage::SOURCE_CONSTANT. This function is an immediate function. (An immediate function directly generates commands and inserts them into the command buffer.)

The following format is used for the data inserted into the command buffer.
0x00000000 (R | (G << 8) | (B << 16) | (A << 24))
0x00000004 [Reserved]
