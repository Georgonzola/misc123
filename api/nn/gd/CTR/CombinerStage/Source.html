nn::gd::CTR::CombinerStage::Source Enumerated Type

Syntax

enum Source;

Value

Value Description
SOURCE_PRIMARY_COLOR The primary color.
SOURCE_FRAGMENT_PRIMARY_COLOR Primary color of the fragment light.
SOURCE_FRAGMENT_SECONDARY_COLOR Secondary color of the fragment light.
SOURCE_TEXTURE0 Texture unit 0.
SOURCE_TEXTURE1 Texture unit 1.
SOURCE_TEXTURE2 Texture unit 2.
SOURCE_TEXTURE3 Texture unit 3.
SOURCE_PREVIOUS_BUFFER Output of the combiner buffer in the previous step.
SOURCE_CONSTANT Constant color set with the gd::CombinerStage::SetTextureCombinerUnitConstantColor function.
SOURCE_PREVIOUS Output of the texture combiner in the previous step.
SOURCE_RESERVED_MAX_U32

Description

The enum values for SrcRgb and SrcAlpha. The default value for the texture combiner unit 0 is SOURCE_CONSTANT. The default value for the other units is SOURCE_PREVIOUS.
