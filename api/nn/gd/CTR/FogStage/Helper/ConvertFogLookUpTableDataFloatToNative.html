nn::gd::CTR::FogStage::Helper::ConvertFogLookUpTableDataFloatToNative Member Function

Syntax

static nnResult ConvertFogLookUpTableDataFloatToNative(
     const f32 * dataValue,
     const f32 * dataDelta,
     u32 countData,
     u32 *__restrict dest
);

Parameters

Name Description
in dataValue Specifies fog data. Set each component of table data within the range 0.0 through 1.0.
in dataDelta Specifies fog difference data. Set each component of table data within the range -1.0 through 1.0.
in countData Specifies the table size. dataValue and dataDelta must have the same table size.[1,128].
out dest Specifies the address storing converted data. The size is the size given by countData.

Return Values

Returns the execution result.

Value Description
Result::IsSuccess Indicates that the operation succeeded.
ResultNullParameter NULL was specified for dataValue, dataDelta, or dest.

Description

Converts float data in a fog lookup table to fixed point format.
