nn::gd::CTR::FogStage Class

Syntax

class FogStage

Description

The fog and gas stage.

Classes

nn::gd::CTR::FogStage::Helper Helper utility for lookup table conversions.

Enumerated Types

Mode Enumerates fog and gas modes. The default value is MODE_NONE.
GasColorLookUpTableInput This is an enum value for an entry in a gas shading table.
GasShadingDensitySource This is an enum value for shading density information.

Member Functions

S SetGasFogMode Sets the gas and fog mode.
S SetGasShadingDensity Sets density information for gas shading.
S SetFogZFlip Enables or disables flipping fog input values.
S SetFogColor This function sets the fog color.
S SetGasAttenuation Sets the density attenuation coefficient for input values to the fog table.
S SetGasAccumulationMax Sets the maximum density value allowed for the additive blend result.
S SetGasAutoAccumulation Enables or disables the auto-update feature for rendering gas density information. Automatically updates the maximum density value when enabled.
S SetGasLightXY Sets the intensity of planar shading for a gas light.
S SetGasLightZ Sets the intensity of line-of-sight shading for a gas light.
S SetGasLightColorLutInput Sets the input values for a shading lookup table.
S SetGasDeltaZ Sets DeltaZ for a gas.
S UploadFogLookUpTableNative Sets the fog lookup table. (Raw mode.) This function is an immediate function. (An immediate function directly generates commands and inserts them into the command buffer.)
S UploadFogLookUpTableFloat Sets the fog lookup table. This function is an immediate function. (An immediate function directly generates commands and inserts them into the command buffer.)
S UploadGasLookUpTableNative Sets the gas shading lookup table. (Raw mode.) This function is an immediate function. (An immediate function directly generates commands and inserts them into the command buffer.)
S UploadGasLookUpTableFloat Sets the gas shading lookup table. This function is an immediate function. (An immediate function directly generates commands and inserts them into the command buffer.)
