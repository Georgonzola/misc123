nn::gd::CTR::FogStage::SetGasAttenuation Member Function

Syntax

static void SetGasAttenuation(
     f32 attenuation
);

Parameters

Name Description
in attenuation Specifies the density attenuation coefficient.

Return Values

None.

Description

Sets the density attenuation coefficient for input values to the fog table.

The default value is 1.0f.
