nn::gd::CTR::FogStage::SetGasLightXY Member Function

Syntax

static void SetGasLightXY(
     u8 lightMinimum,
     u8 lightMaximum,
     u8 lightAttenuation
);

Parameters

Name Description
in lightMinimum Specifies a minimum intensity for controlling planar shading.
in lightMaximum Specifies a maximum intensity for controlling planar shading.
in lightAttenuation Specifies the attenuation applied to the density used for controlling planar shading.

Return Values

None.

Description

Sets the intensity of planar shading for a gas light.

The default value is all 0.
