nn::gd::CTR::FogStage::SetGasShadingDensity Member Function

Syntax

static void SetGasShadingDensity(
     GasShadingDensitySource shadingDensitySrc
);

Parameters

Name Description
in shadingDensitySrc Specifies the density information used for shading.

Return Values

None.

Description

Sets density information for gas shading.

The default value is GAS_SHADING_DENSITY_SOURCE_PLAIN.
