nn::gd::CTR::FogStage::UploadGasLookUpTableNative Member Function

Syntax

static nnResult UploadGasLookUpTableNative(
     u32 lutStartIndex,
     u32 * data,
     u32 countData
);

Parameters

Name Description
in lutStartIndex Specifies the index of the lookup table. The lookup table contains 16 data entries, so a value from 0 to 15 can be specified.
in data Specifies lookup table data after conversion.
in countData Specifies the size of the lookup table.

Return Values

Returns the execution result.

Value Description
Result::IsSuccess Indicates that the operation succeeded.
ResultNullParameter NULL is specified in data.
ResultInvalidParameter 0 was specified in countData.
ResultOutOfRangeParameter The sum of lutStartIndex + countData specifies a value greater than or equal to 16 (the maximum size of a gas lookup table).

Description

Sets the gas shading lookup table. (Raw mode.) This function is an immediate function. (An immediate function directly generates commands and inserts them into the command buffer.)
