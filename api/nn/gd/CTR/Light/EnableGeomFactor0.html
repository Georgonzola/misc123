nn::gd::CTR::Light::EnableGeomFactor0 Member Function

Syntax

void EnableGeomFactor0(
     gdBool value
);

Parameters

Name Description
in value If GD_TRUE is specified, Geometric Factor ０ is used. Not used if GD_FALSE is specified.

Return Values

None.

Description

Sets whether lights use Geometry Factor 0. This function is an immediate function. (An immediate function directly generates commands and inserts them into the command buffer.)

The following format is used for the data inserted into the command buffer.
0x00000000 X
0x00000004 [Reserved]
When bit [2:2] of X is 0, false. When this bit is 1, true.

The default value is GD_FALSE.
