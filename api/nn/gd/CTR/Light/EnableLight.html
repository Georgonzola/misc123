nn::gd::CTR::Light::EnableLight Member Function

Syntax

void EnableLight(
     gdBool value
);

Parameters

Name Description
in value If GD_TRUE is specified, lights are enabled. If GD_FALSE is specified, lights are disabled.

Return Values

None.

Description

Enables or disables each light source.

The default value is GD_FALSE.
