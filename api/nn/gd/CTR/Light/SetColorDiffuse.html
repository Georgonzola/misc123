nn::gd::CTR::Light::SetColorDiffuse Member Function

Syntax

void SetColorDiffuse(
     u8 R,
     u8 G,
     u8 B
);

Parameters

Name Description
in R Specifies red.
in G Specifies green.
in B Specifies blue.

Return Values

None.

Description

Sets the diffuse color of each light source. This function is an immediate function. (An immediate function directly generates commands and inserts them into the command buffer.)

The following format is used for the data inserted into the command buffer.
0x00000000 (B | (G << 10) | (R << 20))
0x00000004 [Reserved]

The default value is all 255.
