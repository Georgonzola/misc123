nn::gd::CTR::Light::SetDistanceAttenuationScaleBias Member Function

Syntax

void SetDistanceAttenuationScaleBias(
     f32 attenuationScale,
     f32 attenuationBias
);

Parameters

Name Description
in attenuationScale Specifies the scale for input values to the distance attenuation lookup table.
in attenuationBias Specifies the scale for input values to the distance attenuation lookup table.

Return Values

None.

Description

Sets the scale and bias for input values to the distance attenuation lookup table. This function is an immediate function. (An immediate function directly generates commands and inserts them into the command buffer.)

The following format is used for the data inserted into the command buffer.
0x00000000 scale
0x00000004 [Reserved]
0x00000008 bias
0x0000000c [Reserved]
scale and bias use a 20-bit floating point number format. This function converts from a 32-bit floating point format to a 20-bit floating point format.

The default value for both is 0.0f.
