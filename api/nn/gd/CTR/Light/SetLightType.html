nn::gd::CTR::Light::SetLightType Member Function

Syntax

void SetLightType(
     SourceType sourceType
);

Parameters

Name Description
in sourceType Specifies the light type.

Return Values

None.

Description

Configures the light type. (Specify SOURCE_TYPE_DIRECTIONAL or SOURCE_TYPE_POINT.) This function is an immediate function. (An immediate function directly generates commands and inserts them into the command buffer.)

The following format is used for the data inserted into the command buffer.
0x00000000 X
0x00000004 [Reserved]
When bit [0:0] of X is 0, sourceType is SOURCE_TYPE_POINT. When this bit is 1, sourceType is SOURCE_TYPE_DIRECTIONAL.

The default value is SOURCE_TYPE_POINT.
