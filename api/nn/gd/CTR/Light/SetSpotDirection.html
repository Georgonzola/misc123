nn::gd::CTR::Light::SetSpotDirection Member Function

Syntax

void SetSpotDirection(
     f32 X,
     f32 Y,
     f32 Z
);

Parameters

Name Description
in X Sets the x-coordinate of the spotlight.
in Y Sets the y-coordinate of the spotlight.
in Z Sets the z-coordinate of the spotlight.

Return Values

None.

Description

Sets the spotlight light direction. You must use SetLightType to set the light type to SOURCE_TYPE_POINT. This function is an immediate function. (An immediate function directly generates commands and inserts them into the command buffer.)

The following format is used for the data inserted into the command buffer.
0x00000000 (Y << 16) | X
0x00000004 [Reserved]
0x00000008 Z
0x0000000c [Reserved]
X, Y, and Z use a signed 13-bit fixed point number format with 11 bits for the fractional part. This function converts from 32-bit to a signed 13-bit fixed point value with an 11-bit fractional portion format.

The default value is all 0.0f.
