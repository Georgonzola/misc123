nn::gd::CTR::LightingStage::Helper::ConvertLookUpTableDataFloatToNative Member Function

Syntax

static nnResult ConvertLookUpTableDataFloatToNative(
     const f32 * valueData,
     const f32 * deltaData,
     u32 dataCount,
     u32 *__restrict destination
);

Parameters

Name Description
in valueData Sets the lookup table data.
in deltaData Sets the lookup table delta data.
in dataCount Specifies the number of data items in the lookup table.
out destination Specifies a pointer to the lookup table after conversion.

Return Values

Returns the result of the function.

Value Description
Result::IsSuccess Indicates that the operation succeeded.
ResultNullParameter NULL was specified for valueData, deltaData, or destination.

Description

Converts the format of lookup table data from floating point to fixed point.

The size of dataValue and deltaData table must be the same.
