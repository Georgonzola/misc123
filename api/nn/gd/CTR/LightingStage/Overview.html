nn::gd::CTR::LightingStage Class

Syntax

class LightingStage

Description

The fragment light stage sets the overall environment for lighting.

Configuration settings are made for the ambient lights of eight light sources.

Classes

nn::gd::CTR::LightingStage::Helper Converts the format of lookup table data from floating point to fixed point.
