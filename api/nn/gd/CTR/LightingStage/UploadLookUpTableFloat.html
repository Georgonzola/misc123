nn::gd::CTR::LightingStage::UploadLookUpTableFloat Member Function

Syntax

static nnResult UploadLookUpTableFloat(
     LightingStage::LookUpTableUploadId lutID,
     u32 lutStartIndex,
     const f32 * valueData,
     const f32 * deltaData,
     u32 dataCount
);

Parameters

Name Description
in lutID Specifies the type of lookup table to set.
in lutStartIndex Specifies the index of the lookup table to be set.
in valueData Specifies lookup table data.
in deltaData Specifies lookup table delta data.
in dataCount Specifies the number of data items in the lookup table.

Return Values

Returns the result of the function.

Value Description
Result::IsSuccess Indicates that the operation succeeded.
ResultNullParameter NULL was specified for valueData or deltadata.
ResultInvalidParameter 0 was specified in dataCount.
ResultOutOfRangeParameter The sum of lutStartIndex + dataCount specifies a value greater than or equal to 256 (the maximum size of a light lookup table.)

Description

Sets the lighting lookup table. The floating point value you specify is converted into fixed point data by this function and then sent to the GPU. This function is an immediate function. (An immediate function directly generates commands and inserts them into the command buffer.)
