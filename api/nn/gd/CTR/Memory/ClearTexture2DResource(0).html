nn::gd::CTR::Memory::ClearTexture2DResource Member Function

Syntax

static nnResult ClearTexture2DResource(
     const Texture2DResource * tex2DResource,
     s32 mipLevelIndexStart,
     s32 mipLevelIndexEnd,
     const u8 Components[4]
);

Parameters

Name Description
in tex2DResource Specifies a pointer to the Texture2DResource resource to be cleared. Only a VRAM resource can be specified.
in mipLevelIndexStart Specifies the index of the first mipmap level to be cleared. (If -1 is specified, the last mipmap level index is used.)
in mipLevelIndexEnd Specifies the index of the last mipmap level to be cleared. (If -1 is specified, the last mipmap level index is used.)
in Components[4] Specifies the component whose resource is to be cleared.

Return Values

Returns the result of the function.

Value Description
Result::IsSuccess Indicates that the operation succeeded.
ResultNullParameter NULL was specified for the resource.
ResultInvalidTextureFormat A format other than ETC and depth formats included in Resource::NativeFormat has been specified for the format of the resource.
ResultInvalidTextureMipLevelIndex A value larger than mipLevelIndexEnd has been specified for mipLevelIndexStart. Alternatively, a value larger than the mipmap level of the specified tex2DResource was specified for mipLevelIndexEnd.
ResultInvalidMemoryRegion The resource memory type is invalid. The specified resource is in FCRAM region memory.

Description

Clears Texture2DResource resources using the color specified in the Components parameter.

Component[0] is the red, luminance, or HI component. Component[1] is the green or LO component. Component[2] is the blue component. Component[3] is the alpha color.
This function adds commands to the command list. For more information, see the Programming Manual.
