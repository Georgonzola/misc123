nn::gd::CTR::OutputStage::AlphaFunction Enumerated Type

Syntax

enum AlphaFunction;

Value

Value Description
ALPHA_FUNCTION_NEVER Never accept fragments.
ALPHA_FUNCTION_ALWAYS Always accept fragments.
ALPHA_FUNCTION_EQUAL Accept a fragment if the fragment alpha equals the reference value.
ALPHA_FUNCTION_NOTEQUAL Accept the fragment if the fragment alpha is not equal to the reference value.
ALPHA_FUNCTION_LESS Accept the fragment if the fragment alpha is less than the reference value.
ALPHA_FUNCTION_LEQUAL Accept the fragment if the fragment alpha is less than or equal to the reference value.
ALPHA_FUNCTION_GREATER Accept the fragment if the fragment alpha is greater than the reference value.
ALPHA_FUNCTION_GEQUAL Accept the fragment if the fragment alpha is greater than or equal to the reference value.
ALPHA_FUNCTION_RESERVED_MAX_U32

Description

Enumerates values for the alpha test function.
