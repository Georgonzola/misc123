nn::gd::CTR::OutputStage::BlendEquation Enumerated Type

Syntax

enum BlendEquation;

Value

Value Description
BLEND_EQUATION_ADD Cs * S + Cd * D
BLEND_EQUATION_SUBTRACT Cs * S - Cd * D
BLEND_EQUATION_REV_SUBTRACT Cd * D - Cs * S
BLEND_EQUATION_MIN min( Cs * S, Cd * D )
BLEND_EQUATION_MAX max( Cs * S, Cd * D )
BLEND_EQUATION_RESERVED_MAX_U32

Description

These enum values are for the blending operators used with m_EqRgb and m_EqAlpha in BlendStateDescription.

Cs and Cd are used for the input color and output color, respectively. S and D are used for the blending coefficient for input and output, respectively.
