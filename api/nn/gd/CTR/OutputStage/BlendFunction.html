nn::gd::CTR::OutputStage::BlendFunction Enumerated Type

Syntax

enum BlendFunction;

Value

Value Description
BLEND_FUNCTION_ZERO RGB coefficient (0, 0, 0), alpha coefficient 0
BLEND_FUNCTION_ONE RGB coefficient (1, 1, 1), alpha coefficient 1
BLEND_FUNCTION_SRC_COLOR RGB coefficient m_SrcRgb, alpha coefficient m_SrcAlpha
BLEND_FUNCTION_ONE_MINUS_SRC_COLOR RGB coefficient 1 - m_SrcRgb, alpha coefficient 1 - m_SrcAlpha
BLEND_FUNCTION_DST_COLOR RGB coefficient m_DstRgb, alpha coefficient m_DstAlpha
BLEND_FUNCTION_ONE_MINUS_DST_COLOR RGB coefficient 1 - m_DstRgb, alpha coefficient 1 - m_DstAlpha
BLEND_FUNCTION_SRC_ALPHA RGB coefficient m_SrcAlpha, alpha coefficient m_SrcAlpha
BLEND_FUNCTION_ONE_MINUS_SRC_ALPHA RGB coefficient 1 - m_SrcAlpha, alpha coefficient 1- m_SrcAlpha
BLEND_FUNCTION_DST_ALPHA RGB coefficient m_DstAlpha, alpha coefficient m_DstAlpha
BLEND_FUNCTION_ONE_MINUS_DST_ALPHA RGB coefficient 1 - m_DstAlpha, alpha coefficient 1 - m_DstAlpha
BLEND_FUNCTION_CONSTANT_COLOR Value set for the nn::gd::OutputStage::SetBlendColor function.
BLEND_FUNCTION_ONE_MINUS_CONSTANT_COLOR 1 - value set for nn::gd::OutputStage::SetBlendColor function.
BLEND_FUNCTION_CONSTANT_ALPHA Alpha value set with the nn::gd::OutputStage::SetBlendColor function.
BLEND_FUNCTION_ONE_MINUS_CONSTANT_ALPHA 1 - alpha value set with nn::gd::OutputStage::SetBlendColor function.
BLEND_FUNCTION_SRC_ALPHA_SATURATE RGB coefficient R,G,B each is min(m_SrcAlpha, 1 - m_DstAlpha), alpha coefficient 1
BLEND_FUNCTION_RESERVED_MAX_U32

Description

These are enum values for the blending coefficients m_SrcRgb, m_DstRgb, m_SrcAlpha, and m_DstAlpha in BlendStateDescription.
