nn::gd::CTR::OutputStage::CreateDepthStencilTarget Member Function

Syntax

static nnResult CreateDepthStencilTarget(
     const Texture2DResource * texture2DResource,
     const DepthStencilTargetDescription * desc,
     DepthStencilTarget ** depthStencil
);

Parameters

Name Description
in texture2DResource Specifies a pointer to an already created Texture2DResource object.
in desc Specifies parameters for creating a DepthStencilTarget.
out depthStencil Specifies the address storing a pointer to the created depth stencil target object.

Return Values

Returns the execution result.

Value Description
Result::IsSuccess Indicates that the operation succeeded.
ResultNullParameter Some parameters were set to NULL.
ResultInvalidTextureMipLevelIndex m_mipLevelIndex in DepthStencilTargetDescription is larger than the mipmap level of the specified resource.
ResultInvalidTextureResolution The height or width of Texture2DResource is 0 or greater than 1024, or the size is not a multiple of 8. Alternatively, the memory layout of the Texture2DResource resource is LAYOUT_BLOCK_32, but the height and width of the Texture2DResource resource is not a multiple of 32.
ResultInvalidTextureFormatForCreatingDepthStencilTarget The pixel format of Texture2DResource is invalid.
ResultInvalidMemoryLocation The memory type of Texture2DResource is not VRAM.
ResultOutOfMemory Cannot allocate memory.
ResultInvalidMemoryLayout The memory layout is invalid. Specify either LAYOUT_BLOCK_8 or LAYOUT_BLOCKE_32.

Description

Creates a depth stencil target from a Texture2DResource.

The following pixel formats can be specified for Texture2DResource.
NATIVE_FORMAT_DEPTH_16
NATIVE_FORMAT_DEPTH_24
NATIVE_FORMAT_DEPTH_24_STENCIL_8

See Also

Resource::CreateTexture2DResource to create a Texture2DResource object.
