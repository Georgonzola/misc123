nn::gd::CTR::OutputStage::CreateRenderTarget Member Function

Syntax

static nnResult CreateRenderTarget(
     const Texture2DResource * texture2DResource,
     const RenderTargetDescription * desc,
     RenderTarget ** renderTarget
);

Parameters

Name Description
in texture2DResource Specifies a pointer to an already created Texture2DResource object.
in desc Specifies parameters for creating a render target.
out renderTarget Specifies the address storing a pointer to the created render target object.

Return Values

Returns the execution result.

Value Description
Result::IsSuccess Indicates that the operation succeeded.
ResultNullParameter NULL was specified in an argument.
ResultInvalidTextureMipLevelIndex m_mipLevelIndex in RenderTargetDescription is larger than the mipmap level of the specified resource.
ResultInvalidTextureResolution The height or width of Texture2DResource is 0 or greater than 1024, or the size is not a multiple of eight. Alternatively, the memory layout of the Texture2DResource resource is LAYOUT_BLOCK_32, but the height and width of the Texture2DResource resource is not a multiple of 32.
ResultInvalidTextureFormatForCreatingRenderTarget The pixel format of Texture2DResource is invalid.
ResultInvalidMemoryLocation The memory type of Texture2DResource is invalid. Specify the VRAM memory type.
ResultOutOfMemory Cannot allocate memory.
ResultInvalidMemoryLayout The memory layout is invalid. Specify either LAYOUT_BLOCK_8 or LAYOUT_BLOCKE_32.

Description

Creates a render target from a Texture2DResource.

The following pixel formats can be specified for Texture2DResource.
NATIVE_FORMAT_RGBA_8888
NATIVE_FORMAT_GAS
NATIVE_FORMAT_SHADOW
NATIVE_FORMAT_RGBA_5551
NATIVE_FORMAT_RGB_565
NATIVE_FORMAT_RGBA_4444

See Also

Resource::CreateTexture2DResource creates a Texture2DResource object.
