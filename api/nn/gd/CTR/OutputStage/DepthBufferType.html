nn::gd::CTR::OutputStage::DepthBufferType Enumerated Type

Syntax

enum DepthBufferType;

Value

Value Description
DEPTHBUFFER_LINEAR In this mode, values stored in the depth buffer are directly linear to the z-coordinates in the eye space. (The hardware does not divide by w.) w buffer mode is enabled and applied on the first shadow pass.
DEPTHBUFFER_RECIPROCAL_FUNCTION In this mode, values stored in the depth buffer are a reciprocal function (1/x) to the z-coordinates in the eye space. (This mode calculates the OpenGL specification depth.) w buffer mode is disabled and the function is applied on the second shadow pass.
DEPTHBUFFER_RESERVED_MAX_U32

Description

Enumerates depth buffer types.
