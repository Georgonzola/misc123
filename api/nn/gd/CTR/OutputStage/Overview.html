nn::gd::CTR::OutputStage Class

Syntax

class OutputStage

Description

The framebuffer operations stage.

This stage sets how the user outputs to the render target.

Enumerated Types

DepthBufferType Enumerates depth buffer types.
FragmentOperationMode Enumerates the fragment operation modes.
BlendType Enumerates the fragment operation blend types.
BlendFunction These are enum values for the blending coefficients m_SrcRgb, m_DstRgb, m_SrcAlpha, and m_DstAlpha in BlendStateDescription.
BlendEquation These enum values are for the blending operators used with m_EqRgb and m_EqAlpha in BlendStateDescription.
LogicOperator Enumerates the opcode for the m_logic members of BlendStateDescription.
ColorWriteMask Enumerates the color write mask.
DepthFunction Enumerates values used in the m_DepthFunc comparison function of DepthStencilStateDescription.
DepthWriteMask Enumerates values for m_DepthMask in DepthStencilDescription.
StencilFunction Enumerates values used in the m_StencilFunc comparison function of DepthStencilStateDescription.
StencilOperation Enumerates test result values for m_StencilFail, m_StencilZFail, and m_StencilZPass in DepthStencilStateDescription.
AlphaFunction Enumerates values for the alpha test function.
