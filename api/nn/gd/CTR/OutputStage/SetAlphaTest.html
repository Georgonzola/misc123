nn::gd::CTR::OutputStage::SetAlphaTest Member Function

Syntax

static void SetAlphaTest(
     gdBool enable,
     AlphaFunction func,
     u8 alphaRef
);

Parameters

Name Description
in enable Enables or disables the alpha test.
in func Specifies the comparison method to be used for the alpha test.
in alphaRef Specifies a reference value to be compared to the fragment's alpha component during the alpha test.

Return Values

None.

Description

Sets the alpha test.

The default values are as follows.

enableGD_FALSE
funcALPHA_FUNCTION_NEVER
alphaRef0
