nn::gd::CTR::OutputStage::SetDepthRangeMode Member Function

Syntax

static void SetDepthRangeMode(
     DepthBufferType type,
     f32 depthRangeNear,
     f32 depthRangeFar,
     s32 offset
);

Parameters

Name Description
in type Specifies the type of depth buffer.
in depthRangeNear If the depth buffer type is DEPTHBUFFER_RECIPROCAL_FUNCTION mode, specify a value that lies in the range [0, 1]. This parameter is not used in DEPTHBUFFER_LINEAR mode.
in depthRangeFar If the depth buffer type is DEPTHBUFFER_RECIPROCAL_FUNCTION mode, specify a value that lies in the range [0, 1]. Specify a scale for the depth value when in DEPTHBUFFER_LINEAR mode.
in offset Specifies the polygon offset. This offset value is divided by 65535 if the bound depth buffer has a 16-bit precision and 16777215 if it has 24-bit precision. If the depth buffer is not bound, the offset value is divided by 16777215 because it is 24-bit precision.

Return Values

None.

Description

Sets the depth buffer mode.

The default values are as follows.

typeDEPTHBUFFER_RECIPROCAL_FUNCTION
depthRangeNear0.0f
depthRangeFar1.0f
offset0
