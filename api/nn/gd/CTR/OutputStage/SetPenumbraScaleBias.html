nn::gd::CTR::OutputStage::SetPenumbraScaleBias Member Function

Syntax

static void SetPenumbraScaleBias(
     f32 Scale,
     f32 Bias
);

Parameters

Name Description
in Scale Specifies the scale to apply when calculating hard shadows from penumbrae.
in Bias Specifies the bias to apply when calculating hard shadows from penumbrae.

Return Values

None.

Description

Sets the Scale and Bias used to control the degree of hardness (or narrowness) to apply to hard-shadow areas.

The default values are as follows.

Scale0.0f
Bias1.0f
