nn::gd::CTR::OutputStage::StencilFunction Enumerated Type

Syntax

enum StencilFunction;

Value

Value Description
STENCIL_FUNCTION_NEVER Never accept fragments.
STENCIL_FUNCTION_ALWAYS Always accept fragments.
STENCIL_FUNCTION_EQUAL Accept a fragment if the stencil alpha value equals the reference value.
STENCIL_FUNCTION_NOTEQUAL Accept a fragment if the stencil alpha value is not equal to the reference value.
STENCIL_FUNCTION_LESS Accept a fragment if the stencil alpha value is less than the reference value.
STENCIL_FUNCTION_LEQUAL Accept a fragment if the stencil alpha value is less than or equal to the reference value.
STENCIL_FUNCTION_GREATER Accept a fragment if the stencil alpha value is greater than the reference value.
STENCIL_FUNCTION_GEQUAL Accept a fragment if the stencil alpha value is greater than or equal to the reference value.
STENCIL_FUNCTION_RESERVED_MAX_U32

Description

Enumerates values used in the m_StencilFunc comparison function of DepthStencilStateDescription.
