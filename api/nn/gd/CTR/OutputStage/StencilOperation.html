nn::gd::CTR::OutputStage::StencilOperation Enumerated Type

Syntax

enum StencilOperation;

Value

Value Description
STENCIL_OPERATION_KEEP Keep the current stencil buffer value.
STENCIL_OPERATION_ZERO Replace with 0.
STENCIL_OPERATION_REPLACE Replace with the reference value.
STENCIL_OPERATION_INCR Increase (no wraparound).
STENCIL_OPERATION_DECR Decrease (no wraparound).
STENCIL_OPERATION_INVERT Invert all bits.
STENCIL_OPERATION_INCR_WRAP Increase (with wraparound).
STENCIL_OPERATION_DECR_WRAP Decrease (with wraparound).
STENCIL_OPERATION_RESERVED_MAX_U32

Description

Enumerates test result values for m_StencilFail, m_StencilZFail, and m_StencilZPass in DepthStencilStateDescription.
