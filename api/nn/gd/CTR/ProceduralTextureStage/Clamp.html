nn::gd::CTR::ProceduralTextureStage::Clamp Enumerated Type

Syntax

enum Clamp;

Value

Value Description
CLAMP_TO_ZERO References the texture image inside the range of [-1.0, 1.0], and references the image of coordinate 0 outside that range.
CLAMP_TO_EDGE The image in the texture is referenced inside the range of [-1.0, 1.0], and pixels at the edge of the texture are referenced outside that range.
CLAMP_SYMMETRICAL_REPEAT Arranges the same image in a grid pattern.
CLAMP_MIRRORED_REPEAT Flips the images in the even-numbered grid squares like in a mirror.
CLAMP_PULSE Picks the pixel at the edge of the nearest texture as the pixel to use for rendering.
CLAMP_RESERVED_MAX_U32

Description

The enum values for procedural texture clamping methods.

The clamping methods are basically the same as the wrapping modes for normal textures, but with the addition of modes like CLAMP_PULSE and CLAMP_TO_ZERO. During the clamping calculations, coordinate values that are not inside the range of [0.0, 1.0] are converted to coordinate values inside that range.
