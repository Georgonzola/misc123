nn::gd::CTR::ProceduralTextureStage::Helper::ConvertColorLookUpTableDataFloatToNative Member Function

Syntax

static nnResult ConvertColorLookUpTableDataFloatToNative(
     f32 ** refArray,
     f32 ** deltaArray,
     u32 lutSize,
     u32 * destRef,
     u32 * destDelta
);

Parameters

Name Description
in refArray Specifies the color lookup table data. Specifies an array of the four lookup tables R, G, B and A. Set each value in the range from 0.0 through 1.0.
in deltaArray Specifies the color lookup table delta data. Specifies an array of the four lookup tables R, G, B and A. Set each value in the range from -1.0 through 1.0.
in lutSize The table size. valueData and deltaData must have the same table size. Specify lutSize in the range from 1 through 256.
out destRef Specifies the address where the converted data is stored. The size is that specified by lutSize.
out destDelta Specifies the address where the converted lookup data is stored. The size is that specified by lutSize.

Return Values

Returns the result of the function.

Value Description
Result::IsSuccess Indicates that the operation succeeded.
ResultNullParameter Some parameters were set to NULL.

Description

Converts color lookup table float data to fixed point format.

refArray and deltaArray are arrays of the four lookup tables R, G. B and A. Each array is the size of the lutSize amount of color component data. refArray[0] = R refArray[1] = G refArray[2] = B refArray[3] = A.
