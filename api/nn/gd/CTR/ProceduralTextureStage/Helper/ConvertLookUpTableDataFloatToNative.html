nn::gd::CTR::ProceduralTextureStage::Helper::ConvertLookUpTableDataFloatToNative Member Function

Syntax

static nnResult ConvertLookUpTableDataFloatToNative(
     f32 * valueData,
     f32 * deltaData,
     u32 lutSize,
     u32 * destination
);

Parameters

Name Description
in valueData Specifies lookup table data. Set each value in the range from 0.0 through 1.0.
in deltaData Specifies lookup table delta data. Set each value in the range from -1.0 through 1.0.
in lutSize Specifies the lookup table size. valueData and deltaData must have the same table size. Specify lutSize in the range from 1 through 128.
out destination Sets the address where the converted data is stored. The size is that specified by lutSize.

Return Values

Returns the result of the function.

Value Description
Result::IsSuccess Indicates that the operation succeeded.
ResultNullParameter Some parameters were set to NULL.

Description

Converts lookup table float data to fixed point format.
