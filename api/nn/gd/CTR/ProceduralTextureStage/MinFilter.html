nn::gd::CTR::ProceduralTextureStage::MinFilter Enumerated Type

Syntax

enum MinFilter;

Value

Value Description
MIN_FILTER_NEAREST Sets the color based on the nearest procedural texture texel. (No miplevel is used.)
MIN_FILTER_LINEAR Sets the color based on interpolation of the two nearest procedural texture texels. (No miplevel is used.)
MIN_FILTER_NEAREST_MIPMAP_NEAREST Sets the color based on the nearest texel of the nearest miplevel.
MIN_FILTER_LINEAR_MIPMAP_NEAREST Sets the color based on interpolation of the two nearest texels of the two nearest miplevel.
MIN_FILTER_NEAREST_MIPMAP_LINEAR Sets the color based on the nearest interpolated texel of the two nearest miplevel.
MIN_FILTER_LINEAR_MIPMAP_LINEAR Sets the color based on the two nearest interpolated texels of the interpolation between the two nearest miplevel.
MIN_FILTER_RESERVED_MAX_U32

Description

The enum values for the MIN filter methods for procedural textures.
