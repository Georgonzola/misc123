nn::gd::CTR::ProceduralTextureStage Class

Syntax

class ProceduralTextureStage

Description

The procedural texture stage contains all functions for creating and using procedural textures.

Texture unit 3 is used for procedural textures.

Classes

nn::gd::CTR::ProceduralTextureStage::Helper Helper utility for lookup table conversions.

Enumerated Types

UvMap The enum values for the functions used for calculating procedural texture mapping.
Clamp The enum values for procedural texture clamping methods.
Shift The enum values for the coordinate shift methods for procedural textures.
MinFilter The enum values for the MIN filter methods for procedural textures.
