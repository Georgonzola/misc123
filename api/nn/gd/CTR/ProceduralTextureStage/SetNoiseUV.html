nn::gd::CTR::ProceduralTextureStage::SetNoiseUV Member Function

Syntax

static void SetNoiseUV(
     f32 noiseU[3],
     f32 noiseV[3]
);

Parameters

Name Description
in noiseU[3] noiseU[0] specifies the frequency of U, noiseU[1] specifies the phase of U, and noiseU[2] specifies the amplitude of U.
in noiseV[3] noiseV[0] specifies the frequency of V, noiseV[1] specifies the phase of V, and noiseV[2] specifies the amplitude of V.

Return Values

None.

Description

Sets the noise for procedural textures.

The default values are 0.0f for all of noiseU[0] to noiseU[2] and noiseV[0] to noiseV[2].
