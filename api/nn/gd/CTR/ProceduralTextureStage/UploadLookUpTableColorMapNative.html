nn::gd::CTR::ProceduralTextureStage::UploadLookUpTableColorMapNative Member Function

Syntax

static nnResult UploadLookUpTableColorMapNative(
     u32 index,
     u32 * Map,
     u32 * MapDelta,
     u32 lutSize
);

Parameters

Name Description
in index Specifies the index of the lookup table for setting data.
in Map Specifies the lookup table data that has been converted to fixed-point data. Specifies an array of the four lookup tables R, G, B and A.
in MapDelta Specifies the delta data that has been converted to fixed-point data. Specifies an array of the four lookup tables R, G, B and A.
in lutSize Specifies the data size of the lookup table. Map and MapDelta must be the same size.

Return Values

Returns the execution result.

Value Description
Result::IsSuccess Indicates that the operation succeeded.
ResultNullParameter NULL was specified for Map or lutSize.
ResultOutOfRangeParameter The sum of the values specified for lutSize and index is equal to or larger than 256 (the maximum size of a color lookup table.)

Description

Sets the color lookup table for procedural textures. The data used by this function has already been converted from floating point to fixed point. This function is an immediate function. (An immediate function directly generates commands and inserts them into the command buffer.)

Map and MapDelta are arrays of four lookup tables pointers. Each pointer is an array of colors values of size lutSize.
(Map[0] = R, Map[1] = G, Map[2] = B, Map[3] = Alpha. MapDelta[0] = R, MapDelta[1] = G, MapDelta[2] = B, MapDelta[3] = Alpha.)
