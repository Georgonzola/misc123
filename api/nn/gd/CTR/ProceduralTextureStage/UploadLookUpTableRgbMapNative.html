nn::gd::CTR::ProceduralTextureStage::UploadLookUpTableRgbMapNative Member Function

Syntax

static nnResult UploadLookUpTableRgbMapNative(
     u32 index,
     u32 * Map,
     u32 lutSize
);

Parameters

Name Description
in index Specifies the index of the lookup table for setting data.
in Map Specifies the lookup table data.
in lutSize Specifies the data size of the lookup table.

Return Values

Returns the execution result.

Value Description
Result::IsSuccess Indicates that the operation succeeded.
ResultNullParameter NULL was specified for Map or lutSize.
ResultOutOfRangeParameter The sum of the values specified for lutSize and index is equal to or larger than 128 (the maximum size of the F function lookup table for RGB mapping).

Description

Sets the F function lookup table for RGB mapping of procedural textures. The data used by this function has already been converted from floating point to fixed point. This function is an immediate function. (An immediate function directly generates commands and inserts them into the command buffer.)
