nn::gd::CTR::RasterizerStage::EarlyDepthFunction Enumerated Type

Syntax

enum EarlyDepthFunction;

Value

Value Description
EARLYDEPTH_FUNCTION_GEQUAL Passes when the depth value of the fragment is greater than or equal to the depth value in the early depth buffer.
EARLYDEPTH_FUNCTION_GREATER Passes when the depth value of the fragment is greater than the depth value in the early depth buffer.
EARLYDEPTH_FUNCTION_LEQUAL Passes when the depth value of the fragment is less than or equal to the depth value in the early depth buffer.
EARLYDEPTH_FUNCTION_LESS Passes when the depth value of the fragment is less than the depth value in the early depth buffer.
EARLYDEPTH_FUNCTION_RESERVED_MAX_U32

Description

The enum values for the comparison functions for the early depth test.

The default value is EARLYDEPTH_FUNCTION_LESS.
