nn::gd::CTR::RasterizerStage::SetClippingPlane Member Function

Syntax

static void SetClippingPlane(
     f32 param1,
     f32 param2,
     f32 param3,
     f32 param4
);

Parameters

Name Description
in param1 Specifies P1 clipping plane 1.
in param2 Specifies P2 clipping plane 2.
in param3 Specifies P3 clipping plane 3.
in param4 Specifies P4 clipping plane 4.

Return Values

None.

Description

Sets the clipping face planes. This function is an immediate function. (An immediate function directly generates commands and inserts them into the command buffer.)

There are four clipping plane coefficients (P1, P2, P3, P4) and the half space defined by the clipping planes must satisfy the following expression:
P1 * x + P2 * y + P3 * z + P4 * w >= 0
The following format is used for the data inserted into the command buffer:
0x00000000 Eq1
0x00000004 [Reserved]
0x00000008 Eq2
0x0000000c Eq3
0x00000010 Eq4
0x00000014 [Reserved]
Eq1, Eq2, Eq3, and Eq4 are 24-bit floating point values. They are obtained by converting P1, P2, P3, and P4.

The default value is all 0.
