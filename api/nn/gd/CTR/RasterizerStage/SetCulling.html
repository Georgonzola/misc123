nn::gd::CTR::RasterizerStage::SetCulling Member Function

Syntax

static void SetCulling(
     Culling culling
);

Parameters

Name Description
in culling Specifies the culling type.

Return Values

None.

Description

Sets the culling mode. This function is an immediate function. (An immediate function directly generates commands and inserts them into the command buffer.)

The following format is used for the data inserted into the command buffer.
0x00000000 culling
0x00000004 [Reserved]

The default value is CULLING_CLOCKWISE.
