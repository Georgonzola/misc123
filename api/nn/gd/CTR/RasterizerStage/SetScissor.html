nn::gd::CTR::RasterizerStage::SetScissor Member Function

Syntax

static void SetScissor(
     Viewport & area
);

Parameters

Name Description
in area Specifies the rectangular region for scissoring.

Return Values

None.

Description

Sets the rectangular region for scissoring. This function is an immediate function. (An immediate function directly generates commands and inserts them into the command buffer.)

The following format is used for the data inserted into the command buffer.
0x00000000 (area.m_Y << 16) | area.m_X
0x00000004 [Reserved]
0x00000008 (endY << 16) | endX
0x0000000c [Reserved]
The values for endX and endY are calculated as follows.
endX = (area.m_X + area.m_Width - 1)
endY = (area.m_Y + area.m_Height - 1))

The default value is a viewport set to (0, 0, 0, 0).
