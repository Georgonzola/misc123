nn::gd::CTR::RasterizerStage::SetViewport Member Function

Syntax

static void SetViewport(
     Viewport & viewport
);

Parameters

Name Description
in viewport Specifies the viewport region.

Return Values

None.

Description

Configures the viewport. This function is an immediate function. (An immediate function directly generates commands and inserts them into the command buffer.)

The following format is used for the data inserted into the command buffer.
0x00000000 Whalf
0x00000004 [Reserved]
0x00000008 W
0x0000000c Hhalf
0x00000010 H
0x00000014 [Reserved]
0x00000018 XY
0x0000001c [Reserved]
The values for Whalf, W, and XY are calculated as follows.
Whalf = Utils::Float32ToFloat24(viewport.m_Width * 0.5f), Hhalf = Utils::Float32ToFloat24(viewport.m_Height * 0.5f)
W = Utils::Float32ToFloat31(2.f / viewport.m_Width) << 1, H = Utils::Float32ToFloat31(2.f / viewport.m_Height) << 1
XY = viewport.m_X | (viewport.m_Y << 16)

The default value is a viewport set to (0, 0, 240, 320).

Utils::Float32ToFloat24
Utils::Float32ToFloat31
