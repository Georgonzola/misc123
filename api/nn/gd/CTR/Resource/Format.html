nn::gd::CTR::Resource::Format Enumerated Type

Syntax

enum Format;

Value

Value Description
FORMAT_RGBA_8888 RGBA 8888 format.
FORMAT_RGB_888 RGB 888 format.
FORMAT_RGBA_5551 RGBA 5551 format.
FORMAT_RGB_565 RGBA 565 format.
FORMAT_RGBA_4444 RGBA 4444 format.
FORMAT_LUMINANCE_ALPHA_88 Format with a 8-bit luminance component and a 8-bit alpha value.
FORMAT_LUMINANCE_8 Format with a 8-bit luminance component.
FORMAT_ALPHA_8 A8 format.
FORMAT_LUMINANCE_ALPHA_44 Format with a 4-bit luminance component and a 4-bit alpha value.
FORMAT_LUMINANCE_4 Format with a 4-bit luminance component.
FORMAT_ALPHA_4 A4 format.
FORMAT_HILO_88 Format configured with R and G components.
FORMAT_RESERVED_MAX_U32

Description

The enum values for texture resource formats.

To use these formats, they must first be converted to native format because they are in OpenGL standard texture format and cannot be used in that format.

See Also

Helper::ConvertTextureResourceToNativeFormat
