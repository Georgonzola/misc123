nn::gd::CTR::SamplerStateDescription Class

Syntax

class SamplerStateDescription

Description

The descriptor for creating a sampler state.

Member Variables

m_MinFilter TextureStage::SamplerMinFilter The minification filter.
m_MagFilter TextureStage::SamplerMagFilter The magnification filter.
m_WrapS TextureStage::SamplerWrapMode WrapS mode.
m_WrapT TextureStage::SamplerWrapMode WrapT mode.
m_BorderColor u8 The border color.
m_LodBias u32 The LOD bias.
m_MinLod u32 The minimum LOD level.
m_MaxLod u32 The maximum LOD level.

Member Functions

SetMinFilter Sets the minification filter in m_MinFilter.
SetMagFilter Sets the magnification filter in m_MagFilter.
SetWrapS Sets the WrapS mode in m_WrapS.
SetWrapT Sets the WrapT mode in m_WrapT.
SetBorderColor Sets the border color in m_BorderColor[4]. Specify values in the range from 0 through 255.
SetLodBias Sets the LOD bias in m_LodBias.
SetMinLod Sets the minimum LOD level in m_MinLod.
SetMaxLod Sets the maximum LOD level in m_MaxLod.
ToDefault Sets the default values.
SetShadow Configures settings for shadow texture.
SetShadowCube Configures settings for shadow cube texture.
SetGas Configures settings for gas texture.
