nn::gd::CTR::SamplerStateDescription::SetBorderColor Member Function

Syntax

void SetBorderColor(
     u8 colorR,
     u8 colorG,
     u8 colorB,
     u8 colorA
);

Parameters

Name Description
in colorR Specifies red.
in colorG Specifies green.
in colorB Specifies blue.
in colorA Specifies the alpha color.

Return Values

None.

Description

Sets the border color in m_BorderColor[4]. Specify values in the range from 0 through 255.
