nn::gd::CTR::ShaderStage::CreateShaderBinary Member Function

Syntax

static nnResult CreateShaderBinary(
     const void * shaderBytecode,
     u32 bytecodeLength,
     ShaderBinary ** shaderBinary
);

Parameters

Name Description
in shaderBytecode Specifies a pointer to shader binary data.
in bytecodeLength Specifies the size of binary data in bytes.
out shaderBinary Specifies the address storing the created shader binary object pointer.

Return Values

Returns the execution result.

Value Description
Result::IsSuccess Indicates that the operation succeeded.
ResultNullParameter NULL was specified for shaderBytecode or shaderBinary.
ResultInvalidShaderSignature shaderBytecode is not correct.
ResultInvalidShaderOperation The file size is not correct.
ResultOutOfMemory Cannot allocate memory.

Description

Loads a shader binary.
