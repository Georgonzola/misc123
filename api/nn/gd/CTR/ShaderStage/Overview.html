nn::gd::CTR::ShaderStage Class

Syntax

class ShaderStage

Description

The shader stage contains all required functionality to create and operate the shader.

Enumerated Types

ShaderType The shader type.

Member Functions

S CreateShaderBinary Loads a shader binary.
S CreateShader Creates a shader object. (Vertex shader or geometry shader.)
S CreateShaderPipeline Creates a shader pipeline with an attached shader object.
S SetShaderPipeline Sets a shader pipeline.
S GetShader Gets a vertex shader or a geometry shader from the shader pipeline.
S ReleaseShaderBinary Releases the shader binary.
S ReleaseShader Releases the shader object.
S ReleaseShaderPipeline Releases the shader pipeline.
S SetShaderPipelineConstantFloat Sets data in the shader floating point constant register.
S SetShaderPipelineConstantBoolean Sets data in the shader Boolean register.
S SetShaderPipelineConstantInteger Sets data in the shader integer register.
S SetFloatConstantBuffer Immediately creates a floating point constant register command in the command buffer.
S GetShaderUniformLocation Gets the location of the shader uniform variable.
