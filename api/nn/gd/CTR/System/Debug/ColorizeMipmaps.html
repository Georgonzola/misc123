Description

Colors the several mipmap levels of a resource with a single color.

This function helps visualize the mipmap level of several textures for user debugging. This function internally calls the nn::gd::CTR::Memory::ClearTexture2DResource function.texture2DResourceSpecifies a pointer to the target Texture2DResource resource. minMipLevelIndexSpecifies the lowest mipmap level index to be colored, in the range [0, texture2DResource->mipLevels]. Specifying -1 uses the last mipmap level. maxMipLevelIndexSpecifies the highest mipmap level index to be colored, in the range [0, texture2DResource->mipLevels]. Specifying -1 uses the last mipmap level. colorsSpecifies an array of colors used to color in mipmap levels. The array specifies the color values used to color each mipmap level. (Specify minMipLevelIndex and maxMipLevelIndex.) If NULL is specified for a color value, the initial array for color values is used.Returns the execution result. Result::IsSuccessIndicates that the operation succeeded. ResultNullParameterNULL was specified for the resource. ResultInvalidTextureFormatA format other than ETC and depth formats included in Resource::NativeFormat has been specified for the format of the resource. ResultInvalidTextureMipLevelIndexA value larger than mipLevelIndexEnd has been specified for mipLevelIndexStart. Alternatively, a value larger than the mipmap level of the specified tex2DResource was specified for mipLevelIndexEnd. ResultInvalidMemoryRegionThe resource memory type is invalid. The specified resource is in FCRAM region memory.
