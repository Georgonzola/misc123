nn::gd::CTR::System::Debug::Filter Enumerated Type

Syntax

enum Filter;

Value

Value Description
FILTER_NONE This filter does nothing. It is provided so that users can disable all filters.
FILTER_TEXTURE_8X8 Using this filter sets causes gd to set all texture units to a unique 8x8 texture. The 8x8 texture used for this filter is allocated in FCRAM.
FILTER_TEXTURE_CUSTOM This filter sets all texture units to a custom texture. This custom texture can be defined by using the System::debug::SetCustomTexture function. If this filter is set without defining a custom texture, an 8x8 texture (used by the System::debug::FILTER_TEXTURE_8X8 filter) is used instead.
FILTER_SAMPLER_STATE_FILTER_NEAREST_MIPMAP_NEAREST This filter sets point sampling between texture mipmap levels and texture filtering.
FILTER_SAMPLER_STATE_FILTER_LINEAR_MIPMAP_NEAREST This filter sets the gd sample state to use the nearest filter for mipmap level sampling and the linear filter for texel sampling.
FILTER_VIEWPORT_1X1 This filter sets a 1x1 viewport.
FILTER_SIMPLE_COMBINER This gd filter is for the simple combiner configuration used to output all pixels as a fixed green color (0, 255, 0).
FILTER_DISABLE_BLENDING This filter disables blending.
FILTER_DISABLE_DEPTH_STENCIL_TEST This filter disables the depth and stencil tests.
FILTER_DISABLE_ALPHA_TEST This filter disables the alpha test.
FILTER_LIGHTING_STAGE_1_LIGHT_MAX This filter sets the maximum number of lights to one. If more than one light is enabled, the light enabled first is used.
FILTER_LIGHTING_STAGE_SIMPLE_LAYER_CONFIGURATION This filter sets the light stage to a simple configuration to minimize the number of cycles.
FILTER_DISABLE_DRAW This filter disables drawing.
FILTER_VISUALIZE_OVERDRAW This filter is used to visualize overdraw. When this filter is used, gd sets the combiner so that both the fixed RGBA color (15,15,15,255) and the additional blend state can both be viewed. The more overdraw, the more pixels appear white. By combining this filter with FILTER_DISABLE_DEPTH_STENCIL_TEST, you can check for overdraw pixels destroyed (or not destroyed) by the depth test.
FILTER_DISABLE_CULLING This filter disables culling.
FILTER_ENABLE_CULLING_CLOCKWISE This filter sets the culling mode to clockwise.
FILTER_ENABLE_CULLING_COUNTERCLOCKWISE This filter sets the culling mode to counterclockwise.
FILTER_RESERVED_MAX_U32

Description

This enumerated type is used by the System::Debug::SetFilter function to specify the filters to use.
