nn::gd::CTR::System::Module Enumerated Type

Syntax

enum Module;

Value

Value Description
MODULE_RASTERIZER Specifies a rasterizer module. This module can be used by the RasterizerStage class.
MODULE_LIGHTING Specifies a light module. This module can be used by the LightingStage class.
MODULE_OUTPUT Specifies an output module. This module can be used by the OutputStage class.
MODULE_TEXTURE_COMBINER Specifies a combiner module. This module can be used by the CombinerStage class.
MODULE_GAS_FOG Specifies a gas fog module. This module can be used by the FogStage class.
MODULE_TEXTURE Specifies a texture module. This module can be used by the TextureStage class.
MODULE_TEXTURE_PROCEDURAL Specifies the procedural texture module. This module can be used by the ProceduralTextureStage class.
MODULE_VERTEX_INPUT Specifies a vertex input module. This module can be used by the VertexInputStage class.
MODULE_SHADER Specifies a shader module. This module can be used by the ShaderStage class.
MODULE_ALL Specifies all modules.

Description

The enum value of each gd API module.
