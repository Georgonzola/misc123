nn::gd::CTR::TextureStage::CreateTexture2D Member Function

Syntax

static nnResult CreateTexture2D(
     const Texture2DResource * tex2DResource,
     Texture2DDescription * desc,
     Texture2D ** texture2D
);

Parameters

Name Description
in tex2DResource Specifies a pointer to an already created Texture2DResource object.
in desc Specifies the parameters of the 2D texture.
out texture2D Specifies the address of the pointer to use for the 2D texture.

Return Values

Returns the execution result.

Value Description
Result::IsSuccess Indicates that the operation succeeded.
ResultNullParameter Some parameters were set to NULL.
ResultInvalidTextureResolution The height or the width of the texture is not the correct size. Specify a value that is a power of 2. The maximum size is 1024x1024.
ResultInvalidMemoryLayout The memory layout is not Memory::LAYOUT_BLOCK_8.
ResultInvalidTextureFormat The pixel format is invalid.
ResultInvalidTextureMipLevelIndex The mipmap level index is incorrect.
ResultOutOfMemory Memory allocation failed.

Description

Creates a 2D texture object from a Texture2DResource.

The memory layout must be Memory::LAYOUT_BLOCK_8.

See Also

Resource::CreateTexture2DResource to create a Texture2DResource object.
