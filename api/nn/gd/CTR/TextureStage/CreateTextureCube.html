nn::gd::CTR::TextureStage::CreateTextureCube Member Function

Syntax

static nnResult CreateTextureCube(
     Texture2DResource ** tex2DResources,
     TextureCubeDescription * desc,
     TextureCube ** textureCube
);

Parameters

Name Description
tex2DResources
in desc Specifies the parameters of the texture cube.
out textureCube Specifies the address storing the pointer to the created texture cube object.
in tex2DResource Specifies a pointer array to the already created Texture2DResource objects for the six faces of a cube. This array must be specified in the following order: (X_POS, X_NEG, Y_POS, Y_NEG, Z_POS, Z_NEG)

Return Values

Returns the execution result.

Value Description
Result::IsSuccess Indicates that the operation succeeded.
ResultNullParameter Some parameters were set to NULL.
ResultInvalidTextureFormat The gas texture format has been specified, or the pixel format is invalid.
ResultInvalidMemoryLayout The memory layout is not Memory::LAYOUT_BLOCK_8.
ResultInvalidMemoryLocation The resources are not all located in the same memory region (FCRAM, VRAM, and so on).
ResultInvalidTextureMipLevelIndex The mipmap level is not correct. This result is also returned if the texture is a shadow texture and cannot use mipmap levels. You must specify 1.
ResultInvalidTextureResolution The resolution width or height is incorrect. You must specify a power of 2. The maximum size is 1024x1024. This error is also returned if the resolution is a different resource.
ResultInvalidMemoryLocation The addresses for the resources for the six faces differ among the most-significant 7 bits. The addresses for all six faces must be the same for the most-significant 7 bits.
ResultOutOfMemory Memory allocation failed.

Description

Creates a texture cube object from a Texture2DResource.

The six faces of the texture cube must use Texture2DResource objects that all have the same resolution, pixel format, and mipmap level. In addition, all six faces must fit within a 32-MB boundary. The memory layout must be set to Memory::LAYOUT_BLOCK_8, and the resolution of each Texture2DResource must define a square.
