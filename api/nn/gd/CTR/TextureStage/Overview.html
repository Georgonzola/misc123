nn::gd::CTR::TextureStage Class

Syntax

class TextureStage

Description

The texture stage contains all functions for creating and using textures.

Texture unit 3 is used only for procedural textures.

Enumerated Types

TextureUnitId The enum values for texture units.
SamplerWrapMode The enum values for Wrap modes.
SamplerMinFilter The enum values for MinFilter.
SamplerMagFilter The enum values for MagFilter.
TextureCoordinateSourceUnit2 The enum values for the texture coordinates of texture unit 2.
TextureCoordinateSourceUnit3Procedural The enum values for the texture coordinates of texture unit 3, which is used for procedural textures.

Member Functions

S CreateTexture2D Creates a 2D texture object from a Texture2DResource.
S GetTexture2DProperties Gets detailed information about a 2D texture.
S ReleaseTexture2D Releases a 2D texture.
S CreateTextureCube Creates a texture cube object from a Texture2DResource.
S GetTextureCubeProperties Gets detailed information about a texture cube.
S ReleaseTextureCube Releases the texture cube.
S SetTexture Sets a 2D texture object in a texture unit.
S ClearTextureUnit Clears the 2D texture object set in the specified texture unit.
S SetTextureUnit0 Sets a texture cube object to texture unit 0.
S SetTexture2DProjectionForUnit0 Sets a projection texture to texture unit 0.
S SetTextureCoordinateSourceForUnit2 Sets the texture coordinate in texture unit 2.
S SetTextureCoordinateSourceForUnit3Procedural Sets the texture coordinate in texture unit 3. (Used for procedural textures.)
S CreateSamplerState Creates a sampler state object.
S SetSamplerState Sets a sampler state object to a texture unit.
S ReleaseSamplerState Releases the sampler state.
S SetPerspectiveShadow Sets the use of perspective projection for generating texture coordinates for shadow texture lookups.
S SetShadowZBias Sets the bias for shadow depth texture.
