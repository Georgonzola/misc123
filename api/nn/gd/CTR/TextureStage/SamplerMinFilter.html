nn::gd::CTR::TextureStage::SamplerMinFilter Enumerated Type

Syntax

enum SamplerMinFilter;

Value

Value Description
SAMPLER_MIN_FILTER_NEAREST Samples texels from the texture nearest those texture coordinates.
SAMPLER_MIN_FILTER_LINEAR Samples texels from the nearest two main textures (no miplevel is used), and sets the color after interpolating between those texels.
SAMPLER_MIN_FILTER_NEAREST_MIPMAP_NEAREST Samples texels from the mipmap texture nearest those texture coordinates.
SAMPLER_MIN_FILTER_LINEAR_MIPMAP_NEAREST Samples texels from the two nearest mipmap level textures and sets the color after interpolating between them.
SAMPLER_MIN_FILTER_NEAREST_MIPMAP_LINEAR Sets the color based on the texel that is nearest to the texel interpolated between the two nearest miplevel.
SAMPLER_MIN_FILTER_LINEAR_MIPMAP_LINEAR Sets the color based on the two nearest interpolated texels between the two nearest miplevel.
SAMPLER_MIN_FILTER_RESERVED_MAX_U32

The enum values for MinFilter.
