Value

Value Description

SAMPLER_WRAP_CLAMP_TO_BORDER The region clamped from the texture becomes the color specified as the border color.
SAMPLER_WRAP_REPEAT Repeats the texture.
SAMPLER_WRAP_MIRRORED_REPEAT Flips and repeats the texture.
