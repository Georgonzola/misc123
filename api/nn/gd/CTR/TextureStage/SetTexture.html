nn::gd::CTR::TextureStage::SetTexture Member Function

Syntax

static nnResult SetTexture(
     TextureUnitId textureUnitId,
     Texture2D & texture2D
);

Parameters

Name Description
in textureUnitId Specifies the ID of the texture unit. (You must specify TEXTURE_UNIT_0, TEXTURE_UNIT_1 or TEXTURE_UNIT_2.)
in texture2D Specifies the 2D texture object to set.

Return Values

Returns the execution result.

Value Description
Result::IsSuccess Indicates that the operation succeeded.
ResultInvalidTextureUnitId The texture unit ID is invalid.
ResultInvalidTextureFormat The texture object type is invalid. Specify a Texture2D-type object.

Description

Sets a 2D texture object in a texture unit.

See Also

TextureStage::CreateTexture2D Creates a 2D texture object.
