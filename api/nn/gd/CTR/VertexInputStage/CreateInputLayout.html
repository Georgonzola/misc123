nn::gd::CTR::VertexInputStage::CreateInputLayout Member Function

Syntax

static nnResult CreateInputLayout(
     InputElementDescription * inputElementDescs,
     u32 numElements,
     u32 * strides,
     Shader * vertexShader,
     InputLayout ** inputLayout
);

Parameters

Name Description
in inputElementDescs Sets a parameter for creating an input layout.
in numElements Specifies the number of elements in the input layout. Up to 12 elements can be specified.
in strides Specifies an array that specifies the size of each element.
in vertexShader Specifies a pointer to a vertex shader object.
out inputLayout Specifies the address storing a pointer to the created input layout object.

Return Values

Returns the execution result.

Value Description
Result::IsSuccess Indicates that the operation succeeded.
ResultNullParameter NULL was specified in an argument other than numElements.
ResultInvalidShaderUniformName The value of numElements is incorrect. Specify a value in the range from 0 to 12. Or, the InputElementDescription parameter or the shader input element name is incorrect.
ResultOutOfMemory Could not allocate memory.
ResultInvalidParameter The parameter is invalid.
ResultInvalidShaderOperation numElements is invalid. Specify a value in the range from 1 to 12.
ResultInputLayoutInvalidStreamSlots Input slots must start from 0 and be consecutive.

Description

Creates an input layout.
