nn::gd::CTR::VertexInputStage Class

Syntax

class VertexInputStage

Description

The vertex input stage.

This class makes vertex buffer and index buffer layout settings passed from the application, assembles primitives, and makes shader settings.

Enumerated Types

ShaderStreamFormatType Enumerated values for shader stream format types.
IndexFormat Enumerated values for index buffer data types.
PrimitiveTopology Enumerated values for primitive types.

Member Functions

S SetIndexBuffer Sets an index buffer to a vertex input stage.
S SetInputLayout Sets an input layout to a vertex input stage.
S SetPrimitiveTopology Sets the primitive type to be rendered.
S SetVertexBuffers Sets vertex buffers. Up to 12 slots can be set.
S CreateInputLayout Creates an input layout.
S ReleaseInputLayout Releases an input layout.
