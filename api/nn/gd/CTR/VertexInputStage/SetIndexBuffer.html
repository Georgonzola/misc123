nn::gd::CTR::VertexInputStage::SetIndexBuffer Member Function

Syntax

static nnResult SetIndexBuffer(
     VertexBufferResource * indexBuffer,
     IndexFormat format,
     u32 offset
);

Parameters

Name Description
in indexBuffer Specifies a pointer to the index buffer to be set to the pipeline created by Resource::CreateVertexBufferResource.
in format Specifies the data format. (See IndexFormat.)
in offset Specifies the byte offset to the first index to be used.

Return Values

Returns the execution result.

Value Description
Result::IsSuccess Indicates that the operation succeeded.
ResultNullParameter NULL was specified for indexBuffer.

Description

Sets an index buffer to a vertex input stage.

See Also

Resource::CreateVertexBufferResource
