nn::gd::CTR::VertexInputStage::SetVertexBuffers Member Function

Syntax

static nnResult SetVertexBuffers(
     u32 startSlot,
     u32 numBuffers,
     VertexBufferResource ** vertexBuffers,
     u32 * offsets
);

Parameters

Name Description
in startSlot Specifies the first input slot number to use in setting a vertex buffer.
in numBuffers Specifies the number of elements in the array of vertex buffers to be set.
in vertexBuffers Specifies the array of pointers to the vertex buffers, created by CreateVertexBufferResource.
in offsets Specifies an array of offset values from the first element in the vertex buffer to the element to be used first in each vertex buffer.

Return Values

Returns the execution result.

Value Description
Result::IsSuccess Indicates that the operation succeeded.
ResultNullParameter NULL was specified for vertexBuffers.

Description

Sets vertex buffers. Up to 12 slots can be set.

See Also

Resource::CreateVertexBufferResource.
