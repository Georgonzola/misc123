nn::gd Namespace

Description

The namespace of the GD (graphics) API.

The GD API allows you to control the data flow in hardware rendering with specifications at each stage.
The GD coexists with the NNGX layer. To use the GD for programming, you must understand the mechanisms of the NNGX layer. For more information, see the Programming Guide.
In addition, gd is not thread-safe.

Namespaces

nn::gd::CTR The namespace for the CTR's GD (graphics) API. (It can also be written without the CTR portion.)
