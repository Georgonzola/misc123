nn::gr::CTR::BindSymbol::SymbolType Enumerated Type

Syntax

enum SymbolType;

Value

Value Description
SYMBOL_TYPE_INVALID Invalid type.
SYMBOL_TYPE_INPUT Indicates shader input. Supports v0 through v15.
SYMBOL_TYPE_FLOAT Indicates floating-point registers. Supports c0 through c95.
SYMBOL_TYPE_INTEGER Integer registers. Supports i0 through i3.
SYMBOL_TYPE_BOOL BOOL registers. Supports b0 through b15.

Description

Represents symbol type information.
