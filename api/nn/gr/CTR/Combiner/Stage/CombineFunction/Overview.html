nn::gr::CTR::Combiner::Stage::CombineFunction Structure

Syntax

struct CombineFunction
{
   util::SizedEnum1< PicaDataTexEnvCombine > combine;
   util::SizedEnum1< PicaDataTexEnvOperand > operand[3];
   util::SizedEnum1< PicaDataTexEnvSrc > source[3];
   util::SizedEnum1< PicaDataTexEnvScale > scale;
   util::SizedEnum1< PicaDataTexEnvBufferInput > bufferInput;
};

Description

Structure representing I/O settings for combine functions.

Member Variables

combine util::SizedEnum1< PicaDataTexEnvCombine > Sets the combine function. The type is PicaDataTexEnvCombine. Use a PicaDataTexEnvCombine enumerated constant defined in the PICA macro header. The initial value is PICA_DATA_TEX_ENV_COMBINE_REPLACE.
operand util::SizedEnum1< PicaDataTexEnvOperand > Sets the operand. The type is PicaDataTexEnvOperand[]. Use a PicaDataTexEnvOperand enumerated constant defined in the PICA macro header. The color operand is PICA_DATA_OPE_RGB_SRC_COLOR by default. The alpha operand is PICA_DATA_OPE_ALPHA_SRC_ALPHA by default.
source util::SizedEnum1< PicaDataTexEnvSrc > Sets Source 0, Source 1, and Source 2. The type is PicaDataTexEnvSrc. Use a PicaDataTexEnvSrc enumerated constant defined in the PICA macro header. The setting for Stage 0 is PICA_DATA_TEX_ENV_SRC_RGBA_CONSTANT by default. The setting for stages other than Stage 0 is PICA_DATA_TEX_ENV_SRC_RGBA_PREVIOUS by default.
scale util::SizedEnum1< PicaDataTexEnvScale > Sets the scale. The type is PicaDataTexEnvScale. Use a PicaDataTexEnvScale enumerated constant defined in the PICA macro header. The default value is PICA_DATA_TEX_ENV_SCALE_1.
bufferInput util::SizedEnum1< PicaDataTexEnvBufferInput > Sets the combiner buffer source. The type is PicaDataTexEnvBufferInput. Use a PicaDataTexEnvBufferInput enumerated constant defined in the PICA macro header. The default value is PICA_DATA_TEX_ENV_BUFFER_INPUT_PREVIOUS_BUFFER_DMP.

Member Functions

protected CombineFunction Sets initial values for combiner function settings.
