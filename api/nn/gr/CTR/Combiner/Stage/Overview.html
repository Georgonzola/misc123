nn::gr::CTR::Combiner::Stage Class

Syntax

class Stage

Description

Class for setting combiner stages.

Structures

nn::gr::CTR::Combiner::Stage::CombineFunction Structure representing I/O settings for combine functions.

Member Variables

rgb CombineFunction Combiner function for colors. The type is nn::gr::CTR::Combiner::Stage::CombineFunction.
alpha CombineFunction Combiner function for alpha. The type is nn::gr::CTR::Combiner::Stage::CombineFunction.
constColorR u8 Red component of the constant color. The type is u8.
constColorG u8 Green component of the constant color. The type is u8.
constColorB u8 Blue component of the constant color. The type is u8.
constColorA u8 Alpha component of the constant color The type is u8.
headRegister util::SizedEnum2< PicaReg > Sets the starting register for the stage being set. Type is PicaReg.

Member Functions

SetupPrimary Sets to replace with PRIMARY_COLOR.
SetupFragmentPrimary Sets to replace with FRAGMENT_PRIMARY_COLOR.
SetupTexture0 Sets to replace with TEXTURE0.
SetupPrevious Sets to scale using the color of the previous stage.
SetupPrimaryModulateTexture0 Sets to multiply PRIMARY_COLOR and TEXTURE0.
SetupFragmentPrimaryModulateTexture0 Sets to multiply FRAGMENT_PRIMARY_COLOR and TEXTURE0.
protected Stage Sets the initial value for the stage.
MakeCommand Generates a command for setting the combiner stage based on information already set. Note that this does not include commands for the combiner buffer.
MakeConstantColorCommand Generates only Combiner stage Constant Color commands.
