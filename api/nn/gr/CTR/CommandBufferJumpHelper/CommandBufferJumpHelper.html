Description of CommandBufferJumpHelper ( )

To use CommandBufferJumpHelper, which is created by the default constructor, you must initialize it. Register the starting address of the command buffer to write to using nn::gr::CTR::CommandBufferJumpHelper::Initialize.

Description of CommandBufferJumpHelper ( bit32 * )

In the address parameter, specify the starting address of the command buffer to write to.
You can get the command buffer starting address using the nngxGetCmdlistParameteri function.
