nn::gr::CTR::CommandBufferJumpHelper::Initialize Member Function

Syntax

void Initialize(
     bit32 * address
);

Parameters

Name Description
in address Starting address of the command buffer to write to.

Return Values

None.

Description

Registers the starting address of the command buffer to write to and initializes the internal state accordingly.

In the address parameter, specify the starting address of the command buffer to write to.
You can get the command buffer starting address using the nngxGetCmdlistParameteri function.

You can use this function to initialize the instance of CommandBufferJumpHelper created by the default constructor, or to register a new command buffer.

Calling this function clears the internal state that was stored to generate a command buffer jump command.
