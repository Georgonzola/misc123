nn::gr::CTR::CopyMtx44WithHeader Function

Syntax

void CopyMtx44WithHeader(
     f32 * dst,
     const nn::math::MTX44 * src,
     bit32 header
);

Parameters

Name Description
in dst Pointer to the start address to write results to.
in src A 4x4 matrix.
in header The header.

Return Values

None.

Description

Inverts the wHzyx order of a 4x4 matrix and makes a copy. (Includes the header.)
