nn::gr::CTR::FragmentLight Class

Syntax

class FragmentLight

Description

Class for configuring fragment lighting settings.

Classes

nn::gr::CTR::FragmentLight::LutConfig Class for adjusting lookup table values.
nn::gr::CTR::FragmentLight::Source Class for making settings per light source.

Member Constants

S LIGHT_SOURCE_MAX const u32 The maximum number of light sources is 8.

Member Variables

globalAmbientR u8 Red component setting for global ambient. The type is u8. Values in the range [0.f, 1.f] are mapped to [0, 255] and then set.
globalAmbientG u8 Green component setting for global ambient. The type is u8. Values in the range from 0.f to 1.f are mapped to the range from 0 to 255 and then set.
globalAmbientB u8 Blue component setting for global ambient. The type is u8. Values in the range from 0.f to 1.f are mapped to the range from 0 to 255 and then set.
source Source Stores light source settings. The type is nn::gr::CTR::FragmentLight::Source[].
isEnable bool Enables or disables light sources. The type is bool[]. The upper limit value of the array is LIGHT_SOURCE_MAX.
isEnableSpot bool Sets whether spotlights are enabled. The type is bool[]. The upper limit value of the array is LIGHT_SOURCE_MAX.
isEnableDistAttn bool Sets whether distance attenuation is enabled. The type is bool[]. The upper limit value of the array is LIGHT_SOURCE_MAX.
isShadowed bool Sets shadow adjustment elements. The type is bool[]. The upper limit value of the array is LIGHT_SOURCE_MAX.
layerConfig PicaDataFragLightEnvLayerConfig Sets the layer config. The type is PicaDataFragLightEnvLayerConfig.
fresnelSelector PicaDataFragLightEnvFresnel Sets where fresnel is applied. The type is PicaDataFragLightEnvFresnel.
shadowSelector PicaDataFragLightEnvTexture Sets textures whose shadow adjustment element is to be sampled. The type is PicaDataFragLightTexture.
bumpMode PicaDataFragLightEnvBump Sets bump mapping. The type is PicaDataFragLightEnvBump.
bumpSelector PicaDataFragLightEnvTexture Sets textures to be used as normal maps and tangent maps. The type is PicaDataFragLightTexture.
isEnableShadowPrimary bool Sets whether shadow attenuation is to be applied to the primary color. The type is bool.
isEnableShadowSecondary bool Sets whether shadow attenuation is to be applied to the secondary color. The type is bool.
isEnableShadowAlpha bool Sets whether shadow attenuation is to be applied to the alpha component. The type is bool.
isInvertShadow bool Sets whether shadow attenuation is to be inverted. The type is bool.
isEnableBumpRenorm bool Sets whether the z-component of the normal map is to be regenerated. The type is bool.
isEnableClampHighLights bool Sets whether the specular color is to be clamped to 0 when L.n is less than 0. The type is bool.
isEnableLutD0 bool If true, the lookup table value or 1 is set for Distribution 0. The type is bool. The initial value is false.
isEnableLutD1 bool If true, the lookup table value or 1 is set for Distribution 1. The type is bool. The initial value is false.
isEnableLutRefl bool If true, the lookup table value or 1 is set for Reflection. The type is bool. The initial value is true.
lutConfigD0 LutConfig Sets the lookup table Distribution 0. The type is nn::gr:CTR::FragmentLight::LutConfig.
lutConfigD1 LutConfig Sets the lookup table Distribution 1. The type is nn::gr:CTR::FragmentLight::LutConfig.
lutConfigSP LutConfig Sets the lookup table Specular. The type is nn::gr:CTR::FragmentLight::LutConfig.
lutConfigFR LutConfig Sets the lookup table FragmentLight. The type is nn::gr:CTR::FragmentLight::LutConfig.
lutConfigRB LutConfig Sets the lookup table RB. The type is nn::gr:CTR::FragmentLight::LutConfig.
lutConfigRG LutConfig Sets the lookup table RG. The type is nn::gr:CTR::FragmentLight::LutConfig.
lutConfigRR LutConfig Sets the lookup table RR. The type is nn::gr:CTR::FragmentLight::LutConfig.
