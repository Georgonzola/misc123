nn::gr::CTR::FragmentLight::Source::SetDistAttnScaleBiasDirectly Member Function

Syntax

void SetDistAttnScaleBiasDirectly(
     const f32 scale_float20,
     const f32 bias_float20
);

Parameters

Name Description
in scale_float20 The distance attenuation scale (20-bit floating point number).
in bias_float20 The distance attenuation bias (20-bit floating point number).

Return Values

None.

Description

Sets the scale and bias for distance attenuation. You must specify a value already converted to a 20-bit floating point number as an argument.

No internal argument check is made.

Normally, you should use the nn::gr::CTR::FragmentLight::Source::SetDistAttnScaleBias function.
