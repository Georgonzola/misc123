nn::gr::CTR::FrameBuffer::DepthStencilBuffer Class

Syntax

class DepthStencilBuffer

Description

Class for configuring depth and stencil buffer settings.

Member Variables

virtualAddr uptr The depth and stencil buffer virtual address. The type is uptr.
format util::SizedEnum1< PicaDataDepth > The depth and stencil buffer format. The type is PicaDataDepth.
width s32 The depth and stencil buffer width. The type is f32.
height s32 The depth and stencil buffer height. The type is f32.
clearDepth f32 Depth buffer clear value. The type is f32. The initial value is 1.0f.
clearStencil u8 Stencil buffer clear value. The type is u8. The initial value is 0.

Member Functions

DepthStencilBuffer Initializes the depth and stencil buffers to the initial values.
MakeCommand Generates a rendering command based on information already set.
