nn::gr::CTR::LookUpTable::SetAbsTable Member Function

Syntax

void SetAbsTable(
     const f32 table[LOOKUP_TABLE_ELEMENT_NUM],
     const f32 lastDiffValue = 0.0f
);

Parameters

Name Description
in table[LOOKUP_TABLE_ELEMENT_NUM] 256 units of lookup data. Make the 0th item correspond to an input value of 0.0.0 and the 255th item correspond to an input value of 1.0.
in lastDiffValue The last difference value of the lookup data.

Return Values

None.

Description

Configures a lookup table with input values in the range from 0.0 through 1.0.
