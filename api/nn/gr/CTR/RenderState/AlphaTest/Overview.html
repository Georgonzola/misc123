nn::gr::CTR::RenderState::AlphaTest Class

Syntax

class AlphaTest

Description

Class for setting the alpha test.

Member Variables

isEnable bool Sets enable/disable for the alpha test. The type is bool. Always passes if disabled. The initial value is false (disabled).
refValue u8 Sets the constant to be referenced by the alpha test. The type is u8. Sets the value resulting from mapping the uniform value [0.f, 1.f] to an 8-bit integer. The initial value is 0.
func util::SizedEnum1< PicaDataAlphaTest > Sets the comparison method to be used in the alpha test. The type is PicaDataAlphaTest. The initial value is PICA_DATA_ALPHA_TEST_ALWAYS.

Member Functions

AlphaTest Initializes alpha test settings to the initial values.
MakeCommand Generates a rendering command based on information already set.
S MakeDisableCommand Generates commands to disable this feature.
