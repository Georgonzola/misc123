nn::gr::CTR::RenderState::Blend Class

Syntax

class Blend

Description

Class for configuring blend settings. By setting each value and calling the MakeCommand function, you can generate the render command. Because registers are shared, this cannot be used at the same time as logical operation settings.

Member Variables

isEnable bool Setting to enable or disable blending. The type is bool. The initial value is false (disabled). You can only enable logical operations or blending, and not both.
eqRgb util::SizedEnum1< PicaDataBlendEquation > Sets the blend method for RGB components. The type is PicaDataBlendEquation. The initial value is PICA_DATA_BLEND_EQUATION_ADD.
eqAlpha util::SizedEnum1< PicaDataBlendEquation > Sets the blend function for the alpha component. The type is PicaDataBlendEquation. The initial value is PICA_DATA_BLEND_EQUATION_ADD.
srcRgb util::SizedEnum1< PicaDataBlendFunc > Sets the blend RGB component source. The type is PicaDataBlendFunc. The initial value is PICA_DATA_BLEND_FUNC_SRC_ALPHA.
srcAlpha util::SizedEnum1< PicaDataBlendFunc > Sets the blend alpha component source. The type is PicaDataBlendFunc. The initial value is PICA_DATA_BLEND_FUNC_SRC_ALPHA.
dstRgb util::SizedEnum1< PicaDataBlendFunc > Sets the blend RGB component destination. The type is PicaDataBlendFunc. The initial value is PICA_DATA_BLEND_FUNC_ONE_MINUS_SRC_ALPHA.
dstAlpha util::SizedEnum1< PicaDataBlendFunc > Sets the blend ALPHA component destination. The type is PicaDataBlendFunc. The initial value is PICA_DATA_BLEND_FUNC_ONE_MINUS_SRC_ALPHA.
colorR u8 Sets the red component to use for the blend constant color. The type is u8. Specify a value in the range [0, 255]. The initial value is 255.
colorG u8 Sets the green component to use for the blend constant color. The type is u8. Specify a value in the range [0, 255]. The initial value is 255.
colorB u8 Sets the blue component to use for the blend constant color. The type is u8. Specify a value in the range [0, 255]. The initial value is 255.
colorA u8 Sets the alpha component to use for the blend constant color. The type is u8. Specify a value in the range [0, 255]. The initial value is 255.

Member Functions

Blend Initializes the blend settings to the initial values.
MakeCommand Generates a rendering command based on information already set.
S MakeDisableCommand Disables blending.
