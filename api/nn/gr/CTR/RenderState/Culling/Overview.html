nn::gr::CTR::RenderState::Culling Class

Syntax

class Culling

Description

Class for setting culling.

Enumerated Types

FrontFace Specifies the front-facing direction for polygons.
CullFace Specifies the polygon face to be culled.

Member Variables

isEnable bool Enables or disables culling. The type is bool. Both faces are drawn if disabled. The initial value is true (enabled).
frontFace util::SizedEnum1< FrontFace > Sets the front-facing direction for polygons. The type is nn::gr::CTR::RenderState::Culling::FrontFace. The initial value is FRONT_FACE_CCW (counterclockwise).
cullFace util::SizedEnum1< CullFace > Sets the polygon face to be culled. The type is nn::gr::CTR::RenderState::Culling::CullFace. The initial value is CULL_FACE_BACK.

Member Functions

Culling Initializes using the initial values of the culling setting.
MakeCommand Generates a rendering command based on information already set.
S MakeDisableCommand Disables culling.
