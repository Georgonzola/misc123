nn::gr::CTR::RenderState::WBuffer Class

Syntax

class WBuffer

Description

Class for setting the w buffer.

Member Variables

wScale f32 Scale value when the w buffer is enabled. The type is f32. If wScale is 0, the w buffer is disabled. The initial value is 0.
isEnablePolygonOffset bool Enables or disables the polygon offset. The type is bool. The polygon offset can generally be used at the same time as the w buffer.
polygonOffsetUnit f32 The scaling value corresponding to the minimum resolvable depth value. The type is f32. The minimum resolvable depth value depends on the bit value of the depth buffer. The initial value is 0. When the w buffer is enabled and the depth buffer is 24 bits, this applies a constant value to the polygon offset to prevent disappearing due to precision errors.
depthRangeNear f32 Sets the depth value for the near plane in the clip space. The type is f32. The initial value is 0. Set so that 0 <= depthRangeNear <= depthRangeFar <= 1. If the w buffer is enabled, this setting is ignored and 0 is set.
depthRangeFar f32 Sets the depth value for the far plane in the clip space. The type is f32. The initial value is 1. Set so that 0 <= depthRangeNear <= depthRangeFar <= 1. If the w buffer is enabled, this setting is ignored and 1 is set.
depthRangeBit u8 Sets the bit value of the depth buffer. The type is f32. The initial value is 24.

Member Functions

WBuffer Initializes w buffer settings to the initial values.
MakeCommand Generates a rendering command based on information already set.
