nn::gr::CTR::Shader::SearchBindSymbolNum Member Function

Syntax

u32 SearchBindSymbolNum(
     const BindSymbol::ShaderType shader_type,
     const BindSymbol::SymbolType symbol_type
) const;

Parameters

Name Description
in shader_type The type of shader to search for.
in symbol_type The type of symbol to search for.

Return Values

Returns the number of symbols that were found.

Description

Finds the number of symbols of the specified type. Finds the number of symbols of the specified type among those set by pragma_bind_symbol in the shader code. Call this function after you have configured the shader binary with the SetupBinary function.
