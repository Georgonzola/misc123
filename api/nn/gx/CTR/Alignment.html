nn::gx::CTR::Alignment Enumerated Type

Syntax

enum Alignment;

Value

Value Description
TEXTURE_ALIGNMENT Specifies the number of bytes that textures (2D textures and environment maps) must be aligned to. Defined as 128 bytes.
VERTEXBUFFER_ALIGNMENT_FLOAT Specifies the number of bytes that the GLfloat vertex buffers must be aligned to. Defined as 4 bytes.
VERTEXBUFFER_ALIGNMENT_SHORT Specifies the number of bytes that the GLshort and GLushort vertex buffers must be aligned to. Defined as 2 bytes.
VERTEXBUFFER_ALIGNMENT_BYTE Specifies the number of bytes that the GLbyte and GLubyte vertex buffers must be aligned to. Defined as 1 bytes.
RENDERBUFFER_ALIGNMENT_COLOR Specifies the number of bytes that the render buffers (color buffers) must be aligned to. Defined as 64 bytes.
RENDERBUFFER_ALIGNMENT_DEPTH16 Specifies the number of bytes that the render buffers (16-bit depth) must be aligned to. Defined as 32 bytes.
RENDERBUFFER_ALIGNMENT_DEPTH24 Specifies the number of bytes that the render buffers (24-bit depth) must be aligned to. Defined as 96 bytes.
RENDERBUFFER_ALIGNMENT_DEPTH24_STENCIL8 Specifies the number of bytes that the render buffers (24-bit depth + 8-bit stencil) must be aligned to. Defined as 64 bytes.
DISPLAYBUFFER_ALIGNMENT Specifies the number of bytes that the display buffers must be aligned to. Defined as 16 bytes.
COMMANDBUFFER_ALIGNMENT Specifies the number of bytes that the 3D command buffers must be aligned to. Defined as 16 bytes.
SYSTEMBUFFER_ALIGNMENT_MULTIPLEOF4 Specifies the number of bytes that the system buffers for which the allocation size is a multiple of 4 bytes must be aligned to. Defined as 4 bytes.
SYSTEMBUFFER_ALIGNMENT_MULTIPLEOF2 Specifies the number of bytes that the system buffers for which the allocation size is not a multiple of 4 bytes but is a multiple of 2 bytes must be aligned to. Defined as 2 bytes.
SYSTEMBUFFER_ALIGNMENT_OTHER Specifies the number of bytes that the other system buffers must be aligned to. Defined as 1 bytes.
BUFFER_ALIGNMENT_SWAP_BUFFERS Specifies the number of bytes that the buffer address displayed with the nngxSwapBuffersByAddress function must be aligned to. Defined as 16 bytes.
BUFFER_ALIGNMENT_B2L_TRANSFER Specifies the number of bytes that the transfer source image and transfer destination image addresses specified in the nngxAddB2LTransferCommand function must be aligned to. Defined as 16 bytes.
BUFFER_ALIGNMENT_L2B_TRANSFER Specifies the number of bytes that the transfer source image and transfer destination image addresses specified in the nngxAddL2BTransferCommand function must be aligned to. Defined as 16 bytes.
BUFFER_ALIGNMENT_BLOCK_IMAGE_COPY Specifies the number of bytes that the transfer source image and transfer destination image addresses specified in the nngxAddBlockImageCopyCommand function must be aligned to. Defined as 16 bytes.
BUFFER_ALIGNMENT_MOMORY_FILL Specifies the number of bytes that the target region starting address specified in the nngxAddMemoryFillCommand function must be aligned to. Defined as 16 bytes.
BUFFER_ALIGNMENT_FILTER_BLOCK_IMAGE Specifies the number of bytes that the transfer source image and transfer destination image addresses specified in the nngxFilterBlockImage function must be aligned to. Defined as 8 bytes.
BUFFER_ALIGNMENT_3DCOMMAND Specifies the number of bytes that the address of the region that contains the 3D commands specified in the nngxAdd3DCommand function must be aligned to. Defined as 16 bytes.
COMMANDBUFFER_ALIGNMENT_JUMP Specifies the number of bytes that the address of the region storing the command buffer specified in the nngxAddJumpCommand function must be aligned to. Defined as 16 bytes.
COMMANDBUFFER_ALIGNMENT_SUBROUTINE Specifies the number of bytes that the address of the region storing the command buffer specified in the nngxAddSubroutineCommand function must be aligned to. Defined as 16 bytes.
COMMANDREQUEST_ALIGNMENT Specifies the number of bytes that the command request buffer address specified in the nngxCmdlistStorageRaw function must be aligned to. Defined as 4 bytes.

Description

Alignment restrictions.
