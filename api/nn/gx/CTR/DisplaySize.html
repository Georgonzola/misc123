nn::gx::CTR::DisplaySize Enumerated Type

Syntax

enum DisplaySize;

Value

Value Description
DISPLAY0_WIDTH Width of the upper LCD screen (length of the short side).
DISPLAY0_HEIGHT Height of the upper LCD screen (length of the long side).
DISPLAY1_WIDTH Width of the lower LCD screen (length of the short side).
DISPLAY1_HEIGHT Height of the lower LCD screen (length of the long side).

Description

Resolutions of the upper and lower LCD screens.

The "normal position" of the CTR LCD screens is with the short side of the screens positioned horizontally and the lower screen positioned to the left of the upper screen (the CTR turned 90 degrees to the right from the front of the controllers).

The values are defined in reference to this position as follows.

ValueNumerical value
DISPLAY0_WIDTH240
DISPLAY0_HEIGHT400
DISPLAY1_WIDTH240
DISPLAY1_HEIGHT320
