nn::gx::CTR::IsStereoVisionAllowed Function

Syntax

bool IsStereoVisionAllowed(
     void
);

Parameters

None.

Return Values

Returns true if stereoscopic display is allowed, and false otherwise.

Description

Gets whether stereoscopic display is allowed.

Checks whether stereoscopic display is allowed and gets this result. Returns TRUE if allowed and FALSE otherwise.

Stereoscopic display is forcibly disabled if the 3D depth slider position is OFF or it is disallowed in System Settings (restricted by Parental Controls). In other words, when this function returns false, the display is not in 3D even if the application specifies nngxSetDisplayMode(NN_GX_DISPLAYMODE_STEREO). At such times, the display buffer specified in NN_GX_DISPLAY0 is used for display.

Note that the return value of this function does not depend on the display mode state configured with the nngxSetDisplayMode function.
