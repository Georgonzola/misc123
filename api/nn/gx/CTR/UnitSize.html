nn::gx::CTR::UnitSize Enumerated Type

Syntax

enum UnitSize;

Value

Value Description
TOTAL_UNITSIZE_BLOCK_IMAGE_COPY Specifies the number of bytes that the transfer size specified in the nngxAddBlockImageCopyCommand function must be a multiple of. Defined as 16 bytes.
UNIT_UNITSIZE_BLOCK_IMAGE_COPY Specifies the number of bytes that the read/write unit size specified in the nngxAddBlockImageCopyCommand function must be a multiple of. Defined as 16 bytes.
INTERVAL_UNITSIZE_BLOCK_IMAGE_COPY Specifies the number of bytes that the skip size specified in the nngxAddBlockImageCopyCommand function must be a multiple of. Defined as 16 bytes.
BUFFER_UNITSIZE_MOMORY_FILL Specifies the number of bytes that the target region size specified in the nngxAddMemoryFillCommand function must be a multiple of. Defined as 16 bytes.
BUFFER_UNITSIZE_3DCOMMAND_ADD_BUFFER If you specify GL_TRUE for copycmd and copy the data to the 3D command buffer, specifies the number of bytes that the size of the region that contains the 3D commands specified in the nngxAdd3DCommand function. Defined as 4 bytes.
BUFFER_UNITSIZE_3DCOMMAND_ADD_REQUEST If you specify GL_FALSE for copycmd and a 3D command request, specifies the number of bytes that the size of the region that contains the 3D commands specified in the nngxAdd3DCommand function must be a multiple of. Defined as 16 bytes.
BUFFER_UNITSIZE_3DCOMMAND_NO_CACHE_FLUSH Specifies the number of bytes that the size of the region that contains the 3D commands specified in the nngxAdd3DCommandNoCacheFlush function must be a multiple of. Defined as 16 bytes.
COMMANDBUFFER_UNITSIZE_JUMP Specifies the number of bytes that the size of the region storing the command buffer specified in the nngxAddJumpCommand function must be a multiple of. Defined as 16 bytes.
COMMANDBUFFER_UNITSIZE_SUBROUTINE Specifies the number of bytes that the size of the region storing the command buffer specified in the nngxAddSubroutineCommand function must be a multiple of. Defined as 16 bytes.
COMMANDBUFFER_UNITSIZE_FLUSH_3DCOMMAND_PARTIALLY Specifies the number of bytes that the size of the command buffer specified in the nngxFlush3DCommandPartially function must be a multiple of. Defined as 16 bytes.

Description

Size restrictions.
