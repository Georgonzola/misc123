nn::gx::CTR::UpdateBuffer Function

Syntax

void UpdateBuffer(
     const void * pBuffer,
     size_t size
);

Parameters

Name Description
in pBuffer Pointer to the start of the buffer.
in size The buffer size.

Return Values

None.

Description

Applies to the graphics processor the updates that have been made to the contents of the vertex buffer and texture buffer in main memory.

When the GPU directly references texture, vertex, and other buffer data that has been expanded, copied, or overwritten into device memory by the CPU, use this function to reflect those CPU-side changes in the GPU. In addition to these operations, this function invalidates the CPU cache.

Specifically, internal processing consists of flushing the cache, but Nintendo does not support using this function as a generic means of flushing the cache.

Only use this function to reflect CPU changes to the vertex and texture buffers in the GPU.

Caching operations now take longer in proportion to the size of the cache when this function is called from the extended applications that run on SNAKE. Generally use the new nn::gx::CTR::UpdateBufferLight function, which skips cache invalidation for faster execution. Use this function only when the CPU cache must be invalidated, such as when the CPU accesses the GPU rendering results.
