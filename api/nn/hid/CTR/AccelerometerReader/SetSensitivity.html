nn::hid::CTR::AccelerometerReader::SetSensitivity Member Function

Syntax

void SetSensitivity(
     s16 play,
     s16 sensitivity
);

Parameters

Name Description
in play Play. Specify a value from 0 to nn::hid::CTR::MAX_OF_ACCELEROMETER_PLAY, inclusive.
in sensitivity Represents detection sensitivity. Specify a value in the range from 0 to nn::hid::CTR::MAX_OF_ACCELEROMETER_SENSITIVITY. If you specify a value of 0, response ceases completely. If you specify a value of nn::hid::CTR::MAX_OF_ACCELEROMETER_SENSITIVITY, response is based directly on the values obtained from the device.

Return Values

None.

Description

Sets the sensitivity and play tolerance for the change in acceleration between samples.

By default, play is set to 0 and sensitivity is set to nn::hid::CTR::MAX_OF_ACCELEROMETER_SENSITIVITY.
