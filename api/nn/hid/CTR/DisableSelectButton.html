nn::hid::CTR::DisableSelectButton Function

Syntax

void DisableSelectButton();

Parameters

None.

Return Values

None.

Description

Disables sampling of SELECT.

After calling this function, SELECT input can no longer be obtained from the values retrieved by the nn::hid::CTR::PadReader::Read and nn::hid::CTR::PadReader::ReadLatest functions. SELECT is thereafter handled in the same way as START.
