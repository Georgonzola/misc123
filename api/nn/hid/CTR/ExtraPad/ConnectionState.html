nn::hid::CTR::ExtraPad::ConnectionState Enumerated Type

Syntax

enum ConnectionState;

Value

Value Description
CONNECTION_STATE_NO_CONNECTION Not connected to the Circle Pad Pro.
CONNECTION_STATE_CONNECTED A connection to the Circle Pad Pro has been established and its input state is being sampled.
CONNECTION_STATE_STOPPED An established connection with the Circle Pad Pro was lost for some external reason, such as physical removal or a depleted battery. To recover to CONNECTION_STATE_NO_CONNECTION, you must call the StopSampling function. On SNAKE, the system never transitions to this state.

Description

Enumerated type representing the connection states of the Circle Pad Pro.
