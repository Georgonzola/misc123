nn::hid::CTR::ExtraPad::GetConnectionState Member Function

Syntax

static ConnectionState GetConnectionState();

Parameters

None.

Return Values

Returns the connection state of the Nintendo 3DS Circle Pad Pro.

Value Description
CONNECTION_STATE_NO_CONNECTION A connection has not been established between the Nintendo 3DS Circle Pad Pro and the CTR system.
CONNECTION_STATE_CONNECTED A connection has been established between the Nintendo 3DS Circle Pad Pro and the CTR system.
CONNECTION_STATE_STOPPED The Circle Pad Pro was disconnected from the CTR system for some external reason. (It may have been physically removed or its battery may be empty.) (This state is never returned on SNAKE.)

Description

Returns the connection state of the Nintendo 3DS Circle Pad Pro.
