nn::hid::CTR::ExtraPad::Initialize Member Function

Syntax

static void Initialize(
     void * workingMemory,
     size_t workingMemorySize
);

Parameters

Name Description
in workingMemory A WORKING_MEMORY_ALIGNMENT-byte aligned buffer (4096-byte aligned) WORKING_MEMORY_SIZE bytes (12,288 bytes).
in workingMemorySize Specifies the size of the buffer specified by the workingMemory parameter.

Return Values

None.

Description

Performs initialization for using the Circle Pad Pro.

You must execute this function before calling the StartSampling or StopSampling function. If called multiple times when Finalize has not been called, this function does nothing on the second and subsequent calls. Pass the buffer to use for connections and sampling as a parameter.

This buffer must be WORKING_MEMORY_SIZE bytes in size (12,288 bytes) and aligned according to WORKING_MEMORY_ALIGNMENT (4096-byte aligned).

If the buffer passed is less than WORKING_MEMORY_SIZE bytes (12,288 bytes), the function returns PANIC.
