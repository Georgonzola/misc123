nn::hid::CTR::ExtraPad::WORKING_MEMORY_SIZE Member Constant

Syntax

static const size_t WORKING_MEMORY_SIZE =  EXTRA_PAD_WORKING_MEMORY_SIZE;

Description

Constant that specifies the working memory size specified in the Initialize function. Defined as 12,288 bytes.
