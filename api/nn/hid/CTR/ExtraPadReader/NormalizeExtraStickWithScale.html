nn::hid::CTR::ExtraPadReader::NormalizeExtraStickWithScale Member Function

Syntax

void NormalizeExtraStickWithScale(
     f32 * normalized_x,
     f32 * normalized_y,
     s16 x,
     s16 y
);

Parameters

Name Description
out normalized_x x after normalization.
out normalized_y y after normalization.
in x x before normalization.
in y y before normalization.

Return Values

None.

Description

Normalizes and returns the Right Circle Pad values obtained by the Read and ReadLatest functions to floating-point values between -1.0 and 1.0. Includes a sensitivity correction feature.

Includes special internal processing to adjust sensitivity. For more information, see the description of the SetNormalizeExtraStickScaleSettings function.
