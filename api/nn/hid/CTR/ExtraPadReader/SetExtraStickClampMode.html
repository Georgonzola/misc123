nn::hid::CTR::ExtraPadReader::SetExtraStickClampMode Member Function

Syntax

void SetExtraStickClampMode(
     StickClampMode mode
);

Parameters

Name Description
in mode The clamp mode to be set.

Return Values

None.

Description

Sets the clamp mode for the Right Circle Pad.

The configured mode is used to clamp Right Circle Pad values obtained with the Read and ReadLatest functions. For more information about clamping, see PadReader::ExtraStickClampMode.
The default clamp mode is circular clamping.
