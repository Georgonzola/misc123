nn::hid::CTR::GyroscopeReader::GetAccReviseEffect Member Function

Syntax

f32 GetAccReviseEffect() const;

Parameters

None.

Return Values

Always returns 0 if acceleration correction is disabled. Returns 0 or greater if enabled.

Description

Gets the strength of acceleration correction.

The value indicates the strength of correction. The closer this value is to 1, the more the GyroscopeStatus structure's direction obtained by the Read or ReadLatest function is being corrected toward the direction of acceleration.
