nn::hid::CTR::GyroscopeReader::ResetAccReviseParam Member Function

Syntax

void ResetAccReviseParam();

Parameters

None.

Return Values

None.

Description

Initializes the weight and enabled range for acceleration correction.

By default, the weight of acceleration correction (revisePower) is 0.030f and the valid radius of acceleration correction (reviseRange) is 0.400f.
