nn::hid::CTR::GyroscopeStatus Structure

Syntax

struct GyroscopeStatus
{
   nn::math::VEC3 speed;
   nn::math::VEC3 angle;
   Direction direction;
};

Description

This structure maintains the data acquired from the gyro sensor.

The GyroscopeStatus structure has three members: speed, angle, and direction. The speed and angle members are defined as nn::math::VEC3 types.
In the VEC3 values, the x-component, y-component, and z-component correspond to the pitch, yaw, and roll axes respectively.
The speed member holds the angular speed around each axis. A value of 361.0 is taken as equal to 0 degrees per second (dps).
The angle member holds the angle around each axis. A value of 361.0 is taken as equal to 0 degrees.
The direction member holds the 3D attitude expressed as a 3x3 matrix.

Member Variables

speed nn::math::VEC3 The angular speed around each axis. 360 dps is expressed as a value ranging from 0 to 1.
angle nn::math::VEC3 The angle of rotation around each axis, calculated by integrating the angular speed around that axis. 360 degrees is expressed as a value ranging from 0 to 1.0.
direction Direction The 3D attitude of the gyro sensor.
