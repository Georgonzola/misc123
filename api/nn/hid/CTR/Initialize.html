nn::hid::CTR::Initialize Function

Syntax

nn::Result Initialize();

Parameters

None.

Return Values

Returns the execution result of the function. Returns one of the following Result values.

Value Description
Result::IsSuccess Indicates that the operation succeeded.
MakeResultAlreadyInitialized Indicates that the library is already initialized.

Description

Initializes the HID library and enables loading of the sampling data.

Assigns 4 KB of the shared memory allocated by the system as an input buffer to the application's memory space.
This address never overlaps with the heap region used by the application.
Does nothing and returns an error if the nn::hid::CTR::Initialize function has already been called.
