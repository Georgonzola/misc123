nn::hid::CTR::TouchPanelReader Class

Syntax

class TouchPanelReader : private nn::util::NonCopyable< TouchPanelReader >

Description

Class for loading sampling data from the Touch Screen.

The Touch Screen is sampled every 4 ms.

Member Constants

S MAX_READ_NUM const s8 The maximum number of data samples that can be read.

Member Functions

TouchPanelReader Constructor.
~TouchPanelReader Destructor.
Read Reads sampling data from the Touch Screen, starting with the latest samples. Does not load any data that has been previously loaded.
ReadLatest Loads the most recent sampling data from the Circle Pad. Unlike the nn::hid::CTR::TouchPanelReader::Read function, this function can load the same sampling data more than once.
