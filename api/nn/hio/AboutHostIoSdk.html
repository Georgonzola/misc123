About the HostIO Host Interface

The host interface has been included as an SDK in the PARTNER-CTR HIO-Daemon provided by KMC.
For more information about the host interface, see the SDK documentation for the PARTNER-CTR HIO-Daemon. IS-CTR HIO Manager also implements a completely compatible library, so see the documentation listed above.
