nn::hio::CTR::HostDirectoryA Class

Header File: nn/hio.h

Syntax

class HostDirectoryA

Description

Represents directories in the host file system. It can be used to get information about the entries that belong to directories.

Member Functions

HostDirectoryA Constructor. You must make a separate call to Open to open the directory you will be working with.
~HostDirectoryA Destructor. Closes a directory if it is open.
Open Opens a directory. After opening a directory, you can use GetNextEntry to get directory entries that belong to a directory that has already been opened.
Close Closes a directory.
GetNextEntry Gets directory entries.
