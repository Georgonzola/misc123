nn::hio::CTR::HostDirectoryW::Open Member Function

Syntax

#include <nn/hio.h>
Result Open(
     const wchar_t * path
);

Parameters

Name Description
in path Specifies the path of the directory to open.

Return Values

Returns the result of the function.

Description

Opens a directory. After opening a directory, you can use GetNextEntry to get directory entries that belong to a directory that has already been opened.
