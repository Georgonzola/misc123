nn::hio::CTR::Rename Function

Syntax

Result Rename(
     const wchar_t * newName,
     const wchar_t * oldName
);

Parameters

Name Description
in newName New name.
in oldName Path of the file or directory to rename.

Return Values

Returns the execution result.

Description

Changes the file and directory name and returns the result of the process. This is the overloaded member function when the name is specified in Unicode.

Changes the name of the file or directory specified by oldName to that specified by newName.
This is the overloaded member function when the name is specified in Unicode.
