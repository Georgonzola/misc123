nn::hio::CTR::Rename Function

Syntax

Result Rename(
     const char * newName,
     const char * oldName
);

Result Rename(
     const wchar_t * newName,
     const wchar_t * oldName
);

Overloaded Member Functions

Rename ( const char *, const char * ) Changes the file and directory name and returns the result of the process.
Rename ( const wchar_t *, const wchar_t * ) Changes the file and directory name and returns the result of the process. This is the overloaded member function when the name is specified in Unicode.

Description of Rename ( const char *, const char * )

Changes the name of the file or directory specified by oldName to that specified by newName.

Description of Rename ( const wchar_t *, const wchar_t * )

Changes the name of the file or directory specified by oldName to that specified by newName.
This is the overloaded member function when the name is specified in Unicode.
