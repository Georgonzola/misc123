nn::http::Finalize Function

Syntax

Result Finalize(
     void
);

Parameters

None.

Return Values

Returns the execution result of the function. Returns one of the following Result values.


Value Description
Result::IsSuccess Indicates that the operation succeeded.
ResultNotInitializedErr Error indicating a lack of initialization. You do not need to call Finalize.
Other Indicates an unexpected error. (For more information, see http_Result.h.)

Description

Finalizes the HTTP library. Processes that link against the HTTP library must call this after they are done using the library.
