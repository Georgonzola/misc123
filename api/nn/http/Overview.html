nn::http Namespace

Description

The namespace of the HTTP communication library.

This library is provided for HTTP initialization and finalization when using NEX.
Contact Nintendo if you want to use this library for direct communication.

Constants

S BUFFER_ALIGNMENT const s32 Specifies the number of bytes that the POST data buffer address specified in the Initialize function must be aligned to. Defined as 4096 bytes.
S BUFFER_UNITSIZE const s32 Specifies the number of bytes that the size of the POST data buffer specified in the Initialize function must be a multiple of. Defined as 4096 bytes.

Functions

Initialize/Finalize the Entire Library
Initialize Initializes the HTTP library. Processes that link against the HTTP library must call this before using the library.
Finalize Finalizes the HTTP library. Processes that link against the HTTP library must call this after they are done using the library.
