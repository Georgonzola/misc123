nn::ir::CTR::Communicator::CalculateBufferSizeToCommunicate Member Function

Syntax

static size_t CalculateBufferSizeToCommunicate(
     size_t dataSize
);

Parameters

Name Description
in dataSize Data size.

Return Values

Required buffer size.

Description

The size of buffer that is required in order to use the Send and Receive functions.

In communication between systems, management information is appended to data transfers. You must pass a buffer that is larger than the size of data you want to send or receive. This function calculates the required buffer size based on the size of data you want to send or receive.
