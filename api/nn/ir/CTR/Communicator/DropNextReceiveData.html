nn::ir::CTR::Communicator::DropNextReceiveData Member Function

Syntax

static Result DropNextReceiveData(
     s32 * pRemainCount
);

Parameters

Name Description
out pRemainCount Stores the remaining number of receive packets.

Return Values

Returns the result of the function. Returns one of the following Result values.

Value Description
Result::IsSuccess Indicates that the operation succeeded.
ResultNoData No received data.
ResultNotConnected Not in the connected state.
ResultMachineSleep Sleeping because the system is in Sleep Mode.
ResultFatalError The IR module may be malfunctioning.

Description

Ignores the data that is expected to come in the next call to the Receive function, and moves to the next set of receive data after that.
