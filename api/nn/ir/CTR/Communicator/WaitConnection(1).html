nn::ir::CTR::Communicator::WaitConnection Member Function

Syntax

static Result WaitConnection(
     nn::fnd::TimeSpan sendReplyDelay
);

Parameters

Name Description
in sendReplyDelay Specifies the amount of time to delay before sending a connection reply.

Return Values

Returns the result of the function. Returns one of the following Result values.

Value Description
Result::IsSuccess Indicates that the operation succeeded.
ResultBufferInsufficient The buffer was insufficient, so the connection process could not proceed.
ResultAlreadyConnected Already in the connected state.
ResultAlreadyTryingToConnect Connection is being authenticated.
ResultMachineSleep Sleeping because the system is in Sleep Mode.
ResultFatalError The IR module may be malfunctioning.

Description

Waits for a connection request and then tries making an authenticated connection. When a connection request is received, waits to send a connection reply for at least the time specified by sendReplyDelay.
