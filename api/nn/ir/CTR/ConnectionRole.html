nn::ir::CTR::ConnectionRole Enumerated Type

Syntax

enum ConnectionRole;

Value

Value Description
CONNECTION_ROLE_NONE Connected via AnyConnection.
CONNECTION_ROLE_TO_REQUIRE Made the connection request and connected.
CONNECTION_ROLE_TO_WAIT Received the connection request and connected.

Description

Enumerated type that represents whether the local system is server or client in the established connection.
