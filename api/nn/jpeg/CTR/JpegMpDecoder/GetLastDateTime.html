nn::jpeg::CTR::JpegMpDecoder::GetLastDateTime Member Function

Syntax

#include <nn/jpeg.h>
size_t GetLastDateTime(
     char * pBuffer
) const;

Parameters

Name Description
out pBuffer Buffer storing date and time information. The size of the buffer in bytes is DATE_TIME_SIZE (20).
The date and time are obtained as a string with the format "YYYY:MM:DD HH:MM:DD" terminated with 0x00 (20 characters in total).

Return Values

Returns DATE_TIME_SIZE (20) on success. Returns 0 if unsuccessful.

Description

Gets the time stamp (the time of capture) of the image that was either decoded or had Exif information extracted immediately prior to this call.

If that processing fails, this function also fails.
This function needs image data to get the information. Do not destroy the GPS IFD structure.
This information is the information registered in the IFD0 DateTime tag.

If you use GetLastDateTimePointer to change the time stamp of the image, this function returns the modified time stamp.
