nn::jpeg::CTR::JpegMpDecoder::GetLastOrientation Member Function

Syntax

#include <nn/jpeg.h>
bool GetLastOrientation(
     u16 * pBuffer
);

Parameters

Name Description
out pBuffer Buffer storing the image orientation.

Return Values

Returns true on success. Returns false on failure.

Description

Gets the image orientation from the Exif IFD of the image that was either decoded or had Exif information extracted immediately prior to this call.

If that processing fails, this function also fails.
This function needs image data to get the information. Do not destroy the GPS IFD structure.
This information is the information registered in the IFD0 Orientation tag.
