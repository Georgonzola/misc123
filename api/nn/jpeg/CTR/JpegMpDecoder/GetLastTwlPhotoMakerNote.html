nn::jpeg::CTR::JpegMpDecoder::GetLastTwlPhotoMakerNote Member Function

Syntax

#include <nn/jpeg.h>
bool GetLastTwlPhotoMakerNote(
     u8 * pBuffer
) const;

Parameters

Name Description
out pBuffer Buffer used to store the Maker Note. The size of the buffer is TWL_PHOTO_MAKER_NOTE_SIZE (8 bytes).

Return Values

Returns true on success. Returns false on failure.

Description

Gets the Maker Note for the Nintendo DSi Camera of the image that was either decoded or had Exif information extracted immediately prior to this call. ("Nintendo DSi Camera" is the name of a Nintendo DSi built-in application.)

If that processing fails, this function also fails.
This function needs image data to get the information. Do not destroy the GPS IFD structure.

Success of this function does not guarantee that the image was taken with the Nintendo DSi Camera application.
