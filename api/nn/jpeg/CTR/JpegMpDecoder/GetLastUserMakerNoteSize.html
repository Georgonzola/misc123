nn::jpeg::CTR::JpegMpDecoder::GetLastUserMakerNoteSize Member Function

Syntax

#include <nn/jpeg.h>
size_t GetLastUserMakerNoteSize() const;

Parameters

None.

Return Values

If successful, returns the size of the Maker Note in bytes. Returns 0 if unsuccessful.

Description

Gets the size in bytes of the Maker Note for CTR applications of the image that was either decoded or had Exif information extracted immediately prior to this call.

If that processing fails, this function also fails.
This function needs image data to get the information. Do not destroy the GPS IFD structure.

Use the GetLastUserMakerNotePointer function to get a pointer to the Maker Note data.
This function fails if the pointer to data is NULL.

Success of this function does not guarantee that the image was taken with a CTR application.
