nn::jpeg::CTR::JpegMpDecoder::GetLastWidth Member Function

Syntax

#include <nn/jpeg.h>
u32 GetLastWidth() const;

Parameters

None.

Return Values

Image width, in pixels.

Description

Gets the width of the image that was either decoded or had Exif information extracted immediately prior to this call.

If that previous operation failed, returns 0.
