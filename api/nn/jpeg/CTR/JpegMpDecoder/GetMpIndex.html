nn::jpeg::CTR::JpegMpDecoder::GetMpIndex Member Function

Syntax

#include <nn/jpeg.h>
bool GetMpIndex(
     MpIndex * pIndex,
     const u8 * src,
     size_t srcSize
);

Parameters

Name Description
out pIndex Specifies the buffer storing the MP index IFD.
The stored data has been converted to a format that is easier for the library to process, and so the binary representation of a stored MP index IFD differs from that of the original data.
in src Specifies the start of the MP-format data.
The MP index IFD only exists in the starting image, so specify the starting image here.
in srcSize Specifies the size of src in bytes.

Return Values

Returns true on success. Returns false on failure.
Use the GetLastError function to get the reason for failure.

Description

Gets the MP index image file directories (IFDs) included in the data stored in MP format.

Obtained MP index IFDs are accessed using functions such as GetMpEntry or GetMpAttribute.
The region specified by the src and srcSize parameters must not be destroyed until the obtained MP index IFDs are no longer needed.

This function only partially performs decoding and shared processing. Consequently, you cannot call some functions immediately after calling this function: the GetLastWidth, GetLastHeight, GetLastOutputBufferWidth, and GetLastOutputBufferHeight functions, and all functions that get EXIF information, such as the GetLastDateTime function. Calls to these functions have undefined return values.
