nn::jpeg::CTR::JpegMpDecoder::GetMpPanOrientation Member Function

Syntax

#include <nn/jpeg.h>
static bool GetMpPanOrientation(
     u32 * pBuffer,
     const MpAttribute * pAttr
);

Parameters

Name Description
out pBuffer Specifies the buffer that stores the image placement.
in pAttr Specifies the MP attribute IFD. To get this value ahead of time, use the GetMpAttribute function.

Return Values

Returns true on success. Returns false on failure.

Description

Gets the image placement, which is included in the MP attribute IFD.
