nn::jpeg::CTR::JpegMpDecoder Class

Header File: nn/jpeg.h

Syntax

class JpegMpDecoder : private nn::util::NonCopyable< JpegMpDecoder >

Description

This class performs JPEG decoding.

Member Constants

S BUFFER_ALIGNMENT const s32 Specifies the number of bytes that the address of the work buffer specified in Initialize must be aligned to. It is defined as 4 bytes.
S DST_ALIGNMENT const s32 Specifies the number of bytes that the address storing the decoded result of StartJpegDecoder, StartJpegDecoderShrink or StartMpDecoderLR must be aligned to. It is defined as 4 bytes.
