nn::jpeg::CTR::JpegMpEncoder::BUFFER_ALIGNMENT Member Constant

Syntax

#include <nn/jpeg.h>
static const s32 BUFFER_ALIGNMENT =  4;

Description

Specifies the number of bytes that the address of the work buffer specified in Initialize must be aligned to. It is defined as 4 bytes.
