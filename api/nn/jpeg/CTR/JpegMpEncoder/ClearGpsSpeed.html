nn::jpeg::CTR::JpegMpEncoder::ClearGpsSpeed Member Function

Syntax

#include <nn/jpeg.h>
static void ClearGpsSpeed(
     GpsData * pGps
);

Parameters

Name Description
out pGps Specifies the GPS IFD.

Return Values

None.

Description

By default, GPS IFD is instructed not to record the velocity.

If you are using the structure specified by the pGps argument (to store or remove information) for the first time, make sure to initialize it first using InitializeGpsData before calling this function.
If no further operations will be performed on the structure, confirm stored data by calling SetGpsData after calling this function.
