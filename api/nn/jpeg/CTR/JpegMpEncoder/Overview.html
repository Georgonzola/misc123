nn::jpeg::CTR::JpegMpEncoder Class

Header File: nn/jpeg.h

Syntax

class JpegMpEncoder : private nn::util::NonCopyable< JpegMpEncoder >

Description

This class performs JPEG encoding.

Member Constants

S BUFFER_ALIGNMENT const s32 Specifies the number of bytes that the address of the work buffer specified in Initialize must be aligned to. It is defined as 4 bytes.
S SRC_ALIGNMENT const s32 Specifies the number of bytes that the address of the buffer for image data specified by StartJpegEncoder, StartMpEncoderLR, StartMpEncoderFirst, or StartMpEncoderNext must be aligned to. It is defined as 4 bytes.

Encoding
