nn::jpeg::CTR::JpegMpEncoder::SetGpsDestLatitude Member Function

Syntax

#include <nn/jpeg.h>
static void SetGpsDestLatitude(
     GpsData * pGps,
     char ref,
     const Rational * pValue
);

Parameters

Name Description
out pGps Specifies the GPS IFD.
in ref Specifies the North ('N') or South ('S') latitude of the destination.
This information is not stored if 0x00 is specified.
in pValue Specifies a pointer to the latitude of the destination (numeric value).
This information is not stored if NULL is specified.

Return Values

None.

Description

Stores the latitude of the destination in the GPS IFD.

If you are using the structure specified by the pGps argument (to store or remove information) for the first time, make sure to initialize it first using InitializeGpsData before calling this function.
If no further operations will be performed on the structure, confirm stored data by calling SetGpsData after calling this function.
