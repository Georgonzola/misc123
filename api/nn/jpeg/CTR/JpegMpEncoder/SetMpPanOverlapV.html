nn::jpeg::CTR::JpegMpEncoder::SetMpPanOverlapV Member Function

Syntax

#include <nn/jpeg.h>
void SetMpPanOverlapV(
     const Rational * pValue
);

Parameters

Name Description
in pValue Specifies the vertical overlap.
Specify NULL to clear the value.

Return Values

None.

Description

Registers the vertical overlap to embed in the MP Attribute IFD.

This function is enabled if the StartMpEncoderFirst or StartMpEncoderNext function is used as the encoding function. This function is disabled if the StartMpEncoderLR function is used.

If this function is not called, or if the image's MP type (MpTypeCode) is not MP_TYPE_CODE_MULTI_VIEW_PANORAMA_IMAGE (panoramic image), this value is not embedded.

Call this function before calling the encoding function. The values specified by this function are cleared after the encoding function completes, regardless of whether the encoding function succeeded or failed. When encoding multiple times, you must call this function before each encoding function call. The values specified by this function are also cleared if you specify NULL for the pValue argument, if you call the ClearMpPanOverlapV function, or if you reinitialize the encoder object using the Initialize function.
