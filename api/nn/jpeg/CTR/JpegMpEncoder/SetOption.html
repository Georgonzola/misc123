nn::jpeg::CTR::JpegMpEncoder::SetOption Member Function

Syntax

#include <nn/jpeg.h>
void SetOption(
     u32 option
);

Parameters

Name Description
in option Specifies the options
Use a bitwise OR of EncoderOption values.
Specify JPEG_ENCODER_OPTION_NONE (0) to restore the default operations of the encoding function.

Return Values

None.

Description

Specifies JPEG encoding options.

You can change the operations of the encoding function by specifying options.

The specifications are enabled for two primary images when encoding with StartMpEncoderLR.

Call this function before calling the encoding function. The values specified by this function are cleared after the encoding function completes, regardless of whether the encoding function succeeded or failed. When encoding multiple times, you must call this function before each encoding function call. The values specified by this function are cleared if you reinitialize the encoder object using the Initialize function.

If this function is repeatedly read before calling the encoding function, the last specification in enabled.

The following data storing functions affect APP1 recorded data. These functions do nothing if APP1 is not recorded.

SetDateTimeSetSoftwareSetUserMakerNoteSetImageUidSetOrientationSetGpsData

The data storing functions related to the following MP format affect APP2 recorded data. They have an effect even if APP1 is not recorded.

SetMpTypeFlagsSetMpIndividualNumSetMpPanOrientationSetMpPanOverlapHSetMpPanOverlapVSetMpBaseViewpointNumSetMpConvergenceAngleSetMpBaselineLengthSetMpVerticalDivergenceSetMpAxisDistanceXSetMpAxisDistanceYSetMpAxisDistanceZSetMpYawAngleSetMpPitchAngleSetMpRollAngle
