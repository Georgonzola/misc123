nn::math::ExtractBits Function

Syntax

template <typename T>T ExtractBits(
     bit64 v,
     int pos,
     int len
);

Template Arguments

Name Description
T Return value type.

Parameters

Name Description
in v Value to operate on.
in pos A position relative to the lowest bit.
in len Range to extract.

Return Values

Returns the extracted bit string.

Description

Extracts the bitstring used to maintain the position.

Extracts the number of bits specified by len, starting from the bit specified by pos ([pos+len-1,pos]) from the low-order bits of v.

You can also make it so that 0s are created for bits other than those specified by len, starting from the bit given by pos from the low-order bits of v.
