nn::math::F32ToS16 Function

Syntax

s16 F32ToS16(
     f32 x
);

Parameters

Name Description
in x Value to be converted to s16-type.

Return Values

Returns the value of an s16 type that has the same value as x.

Description

Converts an f32 type into an s16 type.
