nn::math::FCopySign Function

Syntax

f32 FCopySign(
     f32 abs,
     f32 sign
);

Parameters

Name Description
in abs Number to use for the absolute value.
in sign Number to use for the sign.

Return Values

Returns a number that has the absolute value of abs and the sign of sign.

Description

Copies the sign of one number to another number.
