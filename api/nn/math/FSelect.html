nn::math::FSelect Function

Syntax

f32 FSelect(
     f32 cond,
     f32 ifPos,
     f32 ifNeg
);

Parameters

Name Description
in cond Condition for selecting a return value.
in ifPos The return value if cond is 0 or greater.
in ifNeg The return value if cond is negative.

Return Values

Returns ifPos if cond is 0 or more; otherwise, returns ifNeg.

Description

Selects a value depending on whether a number is positive or negative.
