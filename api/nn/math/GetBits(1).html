nn::math::GetBits Function

Syntax

template <typename T>T GetBits(
     bit64 v,
     int pos,
     int len
);

Template Arguments

Name Description
T Return value type.

Parameters

Name Description
in v Value to operate on.
in pos Position from which to start extracting, relative to the lowest bit.
in len Range to extract.

Return Values

Returns the extracted bit string.

Description

Gets a bit string.

Extracts the number of bits specified by len, starting from the bit specified by pos ([pos+len-1,pos]) from the low-order bits of v. The bit position in v specified by pos is taken as the 0th bit.
