Description of GetBits ( bit32, int, int )

Extracts the number of bits specified by len, starting from the bit specified by pos ([pos+len-1,pos]) from the low-order bits of v. The bit position in v specified by pos is taken as the 0th bit.

Description of GetBits ( bit64, int, int )

Extracts the number of bits specified by len, starting from the bit specified by pos ([pos+len-1,pos]) from the low-order bits of v. The bit position in v specified by pos is taken as the 0th bit.
