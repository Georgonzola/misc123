nn::math::MTX23Scale Function

Syntax

NN_MATH_INLINE MTX23 * MTX23Scale(
     MTX23 * pOut,
     const MTX23 * pM,
     const VEC2 * pS
);

Parameters

Name Description
out pOut Pointer to the buffer that receives the result of the calculation. It may point to the same matrix as pM.
in pM Pointer to the input matrix.
in pS Pointer to the vector that stores the scaling value along the direction of each axis.

Return Values

Returns pOut.

Description

Scales a 2x3 matrix.
