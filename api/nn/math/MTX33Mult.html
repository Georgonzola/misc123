nn::math::MTX33Mult Function

Syntax

template <typename TMatrix>NN_FORCE_INLINE TMatrix * MTX33Mult(
     TMatrix * pOut,
     const TMatrix * p1,
     const TMatrix * p2
);

Parameters

Name Description
out pOut Pointer to the buffer that receives the calculation results. The same matrix may be specified for both p1 and p2.
in p1 Pointer to the left-hand matrix.
in p2 Pointer to the right-hand matrix.

Return Values

Returns pOut.

Description

Calculates the product of two 3x3 matrices.
