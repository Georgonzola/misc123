nn::math::MTX34Mult Function

Syntax

NN_FORCE_INLINE MTX34 * MTX34Mult(
     MTX34 * pOut,
     const MTX34 * p,
     f32 f
);

Parameters

Name Description
out pOut Pointer to the buffer that receives the calculation results. May point to the same matrix as p.
in p Pointer to the original matrix.
in f Number to multiply by.

Return Values

Returns pOut.

Description

Calculates a scalar product of a matrix.
