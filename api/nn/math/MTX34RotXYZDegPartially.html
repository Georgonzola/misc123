nn::math::MTX34RotXYZDegPartially Function

Syntax

NN_FORCE_INLINE MTX34 * MTX34RotXYZDegPartially(
     MTX34 * pOut,
     f32 fDegX,
     f32 fDegY,
     f32 fDegZ
);

Parameters

Name Description
out pOut Pointer to the buffer that receives the calculation results.
in fDegX Angle around the x-axis in degrees.
in fDegY Angle around the y-axis in degrees.
in fDegZ Angle around the z-axis in degrees.

Return Values

Returns pOut.

Description

Updates rotation-related elements in the specified matrix.

This function does not update the translation elements (the 4th column) in the matrix specified by pOut .
