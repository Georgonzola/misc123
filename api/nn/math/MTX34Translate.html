nn::math::MTX34Translate Function

Syntax

NN_FORCE_INLINE MTX34 * MTX34Translate(
     MTX34 * pOut,
     const VEC3 * pT
);

Parameters

Name Description
out pOut Pointer to the buffer that receives the calculation results.
in pT Pointer to the vector storing the amount of translation in each of the three axis directions.

Return Values

Returns pOut.

Description

Creates a translation matrix.
