nn::math::MTX43IsIdentity Function

Syntax

NN_MATH_INLINE bool MTX43IsIdentity(
     const MTX43 * p
);

Parameters

Name Description
in p Pointer to the matrix to evaluate.

Return Values

Returns true if p is the identity matrix, and false otherwise.

Description

Determines whether a matrix is a unitary matrix.
