nn::math::MTX44Transpose Function

Syntax

NN_FORCE_INLINE MTX44 * MTX44Transpose(
     MTX44 * pOut,
     const MTX44 * pSrc
);

Parameters

Name Description
in pOut Pointer to the buffer that receives the calculation results. It is not a problem if this points to the same matrix as pSrc.
in pSrc Pointer to the original matrix.

Return Values

Returns pOut.

Description

Creates the transpose of a matrix.
