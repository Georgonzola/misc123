nn::math::QUATExp Function

Syntax

NN_MATH_INLINE QUAT * QUATExp(
     QUAT * pOut,
     const QUAT * q
);

Parameters

Name Description
out pOut Pointer to the buffer that receives the calculation results. It may point to the same quaternion as q.
in q Pointer to the quaternion to use as the exponent. Must be a pure quaternion (w=0).

Return Values

Returns pOut.

Description

Calculates the natural logarithm for base e to the power of a given quaternion.
