nn::math::QUATScale Function

Syntax

NN_MATH_INLINE QUAT * QUATScale(
     QUAT * pOut,
     const QUAT * q,
     f32 scale
);

Parameters

Name Description
out pOut Pointer to the buffer that receives the calculation results. It may point to the same quaternion as q.
in q Pointer to the left-hand value.
in scale Number to multiply by.

Return Values

Returns pOut.

Description

Calculates a scalar multiple of a quaternion.
