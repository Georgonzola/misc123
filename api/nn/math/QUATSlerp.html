nn::math::QUATSlerp Function

Syntax

NN_MATH_INLINE QUAT * QUATSlerp(
     QUAT * pOut,
     const QUAT * q1,
     const QUAT * q2,
     f32 t
);

Parameters

Name Description
out pOut Pointer to the buffer that receives the calculation results. The same quaternion may be specified for both q1 and q2.
in q1 Pointer to the quaternion taken as the start point of the spherical linear interpolation
in q2 Pointer to the quaternion taken as the end point of the spherical linear interpolation
in t Spherical linear interpolation parameter. If 0.0, q1 is the result; if 1.0, q2 is the result.

Return Values

Returns pOut.

Description

Calculates the spherical linear interpolation between two quaternions.
