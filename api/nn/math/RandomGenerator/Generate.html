nn::math::RandomGenerator::Generate Member Function

Syntax

u32 Generate(
     u32 max = 0
);

Parameters

Name Description
in max The maximum value in the range of numbers to get. Can get numbers in the range from 0 through (max - 1). If 0 has been specified, the entire range of 32-bit values is used.

Return Values

A 32-bit random number.

Description

Gets a pseudorandom number. A different random number can be obtained each time the function runs.
