nn::math::TinyMt::GenerateRandomU64 Member Function

Syntax

#include <nn/math.h>
u64 GenerateRandomU64();

Parameters

None.

Return Values

Returns an unsigned 64-bit random integer.

Description

Generates an unsigned 64-bit random integer.

Generates a random number in the range from 0x00000000 00000000 to 0xFFFFFFFF FFFFFFFF.
