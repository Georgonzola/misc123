nn::math::VEC2Lerp Function

Syntax

NN_MATH_INLINE VEC2 * VEC2Lerp(
     VEC2 * pOut,
     const VEC2 * p1,
     const VEC2 * p2,
     f32 t
);

Parameters

Name Description
out pOut Pointer to the buffer that receives the calculation results. The same vector may be specified for both p1 and p2.
in p1 Pointer to the vector taken as the start point of the linear interpolation
in p2 Pointer to the vector taken as the end point of the linear interpolation
in t Linear interpolation parameter. If 0.0, p1 is the result; if 1.0, p2 is the result.

Return Values

Returns pOut.

Description

Calculates a linear interpolation between two vectors.
