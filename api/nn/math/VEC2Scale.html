nn::math::VEC2Scale Function

Syntax

NN_MATH_INLINE VEC2 * VEC2Scale(
     VEC2 * pOut,
     const VEC2 * p,
     f32 scale
);

Parameters

Name Description
out pOut Pointer to the buffer that receives the calculation results. May point to the same vector as p.
in p Pointer to the left-hand value.
in scale Number to multiply by.

Return Values

Returns pOut.

Description

Calculates a scalar multiple of a vector.
