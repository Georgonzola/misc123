nn::math::VEC3Transform Function

Syntax

NN_FORCE_INLINE VEC4 * VEC3Transform(
     VEC4 * pOut,
     const MTX44 * pM,
     const VEC3 * pV
);

Parameters

Name Description
out pOut Pointer to the buffer that receives the calculation results. It may point to the same vector as pV.
in pM Pointer to the transformation matrix.
in pV Pointer to the original vector.

Return Values

Returns pOut.

Description

Transforms a vector using a matrix. The fourth element of the vector is calculated as 1.
