nn::math::VEC4::Minimize Member Function

Syntax

self_type & Minimize(
     const VEC4 & lhs,
     const VEC4 & rhs
);

Parameters

Name Description
in lhs The lhs vector represents the left side of the equation that calculates the minimum values.
in rhs The rhs vector represents the right side of the equation that calculates the minimum values.

Return Values

Description

Creates the vector formed by the smaller of each component of two input vectors.
