nn::mic::CTR::Finalize Function

Syntax

Result Finalize();

Parameters

None.

Return Values

Returns the execution result.

Value Description
Result::IsSuccess Indicates success.
ResultNotInitialized Indicates that the instance is not initialized.

Description

Finalizes the MIC library. Calling this function stops any sampling in progress, sets the microphone amp gain to nn::mic::CTR::AMP_GAIN_DEFAULT_VALUE, and turns off the microphone power.
