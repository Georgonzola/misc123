nn::mic::CTR::SamplingType Enumerated Type

Syntax

enum SamplingType;

Value

Value Description
SAMPLING_TYPE_8BIT Sampling with an effective bit-width of 8 bits (unsigned). The range of output data is 0 to 255.
SAMPLING_TYPE_16BIT Sampling with an effective bit-width of 16 bits (unsigned). The output data ranges from 0 to 65535.
SAMPLING_TYPE_SIGNED_8BIT Sampling with an effective bit-width of 8 bits (signed). The output data ranges from -128 to 127.
SAMPLING_TYPE_SIGNED_16BIT Sampling with an effective bit-width of 16 bits (signed). The output data ranges from -32768 to 32767.

Description

Enumerated type that indicates the microphone sampling type.
