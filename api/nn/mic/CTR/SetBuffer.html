nn::mic::CTR::SetBuffer Function

Syntax

Result SetBuffer(
     void * address,
     size_t size
);

Parameters

Name Description
in address Specifies the starting address of the memory block.
in size Specifies the size of the memory region.

Return Values

Returns the execution result.

Value Description
Result::IsSuccess Indicates success.
ResultAlreadyInitialized A memory region has already been set.
ResultMisalignedSize The specified memory region size is not 4096-byte aligned.
ResultMisalignedAddress The specified memory region address is not 4096-byte aligned.
ResultInvalidHandle The specified memory region is invalid.
ResultInvalidSize The specified size is 0, which is invalid.

Description

Sets the memory region in which to store the sampling results.

Sets the memory that was obtained by the application. Device memory cannot be used.

This function cannot be called multiple times. To call this function more than once, you must first call the nn::mic::CTR::ResetBuffer function.

For the memory region address, specify a value that is a multiple of BUFFER_ALIGNMENT (4096 bytes). For the size, specify a value that is a multiple of BUFFER_UNITSIZE (4096 bytes). The last four bytes of the specified memory region are used by the mic library for administrative purposes. Do not overwrite the last four bytes of this region. The size of the buffer in which the actual sampling results are saved is the size obtained using nn::mic::CTR::GetSamplingBufferSize.
