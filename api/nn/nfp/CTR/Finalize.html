nn::nfp::CTR::Finalize Function

Syntax

nn::Result Finalize();

Parameters

None.

Return Values



Value Description
ResultSuccess Finalized successfully.
ResultInvalidOperation Already finalized.
Other Unexpected error.

Description

Finalizes the NFC library.

Call this function when you are done using the NFP library. The Unmount and StopDetection functions are automatically called as necessary.
