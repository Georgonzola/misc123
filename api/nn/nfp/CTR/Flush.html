nn::nfp::CTR::Flush Function

Syntax

nn::Result Flush();

Parameters

None.

Return Values



Value Description
ResultSuccess Data was written successfully.
ResultInvalidOperation The library is not in the correct state.
ResultSleep The system is in a sleep state.
ResultWifiOff The Wi-Fi switch is set to off.
ResultNotSupported Non-NFP tag, or invalid tag format. Check the tag being used.
ResultTagNotFound Target tag not found. The tag may have been replaced with another tag during the write process.
ResultOperationFailed An error occurred while writing to the tag. The tag data may have been corrupted.
ResultNeedRetry The writing process failed. Retrying may succeed.
ResultConnectCanceled Lost the connection with the NFC reader and writer due to an internal error.
Other Unexpected error.

Description

Writes the data in an internal buffer to the tag. The data is not actually written to the tag until this function is called.

The tag must be mounted using the Mount() function beforehand. Also, calling this API function can block for about one or two seconds.
