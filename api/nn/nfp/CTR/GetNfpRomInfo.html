nn::nfp::CTR::GetNfpRomInfo Function

Syntax

nn::Result GetNfpRomInfo(
     RomInfo * pInfo
);

Parameters

Name Description
out pInfo Pointer to the RomInfo object in which the retrieved data was stored.

Return Values



Value Description
ResultSuccess Obtained successfully.
ResultInvalidPointer An invalid pointer to the parameter was specified.
ResultInvalidOperation The library is not in the correct state.
ResultSleep The system is in a sleep state.
ResultWifiOff The Wi-Fi switch is set to off.
Other Unexpected error.

Description

Gets the data in the ROM region on the NFP tag.

To call this API function, the tag must be mounted with the Mount() or MountRom() function.
