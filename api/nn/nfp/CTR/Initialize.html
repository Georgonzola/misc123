nn::nfp::CTR::Initialize Function

Syntax

nn::Result Initialize();

Parameters

None.

Return Values



Value Description
ResultSuccess Indicates that initialization succeeded.
ResultInvalidOperation The library is already initialized.
ResultNeedRetry Initialization failed. Retrying may succeed.
ResultIrFunctionError The IrDA module may be malfunctioning. This result can only occur on the CTR. Equivalent to ir::CTR::ResultFatalError.
Removing the IrDA module from the debugger forcibly triggers it.
Other Unexpected error.

Description

Initializes the NFP library.

This function must be called before using any of the features in the NFP library. Normally initialization finishes within a few hundred milliseconds. When you initialize the NFP library on that system for the first time, however, it can take a maximum of about five seconds because it is necessary to allocate memory to save backup data for NFP tags.
