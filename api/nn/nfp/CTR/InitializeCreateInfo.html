nn::nfp::CTR::InitializeCreateInfo Function

Syntax

nn::Result InitializeCreateInfo(
     ApplicationAreaCreateInfo * pCreateInfo
);

Parameters

Name Description
out pCreateInfo The creation information about the application area.

Return Values



Value Description
ResultSuccess Indicates that initialization succeeded.
ResultInvalidPointer An invalid parameter was specified for the pointer.
ResultInvalidOperation The library is not in the correct state.
Other Unexpected error. Use this API function to initialize the parameters passed to CreateApplicationArea. Then specify the necessary values for the members.

Description

Initializes creation information about the application area.
