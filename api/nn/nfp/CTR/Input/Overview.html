nn::nfp::CTR::Input Structure

Syntax

struct Input
{
   s32 mode;
   nn::nfp::TagInfo tagInfo;
   bool isRegistered;
   nn::nfp::RegisterInfo registerInfo;
   nn::nfp::CommonInfo commonInfo;
};

Description

Parameter structure to pass to the applet.

Used as a member variable of the Parameter structure. You must always call the InitializeParameter function to initialize this structure before setting its member variables.

Member Variables

mode s32 The mode amiibo Settings starts in. Specify the value of AmiiboSettingsMode.
tagInfo nn::nfp::TagInfo Tag information used to prevent tag swapping.
isRegistered bool Flag indicating whether the tag is registered. Specify TRUE when the GetNfpRegisterInfo returns a value of ResultSuccess.
registerInfo nn::nfp::RegisterInfo Registration information. Specify the value obtained from the GetNfpRegisterInfo function.
commonInfo nn::nfp::CommonInfo Common area information. Specify the value obtained from the GetNfpCommonInfo function.
