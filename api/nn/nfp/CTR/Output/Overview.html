nn::nfp::CTR::Output Structure

Syntax

struct Output
{
   s32 result;
   nn::nfp::TagInfo tagInfo;
   nn::nfp::RegisterInfo registerInfo;
   nn::nfp::CommonInfo commonInfo;
};

Description

Parameter structure received from the applet.

Used as a member variable of the Parameter structure. You must always initialize it first using the InitializeParameter function.

Member Variables

result s32 The processing result. Stores the value of AmiiboSettingsResultCode.
tagInfo nn::nfp::TagInfo The tag information to be changed.
registerInfo nn::nfp::RegisterInfo Registration information. Stores the value changed by amiibo Settings.
commonInfo nn::nfp::CommonInfo Common area information. Stores the value changed by amiibo Settings.
