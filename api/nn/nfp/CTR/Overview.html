nn::nfp::CTR Namespace

Description

The NFP library namespace for CTR (the "::CTR" portion is optional).

The API functions in the NFP library are not intended for multithread use. Do not call NFP library API functions from multiple threads simultaneously.
The NFP library cannot currently be used on CTR.

Structures

nn::nfp::CTR::Input Parameter structure to pass to the applet.
nn::nfp::CTR::Output Parameter structure received from the applet.
nn::nfp::CTR::Parameter Parameter structure to pass to the amiibo Settings applet when it executes.

Enumerated Types

AmiiboSettingsMode The mode that the amiibo Settings starts in.
AmiiboSettingsResultCode The result of amiibo Settings processing.

Functions

Initialization and Shutdown
Initialize Initializes the NFP library.
Finalize Finalizes the NFC library.
Connecting and Disconnecting
Connect Makes the connection to the NFC reader and writer. This function is asynchronous, so get the connection status with the GetTargetConnectionStatus() function.
GetConnectResult Returns the result of the Connect function. When called on SNAKE, the function succeeds without doing anything.
Disconnect Breaks the connection to the NFC reader and writer.
NFP Tag Access
StartDetection Starts detecting a tag.
StopDetection Stops detecting a tag.
Mount Mounts a tag. Can access all of the data on the tag.
MountRom Mounts a tag. Can only access the ROM data on the tag.
Unmount Unmounts a tag.
OpenApplicationArea Enables access to the application area.
CreateApplicationArea Creates the application area on a tag. The application area must be created before using a blank tag for the first time.
ReadApplicationArea Reads the data in the application area on a tag.
WriteApplicationArea Writes data to the application area on a tag.
Flush Writes the data in an internal buffer to the tag. The data is not actually written to the tag until this function is called.
Restore Restores a corrupted tag using the backup data.
GetTagInfo Gets tag information.
GetNfpRegisterInfo Gets initial NFP tag registration info.
GetNfpCommonInfo Gets the data in the common region on the NFP tag.
GetNfpRomInfo Gets the data in the ROM region on the NFP tag.
InitializeCreateInfo Initializes creation information about the application area.
Other
GetNfpState Gets the state of the NFP library.
SetActivateEvent Sets an event indicating that a tag was detected.
SetDeactivateEvent Sets an event indicating that a tag was lost (deactivated).
GetTargetConnectionStatus Gets the connection status for the NFC reader and writer.
GetConnectionStatus Gets the connection status for the tag.
ReplaceOtherRegionName Converts the amiibo nickname for display in the current region.
InitializeParameter Initializes the Parameter structure to default values.
InitializeParameterForUpdate Initializes the Parameter structure to default values. (For updating the NFC target.)
StartAmiiboSettings Starts amiibo Settings.
IsAmiiboSettingsAvailable Checks whether amiibo Settings has been installed.
