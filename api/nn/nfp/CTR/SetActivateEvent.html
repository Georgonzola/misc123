nn::nfp::CTR::SetActivateEvent Function

Syntax

nn::Result SetActivateEvent(
     nn::os::Event * pEvent
);

Parameters

Name Description
in pEvent Specifies an event to use for tag detection notification.

Return Values



Value Description
ResultSuccess The event was set successfully.
ResultInvalidPointer An invalid pointer to the parameter was specified.
ResultInvalidOperation The library is not in the correct state.
Other Unexpected error.

Description

Sets an event indicating that a tag was detected.

Call this function after the NFP library is initialized and before the StartDetection function starts tag detection. When a tag is detected, a notification with the event specified in this function is sent. This event occurs even when non-NFP tags are detected.

The NFC reader and writer can detect amiibo tags. The New Nintendo 3DS can detect amiibo and various (Japanese) transportation-related integrated circuit cards (Suica, PASMO, Kitaca, TOICA, manaca, ICOCA, SUGOCA, nimoca, and Hayakaken).

Other NFC tags may be detectable, but operations with the NFP library are not guaranteed.

The event is managed within the API function, so there is no need for the user to initialize or release it. Note: The event is initialized to isManualReset = false .
