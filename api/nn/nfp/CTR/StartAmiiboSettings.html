nn::nfp::CTR::StartAmiiboSettings Function

Syntax

bool StartAmiiboSettings(
     Parameter * pParameter
);

Parameters

Name Description
inout pParameter Parameter structure.

Return Values

Returns whether the process was successful.

Value Description
true Success.
false Failure. If pParameter is not NULL, it stores the result in pParameter.output.result.

Description

Starts amiibo Settings.

When using the NFP library, use the nn::nfp::CTR::Finalize function to finalize the library before calling this function.

If you specify NULL for pParameter, the function returns false without doing anything.

pParameter sends its data to amiibo Settings, but it is also used as a place to store the data from amiibo Settings.

When using this function, check the various items described in nn::nfp::CTR.
