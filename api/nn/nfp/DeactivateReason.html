nn::nfp::DeactivateReason Enumerated Type

Syntax

enum DeactivateReason;

Value

Value Description
BEFORE_ACTIVATION Indicates that a tag has not yet been found or accessed.
TAG_IS_ACTIVE Connecting to the tag.
DETECT_ERROR Could not be recognized as a valid tag. The tag data could be corrupted.
CONNECTION_LOST Indicates that the connection could not be maintained because the tag or the signal was lost.
UNKNOWN_DEACTIVATE_REASON Unknown reason.
DISCONNECT_REASON_MAX

Description

Enumerated type representing the reasons for disconnection.
