nn::nfp::NfpState Enumerated Type

Syntax

enum NfpState;

Value

Value Description
NONE The NFC library is in an uninitialized state. It starts in this state.
INIT The NFC library is in an initialized state.
RW_SEARCH The NFC library is searching for tags.
RW_ACTIVE The NFC library enters this state when tags are detected with RW_SEARCH.
RW_DEACTIVE The NFC library enters this state when a tag is detected and later removed.
RW_MOUNT The NFC library enters this state when a nearby tag is mounted with RW_ACTIVE.
RW_MOUNT_ROM The NFC library enters this state when the ROM of a nearby tag is mounted with RW_ACTIVE.
UNEXPECTED This state is unexpected.

Description

Enumerates library states.
