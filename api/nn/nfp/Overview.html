nn::nfp Namespace

Description

Namespace for the NFP library.

Namespaces

nn::nfp::CTR The NFP library namespace for CTR (the "::CTR" portion is optional).

Classes

Result
nn::nfp::ResultInvalidUsage An API function is not being used correctly.
nn::nfp::ResultBufferIsSmall The specified buffer is too small. This error belongs to ResultInvalidUsage.
nn::nfp::ResultInvalidArgument Invalid argument. Revise the specified arguments. This error belongs to ResultInvalidUsage.
nn::nfp::ResultInvalidPointer An invalid pointer was specified. This error belongs to ResultInvalidUsage and ResultInvalidArgument.
nn::nfp::ResultInvalidAlignment Invalid alignment. This error belongs to ResultInvalidUsage and ResultInvalidArgument.
nn::nfp::ResultNeedRetry The communication with the tag may be unstable. Call the same function again.
nn::nfp::ResultIsBusy Indicates that the system is busy and cannot currently receive commands. This error belongs to ResultNeedRetry.
nn::nfp::ResultTimeOutError Indicates a timeout error. This error belongs to ResultNeedRetry.
nn::nfp::ResultConnectionError Indicates a connection error. The tag needs to be passed over the reader again. This error belongs to ResultNeedRetry.
nn::nfp::ResultNotFound The process target was not found.
nn::nfp::ResultTagNotFound Tag not found. The tag may have been removed from the reader or replaced during the process. This error belongs to ResultNotFound.
nn::nfp::ResultCharacterNotFound No character with the specified ID was found. Check that the ID is correct. This error belongs to ResultNotFound.
nn::nfp::ResultBackupError Failed to access backup data.
nn::nfp::ResultNoBackupFile This API function cannot be called because there is no backup file. This error belongs to ResultBackupError.
nn::nfp::ResultBackupOpenFailed Failed to open the backup file. This error belongs to ResultBackupError.
nn::nfp::ResultBackupReadFailed Failed to load backup file data. This error belongs to ResultBackupError.
nn::nfp::ResultBackupWriteFailed Failed to write to the backup file. This error belongs to ResultBackupError.
nn::nfp::ResultFailed The process failed.
nn::nfp::ResultOperationFailed The requested process failed. Check that you are using the API function correctly. This error belongs to ResultFailed.
nn::nfp::ResultFatalError A fatal error has occurred.
nn::nfp::ResultUnexpected Unexpected error. A fatal system error may have occurred. This error belongs to ResultFatalError.
nn::nfp::ResultInvalidOperation An invalid request was issued. Check the API function call order. This error can occur even when the API function call order is correct, based on the timing at which tags were released.
nn::nfp::ResultNotSupported The tag is not supported. Check that the tag is correct.
nn::nfp::ResultNeedRestore The tag content is corrupted. Call the Restore function to use the existing backup data to restore the tag content.
nn::nfp::ResultNeedFormat The tag data is corrupted. The tag data cannot be restored because there is no backup data. Prompt the user to initialize amiibo Settings.
nn::nfp::ResultNeedCreate The application area has not been created. Call the CreateApplicationArea function to create the application area.
nn::nfp::ResultNeedRegister The common area has not been created. It must be registered in amiibo Settings.
nn::nfp::ResultAlreadyCreated The application area has already been created.
nn::nfp::ResultAccessIdMisMatch Cannot access the application area because the specified Access ID does not match the ID written to the tag.
nn::nfp::ResultNotBroken Tag data does not need to be restored because it is not corrupted.
nn::nfp::ResultUidMisMatch UID mismatch.
nn::nfp::ResultInvalidFormatVersion Unsupported format version. Notify the user that the application does not support this amiibo.
nn::nfp::ResultNotImplemented Specifies not implemented.
nn::nfp::ResultSleep The system is in a sleep state.
nn::nfp::ResultWifiOff The Wi-Fi switch is set to off.
nn::nfp::ResultUpdateRequired The NFC reader/writer must be updated.
nn::nfp::ResultIrFunctionError The IdDA module may be malfunctioning. Equivalent to ir::CTR::ResultFatalError.
nn::nfp::ResultNfcTargetError The NFC reader/writer could be malfunctioning.
nn::nfp::ResultConnectCanceled The connection to the NFC reader/writer was either canceled or lost.
