Description of MaskProfanityWordsInText ( int *, wchar_t * )

This function processes text strings equal to or longer than a particular length. The maximum number of characters (including the null terminator) for the text is nn::ngc::MAX_TEXT_LENGTH. It cannot process short single words, such as screen names. To check words, use the ProfanityFilter::CheckProfanityWords function instead.

When this function is called, it checks the specified strings to determine whether they contain profanity and masks those parts with asterisks (*).

This operation may take a long time. (It blocks.) Processing time can vary, depending on such factors as the version of the system and the length of the strings. We recommend running this function in a separate thread. Running it in the main thread can reduce the frame rate.

To determine which pattern lists you need to check against, see the UGC guidelines in the latest 3DS Guidelines.
