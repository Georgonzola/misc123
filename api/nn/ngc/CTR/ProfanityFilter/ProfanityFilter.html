Description of ProfanityFilter ( const nn::WithInitialize & )

Uses MemoryBlock to secure a working memory region. This regions consists of ProfanityFilter::WORKMEMORY_SIZE bytes (64 KB) of memory

Description of ProfanityFilter ( uptr )

The specified memory region is used as the working area. Do not write any data to this working area, or release the area while it is in use. This class does not release the specified memory region when it has finished using it, so the area must be managed by the application side. You must allocate ProfanityFilter::WORKMEMORY_SIZE (64 KB) of memory, and the region must be aligned to WORKMEMORY_ALIGNMENT (4 bytes).
