nn::ngc::CTR::ProfanityFilter::SetMaskMode Member Function

Syntax

void SetMaskMode(
     bool bOverWrite
);

Parameters

Name Description
in bOverWrite Specify true (the default setting) to overwrite the entire text with asterisk symbols when profanity is discovered. Specify false to substitute a single asterisk symbol for any discovered profanity word.

Return Values

None.

Description

Configures how to respond when text is checked and profanity is discovered.

In the default mode, the entire text is overwritten with asterisk symbols when profanity is discovered. In some cases, this can result in display of text outside of the area of the box on the screen. For example, the asterisk symbol ("*") is wider than the letter "i" when a proportional font is used. When overwriting the text with asterisk symbols, there is a risk that the result will extend beyond the screen.

By specifying false for this function, only one asterisk is displayed in substitution for a profanity word, regardless of the number of characters in the profanity word. As a result, there is no danger to the integrity of the user interface.

This setting is only enabled when text is checked using the ProfanityFilter::MaskProfanityWordsInText function. It has no effect on operations when words are being checked by the ProfanityFilter::CheckProfanityWords function.

The specified setting is reset to the default state (true) when the ProfanityFilter::Finalize function is called.
