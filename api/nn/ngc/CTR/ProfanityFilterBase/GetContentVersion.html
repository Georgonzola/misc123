nn::ngc::CTR::ProfanityFilterBase::GetContentVersion Member Function

Syntax

virtual u32 GetContentVersion() = 0;

Parameters

None.

Return Values

The version number. Versions begin from 1, with higher numbers indicating newer versions. Returns 0 if the attempt to get the value fails.

Description

Gets the version number of the profanity pattern file currently installed on the system.
