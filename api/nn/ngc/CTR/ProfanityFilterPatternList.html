PATTERNLIST_TAIWAN_ENGLISH This definition cannot be used.
PATTERNLIST_MAX Represents the number of pattern lists.

Description

Enumerated type listing the patterns of invalid strings for each region and language.

For more information about which lists to use for invalid string checking, see the UGC guidelines in the most recent 3DS Guidelines. Profanity words for testing purposes are registered in these lists so that you can check whether the library has been correctly incorporated in the application.
