nn::nstd::ARMv6::MemMove Function

Syntax

void MemMove(
     void * dstp,
     const void * srcp,
     size_t size
);

Parameters

Name Description
out dstp Copy destination address.
in srcp Copy source address.
in size The number of bytes to copy.

Return Values

None.

Description

Uses the CPU to perform memory copy. The function acts normally even if the copy source region overlaps the copy destination region.

When alignment is undefined, this function is faster than std::memmove because at run time it selects the optimal method based on the alignment of the copy source and destination addresses. For u32* and structures, and in other cases where alignment is determined statically and the copy source and destination addresses are aligned the same, use std::memmove.
