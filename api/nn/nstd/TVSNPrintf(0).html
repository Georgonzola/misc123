nn::nstd::TVSNPrintf Function

Syntax

s32 TVSNPrintf(
     char * dst,
     size_t len,
     const char * fmt,
     va_list vlist
);

Parameters

Name Description
out dst Specifies the buffer that stores the result.
in len Specifies the length of the dst buffer.
in fmt Specifies a general printf formatted text string. However, it cannot specify floating point strings (for example, %f).
in vlist Specifies the parameter.

Return Values

Returns the number of characters of the formatted string that is correctly output.

Description

Outputs a string. Memory-conserving version of vsnprintf.
