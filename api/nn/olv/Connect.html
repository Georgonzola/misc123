nn::olv::Connect Function

Syntax

nn::olv::Result Connect(
     nn::olv::ConnectedData * connectedData,
     const nn::olv::ConnectParam * connectParam
);

Parameters

Name Description
out connectedData The class that stores the connection results.
in connectParam The class that stores the parameters used for the connection.

Return Values

Returns the result of the operation.
For information about handling the result, see the section about Error Handling or the descriptions of the nn::olv::Result values.


Value Description
ResultSuccess Succeeded.
ResultUsageError A usage error.
ResultFatalError A fatal error.
ResultCanceled Canceled.
ResultNetworkError A network error.
ResultServerPermanentError A permanent server error.
ResultServerTemporaryError A temporary server error.

Description

Processes OLV library connections.

Before calling this function you must use the nn::cfg::CTR::IsAgreedEula function and confirm that the user has agreed to the EULA.
If the EULA has not been agreed to, this function returns an error. For information about what to do when the EULA has not been agreed to, see the 3DS Guidelines.

Before calling this function, you must initialize the HTTP library by calling the nn::http::Initialize function.
This function initiates Internet communication, during which time processing is blocked.
This function is not thread-safe. Make sure not to execute any OLV library functions or class methods on another thread while calling this function. While this function is being called, you can call the nn::olv::Cancel() function from another thread to cancel communication processes.

This function returns an error if an old service token has been specified for the connectParam parameter. When calling this function or retrying this function, specify the new service token that was just obtained by the nn::act::applet::AcquireOlvServiceToken function.
