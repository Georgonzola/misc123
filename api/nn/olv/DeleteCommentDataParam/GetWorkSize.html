nn::olv::DeleteCommentDataParam::GetWorkSize Member Function

Syntax

u32 GetWorkSize() const;

Parameters

None.

Return Values

Returns the size of the working buffer used by the nn::olv::DeleteCommentData function.

Description

Gets the size of the working buffer used by the nn::olv::DeleteCommentData function.

Call this function after all of the various parameters have been specified, with the exception of the SetWork function.
