nn::olv::DownloadCommentDataListParam::Flags Enumerated Type

Syntax

enum Flags;

Value

Value Description
FLAG_NONE The state when no flags are specified.
FLAG_FILTER_BY_FRIEND This flag cannot be used. It will be deprecated in the next version.
FLAG_FILTER_BY_FOLLOW This flag cannot be used. It will be deprecated in the next version.
FLAG_FILTER_BY_SELF This flag cannot be used. It will be deprecated in the next version.
FLAG_FILTER_BY_BODY_TEXT Download only text comments.
You cannot specify FLAG_FILTER_BY_BODY_MEMO at the same time.
This flag is disabled if the SetCommentId function is used to specify a comment ID.
FLAG_FILTER_BY_BODY_MEMO Download only handwritten memo comments.
You cannot specify FLAG_FILTER_BY_BODY_TEXT at the same time.
This flag is disabled if the SetCommentId function is used to specify a comment ID.
FLAG_DISTINCT_PID Get only a single comment from each user.
This flag is disabled if the SetCommentId function is used to specify a comment ID.
FLAG_WITH_MII_DATA Include Mii data in the downloaded comment data.
FLAG_WITH_EMPATHY_ADDED Includes a flag for whether comments being downloaded have been given Yeahs by the local host.
FLAG_WITH_SPOILER Also downloads comments with spoilers and comments with spoilers in review.
This flag is disabled if the SetCommentId function is used to specify a comment ID.

Description

Flags that can be specified in the SetFlags function.

When FLAG_WITH_SPOILER is not specified, comments with spoilers or comments with spoilers in review are not downloaded.
