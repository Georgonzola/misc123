nn::olv::DownloadCommentDataListParam Class

Syntax

class DownloadCommentDataListParam

Description

Represents the parameters when downloading comment data.

Enumerated Types

Flags Flags that can be specified in the SetFlags function.

Member Functions

DownloadCommentDataListParam Constructor.
Reset Reverts parameters to their initial state.
SetFlags Sets flags.
SetPostId Sets the post ID for the comment data to download.
SetCommentId Sets the comment ID.
SetPostDate Sets the date the comment was made.
SetLanguageId Sets the language ID.
SetBodyTextMaxLength Sets the maximum length of text data.
SetCommentDataMaxNum Sets the maximum number of comments to download.
GetWorkSize Gets the size of the working buffer used by the nn::olv::DownloadCommentDataList function.
SetWork Sets the size of the working buffer used by the nn::olv::DownloadCommentDataList function.
