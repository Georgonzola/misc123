nn::olv::DownloadCommunityDataListParam::Flags Enumerated Type

Syntax

enum Flags;

Value

Value Description
FLAG_NONE The state when no flags are specified.
FLAG_FILTER_BY_FAVORITE Downloads the data for communities registered as favorites. (Communities created by the local host are treated as favorites and downloaded.)
You cannot simultaneously specify both FLAG_FILTER_BY_OFFICIAL and FLAG_FILTER_BY_SELF.
This flag is disabled if the SetCommunityId function is used to specify a community ID.
FLAG_FILTER_BY_OFFICIAL Downloads the community data for official communities.
You cannot simultaneously specify both FLAG_FILTER_BY_FAVORITE and FLAG_FILTER_BY_SELF.
This flag is disabled if the SetCommunityId function is used to specify a community ID.
FLAG_FILTER_BY_SELF Downloads the community data for communities that you have created.
You cannot simultaneously specify both FLAG_FILTER_BY_FAVORITE and FLAG_FILTER_BY_OFFICIAL.
This flag is disabled if the SetCommunityId function is used to specify a community ID.
FLAG_WITH_MII_DATA Includes the Mii data for the user who created the community in the community data to download.
FLAG_WITH_ICON_DATA Includes icon data in the community data to download.

Description

Flags that can be specified in the SetFlags function.If not using the SetCommunityId function, make sure that you specify one of the following: FLAG_FILTER_BY_FAVORITE, FLAG_FILTER_BY_OFFICIAL, or FLAG_FILTER_BY_SELF.
