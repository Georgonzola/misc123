nn::olv::DownloadCommunityDataListParam Class

Syntax

class DownloadCommunityDataListParam

Description

Represents the parameters when downloading community data.

Enumerated Types

Flags Flags that can be specified in the SetFlags function.If not using the SetCommunityId function, make sure that you specify one of the following: FLAG_FILTER_BY_FAVORITE, FLAG_FILTER_BY_OFFICIAL, or FLAG_FILTER_BY_SELF.

Member Functions

DownloadCommunityDataListParam Constructor.
Reset Reverts parameters to their initial state.
SetFlags Sets flags. You must specify this option when the SetCommunityId function is not used.
SetCommunityDataMaxNum Sets the maximum number of communities to download. You must specify this option.
SetCommunityId Sets the community ID. Specifying this option is not required.
If you specify a community ID, only the community with the specified ID is downloaded.
Up to a maximum of nn::olv::COMMUNITY_ID_MAX_NUM community IDs can be specified.
GetWorkSize Gets the size of the working buffer used by the nn::olv::DownloadCommunityDataList function. Call this function after all of the various parameters have been specified, with the exception of the SetWork function.
This function returns fixed values.
SetWork Sets the size of the working buffer used by the nn::olv::DownloadCommunityDataList function. You must specify this option.
Specify a buffer of at least the size obtained with the GetWorkSize function.
