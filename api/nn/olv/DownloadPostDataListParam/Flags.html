nn::olv::DownloadPostDataListParam::Flags Enumerated Type

Syntax

enum Flags;

Value

Value Description
FLAG_NONE The state when no flags are specified.
FLAG_FILTER_BY_FRIEND Download posts from users who are friends.
This flag is disabled if the SetSearchPid function is used to specify a principal ID for the Nintendo Network ID.
This flag is disabled if the SetPostId function is used to specify a post ID.
FLAG_FILTER_BY_FOLLOW Download posts from users you are following.
This flag is disabled if the SetSearchPid function is used to specify a principal ID for the Nintendo Network ID.
This flag is disabled if the SetPostId function is used to specify a post ID.
FLAG_FILTER_BY_SELF Download your own posts.
This flag is disabled if the SetSearchPid function is used to specify a principal ID for the Nintendo Network ID.
This flag is disabled if the SetPostId function is used to specify a post ID.
FLAG_FILTER_BY_BODY_TEXT Download only text posts.
You cannot specify FLAG_FILTER_BY_BODY_MEMO at the same time.
This flag is disabled if the SetPostId function is used to specify a post ID.
FLAG_FILTER_BY_BODY_MEMO Download only handwritten memo posts.
You cannot specify FLAG_FILTER_BY_BODY_TEXT at the same time.
This flag is disabled if the SetPostId function is used to specify a post ID.
FLAG_DISTINCT_PID Get only a single post from each user.
This flag is disabled if the SetPostId function is used to specify a post ID.
FLAG_WITH_MII_DATA Include Mii data in the post data to download.
FLAG_WITH_EMPATHY_ADDED Include flags for whether you have already given Yeahs to posts in the post data to download.
FLAG_WITH_SPOILER Also downloads posts with spoilers and posts with spoilers in review.
This flag is disabled if the SetPostId function is used to specify a post ID.

Description

Flags that can be specified in the SetFlags function.

You can specify FLAG_FILTER_BY_FRIEND / FLAG_FILTER_BY_FOLLOW / FLAG_FILTER_BY_SELF in combinations to filter the posts to download.
When FLAG_WITH_SPOILER is not specified, posts with spoilers or with spoilers in review are not downloaded.
