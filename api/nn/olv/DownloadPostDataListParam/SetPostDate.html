nn::olv::DownloadPostDataListParam::SetPostDate Member Function

Syntax

void SetPostDate(
     const nn::fnd::DateTime postDate
);

Parameters

Name Description
in postDate The date the post was made.

Return Values

None.

Description

Sets the date the post was made.

Specifying this option is not required.
If you specify a date, only posts that were made prior to the specified date are downloaded.
Specify a date that adjusts for the difference from the time zone of the account.
This setting is disabled if the SetPostId function is used to specify a post ID.
