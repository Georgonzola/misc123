nn::olv::DownloadPostDataListParam::SetSearchPid Member Function

Syntax

void SetSearchPid(
     u32 searchPid
);

Parameters

Name Description
in searchPid The principal ID of a user's Nintendo Network ID.

Return Values

None.

Description

Sets the principal ID for the user's Nintendo Network ID.

Specifying this option is not required.
If you specify a principal ID for a Nintendo Network ID, only the posts from the user with the specified principal ID are downloaded.
This setting is disabled if the SetPostId function is used to specify a post ID.
