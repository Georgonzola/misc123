nn::olv::DownloadUserDataListParam::Flags Enumerated Type

Syntax

enum Flags;

Value

Value Description
FLAG_NONE The state when no flags are specified.
FLAG_FRIEND_TO_USER Downloads user data from friends of the specified user. Specify the target user with the SetUserPid function.
You cannot simultaneously specify FLAG_FOLLOW_TO_USER, FLAG_EMPATHY_TO_POST, and FLAG_FAVORITE_TO_COMMUNITY.
FLAG_FOLLOW_TO_USER Downloads user data from the users that the specified user is following. Specify the target user with the SetUserPid function.
You cannot simultaneously specify FLAG_FRIEND_TO_USER, FLAG_EMPATHY_TO_POST, and FLAG_FAVORITE_TO_COMMUNITY.
FLAG_EMPATHY_TO_POST Downloads user data from users who have given a Yeah to the specified post. Specify the target post with the SetPostId function.
You cannot simultaneously specify FLAG_FRIEND_TO_USER, FLAG_FOLLOW_TO_USER, and FLAG_FAVORITE_TO_COMMUNITY.
FLAG_FAVORITE_TO_COMMUNITY Downloads user data from users who have registered the specified community as a favorite. Specify the target community with the SetCommunityId function.
You cannot simultaneously specify FLAG_FRIEND_TO_USER, FLAG_FOLLOW_TO_USER, and FLAG_EMPATHY_TO_POST.
FLAG_WITH_MII_DATA Downloads user data including the Mii data.

Description

Flags that can be specified in the SetFlags function.

Be sure to specify one of these flags: FLAG_FRIEND_TO_USER, FLAG_FOLLOW_TO_USER, FLAG_EMPATHY_TO_POST or FLAG_FAVORITE_TO_COMMUNITY.
