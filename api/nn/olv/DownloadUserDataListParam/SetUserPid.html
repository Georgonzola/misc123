nn::olv::DownloadUserDataListParam::SetUserPid Member Function

Syntax

void SetUserPid(
     u32 userPid
);

Parameters

Name Description
in userPid The principal ID for the Nintendo Network ID.

Return Values

None.

Description

Sets the principal ID for the specified Nintendo Network ID.

Specifying this option is not required.
When the principal ID for the specified Nintendo Network ID is specified, the user data is downloaded for the friends of that user or the people being followed by that user.
Specify either FLAG_FRIEND_TO_USER or FLAG_FOLLOW_TO_USER in the SetFlags function.
