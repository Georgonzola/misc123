nn::olv::DownloadedCommentData::DownloadExternalImageData Member Function

Syntax

nn::olv::Result DownloadExternalImageData(
     void * externalImageData,
     u32 * externalImageDataSize,
     u32 externalImageDataMaxSize
) const;

Parameters

Name Description
out externalImageData The buffer that stores the attached image data.
out externalImageDataSize The buffer that stores the size of the attached image data.
in externalImageDataMaxSize The size of the buffer that stores the attached image data.

Return Values

Returns the result of the operation.
For information about handling the result, see the section on Error Handling or the descriptions for the nn::olv::Result values.


Value Description
ResultSuccess Succeeded.
ResultUsageError A usage error.
ResultFatalError A fatal error.
ResultCanceled Canceled.
ResultNetworkError A network error.

Description

Downloads the attached image data.

This function can only be used when the return value from the TestFlags function is true when FLAG_WITH_EXTERNAL_IMAGE_DATA is specified.
This function initiates Internet communication, during which time processing is blocked.
While this function is being called, you can call the nn::olv::Cancel function from another thread to cancel communication processes.
