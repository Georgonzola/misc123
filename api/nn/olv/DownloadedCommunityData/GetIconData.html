nn::olv::DownloadedCommunityData::GetIconData Member Function

Syntax

nn::olv::Result GetIconData(
     u8 * iconData,
     u32 * iconDataSize,
     u32 iconDataMaxSize
) const;

Parameters

Name Description
out iconData The buffer that stores the icon data.
out iconDataSize The buffer that stores the size of the icon data.
in iconDataMaxSize The size of the buffer that stores the icon data.
Make sure that you specify a size of at least nn::olv::ICON_DATA_BUFF_LENGTH.

Return Values

Returns the result of the operation.
For information about handling the result, see the section on Error Handling or the descriptions for the nn::olv::Result values.


Value Description
ResultSuccess Succeeded.
ResultUsageError A usage error.
ResultNetworkError A network error.

Description

Gets icon data. This function can only be used when the return value from the TestFlags function is true when FLAG_WITH_ICON_DATA is specified.
