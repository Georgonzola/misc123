nn::olv::DownloadedCommunityData Class

Syntax

class DownloadedCommunityData

Description

A class that stores the downloaded community data.

Enumerated Types

Flags Flags that can be specified in the TestFlags function.

Member Functions

DownloadedCommunityData Constructor.
Reset Reverts parameters to their initial state.
TestFlags Tests flags.
GetCommunityId Gets the community ID.
GetCommunityCode Gets the community code. Note that if you call this function in one of the following invalid states, it will stop on an assertion in the debug/development versions, and return without doing anything in the release version.
If NULL is specified for the communityCode argument.
If a value less than that of nn::olv::COMMUNITY_CODE_BUFF_LENGTH is specified for the communityCodeMaxLength argument.
GetOwnerPid Gets the principal ID for the Nintendo Network ID of the user who created the community.
GetTitleText Gets the community name. This function can only be used when the return value from the TestFlags function is true when FLAG_WITH_TITLE_TEXT is specified.

Note that if you call this function in one of the following invalid states, it will stop on an assertion in the debug/development versions, and return without doing anything in the release version.
If calling the TestFlags function with FLAG_WITH_TITLE_TEXT specified yields a return value of false.
If NULL is specified for the titleText argument.
If a value less than that of nn::olv::TITLE_TEXT_BUFF_LENGTH is specified for the titleTextMaxLength argument.
Note:
The community ID might contain characters that are not supported by the application.
Make sure that the presence of unsupported characters does not corrupt the screen display or hinder the progress of your application.
GetDescriptionText Gets the community description. This function can only be used when the return value from the TestFlags function is true when FLAG_WITH_DESCRIPTION_TEXT is specified.

Note that if you call this function in one of the following invalid states, it stops on an assertion in the debug and development versions, and returns without doing anything in the release version.
If calling the TestFlags function with FLAG_WITH_DESCRIPTION_TEXT specified yields a return value of false.
If NULL is specified for the descriptionText argument.
If a value less than that of nn::olv::DESCRIPTION_TEXT_BUFF_LENGTH is specified for the descriptionTextMaxLength argument.
Note:
The description of the community might contain characters that are not supported by the application.
Make sure that the presence of unsupported characters does not corrupt the screen display or hinder the progress of your application.
GetAppData Gets application data. This function can only be used when the return value from the TestFlags function is true when FLAG_WITH_APP_DATA is specified.

Note that if you call this function in one of the following invalid states, it stops on an assertion in the debug and development versions, and returns without doing anything in the release version.
Calling the TestFlags function with FLAG_WITH_APP_DATA specified yields a return value of false.
If NULL is specified for the appData argument.
If a value less than that returned by the GetAppData function is specified for the appDataMaxSize argument.
GetAppDataSize Gets the size of the application data.
GetIconData Gets icon data. This function can only be used when the return value from the TestFlags function is true when FLAG_WITH_ICON_DATA is specified.
GetOwnerMiiData Gets the Mii data of the user who created the community. This function can only be used when the return value from the TestFlags function is true when FLAG_WITH_MII_DATA is specified.

Note that if you call this function in one of the following invalid states, it stops on an assertion in the debug and development versions, and returns without doing anything in the release version.
If calling the TestFlags function with FLAG_WITH_APP_DATA specified yields a return value of false.
If NULL is specified for the storeData argument.
GetOwnerMiiNickname Gets the nickname of the Mii of the user who created the community. Note that if you call this function in one of the following invalid states, it will stop on an assertion in the debug/development versions, and return without doing anything in the release version.
If NULL is specified for the miiNickname argument.
If a value less than that of nn::olv::MII_NICKNAME_BUFF_LENGTH is specified for the miiNicknameMaxLength argument.
Note:
The Mii nickname might contain characters that are not supported by the application.
Handle this so that it does not corrupt the screen display or hinder the progress of your application.
For more information, see the section about the display of nicknames and creator names in the 3DS Guidelines.
