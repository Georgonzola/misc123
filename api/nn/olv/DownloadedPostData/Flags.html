nn::olv::DownloadedPostData::Flags Enumerated Type

Syntax

enum Flags;

Value

Value Description
FLAG_NONE The state when no flags are specified.
FLAG_WITH_BODY_TEXT A flag indicating that the post is a text post.
FLAG_WITH_BODY_MEMO A flag indicating that the post is a handwritten memo post.
FLAG_WITH_EXTERNAL_IMAGE_DATA A flag indicating that the post includes an attached image.
FLAG_WITH_MII_DATA A flag indicating that the post includes Mii data.
FLAG_WITH_EXTERNAL_URL A flag indicating that the post includes an attached URL.
FLAG_WITH_APP_DATA A flag indicating that the post includes application data.
FLAG_EMPATHY_ADDED A flag indicating that you have already given a Yeah to the post.
FLAG_SPOILER A flag indicating that the post includes spoilers. The flag is not set for posts with spoilers that are under review.

Description

Flags that can be specified in the TestFlags function.
