nn::olv::DownloadedPostData::GetBodyText Member Function

Syntax

void GetBodyText(
     wchar_t * bodyText,
     u32 bodyTextMaxLength
) const;

Parameters

Name Description
out bodyText The buffer that stores the text data.
in bodyTextMaxLength The length of the buffer that stores the text data, including the terminating null character.
Make sure that you specify a size of at least nn::olv::BODY_TEXT_BUFF_LENGTH.

Return Values

None.

Description

Gets text data.

This function can only be used when the return value from the TestFlags function is true when FLAG_WITH_BODY_TEXT is specified.

Note that if you call this function in one of the following invalid states, it stops on an assertion in the debug and development versions, and returns without doing anything in the release version.
If calling the TestFlags function with FLAG_WITH_BODY_TEXT specified yields a return value of false.
If NULL is specified for the bodyText parameter.
If a value less than that of nn::olv::BODY_TEXT_BUFF_LENGTH is specified for the bodyTextMaxLength parameter.

Note:
The text data might contain characters that are not supported by the application.
Make sure that the presence of unsupported characters does not corrupt the screen display or hinder the progress of your application.
