nn::olv::DownloadedPostData::GetMiiNickname Member Function

Syntax

void GetMiiNickname(
     wchar_t * miiNickname,
     u32 miiNicknameMaxLength
) const;

Parameters

Name Description
out miiNickname The buffer that stores the nickname of the Mii.
in miiNicknameMaxLength The length of the buffer that stores the nickname of the Mii, including the terminating null character.
Make sure that you specify a size of at least nn::olv::MII_NICKNAME_BUFF_LENGTH.

Return Values

None.

Description

Gets the nickname of the Mii of the user.

Note that if you call this function in one of the following invalid states, it stops on an assertion in the debug and development versions, and returns without doing anything in the release version.
If NULL is specified for the miiNickname parameter.
If a value less than that of nn::olv::MII_NICKNAME_BUFF_LENGTH is specified for the miiNicknameMaxLength parameter.

Note:
The Mii nickname might contain characters that are not supported by the application.
Handle this so that it does not corrupt the screen display or hinder the progress of your application.
For more information, see the section about the display of nicknames and creator names in the 3DS Guidelines.
