nn::olv Namespace

Description

The namespace of the OLV library.

The OLV library provides features for using Miiverse in applications. For an overview of the Miiverse specifications, see 3DS Planning and Operations Manual - MiiverseEdition.

The function references clearly indicate those OLV library functions that are not thread-safe. All other functions are thread-safe.
In addition, classes are thread-safe between instances.

The method of writing code for error handling for the OLV library is different than for other libraries. For more information, see the following.

  • Error Handling

Finish initial startup of Miiverse

The initial startup of Miiverse is the first time the Miiverse application is run on hardware and allowed to progress to the community.
Because information is deleted when the system is initialized, complete the Miiverse initial startup.
If the Miiverse initial startup is not completed and the OLV library used, an error occurs.

Cautions when using Debugger

The amount of memory that the Miiverse application can use is less for a PARTNER-CTR series or IS-CTR series Debugger than for an actual system, so when verifying by starting the Miiverse application, an error may occur.
(There is no problem with the development emulator, or the IS-SNAKE DevKit with the standard 2GB or more.)
