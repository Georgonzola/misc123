nn::olv::Result Class

Syntax

class Result : public nn::Result

Description

Class that indicates the result of an OLV library operation.

When the OLV library interface needs to return an error, it returns an instance of the nn::olv::Result class.
For information about error handling, see the section on Error Handling or the descriptions for the nn::olv::Result values.

typedef Definitions

Level Enumerated type indicating the severity of an error. (Do not use this type for error handling.)

Member Functions

GetDescription Returns the description value for an error.
GetPrintableBits Converts an nn::olv::Result code into a 32-bit array.
