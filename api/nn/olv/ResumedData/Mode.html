nn::olv::ResumedData::Mode Enumerated Type

Syntax

enum Mode;

Value

Value Description
MODE_NONE The default mode.
MODE_START_COMMUNITY The mode specified when the process is started from the community page.
When the GetMode function returns this value, you can get the community ID using the GetCommunityId function.
Currently, this value will never be returned because the Miiverse application does not support this feature.
MODE_START_POST The mode specified when the process is started from the post page.
If the GetMode function returns this value, you can get the community ID using the GetCommunityId function and the post ID using the GetPostId function.
MODE_START_USER The mode specified when the process is started from the user page.
If the GetMode function returns this value, you can get the principal ID for the Nintendo Network ID using the GetUserPid function.
Currently, this value will never be returned because the Miiverse application does not support this feature.

Description

The modes that can be obtained using the GetMode function.
