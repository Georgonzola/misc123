nn::olv::StartPortalAppParam Class

Syntax

class StartPortalAppParam

Description

Represents the parameters when starting the Miiverse application.

Enumerated Types

Mode Modes that can be specified in the SetMode function.
Flags Flags that can be specified in the SetFlags function.

Member Functions

StartPortalAppParam Constructor.
Reset Reverts parameters to their initial state.
SetMode Sets the mode.
SetFlags Sets flags.
SetPostId Sets the post ID.
SetCommunityId Sets the community ID.
SetUserPid Sets the principal ID for the specified Nintendo Network ID.
SetTopicTag Sets the topic tag.
SetCommonData Adds common data.
GetWorkSize Gets the size of the working buffer used by the nn::olv::StartPortalApp function.
SetWork Sets the size of the working buffer used by the nn::olv::StartPortalApp function.
