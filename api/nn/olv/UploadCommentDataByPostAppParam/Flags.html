nn::olv::UploadCommentDataByPostAppParam::Flags Enumerated Type

Syntax

enum Flags;

Value

Value Description
FLAG_NONE The state when no flags are specified.
FLAG_SPOILER Specify this flag when uploading comment data that includes spoilers.
FLAG_ONLY_BODY_TEXT Specify this flag when limiting the body of the comment to text (so that comments cannot be handwritten messages).
You cannot specify FLAG_ONLY_BODY_MEMO at the same time.
FLAG_ONLY_BODY_MEMO Specify this flag when limiting the body of the comment to handwritten messages (so that comments cannot be text).
You cannot specify FLAG_ONLY_BODY_TEXT at the same time.

Description

Flags that can be specified in the SetFlags function.
