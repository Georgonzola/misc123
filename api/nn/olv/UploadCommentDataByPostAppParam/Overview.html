nn::olv::UploadCommentDataByPostAppParam Class

Syntax

class UploadCommentDataByPostAppParam

Description

The class representing the parameters used when starting the posting applet and uploading comments.

Enumerated Types

StampFlags Flags that can be specified in the SetStampData( function.
