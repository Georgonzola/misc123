nn::olv::UploadCommentDataByPostAppParam::SetExternalImageData Member Function

Syntax

void SetExternalImageData(
     const u8 * externalImageData,
     u32 externalImageDataSize
);

Parameters

Name Description
in externalImageData The attached image data.
Note that only the address value is stored, so this data must remain valid until the call to the nn::olv::UploadCommentDataByPostApp function ends.
in externalImageDataSize The size of the attached image data.
The maximum value that can be specified is nn::olv::EXTERNAL_IMAGE_DATA_MAX_LENGTH.

Return Values

None.

Description

Sets the attached image data.

Specifying this option is not required.
A reduced image of the attached image data is used as the thumbnail image.
For more information about the image format, see Attached Image Data and Attached Thumbnail Image Data.
