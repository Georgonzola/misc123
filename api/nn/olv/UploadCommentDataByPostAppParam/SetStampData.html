nn::olv::UploadCommentDataByPostAppParam::SetStampData Member Function

Syntax

void SetStampData(
     const u8 * stampData,
     u32 stampDataSize,
     const u8 * maskData,
     u32 maskDataSize,
     u32 index,
     u32 stampFlags = STAMP_FLAG_NONE
);

Parameters

Name Description
in stampData The stamp image data.
Note that only the address value is stored, so this data must remain valid until the call to the nn::olv::UploadCommentDataByPostApp function ends.
in stampDataSize The size of the stamp image data.
Either nn::olv::STAMP_DATA_100_100_BUFF_LENGTH or nn::olv::STAMP_DATA_200_200_BUFF_LENGTH can be specified for this value.
in maskData The stamp mask image data.
Note that only the address value is stored, so this data must remain valid until the call to the nn::olv::UploadCommentDataByPostApp function ends.
in maskDataSize The size of the stamp mask image data.
Either nn::olv::STAMP_DATA_100_100_BUFF_LENGTH or nn::olv::STAMP_DATA_200_200_BUFF_LENGTH can be specified for this value.
This value must match the size of the stamp image data.
in index The index.
Values in the range of 0 to nn::olv::STAMP_DATA_MAX_NUM - 1 can be specified.
in stampFlags Flags representing stamp attributes (STAMP_FLAG_*).

Return Values

None.

Description

Specifies stamp image data.

Specifying this option is not required.
Up to nn::olv::STAMP_DATA_MAX_NUM can be specified.
For more information about the image format, see Stamp Image Data and Stamp Mask Image Data.
