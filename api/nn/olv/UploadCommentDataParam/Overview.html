nn::olv::UploadCommentDataParam Class

Syntax

class UploadCommentDataParam

Description

The class representing the parameters used when uploading and deleting comments. (For debugging.)

Note: Contact Nintendo support if you plan on using this class for anything other than debugging. (Its use is normally prohibited in retail versions of products.)

Enumerated Types

Flags Flags that can be specified in the SetFlags function.

Member Functions

UploadCommentDataParam Constructor.
Reset Reverts parameters to their initial state.
SetFlags Sets flags.
SetFeeling Sets the mood.
SetBodyText Sets the text data.
SetBodyMemo Sets the handwritten memo data.
SetExternalImageData Sets the attached image data.
SetAppData Sets the application data.
SetPostId Sets the post ID for the comment.
SetCommentId Sets the comment ID when deleting comments.
GetWorkSize Gets the size of the working buffer used by the nn::olv::UploadCommentData function.
SetWork Sets the size of the working buffer used by the nn::olv::UploadCommentData function.
