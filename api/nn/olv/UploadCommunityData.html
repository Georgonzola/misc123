nn::olv::UploadCommunityData Function

Syntax

nn::olv::Result UploadCommunityData(
     nn::olv::UploadedCommunityData * uploadedData,
     const nn::olv::UploadCommunityDataParam * uploadParam
);

Parameters

Name Description
out uploadedData The class that stores the results of creating or updating community data.
You can specify NULL if not needed.
Nothing is stored if the community data is deleted.
in uploadParam The class that stores the parameters used when sending community data.

Return Values

Returns the result of the operation.
For information about handling the result, see the section about Error Handling or the descriptions of the nn::olv::Result values.


Value Description
ResultSuccess Succeeded.
ResultUsageError A usage error.
ResultFatalError A fatal error.
ResultCanceled Canceled.
ResultLibraryPermanentError A permanent library error.
ResultNetworkError A network error.
ResultServerPermanentError A permanent server error.
ResultServerTemporaryError A temporary server error.

Description

Creates, updates, or deletes community data.

This function can only be used when a connection to the OLV library has already been made with the nn::olv::Connect() function.
This function initiates Internet communication, during which time processing is blocked.
While this function is being called, you can call the nn::olv::Cancel() function from another thread to cancel communication processes.
