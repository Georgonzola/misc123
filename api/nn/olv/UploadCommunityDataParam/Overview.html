nn::olv::UploadCommunityDataParam Class

Syntax

class UploadCommunityDataParam

Description

The class representing the parameters used when creating, updating, and deleting community data.

Enumerated Types

Flags Flags that can be specified in the SetFlags function.

Member Functions

UploadCommunityDataParam Constructor.
Reset Reverts parameters to their initial state.
SetFlags Sets flags.
SetCommunityId Sets the community ID.
SetTitleText Sets the community name.
SetDescriptionText Sets the community description.
SetPostTextMaxLength Specifies the maximum number of characters that can be posted from the Miiverse application.
SetCommentTextMaxLength Specifies the maximum number of characters for comments that can be posted from the Miiverse application.
SetIconData Sets the icon data.
SetAppData Sets the application data.
GetWorkSize Gets the size of the working buffer used by the nn::olv::UploadCommunityData function.
SetWork Sets the size of the working buffer used by the nn::olv::UploadCommunityData function.
