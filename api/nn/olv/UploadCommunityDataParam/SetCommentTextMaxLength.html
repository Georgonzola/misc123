nn::olv::UploadCommunityDataParam::SetCommentTextMaxLength Member Function

Syntax

void SetCommentTextMaxLength(
     u32 length
);

Parameters

Name Description
in length The maximum number of characters for comments postable from the Miiverse application.
The maximum number of characters that can be specified is nn::olv::BODY_TEXT_MAX_LENGTH.

Return Values

None.

Description

Specifies the maximum number of characters for comments that can be posted from the Miiverse application.

Specifying this option is not required.
If nothing is specified, 100 characters is set as the default.
This setting is disabled if FLAG_DELETE is specified in the SetFlags function to delete community data.
