nn::olv::UploadCommunityDataParam::SetDescriptionText Member Function

Syntax

void SetDescriptionText(
     const wchar_t * descriptionText
);

Parameters

Name Description
in descriptionText The community description.
Use UTF-16LE for the encoding.
The maximum number of characters that can be specified is nn::olv::DESCRIPTION_TEXT_BUFF_LENGTH - 1.

Return Values

None.

Description

Sets the community description.

Specifying this option is not required.
This setting is disabled if FLAG_DELETE is specified in the SetFlags function to delete community data.
