nn::olv::UploadPostDataByPostAppParam::SetBodyText Member Function

Syntax

void SetBodyText(
     const wchar_t * bodyText
);

Parameters

Name Description
in bodyText The text data.
Use UTF-16LE for the encoding.
Because the posting applet cannot handle them, do not include newline characters.
For the maximum length of text that can be specified, see the description of SetBodyTextMaxLength.

Return Values

None.

Description

Sets a default for text data.

Specifying this option is not required.
The user can edit this setting in the posting applet.
