nn::olv::UploadPostDataParam::SetBodyText Member Function

Syntax

void SetBodyText(
     const wchar_t * bodyText
);

Parameters

Name Description
in bodyText The text data.
Use UTF-16LE for the encoding.
The maximum number of characters that can be specified is nn::olv::BODY_TEXT_MAX_LENGTH.

Return Values

None.

Description

Sets the text data.

If you are specifying handwritten memo data, you are prohibited from specifying text data. Otherwise, you must specify text data.
This setting is disabled if FLAG_DELETE is specified in the SetFlags function to delete posts.
