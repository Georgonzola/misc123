nn::olv::UploadPostDataParam::SetCommunityId Member Function

Syntax

void SetCommunityId(
     u32 communityId
);

Parameters

Name Description
in communityId The community ID.

Return Values

None.

Description

Sets the community ID.

Specifying this option is not required. If not specified, the post is made to the general community by default.
This setting is disabled if FLAG_DELETE is specified in the SetFlags function to delete posts.
