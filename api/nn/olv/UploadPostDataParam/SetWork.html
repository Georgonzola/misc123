nn::olv::UploadPostDataParam::SetWork Member Function

Syntax

void SetWork(
     u8 * work,
     u32 workSize
);

Parameters

Name Description
in work The working buffer.
in workSize The size of the working buffer.

Return Values

None.

Description

Specifies the size of the working buffer used by the nn::olv::UploadPostData function.

You must specify this option.
Specify a buffer of at least the size obtained with the GetWorkSize function.
