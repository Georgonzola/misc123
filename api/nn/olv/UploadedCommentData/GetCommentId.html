nn::olv::UploadedCommentData::GetCommentId Member Function

Syntax

void GetCommentId(
     char * commentId,
     u32 commentIdMaxLength
) const;

Parameters

Name Description
out commentId The buffer that stores the comment ID.
in commentIdMaxLength The length of the buffer that stores the comment ID, including the terminating null character.
Make sure that you specify a size of at least nn::olv::COMMENT_ID_BUFF_LENGTH.

Return Values

None.

Description

Gets the comment ID.

Note that if you call this function in one of the following invalid states, it stops on an assertion in the debug and development versions, and returns without doing anything in the release version.
If NULL is specified for the commentId argument.
If a value less than that of nn::olv::COMMENT_ID_BUFF_LENGTH is specified for the commentIdMaxLength argument.
