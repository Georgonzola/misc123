nn::olv::UploadedCommunityData::GetDescriptionText Member Function

Syntax

void GetDescriptionText(
     wchar_t * descriptionText,
     u32 descriptionTextMaxLength
) const;

Parameters

Name Description
out descriptionText The buffer for storing the community description.
in descriptionTextMaxLength The length of the buffer that stores the community description, including the terminating null character.
Make sure that you specify a size of at least nn::olv::DESCRIPTION_TEXT_BUFF_LENGTH.

Return Values

None.

Description

Gets the community description.

This function can only be used when the return value from the TestFlags function is true when FLAG_WITH_DESCRIPTION_TEXT is specified.

Note that if you call this function in one of the following invalid states, it stops on an assertion in the debug and development versions, and returns without doing anything in the release version.
If calling the TestFlags function with FLAG_WITH_DESCRIPTION_TEXT specified yields a return value of false.
If NULL is specified for the descriptionText argument.
If a value less than that of nn::olv::DESCRIPTION_TEXT_BUFF_LENGTH is specified for the descriptionTextMaxLength argument.
