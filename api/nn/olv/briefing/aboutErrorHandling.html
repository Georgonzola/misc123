Overview

This section describes the handling of OLV library errors.

The functions of the OLV library can generate a variety of errors even in the retail product, including errors related to the Parental Control settings and errors at unspecified times related to Internet communications.
Design applications that support the OLV library to handle the errors returned by the library functions.
Error codes that correspond to the errors returned by the OLV library functions can be used with the Error/EULA Display applet to prompt users to take corrective actions.

Handling nn::olv::Result

To keep actual error handling private, the details about nn::olv::Result are kept private and only a number of definitions encompassing nn::olv::Result are made.
When handling OLV library errors, always use the <= operator to determine the nn::olv::Result type, as in the following example. 

    if (olvResult.IsFailure())
    {
        if (olvResult <= nn::olv::ResultLibraryPermanentError())
        {
            // This can catch errors of the type ResultLibraryPermanentError.
        }
        ...
    }

Error Codes

When an OLV library error has occurred, use the nn::olv::GetErrorCode function to get the error code to pass it to the Error/EULA Display applet.
Follow the example below to get and pass the error code to the Error/EULA Display applet. 

    if (olvResult.IsFailure())
    {
        u32 errorCode = nn::olv::GetErrorCode(olvResult); // This can get the error code to pass to the Error/EULA Display applet.

        ...
    }

Errors That Must Not Occur in Retail Products

The following errors are due to bugs in the application and should not occur in the retail product. Remove them during debugging.

  • nn::olv::ResultFatalError
  • nn::olv::ResultUsageError

If these errors occur, show them if necessary using the Error/EULA Display applet (if the result is due to a user operation), and do not make subsequent use of the library features.

Recoverable Errors

The following errors are recoverable errors, meaning the system might be able to recover from them if the user tries again.

  • nn::olv::ResultLibraryTemporaryError
  • nn::olv::ResultNetworkError
  • nn::olv::ResultServerTemporaryError

If these errors occur, show them if necessary using the Error/EULA Display applet (if the result is due to a user operation), and allow the user to try inputting again if possible.

Other Error

The following error does not need to be shown using the Error/EULA Display applet.

  • nn::olv::ResultCanceled

This is not an error to be displayed, so do not show it using the Error/EULA Display applet. However, allow the user to try inputting again if possible.

Example of Error Handling

The following simplified C++ code provides guidelines for handling OLV library errors. 

    nn::olv::Result olvResult = nn::olv::Initialize(&initializedData, &initializeParam);

    if (olvResult.IsFailure())
    {
        if (olvResult <= nn::olv::ResultLibraryPermanentError() || 
            olvResult <= nn::olv::ResultServerPermanentError())
        {
            // Shows the error if necessary using the Error/EULA Display applet (if the result is due to a user operation).
            olvErrorCode = nn::olv::GetErrorCode(olvResult);

            // Disable subsequent use of the OLV library.
            olvDisable = true;
        }
        else if (olvResult <= nn::olv::ResultLibraryTemporaryError() || 
            olvResult <= nn::olv::ResultNetworkError() || 
            olvResult <= nn::olv::ResultServerTemporaryError())
        {
            // Shows the error if necessary using the Error/EULA Display applet (if the result is due to a user operation).
            olvErrorCode = nn::olv::GetErrorCode(olvResult);

            // Allow the user to try inputting again if possible.
        }
        else if (olvResult <= nn::olv::ResultFatalError() || 
            olvResult <= nn::olv::ResultUsageError())
        {
            // Shows the error if necessary using the Error/EULA Display applet (if the result is due to a user operation).
            olvErrorCode = nn::olv::GetErrorCode(olvResult);

            // To be removed during debugging and ignored in the retail product.
            NN_ASSERTMSG(0, "Error: %u", nn::olv::GetErrorCode(olvResult));
        }
        else
        {
            // This is not an error to be displayed, so do not show it using the Error/EULA Display applet. 
            // Allow the user to try inputting again if possible.
        }
    }
