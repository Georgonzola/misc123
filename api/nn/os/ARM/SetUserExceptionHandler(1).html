nn::os::ARM::SetUserExceptionHandler Function

Syntax

template <typename T>void SetUserExceptionHandler(
     UserExceptionHandler pHandler,
     T * pStack,
     ExceptionBuffer * pExceptionBuffer
);

Template Arguments

Name Description
T Stack type.

Parameters

Name Description
in pHandler Specifies the exception handler that is called when an exception is thrown.
in pStack Specifies the stack to use for the exception handler.
in pExceptionBuffer Specifies a buffer for storing exception information.

Return Values

None.

Description

Specifies the global exception handler.

This function is a wrapper function for accepting the stack used by the exception handler via a stack-type member function.

For more information, see SetUserExceptionHandler(UserExceptionHandler, uptr, ExceptionBuffer*).
