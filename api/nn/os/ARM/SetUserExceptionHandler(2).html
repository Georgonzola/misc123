nn::os::ARM::SetUserExceptionHandler Function

Syntax

void SetUserExceptionHandler(
     UserExceptionHandler pHandler,
     uptr stackBottom
);

Parameters

Name Description
in pHandler Specifies the exception handler called when an exception is thrown.
in stackBottom Specifies the stack to use for the exception handler. You must specify the bottom of the stack.

Return Values

None.

Description

Provided for compatibility with past SDKs. Calls SetUserExceptionHandler (pHandler, stackBottom, EXCEPTION_BUFFER_USE_HANDLER_STACK).

For more information, see SetUserExceptionHandler(UserExceptionHandler, uptr, ExceptionBuffer*).
