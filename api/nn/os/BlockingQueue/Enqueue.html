nn::os::BlockingQueue::Enqueue Member Function

Syntax

#include <nn/os.h>
void Enqueue(
     uptr data
);

Parameters

Name Description
in data Element to insert.

Return Values

None.

Description

Inserts an element at the end of the queue.

This function inserts data at the end of the queue. However, if the queue is full, the thread that called this function is blocked. This thread immediately restarts when a receiving thread operates to remove data from the queue.

If any threads are being blocked by Dequeue or GetFront, these threads are started.
