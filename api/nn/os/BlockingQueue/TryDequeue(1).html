nn::os::BlockingQueue::TryDequeue Member Function

Syntax

#include <nn/os.h>
bool TryDequeue(
     uptr * pOut,
     fnd::TimeSpan timeout
);

Parameters

Name Description
out pOut Buffer in which to write the retrieved element.
in timeout Specifies the timeout.

Return Values

Returns true if an element was retrieved, and false if no element was retrieved.

Description

Removes the first element from the front of the queue. Waits for the specified time if the queue is empty.

If the queue is not empty, retrieves the element at the front of the queue and returns true. Returns false if the queue is empty and the timeout time elapses.

If any threads are being blocked by Dequeue or GetFront when the element is inserted, these threads are started.
