Overloaded Member Functions

TryDequeue ( uptr * ) Removes the first element from the front of the queue.
TryDequeue ( uptr *, fnd::TimeSpan ) Removes the first element from the front of the queue. Waits for the specified time if the queue is empty.

Description of TryDequeue ( uptr * )

If the queue is not empty, retrieves the element at the front of the queue and returns true. If the queue is empty, does nothing and returns false.

If any threads are being blocked by Dequeue or GetFront when the element is inserted, these threads are started.

Description of TryDequeue ( uptr *, fnd::TimeSpan )

If the queue is not empty, retrieves the element at the front of the queue and returns true. Returns false if the queue is empty and the timeout time elapses.

If any threads are being blocked by Dequeue or GetFront when the element is inserted, these threads are started.
