nn::os::BlockingQueue::TryEnqueue Member Function

Syntax

#include <nn/os.h>
bool TryEnqueue(
     uptr data
);

Parameters

Name Description
in data Element to insert.

Return Values

Returns true if the element was inserted, and false otherwise.

Description

Tries to insert an element at the end of the queue.

If the queue is not full, inserts an element at the end of the queue and returns true. If the queue is full, does nothing and returns false.

If any threads are being blocked by Dequeue or GetFront when the element is inserted, these threads are started.
