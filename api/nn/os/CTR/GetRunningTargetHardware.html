nn::os::CTR::GetRunningTargetHardware Function

Syntax

TargetHardware GetRunningTargetHardware();

Parameters

None.

Return Values

Returns the kind of hardware that is running.

Description

Gets the kind of hardware that is running.

This function cannot be used for master ROM submissions.
