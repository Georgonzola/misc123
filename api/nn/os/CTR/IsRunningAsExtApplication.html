nn::os::CTR::IsRunningAsExtApplication Function

Syntax

bool IsRunningAsExtApplication();

Parameters

None.

Return Values

Returns true if running as an extended application and false otherwise.

Description

Gets whether the application is running as an extended application.

This function cannot be used inside nninitStartUp.
