nn::os::GetAppMemorySize Function

Syntax

size_t GetAppMemorySize();

Parameters

None.

Return Values

Gets the maximum total amount of memory in bytes that the application can use.

Description

Gets the maximum total amount of memory that the application can use.

This value does not change during the course of program execution. The application can be allocated a heap (or a combination of heap and device memory) of size (GetAppMemorySize - GetUsingMemorySize) bytes.
