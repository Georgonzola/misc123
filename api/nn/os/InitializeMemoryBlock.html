nn::os::InitializeMemoryBlock Function

Syntax

#include <nn/os.h>
void InitializeMemoryBlock(
     uptr begin,
     size_t size
);

Parameters

Name Description
begin
size

Return Values

None.

Description

Allows use of the MemoryBlock and StackMemoryBlock classes.

You must specify the memory region that will be managed by the MemoryBlock class and the StackMemoryBlock class before you construct the classes. These classes are allocated memory from the region specified by begin and size.beginSpecifies the starting address of the memory region that will be managed by the MemoryBlock class and the StackMemoryBlock class. sizeSpecifies the size of the memory region that will be managed by the MemoryBlock class and the StackMemoryBlock class. size must be a multiple of MEMORY_BLOCK_UNITSIZE (4096 bytes). ---------------------------------------------------------------------------
