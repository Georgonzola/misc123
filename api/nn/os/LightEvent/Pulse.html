nn::os::LightEvent::Pulse Member Function

Syntax

#include <nn/os.h>
void Pulse();

Parameters

None.

Return Values

None.

Description

Cancel threads waiting for the flag to be set. For an automatically resetting event, only the waiting thread with the highest priority is woken up. Its flag is then cleared. This function differs from Signal in that it clears the flag even if no threads are waiting.

For a manually resetting event, all waiting threads are woken up, and their flags are then cleared.
