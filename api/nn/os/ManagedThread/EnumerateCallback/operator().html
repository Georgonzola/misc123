nn::os::ManagedThread::EnumerateCallback::operator() Member Function

Syntax

#include <nn/os.h>
virtual bool operator()(
     ManagedThread * p
) = 0;

Parameters

Name Description
in p Pointer to an enumerated instance.

Return Values

Returns true if enumeration continues, or false if it has stopped.

Description

A user-defined process function.

This function is a pure virtual function, so it cannot be called. The intention is for users to implement their own processes in derived classes.

The pointers to the ManagedThread instances that are the targets of Enumerate are called one at a time as the argument of this function.

If this function has been called by Enumerate, p cannot be finalized by another thread while it is executing.
