nn::os::ManagedThread::FindById Member Function

Syntax

#include <nn/os.h>
static ManagedThread * FindById(
     bit32 id
);

Parameters

Name Description
in id The ID to use as the thread search key.

Return Values

Returns the pointer to the ManagedThread corresponding to id. If not found, returns NULL.

Description

Gets the pointer to the ManagedThread from the thread ID.

Searches among ManagedThread instances that have been initialized but not finalized to find the one that matches the specified ID.

This function is implemented using Enumerate, so it complies with Enumerate in terms of thread safety and internal function blocks.

For more information, see the Enumerate function.
