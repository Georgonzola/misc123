nn::os::ManagedThread::GetStackBufferEnd Member Function

Syntax

#include <nn/os.h>
uptr GetStackBufferEnd() const;

Parameters

None.

Return Values

Returns the ending address of the buffer being used for the stack.

Description

Gets the ending address of the buffer being used for the stack.

ManagedThread uses this function to store management information for the end of the buffer passed as the stack.

This function gets the next address after the entire buffer region passed as the stack, including the region used as management information.

This is the sum of the value obtained by the GetStackBufferBegin function plus the value obtained by the GetStackBufferSize function.
