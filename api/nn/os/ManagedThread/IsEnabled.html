nn::os::ManagedThread::IsEnabled Member Function

Syntax

#include <nn/os.h>
static bool IsEnabled();

Parameters

None.

Return Values

If InitializeEnvironment is called, it returns true. If not, it returns false.

Description

Determines whether ManagedThread can be used.

To be specific, determines whether InitializeEnvironment has been called.
