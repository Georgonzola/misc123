nn::os::ManagedThread::~ManagedThread Destructor

Syntax

#include <nn/os.h>
~ManagedThread();

Parameters

None.

Description

Destructor.

If one of the intialize member functions has been called but Finalize has not been called, this function internally calls Finalize.

For more information about this process, see the Finalize function.
