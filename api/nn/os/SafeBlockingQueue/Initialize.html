nn::os::SafeBlockingQueue::Initialize Member Function

Syntax

#include <nn/os.h>
void Initialize(
     uptr buffer[],
     size_t size
);

Parameters

Name Description
in buffer[] Buffer to use for the queue. Specify an array of type uptr.
in size The buffer size. Specify the number of elements in the array.

Return Values

None.

Description

Initializes a blocking queue using the specified buffer and size.
