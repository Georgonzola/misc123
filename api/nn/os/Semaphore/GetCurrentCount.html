nn::os::Semaphore::GetCurrentCount Member Function

Syntax

#include <nn/os.h>
static s32 GetCurrentCount();

Parameters

None.

Return Values

Returns the number of Semaphore objects that are currently being used.

Description

Gets the number of Semaphore objects that are currently being used.

The number of Semaphore objects that have been initialized but not yet finalized.

The number returned by this function includes those being used internally by the library.
