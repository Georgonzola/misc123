nn::os::Semaphore Class

Header File: nn/os.h

Syntax

class Semaphore : public nn::os::InterruptEvent

Description

Class for handling semaphores. Semaphores are synchronization objects that have counters.

Generally, use the nn::os::LightSemaphore class instead of nn::os::Semaphore. The nn::os::LightSemaphore class is better than the nn::os::Semaphore class in most respects, except that it cannot wait for multiple synchronization objects.

The semaphore Wait operation first waits for the semaphore counter to become greater than zero, and then increments the semaphore counter by one. Wait operations are not released while the semaphore counter is zero.

Call Release to increment a semaphore's counter.

For more information about synchronization objects, see the reference for the nn::os::Event class.

You can create up to eight Semaphore objects. Note also that different functions might use up resources that count against this limit.
