nn::os::SetHeapSize Function

Syntax

#include <nn/os.h>
Result SetHeapSize(
     size_t size
);

Parameters

Name Description
in size Specifies the new heap size. You must specify a value for size that is a multiple of HEAP_UNITSIZE (4096) and is 100,663,296 (96 Mi) or less.

Return Values

Description

Changes the size of the heap.

It is inappropriate to call this function if nninitStartUp is not being replaced in the application. In an environment where assertions are enabled, processing stops if this function is called without replacing nninitStartUp.
