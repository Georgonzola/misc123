nn::os::SimpleAutoStackManager Class

Syntax

class SimpleAutoStackManager : public nn::os::SimpleAutoStackManagerBase

Description

Implementation of AutoStackManager using fnd::IAllocator. Uses the specified nn::fnd::IAllocator derived class instance to allocate memory for the stack, and then uses it as the stack. If this class instance is used with the Thread::SetAutoStackManager function, the area that is actually available to be used as a stack is slightly smaller than the size specified to Thread::StartUsingAutoStack or Thread::TryStartUsingAutoStack, because part of the region is used for management and alignment purposes.
