nn::os::StackMemory Class

Syntax

class StackMemory : public nn::os::MemoryBlockBase

Description

Class that partitions a specified memory region for use as a stack.

The specified memory region is remapped to an address that differs from the region allocated by 4 KB of unmapped space on either side.

Given this unmapped space, it can be detected as a data abort for stack overflows or underflows when the region is used in a stack.

Member Constants

S MEMORY_ALIGNMENT const s32 The address of the memory region buffer specified by Initialize, TryInitialize, or the constructor must be aligned to the number of bytes specified here. It is defined as 4096 bytes.
S MEMORY_UNITSIZE const size_t Specifies the number of bytes that the size of the memory region buffer specified by Initialize, TryInitialize, or the constructor must be aligned to. It is defined as 4096 bytes.
