nn::os::StackMemoryAutoStackManager::Initialize Member Function

Syntax

void Initialize(
     fnd::IAllocator * pAllocator,
     uptr stackBottom
);

Parameters

Name Description
in pAllocator Specifies the allocator used to allocate the stack.
in stackBottom Specifies the bottom of the stack to use when releasing the stack.

Return Values

None.

Description

Initialization.

Specifies the allocator that actually performs memory allocation.

The allocator specified by pAllocator must be maintained until Finalize() is called.

Because the allocator can be accessed from multiple threads, it must be thread-safe.

In addition, the allocator must be able to allocate a memory region that is aligned to 0x1000 bytes.

The Construct() function passes 0x1000 to the second argument for pAllocator->Allocate.

When the stack is released, stackBottom is used as the stack. The stack is replaced with stackBottom inside the Destruct() member function, after which pAllocator->Free is called.

For this reason, the stack size indicated by stackBottom must be determined based on the size of the stack used by pAllocator->Free.
