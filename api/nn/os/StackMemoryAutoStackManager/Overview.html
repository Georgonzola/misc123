nn::os::StackMemoryAutoStackManager Class

Syntax

class StackMemoryAutoStackManager : public nn::os::SimpleAutoStackManagerBase

Description

Implementation of AutoStackManager using StackMemory.

Memory for the stack is allocated using an instance of the specified nn::fnd::IAllocator-derived class, and after the allocated memory region is isolated by StackMemory, it is used as a stack.

When using an instance of this class with Thread::SetAutoStackManager() the stack size specified by Thread::StartUsingAutoStack and Thread::TryStartUsingAutoStack must be a multiple of 0x1000 bytes.

The actual size that can be used for the stack is slightly smaller because stack administrative information is stored at the bottom of the stack.

Member Constants

S STACK_UNITSIZE const size_t Specifies the number of bytes that the stack size specified by Thread::StartUsingAutoStack and Thread::TryStartUsingAutoStack must be a multiple of when using an instance of this class with Thread::SetAutoStackManager. It is defined as 4096 bytes.
