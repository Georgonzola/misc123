nn::os::StackMemoryBlock::MEMORY_UNITSIZE Member Constant

Syntax

#include <nn/os.h>
static const size_t MEMORY_UNITSIZE =  4096;

Description

Specifies the number of bytes that the size specified by Initialize or the constructor must be a multiple of. It is defined as 4096 bytes.
