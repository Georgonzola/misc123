nn::os::Thread::Detach Member Function

Syntax

#include <nn/os.h>
void Detach();

Parameters

None.

Return Values

None.

Description

Detaches a thread so it is no longer subject to management.

The thread that this instance is managing is detached from this instance. As a result, operations for this thread cease, but it is possible to call Finalize and release the resources in use by this instance.

After you have called this function, do not call any other member function other than Finalize.

Detaching the thread eliminates ways to identify thread exits with certainty, thereby losing the ability to appropriately release the stack.

To prevent this, you can no longer call this function when AutoStackManager is not in use, or, in other words, when either Start or TryStart is in use.
