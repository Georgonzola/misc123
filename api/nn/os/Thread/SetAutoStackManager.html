nn::os::Thread::SetAutoStackManager Member Function

Syntax

#include <nn/os.h>
static void SetAutoStackManager(
     AutoStackManager * pManager
);

Parameters

Name Description
in pManager Pointer to the new AutoStackManager.

Return Values

None.

Description

Changes the AutoStackManager.

Sets the instance of the AutoStackManager-derived class used in the allocation and release of stacks with StartUsingAutoStack or TryStartUsingAutoStack.

You cannot use StartUsingAutoStack or TryStartUsingAutoStack until AutoStackManager is set with this function.
