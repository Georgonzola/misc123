nn::os::Thread::Sleep Member Function

Syntax

#include <nn/os.h>
static void Sleep(
     nn::fnd::TimeSpan span
);

Parameters

Name Description
in span Duration to sleep for.

Return Values

None.

Description

Puts the current thread to sleep for the time specified.

If this function is called repeatedly with a short time specified for the short period, it puts a heavy load on the system core and could diminish overall system performance.
