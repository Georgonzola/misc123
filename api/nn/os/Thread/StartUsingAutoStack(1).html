nn::os::Thread::StartUsingAutoStack Member Function

Syntax

#include <nn/os.h>
template <typename T, typename U>void StartUsingAutoStack(
     void(*)(T) f,
     U param,
     size_t stackSize,
     s32 priority = DEFAULT_THREAD_PRIORITY,
     s32 coreNo = CORE_NO_USE_PROCESS_VALUE
);

Template Arguments

Name Description
T Data type of the arguments of the function to run. (This type must be one that can be copied.)
U A type that can be converted to T.

Parameters

Name Description
in f Pointer to the function to begin running.
in param Argument to pass to the function that begins execution.
in stackSize The stack size.
in priority Priority of the thread.
in coreNo Preferred processor for the thread.

Return Values

None.

Description

Initializes and runs a thread.

Allocates a stack of the specified size using AutoStackManager. The thread is automatically destroyed when it exits.
