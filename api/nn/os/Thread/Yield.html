nn::os::Thread::Yield Member Function

Syntax

#include <nn/os.h>
static void Yield();

Parameters

None.

Return Values

None.

Description

Gives threads with the same priority level as the current thread a chance to run.

The scheduler performs priority scheduling. The thread with the highest priority (of all executable threads) is run.

When multiple threads have the same highest priority, the scheduler attempts to continue to run the current thread. No other threads will be executed as long as the current thread remains executable.

This function vacates execution rights for other threads at the same priority level while keeping the current thread executable.

If there are no threads of the same priority, nothing happens.
