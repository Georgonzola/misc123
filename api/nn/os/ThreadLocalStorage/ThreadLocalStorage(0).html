nn::os::ThreadLocalStorage::ThreadLocalStorage Constructor

Syntax

#include <nn/os.h>
ThreadLocalStorage();

Parameters

None.

Description

Creates a thread-local storage object and allocates one slot.

In the present implementation, up to 16 slots can be secured. If the upper limit is exceeded and securing a slot fails, the program stops. (In the retail product environment, this is handed as a fatal error.)
