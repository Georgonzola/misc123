nn::os::Tick::ToSeconds Member Function

Syntax

#include <nn/os.h>
s64 ToSeconds() const;

Parameters

None.

Return Values

Returns the value of this Tick object converted to seconds.

Description

Converts to seconds.

Performs processing equivalent to this->ToTimeSpan().GetSeconds faster. Note that because some midway calculations are omitted, some errors occur compared to the result from the method mentioned on the left.
