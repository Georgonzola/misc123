Syntax

void ConvertToParam(
     s32 * pYear,

     s32 * pHour,
     s32 * pMinute,
     s32 * pSecond

Parameters

Name Description
out pYear Stores the year. (Specify NULL if not needed.)

out pSecond Stores the second. (Specify NULL if not needed.)

None.

Description
