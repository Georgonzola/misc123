nn::phtsel::CTR::DateTimeConvert::GetMinSeconds Member Function

Syntax

static const DateTimeConvert & GetMinSeconds();

Parameters

None.

Return Values

Elapsed seconds.

Description

Gets the minimum value that can be set for elapsed seconds.

The number of seconds that elapsed as of 1900/01/01 00:00:00.
