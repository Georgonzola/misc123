nn::phtsel::CTR::PhtselInput Structure

Syntax

struct PhtselInput

Description

Parameter structure to pass to the applet.

Used as a member variable of the Parameter structure. You must always call the InitializeParameter function to initialize this structure before setting its member variables.

Enumerated Types

Constant Definitions
ImageKind The kind of image data.
ImageKindBit Bit flag for the kind of photo data.
ScreenShotType Filters for screenshots.
FaceType Filters for face information.
DistributeType Filters for distributability.
STSize Thumbnail size.

Constant Definitions

S TITLE_UNIQUE_ID_MAX_NUM const size_t The maximum number of title unique IDs that can be handled.
