nn::phtsel::CTR::PhtselInput::SetSecStart Member Function

Syntax

void SetSecStart(
     const DateTimeConvert & secondsStart
);

Parameters

Name Description
in secondsStart Starting time for when a photo was taken or video was recorded.

Return Values

None.

Description

Sets the starting time for when a photo was taken or video was recorded.

Photos are displayed if they were taken sometime within the range of time set by the functions SetSecStart and SetSecEnd.
