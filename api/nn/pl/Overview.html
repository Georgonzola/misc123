nn::pl Namespace

Description

The namespace of the common font library, pedometer library, and Play Coins library.

Namespaces

nn::pl::CTR The namespace of the CTR common font library, pedometer library, and Play Coins library. (It can also be written without the CTR portion.)

Classes

nn::pl::ResultSharedFontNotFound Indicates that shared font could not be found.
nn::pl::ResultGameCoinDataReset The result is success, but the save data has been initialized and the number of Play Coins was reset to 0.
nn::pl::ResultLackOfGameCoin Not enough Play Coins. Use the nn::pl::CTR::GetGameCoinCount function to determine the number of Play Coins owned, and set a value equal to or less than this number for the useCount parameter.
