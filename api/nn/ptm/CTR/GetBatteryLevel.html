nn::ptm::CTR::GetBatteryLevel Function

Syntax

BatteryLevel GetBatteryLevel();

Parameters

None.

Return Values

Returns the remaining battery charge level.

Description

Gets the remaining battery charge level.

A maximum time lag of 10 seconds occurs from the moment there is an internal change in the remaining battery level until a state that reflects that value is available.
