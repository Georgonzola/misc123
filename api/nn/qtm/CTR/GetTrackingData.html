nn::qtm::CTR::GetTrackingData Function

Syntax

nn::Result GetTrackingData(
     TrackingData * data
);

Parameters

Name Description
out data Specifies a data buffer for storing the tracking results.

Return Values

Returns the result of the function. Returns one of the following Result values.

Value Description
Result::IsSuccess Indicates that the operation succeeded.
ResultNoData Failed to get the data.
ResultInvalidArgument Indicates that an invalid buffer was passed as an argument.

Description

Gets the face tracking results. This function gets the positions of the left and right eyes of the user in front of the system. The output data includes the eye position data and state data to support the handling of the position data. This data cannot be retrieved while the camera library is in use.
