nn::qtm::CTR Namespace

Description

The namespace of the CTR face tracking library. (The namespace nn::uds::CTR can also be shortened to nn::uds.)

Classes

nn::qtm::CTR::ResultInvalidArgument Indicates an invalid argument.
nn::qtm::CTR::ResultNoData Indicates an inability to get face detection data.

Structures

nn::qtm::CTR::TrackingData Structure that stores face detection results.

Functions

Other
Initialize Initializes the QTM library and prepares it for use.
Finalize Finalizes the QTM library.
GetTrackingData Gets the face tracking results. This function gets the positions of the left and right eyes of the user in front of the system. The output data includes the eye position data and state data to support the handling of the position data. This data cannot be retrieved while the camera library is in use.
