nn::qtm::CTR::TrackingData Structure

Syntax

struct TrackingData
{
   bool isValid;
   bool isClipped;
   nn::math::VEC2 leftEyePlanar;
   nn::math::VEC2 rightEyePlanar;
   nn::math::VEC2 leftEyeTangent;
   nn::math::VEC2 rightEyeTangent;
};

Description

Structure that stores face detection results.

Member Variables

isValid bool This flag indicates whether the retrieved coordinates are valid.
isClipped bool This flag indicates whether the retrieved data is clipped within the valid range.
leftEyePlanar nn::math::VEC2 Normalized x-coordinate and y-coordinate for the left eye. The x-axis uses a range of [-1.0,1.0] and the y-axis uses a range of [-0.75, 0.75] with the center of the image as the origin (0, 0).
rightEyePlanar nn::math::VEC2 Normalized x-coordinate and y-coordinate for the right eye. The x-axis uses a range of [-1.0,1.0] and the y-axis uses a range of [-0.75, 0.75] with the center of the image as the origin (0, 0).
leftEyeTangent nn::math::VEC2 Tangent value for the z-axis (camera optic axis) of the left eye position. Represents the angle from the optic axis of the IN camera.
rightEyeTangent nn::math::VEC2 Tangent value for the z-axis (camera optic axis) of the right eye position. Represents the angle from the optic axis of the IN camera.
