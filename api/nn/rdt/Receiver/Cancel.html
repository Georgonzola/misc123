nn::rdt::Receiver::Cancel Member Function

Syntax

void Cancel(
     void
);

Parameters

None.

Return Values

None.

Description

Cancels the operation.

Call this function when you want to forcibly stop communication based on a command by the player. Calling this function makes the Receiver instance transition to the RECEIVER_STATE_CLOSED state. This function is an exception in that it runs its operation right away, without waiting for the Process function to run it.
