nn::rdt::Receiver::Receive Member Function

Syntax

nn::Result Receive(
     void * pBuf,
     size_t * pRecvSize,
     size_t bufSize
);

Parameters

Name Description
out pBuf Specifies the starting address of the buffer where the received data is written.
out pRecvSize Stores the number of bytes that were read from the receive buffer. If no data was readable, this parameter is set to 0.
in bufSize Holds the size, in bytes, of the buffer where the received data is written. If this parameter is set to 0, the function does nothing and returns a non-error value.

Return Values

Returns the result of the function. Specifically, this function may return the following: a value for which the nn::Result::IsSuccess() function returns true, or the ResultDoNothing, ResultNotInitialized, ResultNullPointer, or ResultUntimelyFunctionCall result code.

Description

Reads the data that has accumulated in the receive buffer. Reads the byte string that was sent from the Sender instance and has accumulated in the receive buffer. This byte string is guaranteed to be obtainable in a form that maintains the byte sequence sent by nn::rdt::Sender::Send. We recommend calling this function every game frame to improve throughput.
