nn::rdt::Sender::Send Member Function

Syntax

nn::Result Send(
     const void * pBuf,
     size_t bufSize
);

Parameters

Name Description
in pBuf Pointer to the start of the data to send. It must not be a null pointer.
in bufSize The size of the data to send (in bytes). If this parameter is set to 0, the function does nothing and returns a non-error value.

Return Values

Returns a value indicating whether data was successfully written to the send buffer. Specifically, this function may return the following: a value for which the nn::Result::IsSuccess function returns true, or the result code ResultNotInitialized, ResultDoNothing, ResultSendBufferIsNotAvailable, or ResultUntimelyFunctionCall.

Description

Writes data to the send buffer.

After you confirm that this call has succeeded, you can free the memory pointed to by pBuf because the data it points to has already been copied to the send buffer. Calls to this function fail if it cannot write all of the requested data because the capacity of the send buffer is too small. No data is written to the send buffer at all in such cases. If Send fails, wait a short time and then try again. The act of sending the data in the send buffer is actually executed by the Process function.
