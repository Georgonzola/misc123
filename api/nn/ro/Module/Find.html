nn::ro::Module::Find Member Function

Syntax

static Module * Find(
     const char * pName
);

Parameters

Name Description
in pName Name of the module to get a pointer to.

Return Values

Returns a pointer to the module specified by pName if it can be found.

Description

Searches for a module.

Searches for the module with the specified name from among all modules that were loaded by the LoadModule function with true specified for the doRegister parameter, and returns a pointer to that module.

If not found, returns NULL.
