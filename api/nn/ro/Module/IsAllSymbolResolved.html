nn::ro::Module::IsAllSymbolResolved Member Function

Syntax

bool IsAllSymbolResolved() const;

Parameters

None.

Return Values

Returns true if all references from this module to other modules have been resolved.

Description

Determines if all references are resolved.

Determines whether all references from this module to other modules have been resolved.

There is no effect on results regardless of whether references from other modules to this module have been resolved.

Returns false if even one reference remains unresolved.
