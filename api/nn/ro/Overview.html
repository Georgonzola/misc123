nn::ro Namespace

Description

The namespace of the library providing DLL functionality.

Also see nnro.

For more information about how to create the DLL, see the DLL Manual.

Classes

nn::ro::Module Class for representing modules.
nn::ro::RegistrationList Class for representing crr objects.

Structures

nn::ro::SizeInfo This structure stores memory size information required to load a cro.
nn::ro::RegionInfo This structure stores information about the memory region being used by a cro.

Enumerated Types

FixLevel This enumerated type is used to represent the level of memory to be released when LoadModule executes successfully.

Constants

S RS_ALIGNMENT const s32 Specifies the number of bytes that the starting address of the crs specified in Initialize must be aligned to. It is defined as 4096 bytes.
S RS_UNITSIZE const s32 Specifies the number of bytes that the size of the crs specified in Initialize must be a multiple of. It is defined as 4096 bytes.
S RR_ALIGNMENT const s32 Specifies the number of bytes that the starting address of the crr specified in RegisterList must be aligned to. It is defined as 4096 bytes.
S RR_UNITSIZE const s32 Specifies the number of bytes that the size of the crr specified in RegisterList must be a multiple of. It is defined as 4096 bytes.
S RO_ALIGNMENT_GET_SIZE_INFO const s32 Specifies the number of bytes that the starting address of the cro specified in GetSizeInfo must be aligned to. It is defined as 4 bytes.
S RO_ALIGNMENT_LOAD_MODULE const s32 Specifies the number of bytes that the starting address of the cro specified in LoadModule must be aligned to. It is defined as 4096 bytes.
S RO_UNITSIZE_LOAD_MODULE const s32 Specifies the number of bytes that the size of the cro specified in LoadModule must be a multiple of. It is defined as 4096 bytes.
S BUFFER_ALIGNMENT const s32 Specifies the number of bytes that the address of the buffer specified in LoadModule must be aligned to. It is defined as 8 bytes.

Functions

Initialize Initializes the ro library.
Finalize Finalizes the ro library.
RegisterList Registers a crr.
GetSizeInfo Gets the memory size information required to load the target cro.
GetAddress Searches for a symbol.
GetPointer Searches for a symbol.
FindModule Searches for a module.
LoadModule Loads a cro.
