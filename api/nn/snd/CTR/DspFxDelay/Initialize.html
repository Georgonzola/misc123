nn::snd::CTR::DspFxDelay::Initialize Member Function

Syntax

bool Initialize(
     uptr buffer,
     size_t size
);

Parameters

Name Description
in buffer Delay buffer pointer.
in size Delay buffer size.

Return Values

Returns true if successful; otherwise returns false.

Description

Initializes a DspFxDelay object and registers a work buffer.

The memory specified for buffer must be 32-byte aligned. Otherwise, this feature will still work but performance may degrade.
