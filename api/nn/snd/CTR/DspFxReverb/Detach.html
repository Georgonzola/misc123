nn::snd::CTR::DspFxReverb::Detach Member Function

Syntax

void Detach();

Parameters

None.

Return Values

None.

Description

Detaches an instance of this class from the AUX bus.

This function automatically calls the nn::snd::CTR::DspFxReverb::Disable function as necessary.
