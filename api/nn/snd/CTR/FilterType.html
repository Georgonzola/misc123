nn::snd::CTR::FilterType Enumerated Type

Syntax

enum FilterType;

Value

Value Description
FILTER_TYPE_NONE Indicates no filter.
FILTER_TYPE_MONOPOLE Indicates a monopole filter.
FILTER_TYPE_BIQUAD Indicates a biquad filter.
FILTER_TYPE_BOTH Combines both the monopole and biquad filters.

Description

Enumerated type representing the filter type.
