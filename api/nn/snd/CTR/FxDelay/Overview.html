nn::snd::CTR::FxDelay Class

Syntax

class FxDelay

Description

The delay effect class.

Call the SetEffect function to execute an effect.

Effects are processed in the CPU (not in the DSP).

You can use the m_IsEnableSound flag in the Param structure to choose whether effects are processed only with the front two channels or with all four channels, including the two in the rear.
