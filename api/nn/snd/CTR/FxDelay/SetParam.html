nn::snd::CTR::FxDelay::SetParam Member Function

Syntax

bool SetParam(
     const FxDelay::Param & param
);

Parameters

Name Description
in param The delay parameter.

Return Values

Returns true if settings are successful. Returns false if settings fail (such as a value is out of bounds).

Description

Configures the delay parameters.

The GetRequiredMemSize function calculates the memory size required based on the delay parameters that have been set.

The Param structure is copied inside the instance of FxDelay.

The operations of FxDelay differ depending on whether it is before or after you pass it to the SetEffect function.

Before calling the SetEffect function, range checks are performed on the m_FeedbackGain and m_Damping members of the Param structure.

When out of range, Debug and Development versions halt on an assert, while false is returned in the Release version.

After calling the SetEffect, function, a check is performed to determine whether additional memory is needed, in addition to the range check performed before calling.

In other words, if a larger value is configured for m_DelayTime than was configured when the SetEffect function was called, or if m_IsEnableSurround is set to true even though it was set to false when the SetEffect function was called, this function fails and returns false.

If these conditions are not met, subsequent parameter changes can be made with the SetEffect function.
