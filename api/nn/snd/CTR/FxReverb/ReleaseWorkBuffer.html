nn::snd::CTR::FxReverb::ReleaseWorkBuffer Member Function

Syntax

void ReleaseWorkBuffer();

Parameters

None.

Return Values

None.

Description

Frees the memory that was used by the effect.

Before deallocating with this function, you must clear the effect first using the SoundSystem::ClearEffect function.
