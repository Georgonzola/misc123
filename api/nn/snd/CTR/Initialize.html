nn::snd::CTR::Initialize Function

Syntax

nn::Result Initialize();

Parameters

None.

Return Values

Returns the result of the function. Returns one of the following Result values.

Value Description
Result::IsSuccess Indicates that the operation succeeded.
ResultAlreadyInitialized Specifies that the library is already initialized.
ResultNoDspComponentLoaded dspcomponent is not loaded.

Description

Initializes the sound library.

Before calling this function you must call both nn::dsp::CTR::Initialize and nn::dsp::CTR::LoadDefaultComponent.

In addition, you must call this function before calling any of the functions in the nn::snd::CTR namespace.
