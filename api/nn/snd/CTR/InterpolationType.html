nn::snd::CTR::InterpolationType Enumerated Type

Syntax

enum InterpolationType;

Value

Value Description
INTERPOLATION_TYPE_POLYPHASE Indicates polyphase (4-point) interpolation.
INTERPOLATION_TYPE_LINEAR Indicates linear interpolation.
INTERPOLATION_TYPE_NONE Indicates no interpolation.

Description

An enumerated type representing the interpolation method.

Indicates which interpolation algorithm is used during sampling rate conversion. To get or set this value, use the nn::snd::CTR::Voice::SetInterpolationType or nn::snd::CTR::Voice::GetInterpolationType function respectively.

When INTERPOLATION_TYPE_NONE is specified, neighboring sample values are used during interpolation. There may be noise in the interpolated results.

When INTERPOLATION_TYPE_LINEAR is specified, linear interpolation is used.

When INTERPOLATION_TYPE_POLYPHASE is specified, 4-point interpolation is used. The appropriate polyphase filter to use for interpolation is selected from the configured sample rate and pitch. Of the three algorithms, this one has the best sound quality.
