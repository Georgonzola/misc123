nn::snd::CTR::MixParam Structure

Syntax

struct MixParam
{
   f32 mainBus[CHANNEL_INDEX_NUM];
   f32 auxBusA[CHANNEL_INDEX_NUM];
   f32 auxBusB[CHANNEL_INDEX_NUM];
};

Description

A structure that stores the gain for each channel.

Member Variables

mainBus f32 The gain for each channel on the main bus (a value of 1.0 indicates unity gain).
auxBusA f32 The gain for each channel on effect bus 0 (a value of 1.0 indicates unity gain).
auxBusB f32 The gain for each channel on effect bus 1 (a value of 1.0 indicates unity gain).

Member Functions

MixParam Instantiates an object.
