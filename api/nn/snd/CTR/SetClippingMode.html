nn::snd::CTR::SetClippingMode Function

Syntax

bool SetClippingMode(
     ClippingMode mode
);

Parameters

Name Description
in mode Mode.

Return Values

Returns true if configuration succeeded; otherwise, returns false.

Description

Sets the clipping mode.

The default value is set to nn::snd::CTR::CLIPPING_MODE_SOFT.

For details on the clipping mode, see nn::snd::CTR::ClippingMode.
