nn::snd::CTR::SetEffect Function

Syntax

bool SetEffect(
     AuxBusId busId,
     FxReverb * fx
);

Parameters

Name Description
in busId Bus ID.
in fx The reverb effect to be configured.

Return Values

Returns true when the configuration was successful. When varfx/var is ttNULL/tt, the configuration fails and ttfalse/tt is returned.

Description

Configures the effect. Configures the reverb effect. To process data that is to be output by AUX, you can use this function to apply delay and reverb effects in addition to configuring advance processing with the RegisterAuxCallback function. Note that you can only configure one delay (FxDelay) and one reverb (FxReverb) for each bus. When you configure more than one, only the settings for the last one are valid. The older configured effects are cleared. If you have called ClearEffect to clear older configurations as noted previously, you can deallocate the memory used for those effects using the FxDelay::ReleaseWorkBuffer and FxReverb::ReleaseWorkBuffer functions. Before calling this function, you must call the FxReverb::SetParam, FxReverb::GetRequiredMemSize, and FxReverb::AssignWorkBuffer functions from the FxReverb class to finish preparing for the reverb effect.
