nn::snd::CTR::SetMasterVolume Function

Syntax

void SetMasterVolume(
     f32 volume
);

Parameters

Name Description
in volume The master volume (a value of 1.0 indicates no change in volume).

Return Values

None.

Description

This function sets the master volume.

The valid values for volume are in the range from 0.0 to 1.0. If a value greater than 1.0 has been specified, it is rounded down to 1.0 by an internal process.
