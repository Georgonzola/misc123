nn::snd::CTR::SetSurroundDepth Function

Syntax

bool SetSurroundDepth(
     f32 depth
);

Parameters

Name Description
in depth Surround sound depth (0 <= depth <= 1.0f).

Return Values

Returns true if configuration succeeded; otherwise, returns false.

Description

Sets the surround sound depth. The surround sound depth parameter indicates the intensity of the 3D surround sound effect when sound is output through the speakers. Larger values create a more pronounced effect. The surround sound depth has no effect when sound is output through headphones. The default value is 1.0f.
