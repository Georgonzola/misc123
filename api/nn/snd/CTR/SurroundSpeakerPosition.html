nn::snd::CTR::SurroundSpeakerPosition Enumerated Type

Syntax

enum SurroundSpeakerPosition;

Value

Value Description
SURROUND_SPEAKER_POSITION_SQUARE Indicates square mode (with speakers positioned at 45° and 135°).
SURROUND_SPEAKER_POSITION_WIDE Indicates wide mode (with speakers positioned at 60° and 110°).
SURROUND_SPEAKER_POSITION_NUM

Description

Enumerated type indicating the virtual speaker position for 3D surround sound.
