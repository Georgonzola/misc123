nn::snd::CTR::Voice::SetBiquadFilterCoefficients Member Function

Syntax

void SetBiquadFilterCoefficients(
     const BiquadFilterCoefficients & coeff
);

Parameters

Name Description
in coeff Coefficient buffer.

Return Values

None.

Description

Sets the coefficients for a biquad filter.

The biquad filter output and frequency characteristics are represented as follows.

y(n) = n0 * x(n) + n1 * x(n-1) + n2 * x(n-2) + d1 * y(n-1) + d2 * y(n-2)

                    n0 + n1(Z^-1) + n2(Z^-2)
           H(z) = ----------------------------
                    1  - d1(Z^-1) - d2(Z^-2)

The coefficients can all be specified as Q14-format 16-bit integers, comprising a 2-bit integer portion and a 14-bit fractional portion. Sounds may be clipped if the filter gain is higher than 1.
