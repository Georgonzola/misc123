nn::snd::CTR::Voice::SetMixParam Member Function

Syntax

void SetMixParam(
     const MixParam & mixParam
);

Parameters

Name Description
in mixParam Reference to a gain structure.

Return Values

None.

Description

Sets the gain for each voice channel.

A ratio of 1.0 represents no change from the original value. This function is used together with the volume set by nn::snd::CTR::Voice::SetVolume to calculate the volume. If the product of the value set by this function and the value set by nn::snd::CTR::Voice::SetVolume exceeds 2.0, it is rounded down to 2.0 by an internal process.

The default values are as follows.

Left FrontRight FrontLeft BackRight Back
mainBus1.01.00.00.0
auxBusA0.00.00.00.0
auxBusB0.00.00.00.0
