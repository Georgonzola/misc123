nn::snd::CTR::Voice::SetMonoFilterCoefficients Member Function

Syntax

void SetMonoFilterCoefficients(
     const MonoFilterCoefficients & coeff
);

Parameters

Name Description
in coeff Coefficient buffer.

Return Values

None.

Description

Sets the coefficients for a monopole filter.

The monopole filter output can be represented as follows.

y(n) = n0 * x(n) - d0 * y(n-1)

The coefficients can all be specified as Q15-format 16-bit integers, comprising a 1-bit integer portion and a 15-bit fractional portion. Sounds may be clipped if the filter gain is higher than 1.
