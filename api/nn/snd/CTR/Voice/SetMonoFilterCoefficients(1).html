nn::snd::CTR::Voice::SetMonoFilterCoefficients Member Function

Syntax

void SetMonoFilterCoefficients(
     u16 cutoff
);

Parameters

Name Description
in cutoff The cut-off frequency.

Return Values

None.

Description

Configures a monopole low-pass filter with the specified cut-off frequency.

The cut-off frequency, cutoff, is designed from the monopole filter and configured to Voice. cutoff must fall within the range 0 <= cutoff <= 16,000.
