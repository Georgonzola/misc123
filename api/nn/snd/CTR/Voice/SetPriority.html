nn::snd::CTR::Voice::SetPriority Member Function

Syntax

void SetPriority(
     s32 priority
);

Parameters

Name Description
in priority Priority.

Return Values

None.

Description

Sets the priority of a voice.

The priority takes a value between 0 (the lowest) and VOICE_PRIORITY_NODROP (the highest).

The priority affects the behavior when getting voices and when controlling the DSP processing load. See the description of the nn::snd::CTR::AllocVoice function.

Among voices that are set to the same level of priority, the voice that was set last takes the highest priority. Because the nn::snd::CTR::AllocVoice function sets the level of priority internally, nn::snd::CTR::AllocVoice is also included in the above description.
