nn::socket::AddrInfo Structure

Syntax

struct AddrInfo
{
   int family;
   int sockType;
   int protocol;
   unsigned addrLen;
   char * canonName;
   void * addr;
   AddrInfo * next;
};

Description

Structure for use with the GetAddrInfo() function.

Member Variables

family int Sets flags.
sockType int Socket address family.
protocol int Socket type.
addrLen unsigned Socket protocol.
canonName char * Size of the socket struct.
addr void * Canonical name.
next AddrInfo * Pointer to the socket struct.
