nn::socket::HostEnt Structure

Syntax

struct HostEnt
{
   char * name;
   char ** aliases;
   s16 addrType;
   s16 length;
   u8 ** addrList;
};

Description

Structure for use with the GetHostByAddr() and GetHostByName() functions.

Member Variables

name char * The host's official name.
aliases char ** A pointer to a list of aliases.
addrType s16 The address type. PF_INET
length s16 Size of the socket struct.
addrList u8 ** A pointer to a list of socket structures.
