nn::socket::Linger Structure

Syntax

struct Linger
{
   int onoff;
   int linger;
};

Description

A structure used with SO_LINGER in the GetSockOpt() and SetSockOpt() functions.

Member Variables

onoff int 1: Enable linger behavior. 0: Disable linger behavior.
linger int Linger time (in seconds).
