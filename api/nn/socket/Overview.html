nn::socket Namespace

Description

The namespace of the socket library.

This library is provided for initializing and finalizing sockets when NEX features are used.
Contact Nintendo if you want to use this library for direct communication.

Structures

nn::socket::SockAddrIn An IPv4 socket structure.
nn::socket::SockAddr A socket structure.
nn::socket::Linger A structure used with SO_LINGER in the GetSockOpt() and SetSockOpt() functions.
nn::socket::IpMreq A structure for when IP_ADD_MEMBERSHIP or IP_DROP_MEMBERSHIP is used in the GetSockOpt() and SetSockOpt() functions.
nn::socket::PollFd Structure for use with the Poll() function.
nn::socket::AddrInfo Structure for use with the GetAddrInfo() function.
nn::socket::HostEnt Structure for use with the GetHostByAddr() and GetHostByName() functions.
nn::socket::DnsAddr A structure for DNS server addresses.

Constants

S BUFFER_ALIGNMENT const s32 Specifies the number of bytes that the address of the working region specified in Initialize must be aligned to. It is defined as 4096 bytes.
