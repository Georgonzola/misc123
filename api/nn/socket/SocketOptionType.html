nn::socket::SocketOptionType Enumerated Type

Syntax

enum SocketOptionType;

Value

Value Description
IP_TOS The value of an IP header's TOS field (int).
IP_TTL The value of an IP header's TTL field (int).
IP_MULTICAST_LOOP (Unsupported) Loop multicast packets (unsigned char).
IP_MULTICAST_TTL (Unsupported) The value of a multicast packet's TTL field (unsigned char).
IP_ADD_MEMBERSHIP (Unsupported) Add a multicast group (IpMreq).
IP_DROP_MEMBERSHIP (Unsupported) Drop a multicast group (IpMreq).
SO_REUSEADDR Allow a socket to be reused (bool).
SO_LINGER Wait for a TCP disconnection to complete (Linger).
SO_OOBINLINE Handle urgent data as online data (bool).
SO_SNDBUF Size of the TCP send buffer (int).
SO_RCVBUF Size of the TCP receive buffer (int).
SO_SNDLOWAT Minimum usable value for the TCP send buffer (int).
SO_RCVLOWAT Minimum usable value for the TCP receive buffer (int).
SO_TYPE Socket type (int).
SO_ERROR Error set in a socket (int).
TCP_NODELAY Do not use delayed ACK messages (bool).
TCP_MAXSEG TCP MSS value (int).
TCP_STDURG Make the TCP urgent pointer conform to the RFC specifications (bool).
TCP_R2 R2 timer value for TCP (int).
TCP_R2_SYN R2 timer value for TCP SYN (int).

Description

Option numbers for the GetSockOpt() and SetSockOpt() functions.
