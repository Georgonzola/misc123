nn::swkbd::CTR::APP_TEXT_CHECK_DIALOG_TEXT_LENGTH_MAX Constant

Syntax

const u16 APP_TEXT_CHECK_DIALOG_TEXT_LENGTH_MAX =  256;

Description

The maximum number of characters in the text of the application's text checking dialog box.

This is the maximum number of characters displayed when the application's text check result indicates a problem.
The text is specified by the ppDialogText argument of the nn::swkbd::CTR::AppTextCheckCallback callback function, which stores the starting address of the string.
