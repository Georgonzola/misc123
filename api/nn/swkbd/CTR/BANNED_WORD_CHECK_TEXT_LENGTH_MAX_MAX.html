nn::swkbd::CTR::BANNED_WORD_CHECK_TEXT_LENGTH_MAX_MAX Constant

Syntax

const u16 BANNED_WORD_CHECK_TEXT_LENGTH_MAX_MAX =  64;

Description

The maximum value for the maximum number of characters that can be entered when input of profanity is prohibited.

The profanity check is used in the following function.

nn::ngc::CTR::ProfanityFilter::CheckProfanityWords(bit32 *, bool, const wchar_t **, size_t)

Due to the specifications of this function, the maximum number of characters for profanity words includes the NULL terminator. When strings longer than BANNED_WORD_CHECK_TEXT_LENGTH_MAX_MAX are specified, the text after the maximum character limit is not checked.

When the TEXT_CHECK_BANNED_WORDS flag is set in the textCheckFlag member variable of the nn::swkbd::CTR::Config structure, the value set in the textLengthMax member variable is clamped not to exceed this constant's value.
