nn::swkbd::CTR::BottomButton Enumerated Type

Syntax

enum BottomButton;

Value

Value Description
BOTTOM_BUTTON_0 (An enumerator used for internal processing. Cannot be used by the application.)
BOTTOM_BUTTON_1 (An enumerator used for internal processing. Cannot be used by the application.)
BOTTOM_BUTTON_2 (An enumerator used for internal processing. Cannot be used by the application.)
BOTTOM_BUTTON_MAX (An enumerator used for internal processing. Cannot be used by the application.)
BOTTOM_BUTTON_1BUTTON A 1-button button.
BOTTOM_BUTTON_2BUTTON_LEFT A 2-button's left button.
BOTTOM_BUTTON_2BUTTON_RIGHT A 2-button's right button.
BOTTOM_BUTTON_3BUTTON_LEFT A 3-button's left button.
BOTTOM_BUTTON_3BUTTON_MIDDLE A 3-button's middle button.
BOTTOM_BUTTON_3BUTTON_RIGHT A 3-button's right button.
BOTTOM_BUTTON_MAX_BIT (An enumerator used for internal processing. Cannot be used by the application.)

Description

Enumerated type for the indices of the buttons on the bottom row.
