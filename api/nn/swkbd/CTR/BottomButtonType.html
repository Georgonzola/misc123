nn::swkbd::CTR::BottomButtonType Enumerated Type

Syntax

enum BottomButtonType;

Value

Value Description
BOTTOM_BUTTON_TYPE_1BUTTON 1-button
BOTTOM_BUTTON_TYPE_2BUTTON 2-button
BOTTOM_BUTTON_TYPE_3BUTTON 3-button
BOTTOM_BUTTON_TYPE_MAX Total number of enumerators.

Description

Enumerated type for the types of buttons on the bottom row.
