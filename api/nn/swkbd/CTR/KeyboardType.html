nn::swkbd::CTR::KeyboardType Enumerated Type

Syntax

enum KeyboardType;

Value

Value Description
KEYBOARD_TYPE_FULL Full keyboard.
KEYBOARD_TYPE_QWERTY QWERTY keyboard.
KEYBOARD_TYPE_TENKEY Numeric keypad.
KEYBOARD_TYPE_NO_JAPANESE Keyboard without Japanese.
KEYBOARD_TYPE_NNID (This enumerator is for built-in applications. It cannot be used from general applications.)
KEYBOARD_TYPE_ACCOUNT_PASSWORD (This enumerator is for built-in applications. It cannot be used from general applications.)
KEYBOARD_TYPE_MAX Total number of enumerators.
KEYBOARD_TYPE_MAX_BIT (An enumerator used for internal processing. Cannot be used by the application.)

Description

Enumerated type indicating the type of keyboard.

In the KEYBOARD_TYPE_FULL type, all the key layout types are available including the QWERTY layout, the symbol layout, and the cell phone layout.

In the KEYBOARD_TYPE_QWERTY type, only the QWERTY layout is available.
Systems designed for the Japan region do not allow romaji-based kana input.
The characters that can be input vary depending on the region and language settings of the system.

In the KEYBOARD_TYPE_TENKEY type, only the numeric keypad is available.

If the system region is Japan, the KEYBOARD_TYPE_NO_JAPANESE type gives access to the QWERTY and symbol layouts only. No Japanese input is possible from this keyboard, because it does not allow romaji-based kana input.
In other regions, this is the same as KEYBOARD_TYPE_FULL except that predictive text input is not available.
