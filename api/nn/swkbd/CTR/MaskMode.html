nn::swkbd::CTR::MaskMode Enumerated Type

Syntax

enum MaskMode;

Value

Value Description
MASK_MODE_DISABLE Do not mask passwords.
MASK_MODE_IMMEDIATE Mask immediately after input.
MASK_MODE_WAIT_1SEC Wait one second after input or until the next character is input before masking passwords.
MASK_MODE_MAX Total number of enumerators.
MASK_MODE_MAX_BIT (An enumerator used for internal processing. Cannot be used by the application.)

Description

Enumerated type that indicates the masking mode for input characters.
