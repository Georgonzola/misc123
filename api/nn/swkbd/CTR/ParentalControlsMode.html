nn::swkbd::CTR::ParentalControlsMode Enumerated Type

Syntax

enum ParentalControlsMode;

Value

Value Description
PARENTAL_CONTROLS_MODE_DISABLE Do not verify the Parental Controls PIN code.
PARENTAL_CONTROLS_MODE_ENABLE Verify the Parental Controls PIN code.
PARENTAL_CONTROLS_MODE_SYSTEM_SETTINGS (This enumerator is for built-in applications. It cannot be used from general applications.)
PARENTAL_CONTROLS_MODE_MAX Total number of enumerators.
PARENTAL_CONTROLS_MODE_MAX_BIT (An enumerator used for internal processing. Cannot be used by the application.)

Description

Enumerated type that represents the Parental Controls PIN code verification modes.

To check whether the PIN code matches the Parental Controls PIN code, use the same process as for the nn::cfg::CTR::CheckParentalControlPinCode function.
