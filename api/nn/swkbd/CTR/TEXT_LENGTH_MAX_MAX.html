nn::swkbd::CTR::TEXT_LENGTH_MAX_MAX Constant

Syntax

const u16 TEXT_LENGTH_MAX_MAX =  65000;

Description

The maximum value for the maximum number of characters that can be input.

This is the maximum value that can be set in the textLengthMax member variable of the nn::swkbd::CTR::Config structure.
