nn::ubl::GetUserId Function

Syntax

u64 GetUserId(
     void
);

Parameters

None.

Return Values

ID of the CTR system.

Description

Returns the unique ID of each CTR system. This ID is shared by all applications. Use nn::cfg::GetTransferableId if you are identifying a system for a purpose other than allowing or disallowing reception.
