nn::ubl::IsExist Function

Syntax

bool IsExist(
     u64 authorId,
     u32 titleId = 0,
     u64 dataId = 0
);

Parameters

Name Description
in authorId User ID to search for.
in titleId Title ID to search for.
in dataId UGC data ID to search for.

Return Values

Returns true if the specified element matches; otherwise, returns false.

Description

Determines whether the specified ID exists in the blocked user list.

The title ID and UGC data ID are not currently used.

Revision History
