nn::uds::CTR::Attach Function

Syntax

nn::Result Attach(
     EndpointDescriptor * pEndpointDesc,
     u16 srcNodeId,
     u8 port,
     size_t receiveBufferSize = ATTACH_BUFFER_SIZE_DEFAULT
);

Parameters

Name Description
in pEndpointDesc Specifies a descriptor indicating the endpoint to use.
in srcNodeId Specifies the node ID being assigned. Specify BROADCAST_NODE_ID to attach all nodes.
in port Specifies the port being assigned. Port 0x00 is reserved by the system and cannot be used.
in receiveBufferSize Specifies the size of the receive buffer being allocated. Memory is allocated from the memory block retrieved by the Initialize function. Do not try to allocate a receive buffer larger than that total. You must specify at least ATTACH_BUFFER_SIZE_MIN bytes. The default value is ATTACH_BUFFER_SIZE_DEFAULT bytes. When specifying the data size, choosing a multiple of 32 yields the maximum efficiency because memory is allocated in 32-byte chunks.

Return Values

Returns the result of the function. Returns one of the following Result values.

Value Description
Result::IsSuccess Indicates that the operation succeeded.
ResultNotInitialized Indicates that the library is not initialized. Call the Initialize function.
ResultInvalidState Indicates that the function cannot be executed in this state.
ResultOutOfResource This value is returned when the receive buffer specified by the Initialize function is insufficient and when there are insufficient system resources.
ResultOutOfRange Indicates that an argument was outside of the allowed range. The function might succeed if you re-execute with an appropriate value for the argument.
ResultWirelessOff Indicates that the device entered wireless off mode. Reinitialization is required.
Other Indicates failure for other reasons.

Description

Enables the local device to receive packets from the endpoint. Assigns the specified port and node ID to the endpoint. The system creates a receive buffer and prepares to receive packets at the port specified below.

This function is thread-safe.
