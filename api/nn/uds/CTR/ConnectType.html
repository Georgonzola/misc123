nn::uds::CTR::ConnectType Enumerated Type

Syntax

enum ConnectType;

Value

Value Description
CONNECT_AS_CLIENT Connect to the network as a client (a node that can both send and receive data and is given an ID when it connects).
CONNECT_AS_SPECTATOR Connect to the network as a spectator (a node that can only receive data and is not given an ID when it connects).
CONNECT_AS_MASTER Connect to the network as a host (only used internally by the library).

Description

Enumerated type that represents modes used during connection to the network.
