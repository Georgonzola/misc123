nn::uds::CTR::CreateEndpoint Function

Syntax

nn::Result CreateEndpoint(
     EndpointDescriptor * pEndpointDesc
);

Parameters

Name Description
out pEndpointDesc Descriptor indicating the endpoint to create.

Return Values

Returns the result of the function. Returns one of the following Result values.

Value Description
Result::IsSuccess Indicates that the operation succeeded.
ResultNotInitialized Indicates that the library is not initialized. Call the Initialize function.
ResultOutOfResource More than the stipulated number of endpoints were already generated. Destroy an existing endpoint.
Other Indicates failure for other reasons.

Description

Creates a network endpoint and returns the corresponding descriptor.

[Provisional spec] Currently, up to 16 endpoints can be created.

This function is thread-safe.
