nn::uds::CTR::DestroyEndpoint Function

Syntax

nn::Result DestroyEndpoint(
     EndpointDescriptor * pEndpointDesc
);

Parameters

Name Description
in pEndpointDesc Descriptor indicating the endpoint.

Return Values

Returns the result of the function. Returns one of the following Result values.

Value Description
Result::IsSuccess Indicates that the operation succeeded.
ResultNotInitialized Indicates that the library is not initialized. Call the Initialize function.
ResultNotAuthorized Indicates that an invalid endpoint descriptor was specified.
Other Indicates failure for other reasons.

Description

Destroys an endpoint.

Frees the receive buffer if the Attach function has been called with the specified endpoint. The freed receive buffer can be reused with the Attach function, but note that the application can only access it after the Finalize function is executed.

This function is thread-safe.
