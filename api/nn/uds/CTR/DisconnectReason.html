nn::uds::CTR::DisconnectReason Enumerated Type

Syntax

enum DisconnectReason;

Value

Value Description
BEFORE_COMMUNICATION Not yet communicating.
NETWORK_IS_AVAILABLE Connection being maintained.
REQUEST_FROM_MYSELF Disconnected from the network due to local action.
REQUEST_FROM_SYSTEM Disconnected from the network due to a system request (transition to wireless off mode or sleep mode).
DISCARDED_FROM_NETWORK Kicked off the network due to a command from the master device. The client was disconnected because of deteriorating signal strength and the internal state of the master.
CONNECTION_LOST The signal deteriorated and the beacon from the master could not be received during a set time, so the connection could not be maintained.
UNKNOWN_DISCONNECT_REASON Disconnected due to an unknown reason.
DISCONNECT_REASON_MAX
DISCONNECT_REASON_MAX_BIT

Description

Enumerates the reasons for disconnection.
