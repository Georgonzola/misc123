nn::uds::CTR::EndpointDescriptor Structure

Syntax

struct EndpointDescriptor
{
   u32 id;
};

Description

This is an endpoint descriptor. Corresponds to a socket descriptor.

Currently this descriptor contains only an ID, but other fields might be added in the future.

Member Variables

id u32 ID assigned each time CreateEndpoint is called.
