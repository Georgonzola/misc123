nn::uds::CTR::FORCE_DIRECT_BC Constant

Syntax

const bit8 FORCE_DIRECT_BC =  0x02;

Description

When SendTo has been specified, sends using direct broadcast without asking for a recipient. Unicast communication among clients takes place without the use of a host. As a result, you can have low latency communication, but packets may not arrive.
