nn::uds::CTR::GetChannel Function

Syntax

nn::Result GetChannel(
     u8 * pChannel
);

Parameters

Name Description
out pChannel Returns the channel.

Return Values

Returns the result of the function. Returns one of the following Result values.

Value Description
Result::IsSuccess Indicates that the operation succeeded.
ResultNotInitialized Indicates that the library is not initialized. Execute the nn::uds::Initialize function.
Other Indicates failure for other reasons.

Description

Gets the current channel.

This function is thread-safe.
