nn::uds::CTR::GetLinkLevel Function

Syntax

nn::Result GetLinkLevel(
     LinkLevel * pLinkLevel
);

Parameters

Name Description
out pLinkLevel The current link level.

Return Values

Returns the result of the function. Returns one of the following Result values.

Value Description
Result::IsSuccess Indicates that the operation succeeded.
Other Indicates failure for other reasons.

Description

Gets the current link level.

This function is thread-safe.
