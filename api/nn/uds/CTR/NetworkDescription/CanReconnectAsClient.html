nn::uds::CTR::NetworkDescription::CanReconnectAsClient Member Function

Syntax

bool CanReconnectAsClient() const;

Parameters

None.

Return Values

Returns whether client reconnections are allowed.

Value Description
true Allowed.
false Not allowed.Checks whether reconnection is permitted for clients. The determination of whether it is a reconnection is made when the host actually connects. There is no guarantee that reconnection is possible even if this value is true.

Description

Checks whether client reconnections are allowed. The determination of whether it is a reconnection is made when the host actually connects. There is no guarantee that reconnection is possible even if this value is true.
