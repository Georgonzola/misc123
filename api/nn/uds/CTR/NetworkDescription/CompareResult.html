nn::uds::CTR::NetworkDescription::CompareResult Enumerated Type

Syntax

enum CompareResult;

Value

Value Description
SAME_NETWORK_SAME_DATA The NetworkDescription is for the same UDS network and the data also matches completely.
SAME_NETWORK_DIFFERENT_DATA The NetworkDescription is for the same UDS network but the data is different. (Example: The number of people has changed or some other change has updated the data.)
DIFFERENT_NETWORK The NetworkDescription is for a different UDS network.

Description

Result of NetworkDescription::CompareWith.
