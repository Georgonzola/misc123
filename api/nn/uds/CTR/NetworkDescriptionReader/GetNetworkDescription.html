nn::uds::CTR::NetworkDescriptionReader::GetNetworkDescription Member Function

Syntax

nn::Result GetNetworkDescription(
     NetworkDescription * pNetworkDescription
);

Parameters

Name Description
out pNetworkDescription Network information storage location. The data is only stored if the network information was acquired correctly.

Return Values

Returns the execution result of the function. Returns one of the following Result values.

Value Description
Result::IsSuccess The network information has been obtained successfully.
ResultMalformedData The data may have been tampered with and could not be obtained.
Other Indicates failure for other reasons.

Description

Gets the network information from scan results.
