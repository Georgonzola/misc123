nn::uds::CTR::NodeInformation Structure

Syntax

struct NodeInformation
{
   ScrambledLocalFriendCode scrambledLocalFriendCode;
   nn::cfg::CTR::UserName userName;
   u16 nodeId;
};

Description

Stores user information about nodes connected to the network.

Member Variables

scrambledLocalFriendCode ScrambledLocalFriendCode Data that can be converted to a local friend code by using the friends library. This type of data helps protect privacy of users.
userName nn::cfg::CTR::UserName User name data. For more information, see nn::cfg::CTR::UserName.
nodeId u16 Node ID.

Revision History

2010/10/15
To protect privacy, changed the friendCode member of the NodeInformation structure to 96-bit encrypted data. The newly added scrambledLocalFriendCode can convert to a local friend code via the friends library.
