nn::uds::CTR::PowerSaveMode Enumerated Type

Syntax

enum PowerSaveMode;

Value

Value Description
ALWAYS_ACTIVE Mode that emphasizes low latency and high traffic, and that does not use any power-saving features.
POWERSAVE_NORMAL Normal mode. This mode has higher latency, but uses almost 20% less power for communication than when in ALWAYS_ACTIVE mode.
POWERSAVE_HIGH Mode that emphasizes power conservation. This mode has higher latency, but uses almost 50% less power for communication than when in ALWAYS_ACTIVE mode.
POWERSAVE_MODE_MAX

Description

Enumerates the network's power-saving modes.
