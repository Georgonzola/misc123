nn::uds::CTR::ResultWirelessOff Class

Syntax

class ResultWirelessOff

Description

Indicates that the system entered wireless-disabled mode after the UDS library was initialized.

After the system enters wireless-disabled mode and until Finalize is called, the UDS library interprets the wireless state as disabled even if the system subsequently enters wireless-enabled mode. To use UDS communication again, you need to rerun the Initialize function in wireless-enabled mode.
