nn::uds::CTR::SetApplicationDataToBeacon Function

Syntax

nn::Result SetApplicationDataToBeacon(
     const void * pData,
     size_t dataSize
);

Parameters

Name Description
in pData Pointer to the data being set.
in dataSize Size of the data to set. Specify a value no greater than NET_DESC_APPDATA_SIZE_MAX.

Return Values

Returns the result of the function. Returns one of the following Result values.

Value Description
Result::IsSuccess Indicates that the operation succeeded.
ResultNotInitialized Indicates that the library is not initialized. Call the Initialize function.
ResultInvalidState Indicates that the function cannot be executed in this state. Returned when executed in a state other than the host.
ResultTooLarge Indicates that dataSize is greater than NET_DESC_APPDATA_SIZE_MAX.
ResultWirelessOff Indicates that the device entered wireless off mode. Reinitialization is required.
Other Indicates failure for other reasons.

Description

Sets application data in the beacon.

Specifies up to NET_DESC_APPDATA_SIZE_MAX bytes of application data. The data is not encrypted. It can be collected by any general-purpose computing device, such as a desktop computer. Other devices can get this data by calling Scan while they are connected as a Client or Spectator. Only the host can run this function.

This function is thread-safe.
