nn::uds::CTR::State Enumerated Type

Syntax

enum State;

Value

Value Description
STATE_NONE Not initialized.
STATE_PROCESSING_INITIALIZE Internal state indicating that initialization is underway. Applications never see this value.
STATE_PROCESSING_FINALIZE Internal state indicating finalization is currently underway. Applications never see this value.
STATE_DISCONNECTED Not communicating (not connected).
STATE_CREATING_NETWORK Intermediate state between not connected, and building a new network and running as the host. Automatically transitions to STATE_MASTER.
STATE_DESTROYING_NETWORK Intermediate state between no longer running as the host and not connected. Automatically transition to STATE_DISCONNECTED.
STATE_MASTER Running as host.
STATE_CONNECTING_NETWORK Intermediate state between not connected, and building a new network and running as a client or spectator. Transitions to STATE_DISCONNECTED if the connection fails or STATE_CLIENT/STATE_SPECTATOR if the connection succeeds
STATE_DISCONNECTING_NETWORK Intermediate state between disconnecting from the network and not connected. Automatically transitions to STATE_DISCONNECTED.
STATE_CLIENT Running as a client. If the connection degrades to the point where the network connection can no longer be maintained, automatically transitions to STATE_DISCONNECTING_NETWORK.
STATE_SPECTATOR Running as a spectator. If the connection degrades to the point where the network connection can no longer be maintained, automatically transitions to STATE_DISCONNECTING_NETWORK.
STATE_ERROR Specifies that the UDS library must be initialized. The library transitions to this state if wireless communications are turned off or the system enters sleep mode while using UDS.
STATE_MAX
STATE_MAX_BIT

Description

Enumerates UDS states.
