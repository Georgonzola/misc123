nn::ulcd::CTR::DynamicStereoCamera::DynamicStereoCamera Constructor

Syntax

#include <nn/ulcd.h>
DynamicStereoCamera();

Parameters

None.

Description

Instantiates the object with default parameters (default constructor). After the class object is generated, it must be configured with at least one of the nn::ulcd::CTR::DynamicStereoCamera::SetView functions and one of the nn::ulcd::CTR::DynamicStereoCamera::SetWindowFromSize, nn::ulcd::CTR::DynamicStereoCamera::SetWindowFromTangent, nn::ulcd::CTR::DynamicStereoCamera::SetWindowFromViewport, or nn::ulcd::CTR::DynamicStereoCamera::SetWindowFromProjectionMatrix functions to define the space. The nn::ulcd::CTR::DynamicStereoCamera::ComputeViewsAndProjections function can then be called to get the view matrix and projection matrix thereafter.
