nn::ulcd::CTR::DynamicStereoCamera::SetView Member Function

Syntax

#include <nn/ulcd.h>
void SetView(
     const nn::math::MTX34 * viewMatrix
);

Parameters

Name Description
in viewMatrix Base view matrix.

Return Values

None.

Description

Sets the view space using a view matrix as input. The view matrix is generated using a function such as nn::math::MTX34LookAt.
