nn::ulcd::CTR::DynamicStereoCamera::SetWindowFromProjectionMatrix Member Function

Syntax

#include <nn/ulcd.h>
void SetWindowFromProjectionMatrix(
     const nn::math::Matrix44 * proj,
     const f32 windowDepth
);

Parameters

Name Description
in proj Projection matrix to the base camera.
in windowDepth Distance from the camera to the window base plane.

Return Values

None.

Description

Sets the virtual view volume directly from a projection matrix. The projection matrix is generating using functions such as nn::math::MTX44Frustum and nn::math::MTX44PerspectiveDeg.
