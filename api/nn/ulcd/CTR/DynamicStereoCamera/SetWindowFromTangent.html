nn::ulcd::CTR::DynamicStereoCamera::SetWindowFromTangent Member Function

Syntax

#include <nn/ulcd.h>
void SetWindowFromTangent(
     const f32 tangentLeft,
     const f32 tangentRight,
     const f32 tangentBottom,
     const f32 tangentTop,
     const f32 nearDepth,
     const f32 windowDepth,
     const f32 farDepth
);

Parameters

Name Description
in tangentLeft Tangent value for the left edge of the window.
in tangentRight Tangent value for the right edge of the window.
in tangentBottom Tangent value for the bottom edge of the window.
in tangentTop Tangent value for the top edge of the window.
in nearDepth Distance from the camera to the near clipping plane.
in windowDepth Distance from the camera to the window base plane.
in farDepth Distance from the camera to the far clipping plane.

Return Values

None.

Description

Sets the virtual view volume based on the window tangent values and three kinds of depth information. The window origin is set from the tangent values passed as arguments and is unrelated to the view space settings. The planes, in order of distance from the camera, are assumed to be as follows: near clipping plane, window plane, and far clipping plane.
