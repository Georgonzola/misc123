nn::ulcd::CTR::DynamicStereoCamera::SetWindowFromViewport Member Function

Syntax

#include <nn/ulcd.h>
void SetWindowFromViewport(
     const f32 screenHorizontalHalfFovTangent,
     const f32 viewportLeftInUnitScreen,
     const f32 viewportRightInUnitScreen,
     const f32 viewportBottomInUnitScreen,
     const f32 viewportTopInUnitScreen,
     const f32 nearDepth,
     const f32 windowDepth,
     const f32 farDepth
);

Parameters

Name Description
in screenHorizontalHalfFovTangent Horizontal FOV.
in viewportLeftInUnitScreen Coordinates of the left side of the viewport. The range is -1 to 1.
in viewportRightInUnitScreen Coordinates of the right side of the viewport. The range is -1 to 1.
in viewportBottomInUnitScreen Coordinates of the bottom of the viewport. The range is -1 to 1.
in viewportTopInUnitScreen Coordinates of the top of the viewport. The range is -1 to 1.
in nearDepth Distance from the camera to the near clipping plane.
in windowDepth Distance from the camera to the window base plane.
in farDepth Distance from the camera to the far clipping plane.

Return Values

None.

Description

Sets the virtual view volume based on the screen horizontal FOV, the viewpoint position, and three kinds of depth information. The planes, in order of distance from the camera, are assumed to be as follows: near clipping plane, window plane, and far clipping plane.
