nn::ulcd::CTR Namespace

Description

Namespace of the CTR stereoscopic view library. (It can also be written without the CTR portion.))

Classes

nn::ulcd::CTR::DynamicStereoCamera Class that supports the creation of camera matrices for rendering images that take binocular parallax into account, based on the face tracking feature. This class provides a new stereoscopic image based on eye position information retrieved from the face tracking feature of the QTM library. When the face tracking feature is not used, this class works the same as the original nn::ulcd::CTR::StereoCamera class.
nn::ulcd::CTR::StereoCamera Class that supports the creation of camera matrices for rendering images that take binocular parallax into account.
