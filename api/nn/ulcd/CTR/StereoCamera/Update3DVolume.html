nn::ulcd::CTR::StereoCamera::Update3DVolume Member Function

Syntax

#include <nn/ulcd.h>
void Update3DVolume(
     void
);

Parameters

None.

Return Values

None.

Description

Updates the internally cached 3D volume value.

Updates the 3D depth slider value held in the StereoCamera class.

By default, it gets the 3D depth slider value to use in the matrix calculations that are performed with nn::ulcd::CTR::StereoCamera. If your implementation uses the StereoCamera class numerous times per frame to calculate matrices, there is a chance the 3D depth slider value could change during that time, which would affect the rendering results.

In cases like this, call this function at the beginning of each frame, and do not update the 3D depth slider values when specifying them as parameters for functions that calculate matrices (such as the nn::ulcd::CTR::StereoCamera::CalculateMatrices function).

    // Pseudocode 
    nn::ulcd::StereoCamera s_StereoCamera;

    while (1) {    // Rendering loop
        // Update 3D depth slider value
        s_StereoCamera.Update3DVolume();

        // Render model
        s_StereoCamera.CalculateMatrices(..., false);
        // Render another model
        s_StereoCamera.CalculateMatrices(..., false);
        // ...

        // End of one frame 
    }
