nn::ulcd::CTR::ULCD_UPPER_SCREEN_DISTANCE Constant

Syntax

#include <nn/ulcd.h>
const f32 ULCD_UPPER_SCREEN_DISTANCE =  (ULCD_SNAKE_UPPER_SCREEN_DISTANCE + ULCD_CLOSER_UPPER_SCREEN_DISTANCE) / 2.0f;

Description

Normalized optimal viewing distance of the upper screen. The mean value of the upper screen optimal viewing distances for both models. The unit is millimeters.
