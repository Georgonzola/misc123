nn::util::Base64::FromBase64String Member Function

Syntax

static nn::Result FromBase64String(
     const char * pSrc,
     void * pDst,
     size_t sizeDst,
     size_t * pNum
);

Parameters

Name Description
in pSrc Pointer to input data in base64 format.
out pDst Pointer to the buffer that holds the output data.
in sizeDst Size of the buffer indicated by pDst.
out pNum Buffer for storing the number of characters actually output.

Return Values

Returns whether the resource was successfully converted.

Description

Converts a base 64 format into binary data.

The number of bytes written to the destination buffer is written to *pNum.

This function parses the entered string according to the following rules:

  • There is no limit to the number of characters in one line.
  • Spaces and linefeed characters (0x09, 0x0a, 0x0d, 0x20) are simply skipped over.
  • Data are treated as invalid if they contain characters that are not suitable for use as base-64 characters nor as space and linefeed characters.
  • The NULL character (0x00) terminates the data. However, if the padding character "=" (0x3d) appears, this is treated as the effective end of the data.

One of the following errors is returned when the conversion fails:

  • nn::util::ResultInvalidPtr - The pointer given as a parameter is invalid.
  • nn::util::ResultBadData - The format of the entered string is invalid.
  • nn::util::ResultBufferFull - The output destination buffer is not large enough.
The conversion process proceeds until the failure occurs, and *pNum gets the number of bytes that were written before the failure.
