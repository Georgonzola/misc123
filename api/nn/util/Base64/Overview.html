nn::util::Base64 Class

Syntax

class Base64

Description

The class for base-64 encoding and decoding.

For base-64 encoding, this class uses the set of character codes defined by RFC2045.

  • For the 62nd character the "+" character (0x2b) is used.
  • For the 63rd character the "/" character (0x2f) is used.
  • For the padding character the "=" character (0x3d) is used.

Member Functions

S ToBase64String Converts the entered binary data into a base 64 format string.
S FromBase64String Converts a base 64 format into binary data.
