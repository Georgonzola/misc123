nn::util::Base64::ToBase64String Member Function

Syntax

static nn::Result ToBase64String(
     const void * pSrc,
     size_t sizeSrc,
     char * pDst,
     size_t sizeDst
);

Parameters

Name Description
in pSrc Pointer to input data.
in sizeSrc The size of input data.
out pDst Pointer to the output buffer for a base-64 format string.
in sizeDst Size of the buffer indicated by pDst.

Return Values

Returns whether the resource was successfully converted.

Description

Converts the entered binary data into a base 64 format string.

This function outputs a string created in the following way:

  • Each line is 76 characters in length, with a linefeed inserted after every 76 characters output.
  • The linefeed code used is CRLF (0x0D, 0x0A).
  • The null character (0x00) terminates the data.

One of the following errors is returned if conversion fails:

  • nn::util::ResultInvalidPtr - The pointer given as a parameter is invalid.
  • nn::util::ResultBufferFull - The output destination buffer is not large enough.
The conversion process proceeds until failure occurs.
