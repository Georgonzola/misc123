nn::util::Crc16::GetHash Member Function

Syntax

u16 GetHash();

Parameters

None.

Return Values

The calculated result.

Description

Gets the final result of calculating the CRC-16 hash value.

After the context has been initialized using the nn::util::Crc16::InitializeContext function, you must provide the input data using the nn::util::Crc16::Update function.
