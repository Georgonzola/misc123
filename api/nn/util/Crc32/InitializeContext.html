nn::util::Crc32::InitializeContext Member Function

Syntax

void InitializeContext(
     u32 context = CRC32_STANDARD_INIT
);

Parameters

Name Description
in context Specifies the default context value. Normally use the default value.

Return Values

None.

Description

Initializes the context used for calculating CRC-32 hash values.

Make sure that this function is called before using the nn::util::Crc32::Update and nn::util::Crc32::GetHash functions.
