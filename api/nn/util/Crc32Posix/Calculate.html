nn::util::Crc32Posix::Calculate Member Function

Syntax

static u32 Calculate(
     const void * input,
     size_t length,
     u32 context = CRC32_POSIX_INIT
);

Parameters

Name Description
in input Pointer to the input data.
in length Specifies the size of the input data.
in context Specifies the default context value. Normally use the default value.

Return Values

The calculated result.

Description

Calls all of the associated functions needed to calculate the CRC-32/POSIX, and gets the calculation result.

Internally, calls the following functions together: nn::util::Crc32Posix::InitializeContext, nn::util::Crc32Posix::Update, and nn::util::Crc32Posix::GetHash. This function cannot be used if you need to split up the data and call nn::util::Crc32Posix::Update multiple times. Any size and alignment position can be used for the input data.
