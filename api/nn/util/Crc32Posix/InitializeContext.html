nn::util::Crc32Posix::InitializeContext Member Function

Syntax

void InitializeContext(
     u32 context = CRC32_POSIX_INIT
);

Parameters

Name Description
in context Specifies the default context value. Normally use the default value.

Return Values

None.

Description

Initializes the context used for calculating CRC-32/POSIX hash values.

Make sure that this function is called before using the nn::util::Crc32Posix::Update and nn::util::Crc32Posix::GetHash functions.
