nn::util::Crc8::Update Member Function

Syntax

void Update(
     const void * input,
     size_t length
);

Parameters

Name Description
in input Pointer to the input data.
in length Specifies the size of the input data.

Return Values

None.

Description

Updates the CRC-8 hash value with the input data.

You must initialize the context in advance with the nn::util::Crc8::InitializeContext function. Call this function as many times as needed, and then use the nn::util::Crc8::GetHash function to get the hash value. Any size and alignment position can be used for the input data.
