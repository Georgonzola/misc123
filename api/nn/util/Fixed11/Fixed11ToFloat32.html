nn::util::Fixed11::Fixed11ToFloat32 Member Function

Syntax

static f32 Fixed11ToFloat32(
     u32 fixed11
);

Parameters

Name Description
in fixed11 An integer value representing the bits of an 11-bit fixed point number.

Return Values

Returns a number that has been converted into an f32 value.

Description

Converts from an 11-bit fixed to a 32-bit float.
