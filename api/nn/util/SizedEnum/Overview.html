nn::util::SizedEnum Class

Syntax

template <typename StorageT, typename EnumT>class SizedEnum

Template Arguments

Name Description
StorageT Type to use for storing the enum value.
EnumT Enumerated type of the target.

Description

Class template for handling enumerated types with a fixed size.

With this class, you can use enum values of explicit size.

The EnumT type enumerated type is stored to a StorageT type variable for handling.

Member Functions

SizedEnum Constructor.
operator EnumT The type conversion operator for the enumerated type.
operator= Assignment operator for assigning an enumerated type.
Set Sets values.
Get Gets a value.
