nn::voicesel::CTR::CancelPreloadVoiceSel Function

Syntax

bool CancelPreloadVoiceSel();

Parameters

None.

Return Values

Description

Cancels the preload of the audio selector applet.

If preloading is not yet complete, this function does nothing.
Before you call this function, you must call either the nn::voicesel::CTR::WaitForPreloadVoiceSel function or the nn::voicesel::CTR::IsRegisteredVoiceSel function to confirm that preloading is complete.

This function cancels all preloads of all library applets.
Note: Preloading of all library applets is canceled in addition to that of the audio selector applet.
