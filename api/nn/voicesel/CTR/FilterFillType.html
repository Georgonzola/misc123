nn::voicesel::CTR::FilterFillType Enumerated Type

Syntax

enum FilterFillType;

Value

Value Description
FILTER_FILL_TYPE_ALL All.
FILTER_FILL_TYPE_100_LT Less than 100% (10.0 seconds).
FILTER_FILL_TYPE_75_LT Less than 75% (7.5 seconds).
FILTER_FILL_TYPE_50_LT Less than 50% (5.0 seconds).
FILTER_FILL_TYPE_25_LT Less than 25% (2.5 seconds).

Description

Enumerated type representing filter conditions on the duration of audio recordings.

Indicates recording duration as a percentage of the maximum savable length of an audio clip. Currently, the maximum savable length is 10 seconds of recording.
