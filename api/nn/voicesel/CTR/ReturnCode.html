nn::voicesel::CTR::ReturnCode Enumerated Type

Syntax

enum ReturnCode;

Value

Value Description
RETURN_CODE_UNKNOWN Abnormal end.
RETURN_CODE_INVALID_CONFIG A config value is invalid.
RETURN_CODE_OUT_OF_MEMORY Ended abnormally because of insufficient memory.
RETURN_CODE_NONE Ended normally (no audio was selected).
RETURN_CODE_SUCCESS Ended normally (audio was selected).
RETURN_CODE_HOME_BUTTON HOME was pressed.
RETURN_CODE_SOFTWARE_RESET A button combination was pressed that causes a software reset.
RETURN_CODE_POWER_BUTTON The POWER Button was pressed.
RETURN_CODE_MAX_BIT (Enumerator used during internal processing. This enumerator cannot be used by applications.)

Description

Enumerated type representing return codes.

The values when ending normally are zero or higher. The values when ending abnormally are negative.
