nn::webbrs::CTR::Config Structure

Syntax

struct Config
{
   char url[URL_LENGTH_MAX];
};

Description

The Config structure storing the data to pass to the Internet Browser.

Member Variables

: Enter
url char The URL of the webpage to display after the browser starts. Specify a null-terminated string that begins with either http:// or https://. If the URL is unspecified or is an empty string, the browser opens using its default behavior.
