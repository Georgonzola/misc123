nn::webbrs::CTR::IsRestrictWebBrowser Function

Syntax

bool IsRestrictWebBrowser();

Parameters

None.

Return Values

Returns a value indicating whether there are restrictions.

Value Description
true Restricted.
false Not restricted.

Description

Checks whether starting the Internet browser is restricted by the Parental Controls.

To use this feature, you must call nn::cfg::Initialize() ahead of time.
