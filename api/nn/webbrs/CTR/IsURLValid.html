nn::webbrs::CTR::IsURLValid Function

Syntax

bool IsURLValid(
     const char * url
);

Parameters

Name Description
in url URL string to check.

Return Values

Returns whether the URL begins with http:// or https://.

Value Description
true The URL begins with either http:// or https://.
false The URL does not begin with either http:// or https://. This URL is invalid.

Description

Checks whether the URL begins with http:// or https://.

Even if this function returns true, the URL might still be invalid for another reason. Prior to this, use the URL entry feature of the browser and confirm that the page for the specified URL can display.
