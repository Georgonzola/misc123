nn::webbrs::CTR::IsWebBrowserAvailable Function

Syntax

bool IsWebBrowserAvailable();

Parameters

None.

Return Values

Returns a value indicating whether the browser has been installed.

Value Description
true Installed.
false Not installed.

Description

Checks whether the Internet browser has been installed.

In some cases, the Internet browser may not be installed on the system. If the system was shipped shortly after release and has never had a system update, it would not have a browser. This function can be used to check whether it is installed.
