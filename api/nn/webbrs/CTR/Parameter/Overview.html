nn::webbrs::CTR::Parameter Structure

Syntax

struct Parameter
{
   Config config;
};

Description

The Parameter structure passed when the Internet browser runs.

Set the config member variable appropriately and pass it to the argument of the nn::webbrs::CTR::StartWebBrowser function.

Member Variables

config Config Configuration data.
