nn::webbrs::CTR::StartWebBrowser Function

Syntax

Result StartWebBrowser(
     Parameter * pParameter
);

Parameters

Name Description
inout pParameter Parameter structure.

Return Values

Returns the result of the function. Returns one of the following Result values.

Value Description
Result::IsSuccess Indicates that the operation succeeded.
ResultLocalCommMode Using local communication.
ResultInvalidUrl Invalid URL.

Description

: Launches the Internet browser.

When using this function, check the various items described in nn::webbrs::CTR.

Will not start when the system mode is set to dev1, dev2, or snake dev1. For more information, see the documentation under CTR Tools > Configuration.

The system runs in extension mode after this function is called.
