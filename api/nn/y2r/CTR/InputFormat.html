nn::y2r::CTR::InputFormat Enumerated Type

Syntax

enum InputFormat;

Value

Value Description
INPUT_YUV422_INDIV_8 YUV 4:2:2 individual input mode (8-bit format)
INPUT_YUV420_INDIV_8 YUV 4:2:0 individual input mode (8-bit format)
INPUT_YUV422_INDIV_16 YUV 4:2:2 individual input mode (16-bit padding format)
INPUT_YUV420_INDIV_16 YUV 4:2:0 individual input mode (16-bit padding format)
INPUT_YUV422_BATCH YUV 4:2:2 batch mode

Description

Enumerated type that indicates the input mode.

The input data formats are as follows.

YUV 4:2:2 / YUV 4:2:0 individual input mode (8-bit format)

Bits 31-24Bits 23-16Bits 15-8Bits 7-0
Y (n+3)Y (n+2)Y (n+1)Y (n)
U (n+3)U (n+2)U (n+1)U (n)
V (n+3)V (n+2)V (n+1)V (n)

YUV 4:2:2 / YUV 4:2:0 individual input mode (16-bit padding format)

Bits 31-24Bits 23-16Bits 15-8Bits 7-0
paddingY (n+1)paddingY (n)
paddingU (n+1)paddingU (n)
paddingV (n+1)paddingV (n)

YUV 4:2:2 batch mode

Bits 31-24Bits 23-16Bits 15-8Bits 7-0
V (n)Y (n+1)U (n)Y (n)
