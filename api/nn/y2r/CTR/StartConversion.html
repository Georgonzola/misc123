nn::y2r::CTR::StartConversion Function

Syntax

nn::Result StartConversion(
     void
);

Parameters

None.

Return Values

Returns the result of the function. Returns one of the following Result values.

Value Description
Result::IsSuccess Indicates that the operation succeeded.
ResultIsSleeping Unusable because the system is in Sleep Mode.

Description

Starts conversion.

Before calling this function, you must configure the transfer with the nn::y2r::CTR::SetReciving and nn::y2r::CTR::SetSendingYuv functions.
