NN_ATTRIBUTE_ALIGN Macro

Syntax

#include <nn/config.h>
#define NN_ATTRIBUTE_ALIGN(n) __attribute__ ((aligned(n)))

Parameters

Name Description
in n Specifies the alignment.

Description

Used to align the boundaries of object addresses.

Ensures that the start addresses of global objects are aligned on n-byte boundaries by padding the ends of the objects.
