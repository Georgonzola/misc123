Environment Macros

Description

These macros represent environment information. They are used to change the implementation depending on the environment.

Macros

Endianness
NN_ENDIAN Defines the endianness of the platform.
NN_ENDIAN_VALUE_BIG Defines big-endian.
NN_ENDIAN_VALUE_LITTLE Defines little-endian.
Other
NN_FILE_NAME Specifies the current source filename.
NN_FUNCTION Specifies the current function name.
NN_ATTRIBUTE_ALIGN Used to align the boundaries of object addresses.
NN_UNUSED_VAR Declares that a variable is not used.
