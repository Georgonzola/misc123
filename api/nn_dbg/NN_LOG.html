NN_LOG Macro

Syntax

#include <nn/dbg.h>
#define NN_LOG(...)

Parameters

Name Description
in ... Specifies variable arguments equivalent to those of the printf function in the standard library.

Description

Outputs strings to the debug window.

Replaces this macro with a dummy process and does nothing if NN_SWITCH_DISABLE_DEBUG_PRINT is defined.
