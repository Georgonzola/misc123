nnfriendsJoinAvailability Enumerated Type

Syntax

enum nnfriendsJoinAvailability;

Value

Value Description
NN_FRIENDS_JOIN_AVAILABILITY_JOINABLE Indicates availability for joining in.
NN_FRIENDS_JOIN_AVAILABILITY_JOINABLE_APP_ONLY Indicates that the application is available for joining in, but that friends cannot join from the friend list.
NN_FRIENDS_JOIN_AVAILABILITY_NOT_JOINABLE Indicates no availability for joining in.

Description

Enumerated type representing the status of accepting participation.
