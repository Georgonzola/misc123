gx

Description

The graphics library (DMPGL 2.0) system API.

With a few exceptions, only a C API has been provided for the gx functions.

Functions

Initialize/Finalize
nngxInitialize Initializes the graphics library (DMPGL).
nngxFinalize Finalizes the graphics library (DMPGL).
nngxGetAllocator Gets a pointer to the allocator function specified with the nngxInitialize function.
nngxGetIsInitialized Checks whether the graphics library has been initialized using the nngxInitialize function.
nngxGetInitializationCommand Gets initialization register setting commands that are executed by the nngxInitialize function.
Command List Execution Controls
nngxGenCmdlists Generates a command list object.
nngxDeleteCmdlists Deletes command list objects.
nngxBindCmdlist Binds a command list object to the current command list.
nngxCmdlistStorage Allocates a data region for the current command list.
nngxRunCmdlist Runs the current command list.
nngxRunCmdlistByID Runs the specified command list.
nngxStopCmdlist Stops execution of the command list.
nngxReserveStopCmdlist Reserve-stops execution of the current command list.
nngxSplitDrawCmdlist Splits the 3D command buffer from the current command list.
nngxClearCmdlist Clears accumulated commands in the current command list.
nngxClearFillCmdlist Clears accumulated commands in the current command list to the specified value.
nngxSetCmdlistCallback Sets the callback function to call upon completion of the current command list.
nngxEnableCmdlistCallback Enables a callback upon completion of the current command list.
nngxDisableCmdlistCallback Disables a callback upon completion of the current command list.
nngxSetCmdlistParameteri Sets a parameter for the current command list.
nngxGetCmdlistParameteri Gets the value set to a current command list parameter (param).
nngxWaitCmdlistDone Waits for completion of the currently running command list object.
nngxSetTimeout Sets the specified value as the timeout for the nngxWaitCmdlistDone function.
nngxSetGasAutoAccumulationUpdate Updates the additive blending result due to rendering gas density data.
nngxGetIsRunning Checks whether a command list is currently running.
VSync Controls
nngxCheckVSync Gets a VSync counter.
nngxWaitVSync Synchronizes with a VSync counter and then returns.
nngxSetVSyncCallback Registers a VSync callback.
Display Buffer Controls
nngxGenDisplaybuffers Generates display buffer objects.
nngxDeleteDisplaybuffers Deletes display buffer objects.
nngxActiveDisplay Activates a display target.
nngxBindDisplaybuffer Binds a display buffer to an active display target.
nngxDisplaybufferStorage Allocates a region for the display buffer bound to an active display target.
nngxDisplayEnv Specifies the display position for an active display target.
nngxTransferRenderImage Transfers the current color buffer image to the display buffer specified for buffer.
nngxSwapBuffers Presets the display buffer bound to the display specified for the display parameter.
nngxSwapBuffersByAddress Presets the specified buffer to the display specified for the display parameter.
nngxGetDisplaybufferParameteri Gets the value of a parameter set for the display buffer that is bound to an active display target.
Command Cache Operations
nngxStartCmdlistSave Starts saving the command list object.
nngxStopCmdlistSave Stops saving the command list object.
nngxUseSavedCmdlist Adds the saved command to the current command request.
nngxUseSavedCmdlistNoCacheFlush Adds the saved command to the current command request. (No cache flush.)
nngxExportCmdlist Exports the specified command list to memory as binary data.
nngxImportCmdlist Imports an exported command list (binary data) to the specified command list.
nngxCopyCmdlist Copies a command list object to another command list object.
nngxValidateState Checks the state and generates a command.
nngxUpdateState Updates the specified state.
nngxGetExportedCmdlistInfo Gets the information for an exported command list (binary data).
nngxSetCommandGenerationMode Sets the command output mode.
nngxGetCommandGenerationMode Gets the command output mode.
nngxAdd3DCommand Adds the data (3D commands) in the specified region to the current 3D command buffer.
nngxAdd3DCommandNoCacheFlush Adds the data (3D commands) in the specified region to the current 3D command buffer.
nngxAddCmdlist Copies and adds the content of the specified command list object to the current command list object.
nngxGetUpdatedState Gets the updated state of the DMPGL library.
nngxInvalidateState Disables updates of the specified DMPGL state.
Low-level Command Request Issuance
nngxAddVramDmaCommand Issues a DMA transfer command to the current command list.
nngxAddVramDmaCommandNoCacheFlush Issues a DMA transfer command to the current command list.
nngxTransferLinearImage Adds a command to transfer the specified render buffer or texture to an image.
nngxFilterBlockImage Issues an antialiasing filter transfer command for a block image.
nngxAddL2BTransferCommand Adds a command to convert and transfer a linear image to a block image.
nngxAddB2LTransferCommand Adds a command to convert and transfer a block image to a linear image.
nngxAddBlockImageCopyCommand Adds a block image transfer command.
nngxAddMemoryFillCommand Adds a memory-fill command.
nngxFlush3DCommand Issues a 3D run command and flushes the accumulated 3D command buffer.
nngxFlush3DCommandNoCacheFlush Issues a 3D run command and flushes the accumulated 3D command buffer (without flushing the CPU cache).
nngxFlush3DCommandPartially Issues a 3D run command and flushes the accumulated 3D command buffer. However, only the specified amount of 3D commands are executed.
nngxAddJumpCommand Adds a jump command to the run address in the specified command buffer to the current command buffer.
nngxAddSubroutineCommand Adds a jump command to the run address in the specified command buffer, in addition to adding a command for setting address information in the current command buffer for returning to the original command buffer.
Helper for Direct Creation of 3D Command Buffers
nngxGetPhysicalAddr Converts a virtual address into a physical address.
nngxUpdateBuffer Applies to the graphics processor the updates that have been made to the contents of the vertex buffer and texture buffer in main memory.
nngxUpdateBufferLight Applies to the graphics processor the updates that have been made to the contents of the vertex buffer and texture buffer in main memory.
nngxMoveCommandbufferPointer Moves the 3D command buffer's current output target pointer.
Profiling
nngxStartProfiling Starts sampling profile information.
nngxStopProfiling Stops sampling profile information.
nngxGetProfilingResult Gets the profiling results.
nngxSetProfilingParameter Sets a profiling feature parameter.
Upper Screen Display Mode Settings
nngxSetDisplayMode Sets the display mode for the upper screen.
Other
nngxStartLcdDisplay Starts displaying an image on an LCD screen.
nngxSetMemAccessPrioMode Sets the priority of access to main memory (FCRAM).
nngxIsStereoVisionAllowed Gets whether stereoscopic display is allowed.
nngxSetInternalDriverPrioMode Sets the priority of internal operations for the graphics driver.
nngxGetVramStartAddr Gets the starting address of the specified VRAM region.
nngxGetVramEndAddr Gets the ending address of the specified VRAM region.
nngxGetVramSize Gets the size of the specified VRAM region.
