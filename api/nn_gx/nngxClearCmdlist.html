nngxClearCmdlist Function

Syntax

NNGX_APICALL void NNGX_APIENTRY nngxClearCmdlist(
     void
);

Parameters

None.

Return Values

No values are returned.

Description

Clears accumulated commands in the current command list.

When the command list is cleared, the 3D command buffer and command requests become unused. (This reverts the data region for the command list to the state immediately after it was allocated.)

Callbacks set with nngxEnableCmdlistCallback are cleared, but callbacks set with nngxEnableCmdlistCallback(-1), which are invoked when all command requests are done executing, are not cleared.

This function generates the following errors.

GL_ERROR_800F_DMPThis function was called on a running command list.
