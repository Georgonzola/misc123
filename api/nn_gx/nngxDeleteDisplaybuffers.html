nngxDeleteDisplaybuffers Function

Syntax

NNGX_APICALL void NNGX_APIENTRY nngxDeleteDisplaybuffers(
     GLsizei n,
     GLuint * buffers
);

Parameters

Name Description
in n Number of display buffer objects to delete.
in buffers Array that stores the display buffer objects to delete.

Return Values

No values are returned.

Description

Deletes display buffer objects.

The n display buffer objects with names that are stored in buffers are deleted.

An error occurs if an attempt is made to delete a display buffer object that is in use. However, the display buffer object in use is affected, and the other display buffer objects are deleted.

This function generates the following errors.

GL_ERROR_801E_DMPA negative value was specified for n.
