nngxDisplayEnv Function

Syntax

NNGX_APICALL void NNGX_APIENTRY nngxDisplayEnv(
     GLint displayx,
     GLint displayy
);

Parameters

Name Description
in displayx X-coordinate at which to start displaying the display buffer.
in displayy Y-coordinate at which to start displaying the display buffer.

Return Values

No values are returned.

Description

Specifies the display position for an active display target.

The display buffer is shown on the LCD screens with the coordinates (displayx, displayy) at the lower-left corner. This function configures the display target rather than each display buffer object.

This function generates the following errors.

GL_ERROR_8026_DMPThe displayx or displayy argument was set to a negative value.
