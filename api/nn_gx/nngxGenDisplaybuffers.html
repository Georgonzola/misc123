nngxGenDisplaybuffers Function

Syntax

NNGX_APICALL void NNGX_APIENTRY nngxGenDisplaybuffers(
     GLsizei n,
     GLuint * buffers
);

Parameters

Name Description
in n Number of display buffer objects to generate.
out buffers Array that stores the generated display buffer objects.

Return Values

No values are returned.

Description

Generates display buffer objects.

Creates n display buffer objects, and stores the object names in buffers.

This function generates the following errors.

GL_ERROR_801C_DMPA negative value was specified for n.
GL_ERROR_801D_DMPFailed to allocate the management region.
