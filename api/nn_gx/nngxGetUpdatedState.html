nngxGetUpdatedState Function

Syntax

NNGX_APICALL void NNGX_APIENTRY nngxGetUpdatedState(
     GLbitfield * statemask
);

Parameters

Name Description
out statemask Variable to store the currently updated state flag.

Return Values

No values are returned.

Description

Gets the updated state of the DMPGL library.

The various DMPGL states are updated by the DMPGL API and by calls to the nngxUpdateState function. Calling this function sets the flags of the currently updated states to 1 in statemask.

If you call this function when states have been validated by nn_gx::glDrawArrays, nn_gx::glDrawElements, nngxValidateState or various other functions, the flags of the validated states are not set in statemask. The NN_GX_STATE_OTHERS state flag is only set by this function when its state has been updated by the nngxUpdateState function.
