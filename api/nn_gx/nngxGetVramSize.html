nngxGetVramSize Function

Syntax

size_t nngxGetVramSize(
     s32 area
);

Parameters

Name Description
in area VRAM region to get (NN_GX_MEM_VRAMA or NN_GX_MEM_VRAMB).

Return Values

Size of the VRAM region that was obtained.

Description

Gets the size of the specified VRAM region.

Always use only this function to get the size of a VRAM region. The size that is obtained may change in the future.
