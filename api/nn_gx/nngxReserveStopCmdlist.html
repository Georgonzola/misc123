nngxReserveStopCmdlist Function

Syntax

NNGX_APICALL void NNGX_APIENTRY nngxReserveStopCmdlist(
     GLint id
);

Parameters

Name Description
in id ID of the accumulated command to stop.

Return Values

No values are returned.

Description

Reserve-stops execution of the current command list.

Tells the command list to stop when execution ends for the idth command accumulated in the command list. You can configure each command accumulated in the command list.

This function generates the following errors.

GL_ERROR_800A_DMPThe current command list is already running.
GL_ERROR_800B_DMPThe id argument is less than or equal to 0, or it is greater than the number of elements allocated in nngxCmdlistStorage.
