nngxSetCmdlistCallback Function

Syntax

NNGX_APICALL void NNGX_APIENTRY nngxSetCmdlistCallback(
     void(*)(GLint) func
);

Parameters

Name Description
in func Callback function.

Return Values

No values are returned.

Description

Sets the callback function to call upon completion of the current command list.

When the command request specified by the nngxEnableCmdlistCallback function finishes, func is called. The callback function receives as an argument the number of the accumulated command that finished executing.

For information about executing command requests and calling callback functions, see the nngxEnableCmdlistCallback function.

Note that the command list is still executing when this callback is invoked by the completion of the last of the accumulated commands. Therefore, as with the nngxClearCmdlist function, you cannot use functions inside the callback that cannot be called for the executing command list.

This function generates the following errors.

GL_ERROR_8010_DMPThe current command list is running.
