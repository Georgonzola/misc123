nngxSetProfilingParameter Function

Syntax

NNGX_APICALL void NNGX_APIENTRY nngxSetProfilingParameter(
     GLenum pname,
     GLuint param
);

Parameters

Name Description
in pname Profiling feature parameter.
in param Value to set.

Return Values

No values are returned.

Description

Sets a profiling feature parameter.

Operation is undefined when starting the profiling features (busy counters) before setting the parameters. Be sure to configure the parameters with appropriate values before starting the profiling features.

The following table lists the parameter names and values that can be specified in pname.

NN_GX_PROFILING_BUSY_SAMPLING_TIMESpecifies the amount of time in clock ticks to spend profiling the busy counter each time it is sampled. Specify a 16-bit value other than 0 in param.
NN_GX_PROFILING_BUSY_SAMPLING_TIME_MICRO_SECONDSet to the same value as NN_GX_PROFILING_BUSY_SAMPLING_TIME in microseconds. Set to a value between 1 and 244.
(The specified value is converted internally to clock cycles for a 268-MHz operation clock and set to the hardware. The value converted to clock cycles must fit within 16 bits.)
NN_GX_PROFILING_BUSY_SAMPLING_TIME_NANO_SECONDSet to the same value as NN_GX_PROFILING_BUSY_SAMPLING_TIME in nanoseconds. Set to a value between 1 and 244537.
(The specified value is converted internally to clock cycles for a 268-MHz operation clock and set to the hardware. The value converted to clock cycles must fit within 16 bits.)
NN_GX_PROFILING_BUSY_SAMPLING_COUNTSpecifies the number of samplings of the busy counter. Specify a 16-bit value for param. If 0 is specified, once sampling starts, it continues until nngxStopProfiling is called.
(Profiling is performed for the duration specified by NN_GX_PROFILING_BUSY_SAMPLING_TIME and is repeated until stopped.)
If the time is set to a value of 1 or higher, and if the results are obtained after waiting for profiling to end, the sum of the values of the profiling results for the various modules calculated by the busy counter feature is equal to the number of times profiling took place with this configuration.

This function generates the following errors.

GL_ERROR_80A5_DMPThe pname argument was set to an invalid value.
GL_ERROR_80A6_DMPThe param parameter was set to an invalid value.
