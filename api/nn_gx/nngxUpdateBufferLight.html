nngxUpdateBufferLight Function

Syntax

void nngxUpdateBufferLight(
     const void * pBuffer,
     size_t size
);

Parameters

Name Description
in pBuffer Pointer to the start of the buffer.
in size The buffer size.

Return Values

None.

Description

Applies to the graphics processor the updates that have been made to the contents of the vertex buffer and texture buffer in main memory.

For more information, see the description of the corresponding C++ function nn::gx::CTR::UpdateBufferLight.
