PicaDataBlendFunc Enumerated Type

Syntax

enum PicaDataBlendFunc;

Value

Value Description
PICA_DATA_BLEND_FUNC_ZERO This is GL_ZERO.
PICA_DATA_BLEND_FUNC_ONE This is GL_ONE.
PICA_DATA_BLEND_FUNC_SRC_COLOR This is GL_SRC_COLOR.
PICA_DATA_BLEND_FUNC_ONE_MINUS_SRC_COLOR This is GL_ONE_MINUS_SRC_COLOR.
PICA_DATA_BLEND_FUNC_DST_COLOR This is GL_DST_COLOR.
PICA_DATA_BLEND_FUNC_ONE_MINUS_DST_COLOR This is GL_ONE_MINUS_DST_COLOR.
PICA_DATA_BLEND_FUNC_SRC_ALPHA This is GL_SRC_ALPHA.
PICA_DATA_BLEND_FUNC_ONE_MINUS_SRC_ALPHA This is GL_ONE_MINUS_SRC_ALPHA.
PICA_DATA_BLEND_FUNC_DST_ALPHA This is GL_DST_ALPHA.
PICA_DATA_BLEND_FUNC_ONE_MINUS_DST_ALPHA This is GL_ONE_MINUS_DST_ALPHA.
PICA_DATA_BLEND_FUNC_CONSTANT_COLOR This is GL_CONSTANT_COLOR.
PICA_DATA_BLEND_FUNC_ONE_MINUS_CONSTANT_COLOR This is GL_ONE_MINUS_CONSTANT_COLOR.
PICA_DATA_BLEND_FUNC_CONSTANT_ALPHA This is GL_CONSTANT_ALPHA.
PICA_DATA_BLEND_FUNC_ONE_MINUS_CONSTANT_ALPHA This is GL_ONE_MINUS_CONSTANT_ALPHA.
PICA_DATA_BLEND_FUNC_SRC_ALPHA_SATURATE This is GL_SRC_ALPHA_SATURATE.

Description

Sets the blending function. Used with the PICA_REG_DEPTH_COLOR_MASK register (0x101[19:16]). Corresponds to the arguments of the functions glBlendFunc and glBlendFuncSeparate.
