PicaDataColorPixelSize Enumerated Type

Syntax

enum PicaDataColorPixelSize;

Value

Value Description
PICA_DATA_COLOR_PIXEL_SIZE16 The color buffer format pixel size is 16 bits.
PICA_DATA_COLOR_PIXEL_SIZE32 The color buffer format pixel size is 32 bits.

Description

The pixel size for a color buffer format. Used with the PICA_REG_RENDER_BUFFER_COLOR_MODE register (0x117[1:0]).
