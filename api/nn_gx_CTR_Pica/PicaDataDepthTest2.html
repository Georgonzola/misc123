PicaDataDepthTest2 Enumerated Type

Syntax

enum PicaDataDepthTest2;

Value

Value Description
PICA_DATA_DEPTH_TEST2_NEVER This is GL_NEVER.
PICA_DATA_DEPTH_TEST2_ALWAYS This is GL_ALWAYS.
PICA_DATA_DEPTH_TEST2_GREATER This is GL_GREATER.
PICA_DATA_DEPTH_TEST2_GEQUAL This is GL_GEQUAL.
PICA_DATA_DEPTH_TEST2_OTHER This is GL_OTHER.

Description

Sets the comparison function for a depth test. Used with the PICA_REG_GAS_DELTAZ_DEPTH register (0x126[25:24]). Corresponds to the glDepthFunc function argument func.
