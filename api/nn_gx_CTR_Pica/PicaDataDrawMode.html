PicaDataDrawMode Enumerated Type

Syntax

enum PicaDataDrawMode;

Value

Value Description
PICA_DATA_DRAW_TRIANGLE_STRIP GL_TRIANGLE_STRIP.
PICA_DATA_DRAW_TRIANGLE_FAN GL_TRIANGLE_FAN.
PICA_DATA_DRAW_TRIANGLES GL_TRIANGLES.
PICA_DATA_DRAW_GEOMETRY_PRIMITIVE GL_GEOMETRY_PRIMITIVE.

Description

Sets the rendering mode. Used with the PICA_REG_VS_OUT_REG_NUM3 register (0x25e[8:8]).
