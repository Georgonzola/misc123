PicaDataFragLightEnvBump Enumerated Type

Syntax

enum PicaDataFragLightEnvBump;

Value

Value Description
PICA_DATA_FRAG_LIGHT_ENV_BUMP_NOT_USED_DMP Normals and tangents are not affected by bumps.
PICA_DATA_FRAG_LIGHT_ENV_BUMP_AS_BUMP_DMP Only normals are affected by bumps.
PICA_DATA_FRAG_LIGHT_ENV_BUMP_AS_TANG_DMP Only tangents are affected by bumps.

Description

Used for the PICA_REG_FRAG_LIGHT_FUNC_MODE register (0x1c3[29:28]), which is used for bump mapping with fragment lighting to set normals and tangents. This corresponds to setting the dmp_LightEnv.bumpMode uniform.
