PicaDataFragLightEnvLutScale Enumerated Type

Syntax

enum PicaDataFragLightEnvLutScale;

Value

Value Description
PICA_DATA_FRAG_LIGHT_ENV_LUTSCALE_1_0 Sets 1.0.
PICA_DATA_FRAG_LIGHT_ENV_LUTSCALE_2_0 Sets 2.0.
PICA_DATA_FRAG_LIGHT_ENV_LUTSCALE_4_0 Sets 4.0.
PICA_DATA_FRAG_LIGHT_ENV_LUTSCALE_8_0 Sets 8.0.
PICA_DATA_FRAG_LIGHT_ENV_LUTSCALE_0_25 Sets 0.25.
PICA_DATA_FRAG_LIGHT_ENV_LUTSCALE_0_5 Sets 0.5.

Description

Used for the PICA_REG_FRAG_LIGHT_LUTSCALE register (0x1d2), which sets the scaling of output values from the lookup table. This corresponds to setting the dmp_LightEnv.lutScale* uniform.
