PicaDataFragLightEnvTexture Enumerated Type

Syntax

enum PicaDataFragLightEnvTexture;

Value

Value Description
PICA_DATA_FRAG_LIGHT_ENV_TEXTURE0 Sets TEXTURE0.
PICA_DATA_FRAG_LIGHT_ENV_TEXTURE1 Sets TEXTURE1.
PICA_DATA_FRAG_LIGHT_ENV_TEXTURE2 Sets TEXTURE2.
PICA_DATA_FRAG_LIGHT_ENV_TEXTURE3 Sets TEXTURE3.

Description

Used for the PICA_REG_FRAG_LIGHT_FUNC_MODE registers (0x1c3[23:22] and 0x1c3[25:24]), which are used for setting fragment lighting textures. This corresponds to setting the dmp_LightEnv.shadowSelector and dmp_LightEnv.bumpSelector uniforms.
