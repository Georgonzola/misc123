PicaDataGSMode Enumerated Type

Syntax

enum PicaDataGSMode;

Value

Value Description
PICA_DATA_GS_OTHER_MODE Some other mode.
PICA_DATA_GS_SUBDIVISION_MODE Uses reserved geometry shader subdivisions (Loop, Catmull-Clark).
PICA_DATA_GS_SUBDIVISION_CATMULL_MODE Uses the reserved geometry shader's Catmull-Clark subdivision algorithm.
PICA_DATA_GS_SUBDIVISION_LOOP_MODE Uses the reserved geometry shader's Loop subdivision algorithm.
PICA_DATA_GS_PARTICLE_MODE Uses the particle system.

Description

Sets the mode of the geometry shader. Used with the PICA_REG_GS_MISC_REG0 register (0x252[31:0]).
