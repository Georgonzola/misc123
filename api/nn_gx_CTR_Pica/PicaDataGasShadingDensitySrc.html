PicaDataGasShadingDensitySrc Enumerated Type

Syntax

enum PicaDataGasShadingDensitySrc;

Value

Value Description
PICA_DATA_GAS_PLAIN_DENSITY_DMP Fog input density information D1 is used for shading.
PICA_DATA_GAS_DEPTH_DENSITY_DMP Fog input density information D2 is used for shading.

Description

Sets the density information to use for gas shading. Used with the PICA_REG_GAS_FOG_MODE register (0x0e0[3:0]). Corresponds to the dmp_Gas.shadingDensitySrc uniform.
