PicaDataLogicOp Enumerated Type

Syntax

enum PicaDataLogicOp;

Value

Value Description
PICA_DATA_LOGIC_CLEAR This is GL_CLEAR.
PICA_DATA_LOGIC_AND This is GL_AND.
PICA_DATA_LOGIC_AND_REVERSE This is GL_AND_REVERSE.
PICA_DATA_LOGIC_COPY This is GL_COPY.
PICA_DATA_LOGIC_SET This is GL_SET.
PICA_DATA_LOGIC_COPY_INVERTED This is GL_COPY_INVERTED.
PICA_DATA_LOGIC_NOOP This is GL_NOOP.
PICA_DATA_LOGIC_INVERT This is GL_INVERT.
PICA_DATA_LOGIC_NAND This is GL_NAND.
PICA_DATA_LOGIC_OR This is GL_OR.
PICA_DATA_LOGIC_NOR This is GL_NOR.
PICA_DATA_LOGIC_XOR This is GL_XOR.
PICA_DATA_LOGIC_EQUIV This is GL_EQUIV.
PICA_DATA_LOGIC_AND_INVERTED This is GL_AND_INVERTED.
PICA_DATA_LOGIC_OR_REVERSE This is GL_OR_REVERSE.
PICA_DATA_LOGIC_OR_INVERTED This is GL_OR_INVERTED.

Description

Sets the logical operation. Used with the PICA_REG_LOGIC_OP register (0x102[3:0]). Corresponds to the glLogicOp function argument opcode.
