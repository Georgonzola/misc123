PicaDataStencilTest Enumerated Type

Syntax

enum PicaDataStencilTest;

Value

Value Description
PICA_DATA_STENCIL_TEST_NEVER This is GL_NEVER.
PICA_DATA_STENCIL_TEST_ALWAYS This is GL_ALWAYS.
PICA_DATA_STENCIL_TEST_EQUAL This is GL_EQUAL.
PICA_DATA_STENCIL_TEST_NOTEQUAL This is GL_NOTEQUAL.
PICA_DATA_STENCIL_TEST_LESS This is GL_LESS.
PICA_DATA_STENCIL_TEST_LEQUAL This is GL_LEQUAL.
PICA_DATA_STENCIL_TEST_GREATER This is GL_GREATER.
PICA_DATA_STENCIL_TEST_GEQUAL This is GL_GEQUAL.

Description

Sets the comparison function for a stencil test. Used with the PICA_REG_STENCIL_TEST register (0x105[6:4]). Corresponds to the glStencilFunc function argument func.
