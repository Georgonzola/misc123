PicaDataTexEnvCombine Enumerated Type

Syntax

enum PicaDataTexEnvCombine;

Value

Value Description
PICA_DATA_TEX_ENV_COMBINE_REPLACE This is GL_REPLACE.
PICA_DATA_TEX_ENV_COMBINE_MODULATE This is GL_MODULATE.
PICA_DATA_TEX_ENV_COMBINE_ADD This is GL_ADD.
PICA_DATA_TEX_ENV_COMBINE_ADD_SIGNED This is GL_ADD_SIGNED.
PICA_DATA_TEX_ENV_COMBINE_INTERPOLATE This is GL_INTERPOLATE.
PICA_DATA_TEX_ENV_COMBINE_SUBTRACT This is GL_SUBTRACT.
PICA_DATA_TEX_ENV_COMBINE_DOT3_RGB This is GL_DOT3_RGB.
PICA_DATA_TEX_ENV_COMBINE_DOT3_RGBA This is GL_DOT3_RGBA.
PICA_DATA_TEX_ENV_COMBINE_MULT_ADD_DMP This is GL_MULT_ADD_DMP.
PICA_DATA_TEX_ENV_COMBINE_ADD_MULT_DMP This is GL_ADD_MULT_DMP.

Description

Sets combineRgb and combineAlpha for the texture combiners. Used with the PICA_REG_TEX_ENV*_COMBINE registers. Corresponds to the dmp_TexEnv[i].combineRgb and dmp_TexEnv[i].combineAlpha uniforms.
