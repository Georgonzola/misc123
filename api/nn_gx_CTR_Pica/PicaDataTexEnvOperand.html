PicaDataTexEnvOperand Enumerated Type

Syntax

enum PicaDataTexEnvOperand;

Value

Value Description
PICA_DATA_OPE_RGB_SRC_COLOR This is GL_SRC_COLOR.
PICA_DATA_OPE_RGB_ONE_MINUS_SRC_COLOR This is GL_ONE_MINUS_SRC_COLOR.
PICA_DATA_OPE_RGB_SRC_ALPHA This is GL_SRC_ALPHA.
PICA_DATA_OPE_RGB_ONE_MINUS_SRC_ALPHA This is GL_ONE_MINUS_SRC_ALPHA.
PICA_DATA_OPE_RGB_SRC_R_DMP This is GL_SRC_R_DMP.
PICA_DATA_OPE_RGB_ONE_MINUS_SRC_R_DMP This is GL_ONE_MINUS_SRC_R_DMP.
PICA_DATA_OPE_RGB_SRC_G_DMP This is GL_SRC_G_DMP.
PICA_DATA_OPE_RGB_ONE_MINUS_SRC_G_DMP This is GL_ONE_MINUS_SRC_G_DMP.
PICA_DATA_OPE_RGB_SRC_B_DMP This is GL_SRC_B_DMP.
PICA_DATA_OPE_RGB_ONE_MINUS_SRC_B_DMP This is GL_ONE_MINUS_SRC_B_DMP.
PICA_DATA_OPE_ALPHA_SRC_ALPHA This is GL_SRC_ALPHA.
PICA_DATA_OPE_ALPHA_ONE_MINUS_SRC_ALPHA This is GL_ONE_MINUS_SRC_ALPHA_DMP.
PICA_DATA_OPE_ALPHA_SRC_R_DMP This is GL_SRC_R_DMP.
PICA_DATA_OPE_ALPHA_ONE_MINUS_SRC_R_DMP This is GL_ONE_MINUS_SRC_R_DMP.
PICA_DATA_OPE_ALPHA_SRC_G_DMP This is GL_SRC_G_DMP.
PICA_DATA_OPE_ALPHA_ONE_MINUS_SRC_G_DMP This is GL_ONE_MINUS_SRC_G_DMP.
PICA_DATA_OPE_ALPHA_SRC_B_DMP This is GL_SRC_B_DMP.
PICA_DATA_OPE_ALPHA_ONE_MINUS_SRC_B_DMP This is GL_ONE_MINUS_SRC_B_DMP.

Description

Sets operandRgb and operandAlpha for the texture combiners. Used with the PICA_REG_TEX_ENV*_OPERAND registers. Corresponds to the dmp_TexEnv[i].srcRgb and dmp_TexEnv[i].srcAlpha uniforms.
