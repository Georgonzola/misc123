PicaDataTexEnvSrc Enumerated Type

Syntax

enum PicaDataTexEnvSrc;

Value

Value Description
PICA_DATA_TEX_ENV_SRC_RGBA_PRIMARY_COLOR This is GL_PRIMARY_COLOR.
PICA_DATA_TEX_ENV_SRC_RGBA_FRAGMENT_PRIMARY_COLOR_DMP This is GL_FRAGMENT_PRIMARY_COLOR.
PICA_DATA_TEX_ENV_SRC_RGBA_FRAGMENT_SECONDARY_COLOR_DMP This is GL_FRAGMENT_SECONDARY_COLOR.
PICA_DATA_TEX_ENV_SRC_RGBA_TEXTURE0 This is GL_TEXTURE0.
PICA_DATA_TEX_ENV_SRC_RGBA_TEXTURE1 This is GL_TEXTURE1.
PICA_DATA_TEX_ENV_SRC_RGBA_TEXTURE2 This is GL_TEXTURE2.
PICA_DATA_TEX_ENV_SRC_RGBA_TEXTURE3 This is GL_TEXTURE3.
PICA_DATA_TEX_ENV_SRC_RGBA_PREVIOUS_BUFFER_DMP This is GL_PREVIOUS_BUFFER_DMP.
PICA_DATA_TEX_ENV_SRC_RGBA_CONSTANT This is GL_CONSTANT.
PICA_DATA_TEX_ENV_SRC_RGBA_PREVIOUS This is GL_PREVIOUS.

Description

Sets srcRgb and srcAlpha for the texture combiners. Used with the PICA_REG_TEX_ENV* registers. Corresponds to the dmp_TexEnv[i].srcRgb and dmp_TexEnv[i].srcAlpha uniforms.
