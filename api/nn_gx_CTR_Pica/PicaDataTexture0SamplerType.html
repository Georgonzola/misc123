PicaDataTexture0SamplerType Enumerated Type

Syntax

enum PicaDataTexture0SamplerType;

Value

Value Description
PICA_DATA_TEXTURE0_SAMPLER_TYPE_TEXTURE_2D This is GL_TEXTURE_2D.
PICA_DATA_TEXTURE0_SAMPLER_TYPE_TEXTURE_CUBE_MAP This is GL_TEXTURE_CUBE_MAP.
PICA_DATA_TEXTURE0_SAMPLER_TYPE_TEXTURE_SHADOW_2D_DMP This is GL_TEXTURE_SHADOW_2D_DMP.
PICA_DATA_TEXTURE0_SAMPLER_TYPE_TEXTURE_PROJECTION_DMP This is GL_TEXTURE_PROJECTION_DMP.
PICA_DATA_TEXTURE0_SAMPLER_TYPE_TEXTURE_SHADOW_CUBE_DMP This is GL_TEXTURE_SHADOW_CUBE_DMP.
PICA_DATA_TEXTURE0_SAMPLER_TYPE_TEXTURE_FALSE This is GL_FALSE.
PICA_DATA_TEXTURE0_SAMPLER_TYPE_FALSE This is GL_FALSE.

Description

Sets the sampler type for texture 0. Used with the PICA_REG_TEXTURE_FUNC register (0x080 [30:28]). Corresponds to the dmp_Texture[0].samplerType uniform.
