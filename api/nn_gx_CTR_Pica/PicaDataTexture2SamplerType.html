PicaDataTexture2SamplerType Enumerated Type

Syntax

enum PicaDataTexture2SamplerType;

Value

Value Description
PICA_DATA_TEXTURE2_SAMPLER_TYPE_FALSE This is GL_FALSE.
PICA_DATA_TEXTURE2_SAMPLER_TYPE_TEXTURE_2D This is GL_TEXTURE_2D.

Description

Sets the sampler type for texture 2. Used with the PICA_REG_TEXTURE_FUNC register (0x080 [2:2]). Corresponds to the dmp_Texture[2].samplerType uniform.
