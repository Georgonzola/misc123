PicaDataTextureWrap Enumerated Type

Syntax

enum PicaDataTextureWrap;

Value

Value Description
PICA_DATA_TEXTURE_WRAP_CLAMP_TO_EDGE This is GL_CLAMP_TO_EDGE.
PICA_DATA_TEXTURE_WRAP_CLAMP_TO_BORDER This is GL_CLAMP_TO_BORDER.
PICA_DATA_TEXTURE_WRAP_REPEAT This is GL_REPEAT.
PICA_DATA_TEXTURE_WRAP_MIRRORED_REPEAT This is GL_MIRRORED_REPEAT.

Description

Sets the texture's wrap mode. Used with the PICA_REG_TEXTURE*_WRAP_FILTER registers (0x83[14:12, 10:8], 0x93[14:12, 10:8], 0x9b[14:12, 10:8]). Corresponds to glTexParameter{if}( TEXTURE_WRAP_*, ...).
