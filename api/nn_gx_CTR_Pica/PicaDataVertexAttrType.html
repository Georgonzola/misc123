PicaDataVertexAttrType Enumerated Type

Syntax

enum PicaDataVertexAttrType;

Value

Value Description
PICA_DATA_SIZE_1_BYTE This is size = 1, type = GL_BYTE.
PICA_DATA_SIZE_1_UNSIGNED_BYTE This is size = 1, type = GL_UNSIGNED_BYTE.
PICA_DATA_SIZE_1_SHORT This is size = 1, type = GL_SHORT.
PICA_DATA_SIZE_1_FLOAT This is size = 1, type = GL_FLOAT.
PICA_DATA_SIZE_2_BYTE This is size = 2, type = GL_BYTE.
PICA_DATA_SIZE_2_UNSIGNED_BYTE This is size = 2, type = GL_UNSIGNED_BYTE.
PICA_DATA_SIZE_2_SHORT This is size = 2, type = GL_SHORT.
PICA_DATA_SIZE_2_FLOAT This is size = 2, type = GL_FLOAT.
PICA_DATA_SIZE_3_BYTE This is size = 3, type = GL_BYTE.
PICA_DATA_SIZE_3_UNSIGNED_BYTE This is size = 3, type = GL_UNSIGNED_BYTE.
PICA_DATA_SIZE_3_SHORT This is size = 3, type = GL_SHORT.
PICA_DATA_SIZE_3_FLOAT This is size = 3, type = GL_FLOAT.
PICA_DATA_SIZE_4_BYTE This is size = 4, type = GL_BYTE.
PICA_DATA_SIZE_4_UNSIGNED_BYTE This is size = 4, type = GL_UNSIGNED_BYTE.
PICA_DATA_SIZE_4_SHORT This is size = 4, type = GL_SHORT.
PICA_DATA_SIZE_4_FLOAT This is size = 4, type = GL_FLOAT.

Description

Sets the type of the internal vertex attribute. Used with the PICA_REG_VERTEX_ATTR_ARRAYS0-1 registers (00x201-0x202). Corresponds to the glVertexAttribPointer function.
