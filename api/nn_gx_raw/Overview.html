gx Raw

Description

DMPGL system API for direct specification of command lists.

The gx Raw API is a group of functions that enable the command list-related controls of the gx API to be processed in the form of directly specified command list objects.

With the existing gx API, command list processing adheres to OpenGL style rules: you specify the IDs for objects allocated in the library and use them in a bound form. The code for the use of command lists looks something like this.

// For saving the command list ID.
GLuint s_MyCmdListId;

{
    // Prepare a new command list.
    nngxGenCmdlists(1, &s_MyCmdListId);
    
    nngxBindCmdlist(s_MyCmdListId);
    nngxCmdlistStorage(MY_CMDLIST_BUFFER_SIZE, MY_CMDLIST_NUM_REQUESTS);
    nngxClearCmdlist();
}

{
    // Run the command list.
    nngxBindCmdlist(s_MyCmdListId);
    nngxRunCmdlist();
}

{
    // Destroy the command list.
    nngxDeleteCmdlists(1, &s_MyCmdListId);
}

In contrast, with this gx Raw function group, the definition of the nngxCommandList structure that maintains command list management information is provided, and the various functions are called with a pointer to an object of this type specified as the first argument. In this case, the code for performing this same kind of process looks something like this.

// Allocate command list object.
nngxCommandList s_MyCmdListObj;

{
    // Prepare a new command list.
    GLsizei reqSize;
    GLvoid* pCmdBuffer;
    GLvoid* pReqBuffer;
    
    // The application allocates the required buffer region.
    reqSize = nngxGetCommandRequestSizeRaw(MY_CMDLIST_NUM_REQUESTS);
    pCmdBuffer = MyAlloc(MY_CMDLIST_BUFFER_SIZE);
    pReqBuffer = MyAlloc(reqSize);
    
    nngxCmdlistStorageRaw(&s_MyCmdListObj, MY_CMDLIST_BUFFER_SIZE, pCmdBuffer, MY_CMDLIST_NUM_REQUESTS, pReqBuffer);
    nngxClearCmdlistRaw(&s_MyCmdListObj);
}

{
    // Run the command list.
    nngxRunCmdlistRaw(&s_MyCmdListObj);
}

// No need to explicitly destroy the command list.

Although this requires that the application take complete responsibility for managing the allocating and deallocating of objects and buffer regions, it gives you more freedom to build a framework that is not dependent on the state inside the library.

The provided APIs are named like the existing functions but with "Raw" appended to the end of the name, and with the added parameter of the pointer to nngxCommandList. As long as there are no special descriptions, you can read details about operations and errors by reading the references for the existing functions. (Wherever the reference says "the current command list," replace it with "the command list specified by nngxCommandList.")

Note:

  • Because the gl and gd APIs operate on the presumption that the command list is bound inside the library, you cannot use those functions in combination with the gx Raw API. It is assumed that the gx Raw API is used mainly in combination with the gr API.
  • For the sake of internal implementation, you cannot use copies of nngxCommandList objects.
  • Operations are not guaranteed if you use normal gx command list processing functions in combination with gx Raw functions. However, the following functions are exceptions in that they can be used at the same time as the gx Raw functions.
    • nngxWaitCmdlistDone
    • nngxSetTimeout
    • nngxStopCmdlist
    • nngxGetIsRunning

Structures

nngxCommandList Structure that holds command list information.

Functions

Command list execution controls.
nngxCmdlistStorageRaw Allocates a data region for the specified command list.
nngxGetCommandRequestSizeRaw Calculates and returns the required size for the buffer for storing command list queue data.
nngxSplitDrawCmdlistRaw Splits the 3D command buffer from the specified command list.
nngxEnableCmdlistCallbackRaw Enables calling a callback function upon completion of the specified command list.
nngxDisableCmdlistCallbackRaw Disables calling a callback function upon completion of the specified command list.
nngxSetCmdlistCallbackRaw Sets the callback function to call upon completion of the specified command list.
nngxRunCmdlistRaw Runs the specified command list.
nngxReserveStopCmdlistRaw Reserve-stops execution of the specified command list.
nngxClearCmdlistRaw Clears accumulated commands from the specified command list.
nngxClearFillCmdlistRaw Clears accumulated commands from the specified command list and replaces them with the specified value.
nngxSetGasAutoAccumulationUpdateRaw Updates the result of additive blending due to rendering the gas density information in the specified command request handler for the specified command list.
nngxSetGasUpdateRaw Specifies whether to update the result of additive blending due to rendering the gas density information in the specified command list.
nngxGetIsRunningRaw Checks whether the specified command list is currently running.
nngxGetUsedBufferSizeRaw Returns the size of the accumulated 3D command buffer for the specified command list.
nngxGetUsedRequestCountRaw Returns the number of accumulated command requests in the specified command list.
nngxGetMaxBufferSizeRaw Returns the maximum size of the 3D command buffer that can be accumulated for the specified command list.
nngxGetMaxRequestCountRaw Returns the maximum number of command requests that can be accumulated in the specified command list.
nngxGetTopBufferAddrRaw Returns the start address of the 3D command buffer set to the specified command list.
nngxGetRunBufferSizeRaw Returns the size of the executed 3D command buffer for the specified command list.
nngxGetRunRequestCountRaw Returns the number of the executed command requests in the specified command list.
nngxGetTopRequestAddrRaw Returns the start address buffer for storing command request queue data set to the specified command list.
nngxGetNextRequestTypeRaw Returns the type of the next command request that will be run out of the command requests accumulated in the specified command list.
nngxGetNextBufferAddrRaw Returns the address of the next buffer that will be executed out of the 3D command buffers accumulated for the specified command list.
nngxGetNextBufferSizeRaw Returns the size of the next buffer that will be executed out of the 3D command buffers accumulated for the specified command list.
nngxGetCurrentBufferAddrRaw Returns the address of the 3D command buffer in which the next 3D command will be accumulated for the specified command list.
nngxGetHWStateRaw Gets a 32-bit piece of data that indicates the state of the hardware.
Command cache operations.
nngxCopyCmdlistRaw Copies a command list object to another command list object.
nngxAdd3DCommandNoCacheFlushRaw Adds the data (3D commands) in the specified region to the 3D command buffer for the specified command list.
nngxAddCmdlistRaw Copies the content of the specified command list object to add to another command list object.
Low-level Command Request Issuance
nngxFlush3DCommandNoCacheFlushRaw Issues a 3D run command to the specified command list and flushes the accumulated 3D command buffer.
nngxFlush3DCommandPartiallyRaw Issues a 3D run command to the specified command list and flushes the accumulated 3D command buffer. However, only the specified amount of 3D commands are executed.
nngxMoveCommandbufferPointerRaw Moves the 3D command buffer's current output target pointer to the specified command list.
nngxAddB2LTransferCommandRaw Adds a command to convert and transfer a block image to a linear image to the specified command list.
nngxAddBlockImageCopyCommandRaw Adds a block image transfer command to the specified command list.
nngxAddL2BTransferCommandRaw Adds a command to convert and transfer a linear image to a block image to the specified command list.
nngxAddMemoryFillCommandRaw Adds a memory-fill command to the specified command list.
nngxAddVramDmaCommandNoCacheFlushRaw Adds a DMA transfer command to the specified command list.
nngxFilterBlockImageRaw Adds a block image antialiasing filter transfer command to the specified command list.
nngxAddJumpCommandRaw Takes a command for jumping to the run address in a specified command buffer and adds it to a specified command list object.
nngxAddSubroutineCommandRaw Takes a command for jumping to the run address in a specified command buffer, and a command for setting the address information for returning back to the command buffer from where the jump took place, and adds them both to a specified command list object.
