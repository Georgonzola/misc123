nngxAddB2LTransferCommandRaw Function

Syntax

NNGX_APICALL void NNGX_APIENTRY nngxAddB2LTransferCommandRaw(
     nngxCommandList * cmdlist,
     const GLvoid * srcaddr,
     GLsizei srcwidth,
     GLsizei srcheight,
     GLenum srcformat,
     GLvoid * dstaddr,
     GLsizei dstwidth,
     GLsizei dstheight,
     GLenum dstformat,
     GLenum aamode,
     GLboolean yflip,
     GLsizei blocksize
);

Parameters

Name Description
in cmdlist Target command list object.
in srcaddr Address of the source block image to transfer.
in srcwidth Width of the source image to transfer.
in srcheight Height of the source image to transfer.
in srcformat Pixel format of the source image to transfer.
out dstaddr Address storing the linear image to which the data is transferred.
in dstwidth Width of the destination image.
in dstheight Height of the destination image.
in dstformat Pixel format of the destination image.
in aamode Antialiasing filter mode.
in yflip Specifies whether to vertically flip the image when transferring it.
in blocksize Block image block size (8 or 32).

Return Values

No values are returned.

Description

Adds a command to convert and transfer a block image to a linear image to the specified command list.

This is the directly specified command list version of nngxAddB2LTransferCommand.
