nngxAddBlockImageCopyCommandRaw Function

Syntax

NNGX_APICALL void NNGX_APIENTRY nngxAddBlockImageCopyCommandRaw(
     nngxCommandList * cmdlist,
     const GLvoid * srcaddr,
     GLsizei srcunit,
     GLsizei srcinterval,
     GLvoid * dstaddr,
     GLsizei dstunit,
     GLsizei dstinterval,
     GLsizei totalsize
);

Parameters

Name Description
in cmdlist Target command list object.
in srcaddr Transfer source start address.
in srcunit Unit size to use when reading the transfer source image (in bytes).
in srcinterval Skip size of the transfer source image, in bytes.
out dstaddr Start address of the location to transfer the image to.
in dstunit Unit size to use when writing the transfer image (bytes).
in dstinterval Skip size of the transfer destination image, in bytes.
in totalsize Total size of the image to transfer, in bytes. (It must be a multiple of 16.)

Return Values

No values are returned.

Description

Adds a block image transfer command to the specified command list.

This is the directly specified command list version of nngxAddBlockImageCopyCommand.
