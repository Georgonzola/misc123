nngxAddL2BTransferCommandRaw Function

Syntax

NNGX_APICALL void NNGX_APIENTRY nngxAddL2BTransferCommandRaw(
     nngxCommandList * cmdlist,
     const GLvoid * srcaddr,
     GLvoid * dstaddr,
     GLsizei width,
     GLsizei height,
     GLenum format,
     GLsizei blocksize
);

Parameters

Name Description
in cmdlist Target command list object.
in srcaddr Address of the source linear image to transfer.
in dstaddr Address that will store the block image to which the data is transferred.
in width Width of the transfer source and destination images.
in height Height of the transfer source and destination images.
in format Pixel format of the image to transfer.
in blocksize Block image block size (8 or 32).

Return Values

No values are returned.

Description

Adds a command to convert and transfer a linear image to a block image to the specified command list.

This is the directly specified command list version of nngxAddL2BTransferCommand.
