nngxAddSubroutineCommandRaw Function

Syntax

NNGX_APICALL void NNGX_APIENTRY nngxAddSubroutineCommandRaw(
     nngxCommandList * cmdlist,
     const GLvoid * bufferaddr,
     GLsizei buffersize
);

Parameters

Name Description
in cmdlist The command list object to add the jump command to.
in bufferaddr Address of the region storing the command buffer to jump to.
in buffersize Size of the command buffer stored in bufferaddr, in bytes.

Return Values

No values are returned.

Description

Takes a command for jumping to the run address in a specified command buffer, and a command for setting the address information for returning back to the command buffer from where the jump took place, and adds them both to a specified command list object.

This function is the directly specified command list version of nngxAddSubroutineCommand.
