nngxClearFillCmdlistRaw Function

Syntax

NNGX_APICALL void NNGX_APIENTRY nngxClearFillCmdlistRaw(
     nngxCommandList * cmdlist,
     GLuint data
);

Parameters

Name Description
in cmdlist Target command list object.
in data Replacement value.

Return Values

No values are returned.

Description

Clears accumulated commands from the specified command list and replaces them with the specified value.

This is the directly specified command list version of nngxClearFillCmdlist.
