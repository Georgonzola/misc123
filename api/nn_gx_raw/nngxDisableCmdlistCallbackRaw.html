nngxDisableCmdlistCallbackRaw Function

Syntax

NNGX_APICALL void NNGX_APIENTRY nngxDisableCmdlistCallbackRaw(
     nngxCommandList * cmdlist,
     GLint id
);

Parameters

Name Description
in cmdlist Target command list object.
in id ID of the accumulated command that disables the callback.

Return Values

No values are returned.

Description

Disables calling a callback function upon completion of the specified command list.

This is the directly specified command list version of nngxDisableCmdlistCallback.
