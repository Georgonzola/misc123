nngxGetTopBufferAddrRaw Function

Syntax

NNGX_APICALL GLvoid *NNGX_APIENTRY nngxGetTopBufferAddrRaw(
     nngxCommandList * cmdlist
);

Parameters

Name Description
in cmdlist Target command list object.

Return Values

The start address of the 3D command buffer.

Description

Returns the start address of the 3D command buffer set to the specified command list.

Corresponds to the process when NN_GX_CMDLIST_TOP_BUFADDR is specified for the pname parameter of nngxGetCmdlistParameteri.
