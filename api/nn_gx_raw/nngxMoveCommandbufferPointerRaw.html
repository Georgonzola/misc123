nngxMoveCommandbufferPointerRaw Function

Syntax

NNGX_APICALL void NNGX_APIENTRY nngxMoveCommandbufferPointerRaw(
     nngxCommandList * cmdlist,
     GLint offset
);

Parameters

Name Description
in cmdlist Target command list object.
in offset Offset by which to move the pointer (in bytes).

Return Values

No values are returned.

Description

Moves the 3D command buffer's current output target pointer to the specified command list.

This is the directly specified command list version of nngxMoveCommandbufferPointer.
