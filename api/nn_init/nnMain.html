nnMain Function

Syntax

void nnMain(
     void
);

Parameters

None.

Return Values

None.

Description

Entry function for applications. This function is not called by applications, but is implemented for the application. It is equivalent to the main function used in standard C. Although there are exceptions such as nninitStartUp, applications typically execute from inside this function. All applications must be implemented by defining this function. If nnMain is not defined for application code, the application build fails.
