List of Usable API Functions

Description

This is a list of libraries that can be used inside nninitStartUp.

The following libraries and classes can be used.

  • Result class
  • init Library
    The following classes and functions can be used.
    • InitializeAllocator
    • GetAllocator
  • os Library
    The following classes and functions can be used.
    • CriticalSection
    • Event
    • LightBarrier
    • LightEvent
    • LightSemaphore
    • Mutex
    • Semaphore
    • SimpleAutoStackManager
    • StackMemory
    • StackMemoryAutoStackManager
    • StackBuffer
    • ThreadLocalStorage
    • Tick
    • Timer
    • Thread::SetAutoStackManager
    • GetAppMemorySize
    • GetUsingMemorySize
    • SetupHeapForMemoryBlock
    • GetDeviceMemoryAddress
    • SetDeviceMemorySize
    • GetDeviceMemorySize
    • GetHeapAddress
    • GetHeapSize
    • SetHeapSize
    • InitializeMemoryBlock
    • SetUserExceptionHandler
    • ManagedThread::InitializeEnvironment
  • fnd Library
    Classes and functions other than the DateTime class can be used.
  • math Library
    All classes and functions can be used.
  • nstd Library
    All classes and functions can be used.
  • util Library
    All classes and functions can be used.
  • hio Library
    The following classes and functions can be used.
    • Initialize
