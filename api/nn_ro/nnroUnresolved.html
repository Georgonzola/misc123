nnroUnresolved Function

Syntax

NN_DLL_EXPORT void nnroUnresolved();

Parameters

None.

Return Values

Description

Placeholder for the function to be used in place of an unresolved symbol.

If you define and make public a function named nnroUnresolved for DLL files ahead of time, this placeholder is used as the substitute address for the unresolved symbol.

It can be used to detect when an unresolved function is called.

(The SDK does not provide an implementation of this function.)
