Basic type definitions.

Description

These are the definitions of basic types used by the SDK.

typedef Definitions

u8 8-bit unsigned integer type.
u16 16-bit unsigned integer type.
u32 32-bit unsigned integer type.
u64 64-bit unsigned integer type.
s8 8-bit signed integer type.
s16 16-bit signed integer type.
s32 32-bit signed integer type.
s64 64-bit signed integer type.
sptr Signed address integer type.
uptr Unsigned address integer type.
char8 8-bit character type.
char16 16-bit character type.
f32 Single-precision floating-point value type.
f64 Double-precision floating-point value type.
bit8 8-bit data type.
bit16 16-bit data type.
bit32 32-bit data type.
bit64 64-bit data type.
