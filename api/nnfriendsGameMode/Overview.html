nnfriendsGameMode Structure

Syntax

struct nnfriendsGameMode
{
   u32 joinAvailabilityFlag;
   u32 matchmakeSystemType;
   u32 joinGameId;
   u32 joinGameMode;
   nnfriendsPrincipalId ownerPrincipalId;
   u32 joinGroupId;
   u8 applicationArg[NN_FRIENDS_APPLICATION_ARG_SIZE];
};

Description

Structure that stores information about the game the local host is currently playing. This information is required in order for friends to join in.

Member Variables

joinAvailabilityFlag u32 Specifies whether the game on the local host is available for friends in the matchmaking group to join in.
matchmakeSystemType u32 The local host's matchmaking group type.
joinGameId u32 The matchmaking game ID. An ID used to distinguish within the group what game titles allow friends to join in.
joinGameMode u32 The join-in game mode. A value used to distinguish what game modes allow friends to join. For partners to be able to join, the bitwise AND of the partners' join game mode mask (1 << join game mode) must be a value other than 0. As a result, a value between 0 and 63 must be specified for the join-in game mode.
ownerPrincipalId nnfriendsPrincipalId The principal ID of the owner of the current matchmaking group.
joinGroupId u32 The local host's matchmaking group ID.
applicationArg u8 Information that can be freely defined by applications.
