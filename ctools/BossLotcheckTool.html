BossLotcheckTool

Overview

BossLotcheckTool is a tool for checking SpotPass task information and NS data information.
In addition to checking information, it can also perform the following operations.
- Immediate execution of tasks (foreground/background)
- Deleting tasks
- Deleting NS data
- Approving/stopping StreetPass relay tasks
- Execution of StreetPass relay tasks

Screen Description

The upper screen displays information such as menus and the list of applications.
The lower screen displays the operation log.

How to Use

Operation for each screen is as follows.

Menu

Button Operation
↑↓ Switch selection
A Confirm selected item

Application List

Button Operation
↑↓ Switch selection
L / R Switch page
A Display task list for the selected application
Y Display NS data list for the selected application
B Return to Menu

Task List

Button Operation
↑↓ Switch selection
L / R Switch page
A Display the information of the selected task
B Return to Application List

Task Info

Button Operation
L / R Switch page
X Delete task
Y Immediate execution of tasks (foreground)
START Immediate execution of tasks (background)
B Return to Task List

NS Data List

Button Operation
↑↓ Switch selection
L / R Switch page
A Display the information of the selected NS data
B Return to Application List

NS Data Details

Button Operation
X Delete NS data
B Return to NS Data List

Check Functions Menu (Check Functions)

Button Operation
↑↓ Switch selection
A Confirm selected item
B Return to Menu

StreetPass Relay Menu (SPRelay)

Button Operation
↑↓ Switch selection
A Confirm selected item
B Return to the Check Functions Menu.

Execute StreetPass Relay Task (Forced Task Start)

Immediately executes StreetPass relay tasks, even in environments that are not Nintendo Zone. Waits until the execution is finished.

Button Operation
B Return to the StreetPass Relay Menu.

Allow StreetPass Relay (Enable)

Allows StreetPass relay in the Nintendo Zone environment.

Button Operation
B Return to the StreetPass Relay Menu.

Stop StreetPass Relay (Disable)

Stops StreetPass relay in the Nintendo Zone environment.

Button Operation
B Return to the StreetPass Relay Menu.

Clear BOSS DB in the system (Shutdown Required)

Clears all BOSS DB information managed by the system. This clears the list of BossLotcheckTool tasks.The application must be shut down after deletion.
When debugging without any BOSS information registered, use this feature with deletion of expanded save data (or initialization of SD card).

Button Operation
A+X Delete BOSS storage data
B Return to Menu
