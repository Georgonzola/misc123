CecLotcheckTool

Overview

Regarding StreetPass in the 3DS Guidelines, CecLotcheckTool is the program used for the following checks.
・Support when the StreetPass BOX cannot be created
・StreetPass ID
・Support when the outbox is full

Screen Description

The upper screen displays a list of StreetPass boxes currently created on the CTR system.
Each box is displayed using an 8-digit StreetPass ID.
You can check the StreetPass ID by creating a box using the application to be checked and then starting this tool.

How to Use

Button Operation
↑↓ Select a StreetPass ID
A Create dummy boxes
X Delete boxes
Y Box operation options
START Reset the filter

Create Dummy Boxes

Create dummy boxes until all slots are full.
By using this tool to fill all slots with dummy boxes and then running the application to try and create a box, you can check how the application handles the inability to create a StreetPass box.
The StreetPass IDs of the dummy boxes all start with D and end with D.

Button Operation
↑↓ Select
A Run
B Cancel

Delete Boxes

Delete all StreetPass boxes created on the CTR system.

Button Operation
↑↓ Select
A Run
B Cancel

Box Operation Options

The following box operations are possible.
・ Browse box information
・ Browse outbox message information
・ Dummy message creation
・ When browsing box information, you can check the maximum outbox size, the size used, the maximum number of messages, and the number of currently saved messages.
When browsing outbox message information, you can check the information of individual messages saved in the outbox.
The available information includes the recipients (friends, non-friends, or anyone), the send mode (exchange, send and receive, send, or receive), the send count, and the propagation count.

Button Operation
↑↓ Select
A Run
B Cancel

Create a Dummy Message

If the application being checked has created a box, fill the send box with a dummy message.
This does not apply to a dummy box.
Make the dummy message for any of the boxes with IDs shown in green in the list of StreetPass IDs.
The box can be filled by dummy messages in two ways: 1) fill it with a dummy message having the maximum size of the send box, or 2) fill it with the maximum number of messages allowed.
Depending on the box settings, you may only be able to use one of these methods.


To check what to do when the send box is full, first create a box using the application being tested, and then create a dummy message with this tool.
Next, try to create a send message using the application being tested.

Button Operation
↑↓ Select the box to be created and the type.
A Run
B Cancel

Resetting the StreetPass MAC Filter

Reset the filter to allow StreetPass communication to happen again with the same peer.
To do so, call nn::cec::CTR::CecControl::StartScanning(true).
Note: Call StopScanning right after to prevent any actual communication from the CecLotcheckTool.
Note: The filter reset feature is not available for release builds.
Note: You only need to reset the filter on one system to allow StreetPass communication to happen again.

Button Operation
START Reset the filter
