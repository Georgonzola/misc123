DlpClient

Overview

DlpClient is the Download Play client program.
A separate server program is also needed for Download Play.

Scan

Start the program and then press the A Button.
The list in the upper screen shows information about the detected servers.
From this list, select the server from which you want to download.
The lower screen displays information about the titles being distributed, distribution server, and node for each server.

Button Operation
↑↓ Select
A Run
B Cancel

Download

When distribution from the server to the clients begins, the upper screen shows the progress of the download and other aspects of the states of the clients.

Button Operation
B Cancel

Reboot

Once all clients have received the distributed program, they receive a signal from the server to reboot and start the downloaded program.
