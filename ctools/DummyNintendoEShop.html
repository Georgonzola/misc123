DummyNintendoEShop

Overview

DummyNintendoEShop is an application that replaces Nintendo eShop and is a tool for verifying the operation of features that jump to Nintendo eShop.

Because Nintendo eShop is not included in development machines after System Updater is applied, DummyNintendoEShop must be imported to verify the feature to jump to Nintendo eShop.

The $CTRSDK_ROOT/tools/TargetTools/CTR-TS/Release directory contains a DummyNintendoEShopXX.cia (XX is the region name) file for each region.
Import the CIA file corresponding to your region from DevMenu.
If the import is successful, a DummyNintendoEShop icon with a pink frame appears in the HOME Menu.

Jump Feature Operation Verification

Verify by jumping from the application to Nintendo eShop.
The applet library has the following functions for jumping to Nintendo eShop.

  • nn::applet::JumpToEShopTitlePage ... Jumps to the software information page of the specified title.
  • nn::applet::JumpToEShopTitlePage ... Jumps to the patch page of the specified title.

    • When these functions are called, the application ends and DummyNintendoEShop starts, so verify the content displayed in the upper screen.
    For more information, see Screen Description.
    If the jump destination page and the title unique ID are as expected, jumping to the actual Nintendo eShop is also possible.

    Screen Description

    The upper screen displays the title and unique ID of the page specified as the jump destination for Nintendo eShop and the unique ID of the application where the jump originates.

    Jump DestinationContent Shown
    Software information pageDummy Nintendo eShop
    Title page uniqueID=Unique ID of title to jump to jump from app (Unique ID of the application where the jump originates)
    JumpType=2(Public) OK
    Patch PageDummy Nintendo eShop
    Patch uniqueID=Unique ID of title to jump to jump from app (Unique ID of the application where the jump originates)
    JumpType=2(Public) OK
    The lower screen shows ways to cancel DummyNintendoEShop (press the HOME button) and exit DummyNintendoEShop (press the POWER Button).

    Notes

    The nn::applet::JumpToEShopPatchPage function is used to guide to the patch page of the Nintendo eShop title when the authentication server or account server returns an error such as nn::act::ResultApplicationUpdateRequired that requires an application update.

    However, when the application update is operated with remastering, the patch page for the title in Nintendo eShop does not exist.
    In this case, an error that the title does not exist on the actual Nintendo eShop is displayed and transition to the Nintendo eShop startup screen.
    When operating a remaster, use the nn::applet::JumpToEShopTitlePage function.

    With DummyNintendoEShop, verify that "Patch page" is displayed when operating with a patch and that "Title page" is displayed when operating with a remaster.
