EcDevTool

Overview

EcDevTool is a tool that can be used to add to the balance of an eShop account, to delete the rights to purchased content or to delete an eShop account, to change the country setting of an account on the eShop server to match the country setting of the CTR system, and to set a tax address.
When the user attempts to add money, change country settings, or set a tax address while the EULA has yet to be agreed to, the EULA applet starts.

Network Settings

DNS must be configured to connect to the development server. The recommended DNS addresses to use can be found at WarioWorld.com.

Connection to the development server begins after you start this program and select one of the menus (such as the one to add money to an account).
Contact Nintendo if the connection fails even when your network configuration is correct.

The upper screen displays the following information about the settings in the system.

How to Operate

All operations can be performed through both touch input and key input.
Here are the basic operations.

Button Operation
+Control Pad Moves the cursor.
A Confirm.
B Back.

Operations differ in some ways depending on whether the eShop account is managed by a Nintendo Network ID, so the two cases are described separately.

How to Operate (When the eShop Account Is Not Managed by a Nintendo Network ID)

Add Money

Money is added to the account balance using the EC applet. To do so, a writable SD card must be inserted in the system.

The following credit card information is required to add to the balance.

Credit card type VISA
Credit card number 4444444444444448
Expiration date 12/2030
Security code 123
Address (U.S. only) Enter your current ZIP code. Example: 12345

Delete Rights

Select a title to delete the rights to the purchased content.
When the rights are deleted, the data title is also deleted.
You can also delete all rights.

In addition to the basic operations, you can also use the L/R Buttons to switch pages.

Button Operation
+Control Pad Moves the cursor.
A Confirm.
B Back.
L, R Switch page.

Delete eShop Account

After confirming the action, the eShop account is deleted.
Deleting your eShop account also deletes all rights to purchased content and sets your balance to zero.

Change Country Setting

Changes the country setting of the account on the eShop server to match the country setting of the CTR system.
To purchase content, the country setting of the CTR system must match the country setting of the account on the eShop server.

After this is selected, the country settings are checked automatically and the country setting of the account on the eShop server is synchronized with the country setting of the CTR system.

You cannot change to a country in a different region. To change regions, first delete the eShop account and then run the System Updater for the appropriate region. After you do this, you can use the system in the countries of this different region.

Setting a Tax Address

When the country setting is the United States or Canada, the EC applet is used to set a tax address.
This is used when using the eShop REST API.

When the country setting is the United States, enter the current ZIP code (example: 12345).
When the country setting is Canada, select a registered region as the address.

How to Operate (When the eShop Account Is Managed by a Nintendo Network ID)

Add Money

Operations are the same as when the eShop account is not managed by a Nintendo Network ID.

Delete Rights

You cannot delete rights when the eShop account is managed by a Nintendo Network ID.
You can instead delete the entire eShop account.

Delete eShop Account

Operations are the same as when the eShop account is not managed by a Nintendo Network ID.

Change Country Setting

If the eShop account is managed by a Nintendo Network ID, first delete the Nintendo Network ID in the System Settings, then change the country setting in the CTR system and run EcDevTool again to change the country setting.

Alternatively, first delete the Nintendo Network ID in the System Settings, and then change the country setting in the CTR system and set a new Nintendo Network ID. This method does not require changing the country setting with EcDevTool.

To change to a country in a different region, first delete the Nintendo Network ID in the System Settings and then use EcDevTool to delete the eShop account.
After that is done, if you run System Updater for the appropriate region you can use the system in the countries of this different region.

Setting a Tax Address

Operations are the same as when the eShop account is not managed by a Nintendo Network ID.
