FriendTestingPartner

Overview

FriendTestingPartner is a tool that creates a high friend presence processing load to verify whether friend presence background processing affects the operations of the application.

The tool provides a way to automatically perform the following actions frequently.

  • Switching between online and offline.
  • Updating the Currently Playing software.
  • Updating the Currently Playing game mode description string.
  • Updating the status message.
  • Updating the Mii.
  • Updating the Favorite Title.
These features enable the CTR running this tool to send frequent notifications to other CTR systems with which friendships are established, indirectly triggering background processing of friend presence.

When the local CTR is disconnected from the wireless AP or friend server, it automatically tries to reconnect and maintain the connection.

How to Use

Button Operation
+Control Pad Move the cursor.
A Select.
START Press and hold down to display only errors in the log.

UI Reference

Master Control

Controls overall tool operation.If this setting is not RUN, no status switching or updates occur even if individual settings are ON. Also, even if this setting is STOP, reconnection to the wireless AP or friend server does not stop.

Login/Logout

Controls switching between online and offline.

Playing Title

Controls updates of the Currently Playing software. Cycles through the following titles (all are Japanese versions).

  1. System Settings
  2. Download Play
  3. Activity Log
  4. Health and Safety Information
  5. Nintendo 3DS Camera
  6. Nintendo 3DS Sound
  7. Mii Maker
  8. StreetPass Mii Plaza
  9. Nintendo Zone
  10. Face Raiders
  11. AR Games

Mode Description

Controls updates of the game mode description string. The updated description string shows the current date and time. When the Currently Playing software is updated, the game mode description string is deleted.

Message

Controls updates of the status message. The updated status message shows the current date and time (using numbers only).

Mii Characters

Controls updates of the Personal Mii. Cycles in order through the four Mii characters prepared in advance. The Mii names are Partner 1 through Partner 4.

Favorite Title

Controls updates to the Favorite Title. The titles cycled through by this setting are the same as those for the Currently Playing software.

Wireless AP

Indicates the connection status with the wireless AP. If the connection fails for some reason, displays the Result code and, if available, the error code.

Friend Presence

Indicates the connection status with the friend server. If the connection fails for some reason, displays the Result code and, if available, the error code.

Cautions

Before running this tool, be sure to configure network settings and establish one or more friendships.

This tool overwrites the following friend presence settings saved on the CTR system. To restore these settings to their original values, use the following procedure.

  • Status message and Favorite Title: Reset from the friend list.
  • Personal Mii: When the friend list is started, this Mii is automatically restored to the Personal Mii created in Mii Maker.

There is a send limitation of once per 10 seconds for each setting in friend presence communication. Even when the local system is online, the timing at which this tool outputs the log might not synchronize with the timing at which the communication is actually performed.

Operation stops while the HOME Menu is open. Even if the local system connects or disconnects to the AP or friend server during this period, the tool does not detect and output that to the log until tool operation resumes.
