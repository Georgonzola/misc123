NetworkSetting

Overview

The NetworkSetting program configures the security mode, SSID, and key (or passphrase) settings for network configurations 1-3.

How to Use

Button Operation
↑↓ Moves between slots to configure
A Select / Run
B Back / Cancel

Slot 1–3 Setting

Here you can set the network configuration of the selected slot. Select Write Setting to apply the settings.

Button Operation
A Select / Confirm
B Back / Cancel
Touch Panel Input of SSID and key (or passphrase)

SecurityMode

Choose from the following: OPEN, WEP 40bit, WEP 104bit, WEP 128bit, WPA TKIP, WPA2 TKIP, WPA AES, and WPA2 AES.

SSID

Enter an SSID via the software keyboard. You can only enter single-byte alphanumeric characters. The setting has a maximum length of 32 characters.

KEY / PASS PHRASE

Use the software keyboard to enter your data.
If you chose OPEN, WEP 40bit, WEP 104bit, or WEP 128bit for the SecurityMode, enter a key. If you chose WPA TKIP, WPA2 TKIP, WPA AES, or WPA2 AES, enter a passphrase.

The maximum length of the string varies as shown below.

SecurityMode Setting Number of characters that can be input
OPEN - -
WEP 40-bit authentication KEY Maximum 5 characters
WEP 104-bit authentication KEY Maximum 13 characters
WEP 128-bit authentication KEY Maximum 16 characters
WPA TKIP authentication PASS PHRASE Minimum 8 characters - maximum 63 characters
WPA2 TKIP authentication PASS PHRASE Minimum 8 characters - maximum 63 characters
WPA AES authentication PASS PHRASE Minimum 8 characters - maximum 63 characters
WPA2 AES authentication PASS PHRASE Minimum 8 characters - maximum 63 characters

AutoDNS

Configures the automatic DNS retrieval.
Setting automatic DNS to false allows the primary DNS and secondary DNS to be specified. Only numeric values can be entered.

Write Settings and AP Connection Test

Apply the setting and run the connection test.
The connection test only checks connections with an access point.

An error code is displayed if the connection test fails.
