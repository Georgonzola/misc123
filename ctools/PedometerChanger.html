PedometerChanger

Overview

PedometerChanger is a tool that allows for the viewing, importing and exporting of pedometer information.

How to Use It

Button Operation
↑↓ Switch the selection.
A Confirm the selected item.

Tool Features

Functionality Description
Import Imports the pedometer information included in the ImportPedometerFile.csv file, at the top level of the SD card.
When it does so, all existing pedometer data is deleted.
Export Exports the system's pedometer information to the ExportPedometerFile.csv file, at the top level of the SD card.
Append Appends the pedometer information included in the ImportPedometerFile.csv file, at the top level of the SD card.
This differs from the Import command in that any pedometer data existing on the system that does not have an equivalent in date and time in the ImportPedometerFile.csv file remains as is.
Display Displays the system's pedometer information on the lower screen.
Initialize Deletes all pedometer information on the system.

File Format

The import file, ImportPedometerFile.csv, and the export file, ExportPedometerFile.csv use the following format.

YEAR, MONTH, DAY, TIME, STEPS TAKEN

For example, if you walked for 100 steps between 11:00 AM and 12:00 PM on 1 December 2000, and for 200 steps between 11:00 PM and 12:00 AM on 1 January 2001, the data would appear as follows in the files.

2000,12,1,11,100
2001,1,1,23,200
