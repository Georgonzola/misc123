PlayCoinSetter

Overview

PlayCoinSetter is a tool to configure the number of Play Coins you have.

How to Use

Button Operation
←→ Set the number of Play Coins
A Apply settings
L+R+X Buttons pressed together The upper limit for the number of Play Coins. (Note)
Note: It is possible to set this as an upper limit for the number of Play Coins that can be added on the current date. However, you can use this tool to increase or decrease this amount. There is no feature to remove this limit. To work around this limit, change the date in System Settings.
