UdsLocalIdChecker

Overview

UdsLocalIdChecker is a tool that searches for a UDS network with a specific local ID.
It is used for verifying the local communication ID set in the Local Communication section of the 3DS Guidelines.

Screen Description

The tool searches for the UDS network having the specified unique ID and gets the network's information. The top screen displays the list of detected UDS networks, and the bottom screen displays the details of each network.
When two UDS networks with the 00F01 parameter in their unique IDs are detected, the following is displayed.

How to Use

UniqueID Configuration Mode

When the tool is started, the cursor is displayed in the first digit of the unique ID in the bottom screen. Specify the network unique ID with the +Control Pad, then press the A Button to start searching.

Button Operation
←→Move between digits in the unique ID
↑↓ Change individual digit values in the unique ID
A Start search, or transition to search result display mode after search mode
R Toggle the isDemo flag

Result Display Mode

When the search ends, a list of the found UDS networks is displayed. The cursor moves to the upper screen, and you can select a network from the list using Up/Down on the +Control Pad. The information for the selected network is displayed on the lower screen.

Button Operation
↑↓ Select network
A Search again
B Return to unique ID configuration mode
