WMTestTool

Overview

WMTestTool is a program originally made to test the TWL and DS wireless features.
With CTR, only Noise Mode is available.
To use this tool, import the CIA file and run it from DevMenu.

Noise Mode

Select items by pressing Up and Down on the +Control Pad. Pressing Left and Right on the +Control Pad changes the parameters.
Start noise generation with the A Button, and end it with the B Button.
Once noise generation has started, the number of occurrences of noise generation and non-generation are displayed on the upper screen.

Item Description
Channel Sets the channel on which to generate noise. A value from 1 – 13 can be set here. Keep in mind that several neighboring channels will also be affected.
Noise rate Sets the noise generation rate. Noise generation or non-generation is determined randomly once per each Frequency interval. Note that this value is the time rate at which radio waves are emitted, not the strength of those waves. This setting can be changed even after noise generation has started. Note that a Noise rate of 100 percent is an extreme case and this state is not sustainable in a real environment.
Frequency The period (in milliseconds) over which to determine whether to generate or not generate noise. A value from 5 – 50 can be set.
Seek Mode This feature changes the channel on which noise is output. It is measured as the time interval (in ms) between channel changes. This feature cycles through channels 1 – 13, incrementing the channel for each channel change.
