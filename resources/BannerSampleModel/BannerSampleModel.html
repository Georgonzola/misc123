Banner Sample Model

Description

This is a sample of the banner model displayed on the upper screen of the HOME Menu, and data used to create the banner model.

This is included under the $CTR_SDK/resources/bannermodel directory as follows.

Folder Name Details
BannerCamera Intermediate camera data used to display banners on the HOME Menu.
BannerMask An ellipsoid model of the same size as the model used to mask the banner. Use this as a reference in sizing the banner.
Note: This model is only for sizing purposes.
BannerModel_00 A bare-bones simple sample of banner data.
Note: If you do not have an environment for creating 3D models, create your data by replacing the textures in this model.
BannerModel_01 A version of BannerModel_00 modified to support replacing textures based on the language.
BannerModel_02 A sample model including additional features such as animation and lighting. Logo data modified to support replacing textures.
BannerModel_03 A sample model including additional features such as animation and lighting. Modified to support model switching and replacing textures.
