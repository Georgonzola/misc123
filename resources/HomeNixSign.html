HOME Menu Disabled Icon

Description

This is the HOME Menu Disabled icon.
It is displayed when pressing the HOME Button does not call the HOME Menu.

How to Use

When Using NW4C Layout

Select which layout to use based on your version of NW4C, as shown in the following table.

NW4C Version Layout Data to Use
Prior to 3.0 Data in the folder $CTR_SDK/resources/icon/HomeNixSign/layout1.
The extensions for the layout data are .clan and .clyt.
3.0 or later Data in the folder $CTR_SDK/resources/icon/HomeNixSign/layout2.
The extensions for the layout data are .flan and .flyt.

When Not Using NW4C Layout

Use $CTR_SDK/resources/icon/HomeNixSign/HomeNixSign_Targa.tga and implement according to the following display rules.

Content Details
Texture to Use HomeNixSign_Targa.tga (Targa format)
Display Location Middle of lower screen
Animation Curve Type Linear curve. (In LayoutEditor, this is the Linear Interpolation for Slope feature.)
Display Time When creating at 60 frames per second
------------------------------------
Fade in: 5 frames
Display: 60 frames
Fade out: 20 frames
------------------------------------
