Character Filter File (xllt)

Description

This is a font converter character filter file (xllt) for filtering characters available for use in CTR system usernames.

The $CTR_SDK/resources/xllt directory contents are as follows.

Filename Details
CTR_userName_JPN.xllt Character filter file for use with usernames only in the Japan region.
CTR_userName_USA_EUR.xllt Combined character filter file for characters available for usernames in the Americas and Europe/Australia regions.
CTR_userName_USA_EUR_JPN.xllt Combined character filter file for characters available for usernames in the Japan, Americas, and Europe/Australia regions.
CTR_userName_CHN.xllt Character filter file for use with user names only in the China region.
CTR_userName_KOR.xllt Character filter file for use with user names only in the Korea region.
CTR_userName_TWN.xllt Character filter file for use with user names only in the Taiwan region.
CTR_userName_ALL.xllt Combined character filter file for characters available for user names in all regions.

See List of Characters Available for the CTR System Settings Username for the characters available for use in usernames for each region and language.
