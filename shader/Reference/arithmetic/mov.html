mov - Move

Calling Format

mov     dest, src

Operands

Name Registers
dest An output register or temporary register.
src A temporary register, input register, or floating-point constant register.

Overview

Copies the contents of src into dest.

Operation

dest = src

Code Example

mov     r0,     c1

Timetable

123
mov read mov write
