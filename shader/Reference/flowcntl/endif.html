endif - End if block

Calling Format

endif

Operands

None.

Overview

Ends a control block started by ifc or ifb.

You must include at least one instruction betweenifb and endif, between ifc and endif, and between else and endif.

Operation

if ( src == true )
{
    ...
}
else
{
    ...
}
endif

Code Example

ifb     b0
    ...
else
    ...
endif
