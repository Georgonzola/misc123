abs - Absolute

Calling Format

abs     dest, src

Operands

Name Registers
dest An output register or temporary register.
src A temporary register or input register.

Overview

Stores the absolute value of src in dest.

Operation

dest.x = abs ( src0.x )
dest.y = abs ( src0.y )
dest.z = abs ( src0.z )
dest.w = abs ( src0.w )

Macro Expansion

max     dest, src, -src

Code Example

abs     r0, r1
abs     o0, r7.yzww
