Miscellaneous Documents

Overview

The various documents needed for application development are located in the CTR_SDK/documents/TechnicalNotes directory.

This document provides a list of those documents, and links to each of them.

List

ID List List of CTR-SDK related IDs, together with their descriptions.
System Application and Applet Specifications Specifications for CTR system applications and applets.
System Programming Guide Programming guide for the CTR-SDK os library and dbg library.
CTR Icon and Banner Authoring Manual This document covers the creation of CTR icons and banners.
Differences Between Test Units and Retail Units This document highlights the differences between development units and retail units as far as the CTR-SDK is concerned.
DLL Manual This document explains the DLL mechanism provided by the CTR-SDK.
Texture Packager Tool External Specifications Specifications for the TexturePackager tool, which converts TGA format image files into texture format for use on the CTR.
Texture Packager Library Runtime Specifications Specifications for the TPL library, which loads textures from the texture packages created by the TexturePackager tool.
Description of the ULCD Library Description of the ULCD library, which supports stereoscopic representation on the CTR.
Patch Manual Documentation that combines patch specifications with the method for creating patches.
Account System: Developers' Guide A manual that lists the specifications for Nintendo Network Accounts in addition to the methods by which applications can use account authentication.
eShop REST API Operating Manual This is a manual describing how to use the eShop REST API.
