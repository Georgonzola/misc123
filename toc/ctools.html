CTR Tools

Overview

The CTR_SDK/tools/TargetTools directory contains the CTR tools used for application development. These CTR tools have a file extension of .csu, but they can be treated just like files with the .cci extension. When they are executed in a debugger, however, they start automatically and do not allow breakpoints or other debugging operations.

DevMenu and Config and imported to the system NAND memory when System Updater is applied.

The following list contains the CTR tools, with links to the pages where each tool is described in detail.

List

BossLotcheckTool A tool for debugging SpotPass.
CecBoxTool A tool for debugging StreetPass.
CecLotcheckTool A Lotcheck tool for debugging StreetPass and checking its operation.
Config Changes the settings stored to system memory.
DevMenu Manages the import applications contained in the CIA file that is imported.
DlpClient The Download Play client used for development.
DummyNintendoEShop A tool for confirming jumps to Nintendo eShop.
EcDevTool This tool can be used to add to the balance of an eShop account, to delete the rights to purchased content or to delete an eShop account, and to change the country setting of an account on the eShop server to match the country setting of the CTR system.
FangateFwUpdater This tool updates or destroys NFC reader/writer firmware. Runs only with CTR.
FragDisk Fragments the free region in a FAT file system on an SD memory card.
FriendAddTool A tool to support the adding and deleting of friends.
FriendTestingPartner Recreates the heavy load state because of friend presence.
IrCommunicatorChecker A tool for checking the communication ID used for infrared communication between systems.
NetworkSetting Changes the access point settings stored to system memory.
NfpManager This tool manages NPF tags.
NfpSdmcTool This tool uses the NFP library and supports debugging of applications.
NfpUtil This tool helps you check whether your application complies with the NFP guidelines.
NoftWriter This tool is for creating a Nintendo Original Format Tag (NOFT) from an NTF file to write to a tag. Runs only with SNAKE.
PedometerChanger Allows for the viewing, importing and exporting of pedometer information.
PlayCoinSetter Configures the Play Coin settings.
SaveDataFiler For viewing, importing, and exporting the save data of various applications.
StereoCameraCalibrator A tool for manually calibrating the stereo camera.
TwlNandFiler A tool for manipulating (TWL region) data in the CTR system NAND memory.
UdsLocalIdChecker Searches for the UDS network configured with the specified local ID.
WMTestTool Generates radio interference.
