Resource Data

Overview

The CTR_SDK/resources directory stores resource data relating to application development.

Below follows a description of this resource data.

Description

banner This is the default banner/icon. You can use it in place of your own banner/icon in the early stages of development.
bannermodel This is a sample of the banner model displayed on the upper screen of the HOME Menu, and data used to create the banner model.
configtool This directory stores a sample configuration file for modifying all settings at one time using Config. For more information, see the Config tool reference.
icon BatteryLevelIcon This directory holds battery-level icon data. Use by the application is optional. For more information about using this icon, see the Battery Level icon documentation.
BrowserIcon This directory holds Internet Browser icon data. Use by the application is optional.
For more information about using this icon, see the Internet Browser icon documentation.
ConnectionIcon This directory holds communication icon data. Use by the application is optional. For more information about using this icon, see the Communication icon documentation.
DlplayIcon This directory holds Download Play icon data. Use by the application is optional.
For more information about using this icon, see the Download Play icon documentation.
FriendIcon This directory holds the Friend icon and Friend Registration icon data. Use by the application is optional.
For more information about using this icon, see the Friend Icon / Friend Registration Icon documentation.
HomeNixSign This directory stores the HOME Menu Disabled icon file. This icon appears when an application is running, and the HOME Menu cannot be displayed even when the HOME Button is pressed. For more information about using this icon, see the HOME Button Disabled icon documentation.
PlayCoinIcon This directory holds Play Coin icon data. Use by the application is optional. For more information about using this icon, see the Play Coin icon documentation.
UpdateIndicator This directory holds Update Indicator icon data. Use by the application is optional.
For more information, see the Update Indicator documentation.
shaders These geometry shaders are provided by DMPGL 2.0.
For more information, see About the reserved geometry shader.
shareddata This directory stores font data. Use by CTR applications is optional.
For more information, see the internal bitmap font documentation.
specfiles This directory stores sample DESC and RSF files. These files are necessary to create a CTR application.
XLLT This font converter character filter file (.xllt) is for filtering the characters available for use in CTR system user names.
