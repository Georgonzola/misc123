Windows Tools

Overview

The Windows tools needed for application development are located in the CTR_SDK/tools/CommandLineTools directory.

Here we provide a list of those Windows tools with links to the pages where each tool is described in detail.

List

ctr_compress Compresses files in formats that can be handled by the cx library.
ctr_loadrun Operates debugging from the command line.
ctr_make_ex_banner Creates contextual banners.
ctr_makebanner Creates banners and icons.
ctr_makecia Creates CIA files.
ctr_makeciaarchive Creates CFA files.
ctr_makecro Creates a CRO file. Also creates and updates XRL files.
ctr_makecrr Creates a CRR file.
ctr_makedlexbanner Creates downloadable contextual banners.
ctr_makerom Creates CXI, CCI, CFA, and CCL files
ctr_BannerModelConverter Creates the CBMD files used with ctr_makebanner.
ctr_FilerDataMigrator Converts data that was saved with SaveDataFiler.
ctr_FontConverterConsole Creates font files in formats that can be handled by the font library.
ctr_FsAccesslogAnalyzer Detects inefficient file access not intended by the developer
ctr_GxCommandAnalyzer Analyzes and outputs the 3D command buffer.
ctr_makeMetaDataContent Creates the required content (metadata content) based on the XML file.
ctr_TexturePackager Creates texture files in formats that can be handled by the tpl library.
ctr_VertexCacheSimulator Simulates the PICA post-vertex cache and outputs the number of vertices that had cache hits.
ctr_VertexShaderAssembler Assembles assembler files written in assembler language conforming to proprietary PICA specifications, and outputs the corresponding object files.
ctr_VertexShaderLinker Links object files output by ctr_VertexShaderAssembler and outputs an executable file.
ctr_WaveConverter Creates sound files in formats that can be handled by the snd library.
WaveCodecCtr.dll Provides an API for DSP ADPCM encoding and decoding.
